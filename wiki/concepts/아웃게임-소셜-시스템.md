# 아웃게임 소셜 시스템

> 출처: [[raw/gdd/02_System/M5_아웃게임_우편]], [[raw/gdd/02_System/M5_아웃게임_친구]], [[raw/gdd/02_System/M5_아웃게임_퀘스트]], [[raw/gdd/02_System/M5_아웃게임_출석부]]

## 개요

유저의 지속 접속과 플레이를 유도하기 위해 설계된 4개 아웃게임 시스템의 통합 정리. 우편·친구·퀘스트·출석부가 각각 독립적으로 동작하며, 행동력·달성 점수·누적 보상 등의 재화를 공급한다. 모든 시스템은 현재 (아이디어 단계, 미구현) 상태이다.

---

## 우편 시스템

> 출처: [[raw/gdd/02_System/M5_아웃게임_우편]]

운영 툴 및 콘텐츠 이벤트에서 유저에게 보상을 전달하는 인프라 시스템. (아이디어 단계, 미구현)

**우편 종류:**

| 종류 | 설명 |
|------|------|
| 시스템 우편 (지정) | 운영 툴에서 특정 유저 또는 다수 유저(CSV)에게 발송 |
| 시스템 우편 (전체) | 서버 또는 전체 대상, 수령 가능 기간(시작일·종료일) 설정 |
| 콘텐츠 우편 | 패키지 상품 구매 시 자동 발송, 향후 콘텐츠 추가에 따라 확장 |

**버전 잠금:**
- 우편에 버전 설정 가능: 해당 버전 미만 유저에게는 편지·보상 아이콘 잠금 상태 출력
- 잠금 우편 선택 시 [업데이트 안내] 팝업 출력 → 스토어 이동 버튼 제공

**만료 설정:**
- 영구(0) 또는 만료(1) 선택, 만료 시 획득 후 N일 경과로 소멸
- 영구 설정 우편은 삭제 불가

**편의 기능:**
- 일괄 받기: 미열람 우편 전부 수령 처리
- 일괄 삭제: 열람 완료 우편 전체 삭제 (미열람 우편만 있으면 비활성화)

## 친구 시스템

> 출처: [[raw/gdd/02_System/M5_아웃게임_친구]]

유저 간 친구 관계를 맺고 매일 행동력을 주고받는 소셜 시스템. (아이디어 단계, 미구현)

**친구 관계:**
- 신청 → 수락 시 성립, 한 쪽이 취소하면 해제
- 최대 친구 인원: 20명(NPC 포함, `friend_max_amount`)
- 최대 신청 수신: 20건(`friend_max_request`), 발신 횟수는 무제한

**행동력 선물 규칙:**

| 항목 | 값 |
|------|-----|
| 1회 선물 수량 | 1개 (`friend_daily_present_energy`) |
| 일일 최대 수령 | 10개 (`friend_max_daily_present_acceptance`) |
| 선물 내역 초기화 | 익일 서버 초기화 시 |
| 행동력 차감 | 없음 (선물 시 본인 행동력 차감 없음) |

**NPC 친구:**
- 친구가 없어도 하루 1개 행동력을 무료 수령하도록 NPC 자동 배치
- NPC와 친구 취소 불가, NPC도 친구 인원에 포함

**자동 전송 방식 (개선안):**
- [일괄 수령 및 선물] 버튼으로 NPC 보상 수령 시 친구들에게 자동 전송
- 기존 레퍼런스(수동 선물)의 부정적 경험을 개선하여 자동화

**추천 친구:**
- 접속일 가깝고 최고 기록 유사한 유저 5명을 기본으로 추천 목록 제공

## 퀘스트 시스템

> 출처: [[raw/gdd/02_System/M5_아웃게임_퀘스트]]

일일 플레이 동선 가이드 및 꾸준한 접속을 유도하는 달성 점수 + 진행도 보상 시스템. (아이디어 단계, 미구현)

**구성:**

| 구분 | 초기화 주기 | 역할 |
|------|-----------|------|
| 일일 퀘스트 | 매일 서버 초기화 | 일일 플레이 동선 가이드 |
| 주간 퀘스트 | 매주 서버 초기화 | 주간 누적 플레이 유도 |

**달성 점수 & 진행도 보상:**
- 퀘스트 완료 시 달성 점수 지급 → 일일/주간 진행도 바에 누적
- 특정 점수 도달 시 진행도 보상 수령 가능 (보상 최대 3종 연결 가능)
- 미수령 보상은 초기화 시 소멸 (보존 없음)

**퀘스트 상태 흐름:**
1. 진행 중: 할당 시 초기 상태, [이동] 버튼으로 해당 콘텐츠 UI 이동
2. 완료: 달성 조건 만족, [보상 받기] 버튼 활성화, 목록 최상단 이동
3. 보상 수령 완료: 달성 점수 획득, 목록 하단 이동
4. 진행 불가: 선행 조건 미충족, 잠금 상태로 최하단 표시

**퀘스트 조건 예시:**

| 조건 | 내용 |
|------|------|
| First_Enter_Daily | 게임 로그인 N회 |
| Try_Draw_Parts | 장비 소환 N회 |
| Try_Energy_Recharge | 에너지 충전 N회 |
| Try_Stage_Quick_Sweep | 빠른 순찰 N회 |
| Get_Ads_Reward | 광고 보상 획득 N회 |

## 출석부 시스템

> 출처: [[raw/gdd/02_System/M5_아웃게임_출석부]]

신규 유저와 일반 유저를 분리하여 매일 접속 보상을 제공하는 출석 시스템. (아이디어 단계, 미구현)

**상시 출석부 구분:**

| 유형 | 조건 | 특징 |
|------|------|------|
| 신규 유저 출석부 | 계정 생성 시 최초 1회 | 고정 ID, 상수값으로 지정 |
| 일반 유저 출석부 | 신규 출석부 종료 후 | 나머지 출석부 중 랜덤 선택, 연속 동일 출석부 방지 |

**운영 규칙:**
- 7일 출석부 고정, 동시에 상시 출석부 1개만 진행
- UI 오픈 시 자동 출석 처리 (별도 버튼 없음)
- 밀린 보상 한 번에 수령 가능

**누적 출석 보상:**
- 출석 2일마다 누적 보상 1회 지급 (`attendance_progress_reward_period_day = 2`)
- 최종 step 보상 소진 시 step1부터 재순환

**이벤트 출석부:**
- 특정 기간 한정, 동시 N개 병행 운영 가능
- 이벤트 관리 테이블에서 일정·ID 관리
- 상시 출석부와 진입 경로 분리 ([이벤트 메뉴])

## 시스템 간 상호작용

| 연관 | 내용 |
|------|------|
| 퀘스트 → 행동력 | 에너지 충전 퀘스트로 행동력 소비 유도 |
| 친구 → 행동력 | 일일 선물로 최대 10개 무료 행동력 획득 → 퀘스트 달성에 활용 |
| 출석부 → 초보자퀘스트 | 신규 유저 출석부와 초보자퀘스트가 동시 진행되며 초반 보상 집중 제공 |
| 우편 → BM | 패키지 구매 보상이 우편으로 전달되므로 과금 흐름의 마지막 단계 |
