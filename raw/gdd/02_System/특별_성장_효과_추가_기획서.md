# 특별 성장 효과 추가

> **요약:** 특별 성장 시스템에 추가되는 효과들을 정의한다. 이자 상한 증가(add_limit_interest_value), 스페셜 택틱 오브젝트 추가 등장(special_tactic_draw), 시너지 보너스 가중치(mul_set_effect_value), 스킬 카드 최대 획득 수 상한(add_limit_skill_tactic_count), 카드 밴 횟수 증가, 초당 경험치 증가 등 6종의 효과를 포함하며, 각각의 기본값과 최대값, 적용 로직을 기술한다.
> **현재 상태:** 미적용

특별 성장 효과 추가

# [확인 완료]이자 상한 증가(add\_limit\_interest\_value)

* 이자율로 얻을 수 있는 미네랄의 상한을 제한한다.
* 스탯으로 만들어서 증가 처리
  + 기본 값 3
  + 최대 값 5 ~ 10 사이로 밸런싱

# [확인 완료료] 스페셜 택틱 오브젝트 추가 등장 (special\_tactic\_draw)

* 신화 택틱 오브젝트 등장 스탯에 단계를 추가한다.
  + 레벨 1일 때 : 7 웨이브에 확정 등장 (기구현)
  + 레벨 2일 때 : 7웨이브, 14 웨이브에 확정 등장 (구현 추가)

# [확인 완료]시너지 보너스(mul\_set\_effect\_value)

* 시너지로 얻는 효과 수치에 가중치를 적용한다.
* 시너지 수치 최종 값 = SetEffectDetailData@effect\_value \* (1 + mul\_set\_effect\_value)
* 시너지 수치 보너스 계산을 먼저 처리한 후에 다른 컨텐츠에서 얻은 스탯과 합연산 한다.

# [확인 완료]스킬 카드 최대 획득 수 상한(add\_limit\_skill\_tactic\_count)

* 인게임에서 얻을 수 있는 스킬 카드 최대 수치
* 해당 횟수만큼 스킬 카드를 선택할 수 있다.
  + 기본 값 : 3
  + 최대 값 : 5
* 최대 횟수 도달 시 해당 유닛의 스킬 카드는 더이상 등장하지 않는다.
  + 최대 획득 시 UI 표현 고민 필요 “파이어 강화 완료!’

# [보류/세부 기획 필요]카드 밴 횟수 증가(add\_limit\_tactic\_ban\_count)

* 카드 도감에서 원하지 않는 카드를 잠금 수 있음.
  + 단, 일반 카드는 기본 카드라서 잠글 수 없다.
* 잠금 상태의 카드는 인게임 택틱 선택지에서 제외된다.
* 기본 횟수는 0이고, 특별 성장에서 횟수를 늘려준다.
* 도감 UI 기능 추가 필요
  + 개별 카드 선택 시 [밴] 버튼 추가
  + 도감 리스트에서 [밴] 상태 표시
  + 도감 리스트에 [밴 리스트 관리] 기능 추가
    - 해당 팝업 UI에서 밴 리스트 확인하고, 제외하는 기능까지는 추가 필요

# [보류]초당 경험치 증가 (add\_encrease\_exp\_by\_auto )

* 특별 성장 찍으면 매 초 마다 add\_encrease\_exp\_by\_auto 만큼 경험치 증가
  + 기본 값 0
    - 값 추가 시 매 초 마다 추가한 수치만큼 경험치 증가
* 게임의 난이도를 보정해주는 의미가 있음.
