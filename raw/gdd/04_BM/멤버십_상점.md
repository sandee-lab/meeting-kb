# 아웃게임 멤버십 상점

> **요약:** 기간제 멤버십 상품의 구성·혜택·보상 지급 규칙을 설계. 광고 제거, 스태미나 추가, 매일 보상 등 멤버십 혜택을 정의한다.
> **현재 상태:** 이후 버전으로 대체됨

**M7\_아웃게임\_멤버십상점**

![](data:image/png;base64...)

# 개요

* 기간제 상품 (이하 멤버십 상품) 상점을 구성한다.

# 멤버십 상품 구성 요소

## 혜택

### 멤버십 효과

* + - 구매 즉시 적용되며, 멤버십 만료 시 해제된다.
    - 효과 종류
      * 광고 제거 (개별 컬럼으로 제어)
      * 스태미나 상한 증가 (스탯 처리 : add\_energy\_default\_capacity)
      * 빠른 순찰 횟수 증가 (스탯 처리 : add\_stage\_quick\_sweep\_daily\_try)
      * 순찰 수익 증가 (스탯 처리 : mul\_stage\_sweep\_reward)
      * 던전 소탕 횟수 증가 (스탯 처리 : - )
      * 인게임 유닛 리롤 횟수 증가 (스탯 처리 : - )
      * (효과 종류는 상품 구성에 따라 추가될 수 있습니다.)

### 우편 보상

* + - 구매 보상
      * 구매 즉시 ~~우편으로~~ 제공되는 보상이다.
      * 테이블 입력 값에 따라 구매 보상이 없거나, 제공될 수 있다.
    - 매일 보상
      * 구매일부터 종료일까지, 매일 수령할 수 있는 보상이다.
      * 지급 횟수는 상품 적용 기간과 동일하다.
        + 30일 적용 상품의 경우, 매일 보상 30회 지급
      * 매일 보상 지급 횟수는 서버 갱신 시간을 기준으로 차감한다.
        + 서버 갱신 시간은 현재 한국 시간 기준 A.M. 09:00 (UTC+9)
      * 매일 보상은 유저 접속시, 당일 보상을 받지 않은 경우에 우편으로 지급된다.
        + 구매일에는 구매 즉시 지급한다.
        + 서버 갱신 시간을기준으로, 당일 보상이 이미 전송된 경우는 지급하지 않는다.
        + 미접속일 보상은 지급하지 않는다. (상품에 안내 필요)
      * 테이블 입력 값에 따라 매일 보상이 없거나, 제공될 수 있다.

## 상품 적용 기간

* + 영구
    - 상품 혜택이 영구 적용된다.
    - 영구 적용 여부는 테이블에서 결정한다.
  + 기간제
    - 상품 혜택이 구매 시점부터 만료 시점까지 적용된다.
      * 구매 시점 : 구매 일시 (시각 확인)
      * 만료 시점 : 구매 일시+(적용 기간(일 기준)\*24시간) (시각 적용)
    - 적용 기간은 테이블에서 결정한다.
  + 일일 (2025-09-30 추가)
    - 무료 상품 대상
    - 다음 날 서버 갱신 시간까지 적용된다.

## 상품 만료

* + 기간제 상품에 한해 만료 시점에 혜택 적용을 만료한다.
    - 서버 처리 확인 필요
      * 멤버십 효과 만료 시점은 구매 시 서버에서 내려줌
  + 기간제 상품은 만료 시점부터 재구매할 수 있다.
    - 만료 n일 전 만료 임박 알림을 출력한다.
      * n일 전은 각 상품별로 관리한다.
    - 특정 상품은 만료 n2일 전부터 재구매를 허용할 수 있다. (아래 “[기간 만료 (임박) 알림](#_a2d2xv34v0nk)” 참고)
      * 만료 전 재구매 가능 상품은 별도로 안내한다.

* + 예외처리 (2025-09-30 추가)
    - 타입이 일일(Daily)인 경우, 서버 시간을 기준으로 처리한다.
      * 일일 서버 갱신 시간에 만료 처리(구매 횟수 초기화)

# 처리

## 멤버십 효과 만료 시

* + 스태미나 상한 증가
    - 만료 시점 이후로 스태미나 상한치를 복원한다.
    - 스태미나 상한치보다 보유하고 있는 스태미나가 많은 경우, 유지한다.
  + 빠른 순찰 횟수 증가
    - 만료 시점 이후로 빠른 순찰 횟수를 기본 횟수로 복원한다.
  + 순찰 수익 증가
    - 순찰 수익은 멤버십 효과를 구분하여 지급한다.
      * 순찰 수익 수령 시 , 공용 보상창에 멤버십 적용 보상을 별도 슬롯으로 표시
      * 광고 멤버십 보상 표현은 광고 제거와 표현 통일하여 사용 희망
        + [참조. 광고 제거](https://docs.google.com/document/d/12Ju70MK1VKQoX_qhm07P8OvcFTh7YjkSxvxM_DEpAF0/edit?usp=sharing)

![](data:image/png;base64...)

* + - 수령 시점에 멤버십이 만료되었다면 보너스 수익을 제공하지 않는다.
  + 던전 소탕 횟수 증가
    - 만료 시점 이후로 던전 소탕 횟수를 기본 횟수로 복원한다.
  + 인게임 유닛 리롤 횟수 증가
    - 만료 시점 이후로 리롤 횟수를 기본 횟수로 복원한다.

## 멤버십 효과 중첩 시

* + 멤버십 혜택 중 중복 효과가 있는 경우, 수치를 합산하여 적용한다.

## 기간 만료 안내 팝업

* + 접속 중일 때 만료하는 경우
    - 로비에 복귀 했을 때 표시
      * 메인 화면 진입 바로가기 아이콘 말풍선
        + “만료된 멤버십이 있습니다”
        + 1회 알림 후 종료
  + 만료 후 접속하는 경우
    - 만료 시점에 푸시 메시지 발송

    - 만료 후 첫 로그인 시 팝업 출력 (재구매 유도)
      * 표시할 멤버십이 2개인 경우 먼저 만료되는 것부터 순차적으로 보여준다.

      * 만료일 가까운 것부터 순차적으로 출력
