# 피해량 표시 UI 개선 제안

> **요약:** 피해량 표시를 웨이브별 초기화하고, 유닛 종류별 합산 피해량과 개체별 피해량(1마리당 가치)을 분리 표시하여 덱 전략 판단 기준을 제공하는 UI 개선 제안. 탭을 종류별/개체별로 교체하고, 14웨이브 기준 덱 저장을 포함한다.
> **현재 상태:** 미적용

**피해량 표시 UI 개선 제안**

문서 유형: UI 개선 제안 | 작성: 윤호민 | 상태: 검토 요청

# **1. 문제**

* 초당 피해, 누적 피해의 의미를 찾을 수 없음
* 유저가 전략을 정할 때 판단의 기준이 될 수 없음

# **2. 해결**

* 웨이브 마다 초기화
* 모든 유닛 종류별로 합쳐서 피해량
  유저는 해당 덱에서 그 유닛이 차지하는 비중을 알 수 있음
  이는 택틱 고를 시 유용함
* 1마리 각각의 피해량

N마리 유닛을 사용할 경우 위에 피해량/N

유저는 슬롯 1칸당 가치를 알 수 있음

이는 유닛 교체 시 유용함

# **3. 표시 스펙**

* 탭 이름 교체

종류 별 피해량

개체 별 피해량

* 상단 피해량 표시를 “N웨이브 피해량”로 변경
* 덱 저장 시 14 웨이브 피해량으로 (15웨이브는 극단적) 저장
  덱 완성은 14웨이브이니까...

# **4. 비고**

* 초당 피해량 필요하면 추가
