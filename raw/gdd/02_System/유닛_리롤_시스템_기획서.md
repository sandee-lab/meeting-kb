# 유닛 새로고침 시스템

> **요약:** 인게임에서 유닛을 드래그하여 새로고침(동일 등급 다른 유닛으로 교체)하거나 판매하는 시스템 기획이다. 새로고침은 횟수 제한이 있으며 택틱으로 추가 획득 가능하고, 유닛별 reroll_price/-1로 가능 여부를 제어한다. 판매는 sell_price만큼 미네랄을 획득한다. 드래그 앤 드롭 UX로 판매/새로고침 영역을 구분하며, 해금 전/횟수 소진/불가능 유닛 등 다양한 예외 상태에 대한 피드백을 정의한다.
> **현재 상태:** 미적용

유닛 새로고침 시스템

### **개요**

* 새로고침 기능을 추가하여 플레이어가 원하는 방향의 빌드를 구성할 수 있다.
* 유닛 새로고침 기능을 추가하고 판매 UI/UX를 개선한다.

### **해금 시점**

* **유닛 판매 (Unit\_Sell):** Contents Unlock 데이터 테이블에 정의된 시점에 해금.
* **유닛 리롤 (Unit\_Reroll):** Contents Unlock 데이터 테이블에 정의된 시점에 해금.

### **해금 연출**

* 해금 아이콘이
* 아이콘과 텍스트 출력
  + "[유닛 새로고침]"Local\_Contents\_Unlock\_UnitReroll
  + "[유닛 판매]"Local\_Contents\_Unlock\_UnitSell
* 종료 및 이동
  + Fly 액션: 중앙의 아이콘이
    - 원래 위치로 날아간뒤 사라짐

### **유닛 데이터 (Static: UnitData)**

각 유닛의 고유 속성값으로, 테이블(Table)에서 관리합니다.

| **변수명** | **타입** | **값의 의미 (Logic)** |
| --- | --- | --- |
| **reroll\_price** | int | **1 이상:** 리롤 가능 (리롤 시 해당 값만큼 차감)  **-1:** 리롤 불가 (잠금 처리) |
| **sell\_price** | int | **0 이상:** 판매 가능 (판매 시 에 해당 값만큼 추가)  **-1:** 판매 불가 (잠금 처리) |

### **게임 중 리롤 횟수**

| **변수명** | **타입** | **설명** | **저장 여부** |
| --- | --- | --- | --- |
| **reroll\_count** | int | **현재 남은 리롤 횟수**  - 기본값: 5 (업그레이드 반영)  - 택틱 획득 시 증가 가능  - 사용 시 reroll\_price 만큼 감소 | **O (필수)** |

### **새로고침 횟수 표시**

리롤 횟수가 택틱 등에 의해 유동적으로 증가할 수 있으므로, 분모(Max)를 제거하고 **잔여 횟수**만 표시합니다.

* **기존:** "15/15"
* **변경:** (예: "15회 남음") Local\_Text\_Count\_left
  + *예시:* 택틱으로 3회 추가 획득 시 → "18회 남음"로 즉시 갱신.

### **UI/UX 상세 기획**

### **기본 동작**

* **진입:** 관리 페이즈에서 유닛을 **터치하고 드래그로 영역에 도달하면**, 화면 하단(또는 지정된 영역)에 상호작용 가능한 UI가 등장.
* **영역 구성:**
  + **판매 영역:** 뽑기(소환) 버튼 왼쪽
  + **새로고침 영역:** 뽑기(소환) 버튼 오른쪽.
* **활성화:**
  + 해당 영역에 올라와있을 때: 말풍선 배경이 색깔이 파란색으로 변경됩니다.

###

###

### **유닛 판매**

1. 유닛을 **드래그로 영역에 도달하면** 화면 하단에 [판매 영역]이 슬라이드되고, 판매 가격이 표시됨 (예: "+5 미네랄")
2. 유닛을 해당 영역 위로 가져가면 말풍선 배경이 파란색으로 바뀜
3. 드롭(Drop) 시 판매 완료 및 재화 획득.

### **유닛 새로고침**

1. 유닛을 **드래그로 영역에 도달하면** 화면 하단에 [새로고침 영역]이 슬라이드되고, 남은 횟수 표시됨 (예: "15회 남음") Local\_Text\_Count\_left
2. 유닛을 해당 영역 위로 가져가면 말풍선 배경이 파란색으로 바뀜
3. 드롭(Drop) 시 해당 유닛이 사라지고, **동일 등급**의 **다른 유닛** (자기 자신 제외) 중 하나가 즉시 소환
4. **위치:** 드래그 전 위치(슬롯)로 새로운 유닛이 생성

### **데이터 처리**

* 현재 판의 남은 새로고침 횟수 추적
* 새로고침 사용 시 카운트 차감
* 게임 종료 시 리셋

### **유닛 리롤이 해금되지 않았을 때**

* **어떻게 표현?**
  + 유닛 **드래그로 영역에 도달하면** 새로고침 영역이 등장하되 [잠금 아이콘]이 덮여 있고 전체적으로 **회색조(Dimmed)** 처리됩니다.
  + **말풍선** “잠김” 텍스트 출력 Local\_Text\_Locked
* **어떤 반응이 나오는가?**
  + **드롭 불가:** 잠긴 영역 위로 유닛을 가져가도 '하이라이트(활성화)' 효과가 나타나지 않습니다.
  + **피드백:** 유닛을 억지로 해당 영역에 드롭하면, 리롤이 실행되지 않고 유닛은 원래 있던 자리로 튕겨져 돌아갑니다.
  + **시스템 메시지:** 화면에 **토스트 메시지** 출력.
    - *문구 예시:* 유닛 드래그 완료 시 콘텐츠를 해금하기 위한 조건이 토스트메시지로 출력. "새로고침 기능은 스테이지 {0} 클리어 후 사용할 수 있습니다." Local\_Toast\_Locked\_Unit\_Reroll Contents Unlock 참조

### **해당 유닛이 '새로고침' 불가능한 속성일 때 (reroll\_price가 -1)**

* **어떻게 표현?**
  + 해당 유닛을 **드래그로 영역에 도달하면**, 말풍선 배경이 **[붉은색 비활성]** 상태로 표시
  + 말풍선 텍스트 **"불가능"**
* **어떤 반응?**
  + **드롭 불가:** 영역 위로 유닛을 가져가도 활성화되지 않습니다.
  + **피드백:** 드롭 시 새로고침이 실행되지 않고 유닛은 **원래 자리로 튕겨져 돌아갑니다.**
  + **시스템 메시지:** 토스트 메시지 출력.
    - *문구 예시:* "새로고침할 수 없는 대상입니다."Local\_Toast\_Cant\_Reroll\_Particular\_Unit

### **유닛 리롤이 가능한 유닛인데, 유닛 리롤 횟수를 다 썼을 때**

* **어떻게 표현?**
  + 말풍선 텍스트가 **빨간색**으로 표시됩니다
  + 영역 내 텍스트: "횟수 부족"Local\_Text\_Useup\_Unit\_Reroll
* **어떤 반응?**
  + **드롭 불가:** 영역 위로 유닛을 가져가도 활성화되지 않습니다.
  + **피드백:** 유닛을 드롭하면 리롤이 실행되지 않고 원래 자리로 튕겨져 돌아갑니다.
  + **시스템 메시지:** 화면에 **토스트 메시지** 출력.
    - *문구 예시:* "남은 새로고침 횟수가 없습니다."Local\_Toast\_Useup\_Unit\_Reroll

### **유닛 판매가 해금되지 않았을 때**

* **어떻게 표현?**
  + 유닛**드래그로 영역에 도달하면** 새로고침 영역이 등장하되 [잠금(Lock) 아이콘]이 덮여 있고 전체적으로 **회색조(Dimmed)** 처리됩니다.

* **어떤 반응?**
  + **드롭 불가:** 잠긴 판매 영역 위에 유닛을 놓을 수 없습니다.
  + **피드백:** 유닛을 드롭하면 판매되지 않고 **원래 자리로 튕겨져 돌아갑니다.**
  + **시스템 메시지:** 토스트 메시지 출력.
    - *문구 예시:* "판매 기능은 스테이지 {0} 클리어 후 사용할 수 있습니다."Local\_Toast\_Locked\_Unit\_Sell

### **해당 유닛이 '판매' 불가능한 속성일 때 (sell\_price가 -1)**

* **어떻게 표현?**
  + 해당 유닛을 **드래그로 영역에 도달하면**, 말풍선 배경이 **[붉은색 비활성]** 상태로 표시

    - *문구 예시:* "판매할 수 없는 대상입니다."Local\_Toast\_Cant\_Sell\_Particular\_Unit
