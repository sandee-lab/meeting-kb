# 시간의 균열 최고 기록 표시 UI 개선 제안

> **요약:** 시간의 균열 도전 탭에서 최고 기록을 확인할 수 없는 문제를 해결하기 위해 타이틀 우측에 인라인으로 최고 기록을 표시하는 UI 개선 제안. 폰트 사이즈, 색상, 갱신 시점 등 표시 스펙을 포함한다.
> **현재 상태:** 미적용

**시간의 균열 — 최고 기록 표시 UI 개선 제안**

문서 유형: UI 개선 제안 | 작성: 윤호민 | 상태: 검토 요청

# **1. 문제**

* 도전 탭 진입 시 내 최고 기록을 확인할 방법이 없음
* 덱 박스의 KP(덱 점수)와 혼동할 여지가 있어 별도 표시 필요

# **2. 해결**

* 타이틀 '시간의 균열' 우측 여백에 최고 기록 인라인 표시

표시 예시: **시간의 균열 🏆 최고 기록 12,030,140점**

* 덱 점수(KP)와 레이블로 명확히 구분

# **3. 표시 스펙**

| **항목** | **내용** |
| --- | --- |
| 표시 텍스트 | 🏆 최고 기록 N,NNN,NNN점 |
| 위치 | 타이틀 '시간의 균열' 우측 인라인 |
| 폰트 사이즈 | 타이틀 대비 70% 수준 (소형 텍스트) |
| 색상 | 흰색 또는 골드 계열 (기존 타이틀 색상 체계 따름) |
| 기록 없을 때 | '최고 기록 없음' 또는 비표시 처리 (개발팀 확인) |
| 갱신 시점 | 플레이 결과 화면 확인 후 즉시 반영 |
| 단위 | 점 (KP 변환 없이 원본 수치 표시) |

# **4. 비고**

* 타이틀 행 공간 부족 시 타이틀 바로 아래 서브텍스트 행으로 대안 검토
* 아이콘(🏆) 리소스 부담 시 텍스트 전용 처리 가능 — 개발팀 협의
