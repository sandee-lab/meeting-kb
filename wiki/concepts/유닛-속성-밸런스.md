## 개요

유닛은 속성(물리/화염/냉기/전기/카오스)별로 구분되며, 각 속성은 고유한 상태이상 효과와 시너지 스탯을 갖는다. 등급은 일반~전설까지 5단계이며, 전설 유닛은 하위 등급 유닛들을 조합하여 제작한다. M5(2025.07)부터 M8(2025.10)을 거쳐 2026.03까지 속성 시너지 체계, 유닛 역할 분담, 등급별 파워 밸런스가 지속적으로 개선되어왔다.

## 상세 내용

### M5 3차 빌드 — 전투 개선 방향 (2025.07.24)

- 넉백 시스템 도입 및 포지션별 특징 부각 (드릴병, 바주카병, 엔지니어)
- 근접 유닛은 전방만 사격하는 것으로 정리
- 시너지를 간단한 방식으로 정리: 배치 시너지, 동일 유닛 버프
- 전투 템포 개선: 이동속도/공격속도 상향, 웨이브에 따라 맵 확장, 피격/장판 대미지 감소, 이동 자유도 향상
- 택틱 종류 다양화 필요
- 에픽 등급 유닛 1마리 추가 필요성 제기
- 재화 정리: 레벨업 삭제, 미네랄은 유닛 소환 재화로, 가스는 택틱 추가 구매용으로, 슬롯머신은 오브젝트로 변경

### M8 유닛 및 택틱 — 유닛 역할 재설계 (2025.10.18~21)

수집형 RPG와 같이 각 유닛 간 역할을 명확히 부여하는 방향으로 수정. 각 유닛의 특징이 명확해지면 해당 유닛의 기능을 합쳐 하나의 전설 유닛으로 "덱 압축"하는 구조.

**등급별 유닛 구성:**

| 등급 | 유닛 예시 | 공격 형태 |
|------|----------|-----------|
| 일반 | 총 주니어, 불 주니어, 버블 주니어, 칼 주니어 | 단일 연사, 근거리 광역, 다중 연사 |
| 고급 | 총, 불, 칼, 버블, 섀퍼, 쉴드, 기계형(신규) | 연사, 광역, 타워/위성 소환 |
| 희귀 | 라이플, 바주카, 블레이즈, 소드마스터, 펀치, 밴쉬 | 다중 연사, 원거리 다중 타겟, 부메랑 투사체 등 |
| 에픽 | 마린, 탱크, 파이어캣, 제다이, 신규(버블스 대체) | 원거리 즉시 관통, 원거리 광역, 확산형 등 |

**전설 유닛 조합식 구조:**
- 각 속성별로 조합식이 정의되어 있으며, 재료 유닛의 공격 형태와 속성을 계승
- 예시: 라이더 = 섀퍼 3 + 바주카 1, 헬파이어 = 파이어캣 2 + 펀치 1, 토르 = 쉴드 3 + 파이어캣 1

### M8 외부 피드백 — 속성별 상태이상 체계 (2025.10.18)

각 속성에 고유 상태이상과 시너지 효과를 부여하는 체계 정의:

| 속성 | 상태이상 | 효과 | 시너지 스탯 | 컨셉 |
|------|---------|------|------------|------|
| 물리 | 출혈 | 1스택 당 추가 물리 대미지 | 이동속도, 공격력 | 메카닉 |
| 불 | 점화 | 1스택 당 추가 화염 대미지 | 체력 | 바이오닉 |
| 냉기 | 동상 | 1스택 당 감속 n% | 크리티컬 | 매지컬 |
| 전기 | 감전 | 1스택 당 추가 감전 대미지, 20스택 이후 주변 2마리에 전이 | 공격속도 | 메카닉 |
| 카오스 | 붕괴 | 남은 체력 10% 미만 시 즉사 | 쿨타임 | 사이언스 퍼실리티 |

**전략적 상호작용 개선 방향:**
- 전설 유닛 단일 성능이 아니라, 전설 유닛까지 쌓은 시너지를 터뜨리는 방향
- 캐릭터가 스킬 역할을 하므로, 단순히 재료로 소진하는 것이 아니라 플레이에 의미있게 배치되어야 함
- 운보다는 의도된 조합의 다양성에 초점
- 피아 식별 및 시인성 개선: 적 유닛 아웃라인 컬러/톤 조정, 충돌 대미지와 HP 개념 정리

### M8 전략성 개선 회의 (2025.10.22)

- 유닛 개별성장은 현재 전설 획득 방식과 맞지 않기 때문에 배제
- 전설 유닛이 재료 유닛의 공격 형태와 속성을 계승 받는 것은 좋으나 단조로워질 수 있으므로 보완 필요
- 전설을 [계승]과 [진화] 두 계열로 나누는 방안 제안
  - 초기 학습이 낮은 단계에서는 계승 전설 위주로 사용
  - [카오스] 계열 유닛이 포함되는 레시피에서 진화가 되는 방식도 고려
  - 특정 택틱으로 유닛 무기 타입/형태를 변화시켜 히든 조합식 활용 가능
- 택틱 종류: [스택]과 [단일] 두 가지
  - 스택: 3단계(일반→고급→레어)로 진화, 마지막 단계에 스페셜 택틱 추가. 리스크 감수 후 리턴을 크게 받는 도파민 구조
  - 단일: 공격 형태에 변화를 주는 즉시 효과. 지휘관/단일 유닛 대상 택틱도 고려
- 배치 위치와 관련된 시너지 제안: 불속성 유닛 4마리를 연속 배치 시 지휘관 근처에 불의 고리 생성, 방패병 2마리 + 불속성 2마리 연속 배치 시 방패에 불 효과 추가 등
- 단일 유닛 시너지 불필요, 대신 배치된 유닛 조합에 따라 지휘관이 스킬을 상속받는 방향

### 속성 시너지 상세 설계 (2026.01.10~12)

모든 속성 덱은 기본 컨셉이 존재하지만, 추후 업데이트 시 다양한 베리에이션이 가능하도록 설계. 상태이상을 다양하게 다루기 위한 [에너지 차지] 메커니즘 도입.

**화염 속성 빌드:**
- 컨셉: 체력을 올리고 이속 버프를 받아 몬스터와 충돌하며 전장을 휩쓸고 다님. 짧은 웨이브에서 강력
- 시너지 스탯: 체력
- 상태이상: [화염] — 대상의 초당 최대 체력 1% 감소, 최대 5스택
- 지휘관 스킬: 운석 포격 — 체력 10% 이하인 화염 대상에게 즉사
- 메인 전설: 헬파이어 — 불주먹, 피격 대상 즉시 화염 3스택. 패시브로 불굴의 에너지 차지 → 아군 이동속도 100% 증가
- 서포트 전설: 간호순 — 아군 최대 체력 100% 증가, 초당 3% 회복
- 유닛 구성: 고급(파이어 뱃)→원거리 범위 화염, 희귀(플레임)→이동경로+주변 화염, 에픽(라이더)→서브딜러

**전기 속성 빌드:**
- 컨셉: 빠른 공격과 누적 디버프로 원거리 단일 딜 특화. 연쇄 반응 도파민
- 시너지 스탯: 공격 속도
- 상태이상: [감전] — 받는 피해 2% 증가, 최대 10스택
- 메인 전설: 스카우터 — 전기 탄환 발사, 아군 공격 적중 시 추가 전기 탄환 생성 → 가로형/세로형 레이저 연쇄
- 서포트 전설: 토르 — 전기 망치, 전기장 생성으로 아군 전기 속성 피해 증가

**냉기 속성 빌드:**
- 컨셉: 대규모 디버프로 광역/단일딜 모두 유리. 동상 스택 쌓아 빙결 터뜨리는 재미
- 시너지 스탯: 크리티컬
- 상태이상: [동상] 5스택 시 [빙결] — 3초간 동결
- 메인 전설: 사토루 — 전방/후방 영역에 얼음 칼날 소환. 영역 에너지 차지 → 광역 빙결 + 크리티컬 대미지 100% 상승
- 서포트 전설: 빙결 술사 — 얼음 파편 생성, 빙결 대상 타격 시 얼음 파편 추가 생성

**카오스 속성 빌드:**
- 컨셉: 광역딜 특화. 리스크를 안고 플레이하지만 소환수로 안정성 확보. 쿨타임 감소가 핵심
- 시너지 스탯: 쿨타임
- 상태이상: [저주] — 1스택 당 이동속도 10% 감소, 최대 5스택
- 메인 전설: 고스트 — 광역 폭탄, 소환된 플라즈마 유령 사라질 때 폭파 대미지 추가
- 서포트 전설: 아비터 — 시간 역장(투사체/몬스터 행동 50% 감속), 암흑 에너지 차지 → 쿨타임 가속

### 3월 2~3주차 밸런스 조정 (2026.03.12~19)

**전설 유닛 합성 재료 간소화:**
- 속성별로 쉬운 유닛(에픽 1 + 고급 1)과 어려운 유닛(에픽 2) 두 가지로 구성

| 속성 | 1티어(에픽1+고급1) | 2티어(에픽2) |
|------|-------------------|-------------|
| 불꽃 | 간호순 | 헬파이어 |
| 냉기 | 섀도 | 빙결술사 |
| 전기 | 스카우트 | 토르 |
| 카오스 | 미스테리오(추가 예정) | 고스트 |
| 물리 | 시즈, 엣지, 캡틴, 포어맨, 오퍼레이터, 메카 | — |

**등급별 파워 밸런스 변경:**

| 등급 | 기존 | 변경 |
|------|------|------|
| 일반 | 1 | 1 |
| 고급 | 2 | 2 |
| 희귀 | 4 | 5 |
| 영웅(에픽) | 8 | 12 |
| 전설 | 10~16 | 16~24 |

**스마트 밴 시스템:**
- 유닛 리롤 시 한번 리롤한 유닛은 결과로 나오지 않음 (같은 등급 내 모두 밴 되면 밴 초기화)
- 택틱도 동일한 스마트 밴 적용
- 리롤 횟수가 쌓일수록 원하는 카드를 찾을 확률이 선형적으로 상승

**기타 밸런스 조정:**
- 물리 속성 → 무속성으로 변경, 범용 서포트 유닛 포지션으로 활용
- 무속성 유닛을 특정 속성으로 바꾸는 택틱을 통해 속성 시너지 강화
- 택틱 등급 확률 동적 제어: 포메이션에 배치한 유닛의 등급 평균이 높아질수록 높은 등급 카드 출현 확률 증가
- 몬스터 이동 속도 하향 (1.2 → 0.6)
- 보스 등급 몬스터 체력 가중치 하향 (100 → 50)
- 유닛 사이즈 정리: 커먼 1→0.9, 언커먼 1→0.95, 지휘관 0.63→0.7

### 전설 유닛 밸런스 재설계 — 기본:시너지 50:50 (2026.03.05)

> 출처: [[raw/meetings/[260305]TCA_2월 빌드 피드백에 대한 개선 방향]], [[raw/feedback/Ncsoft회의록_260310]]

2월 빌드 피드백에서 "전설을 뽑아도 시너지 없이는 쓸모없다"는 문제가 핵심으로 지목되어, 전설 유닛의 기본 성능 비중을 대폭 상향하는 방향으로 재설계.

**변경 전 vs 변경 후:**

| | 기본 성능 | 시너지 성능 |
|--|-----------|------------|
| 기존 | 0~20% | 80~100% |
| 변경 | 50% | 50% |

- 전설 단일 유닛 소환만으로도 명확한 체감과 만족감 제공
- 시너지 조합 완성 시 성능 극대화 (곱연산 구조 유지)
- 포어맨 등 시너지 없으면 '꽝'인 유닛 → 자체 터렛 소환 공격 추가

**카드 등급 확률 배치점수 보정:**

배치된 유닛 등급 합산 점수에 따라 카드 출현 확률이 동적으로 조정됨.

| 등급 | 점수 |
|------|------|
| 일반 | 1 |
| 고급 | 2 |
| 희귀 | 4 |
| 영웅 | 10 |
| 전설 | 10 |

| 배치 점수 | 소환 레벨 | 일반 | 고급 | 희귀 | 영웅~전설 |
|-----------|----------|------|------|------|----------|
| 1~10 | 1 | 45% | 30% | 20% | 5% |
| 21~30 | 3 | 20% | 35% | 35% | 10% |
| 41~50 | 5 | 15% | 20% | 45% | 20% |
| 71~80 | 8 | 5% | 20% | 30% | 45% |

**리롤 풀 보정 (스마트 밴 강화):**

### GDD 기획서 — 전설 유닛 설계 원칙 (2026.03.23, 상태: 구현됨)

> 출처: [[raw/gdd/02_System/전설유닛_설계_논의_맥락]]

**핵심 설계 원칙:**
- 기본 성능 50% + 시너지 성능 50% — 단독 소환 시에도 효능감 확보, 조합 완성 시 극대화
- 10점(무속성) 전설은 이중 효과: ① 하위 유닛 조합 효과, ② 범용 버퍼 효과 (속성 전환 후에도 유지)
- 간호순·포어맨은 이 원칙에 미달 → To-Be 설계 완료 (간호순: 화염 주사기 투척 + 아군 소량 회복 복합형 채택, 포어맨: 자체 터렛 다수 소환)

**속성 전환 시스템 (확정):**
- 게임 내 희귀 확률로 "무속성 → 특정 속성 전환" 전략 카드 등장
- 선택 시 모든 무속성 유닛이 해당 속성으로 전환 (속성 디버프 능력은 미부여, 시너지 수혜만)
- 한번 선택 시 속성 전환 카드 재등장 없음

**기각된 아이디어:** 인챈트/각성/변이로 무속성 덱 강화, 소환 덱 독립 루트화, 광부 → 상위 유닛 2갈래 분기

**전설 조합식:** 1전설 = 1조합식 1:1 대응. 현재 풀(고급6+희귀6+영웅5)로 가능한 조합 161가지 → 전설 50종까지 대응 가능

- 하위 유닛만으로는 일정 웨이브 이상 진행이 힘들도록 밸런싱
- 등장 몬스터 개체 수 증가 및 크기 축소로 플레이 압박감 강화
- 권장 전투력 차이 10~20% 수준에서 유저가 허들을 느끼도록 조정
- 진행에 따라 저등급 유닛은 3~4대, 고등급은 1~2대로 처치 밸런싱
- 한방 딜 중요성 강화

**유닛 역할 부재 (테스터 피드백):**
- 대부분 전설 유닛이 딜러 역할만 강조 → 효율 차이로 버려지는 유닛 발생
- 유틸리티 유닛 수 부족 (광역 딜링, 단일 보스 특화, CC, 진영 모아주기 등)
- 근접 유닛 매력도 낮음 (원거리가 압도적으로 유리한 디자인)

**개선 제안:** 딜링 외 여러 역할 부여 (CC, 탱킹, 진영 조정 등), 전설 유닛도 역할 분화

### 스탯 체계 — add/mul 이중 구조 (2026.04, 상태: 미적용)

> 출처: [[raw/gdd/02_System/TCA_스탯 리스트]]

**기본 방향성:**
- 강화 수치는 기본적으로 곱연산 처리. 다른 이름끼리 곱연산, 같은 이름끼리 합연산
- 물리 피해와 마법 피해로 구분하여 각각 다르게 연산
- 인게임 규칙에 따라 초기화되지 않는 스탯은 `스탯`으로, 초기화되는 것은 `buff_` 스탯으로 처리
- 아웃게임·인게임 공통 스탯은 밸런스에 따라 분리하거나 수치 제한 가능

**아웃게임·인게임 구분 원칙:**

| 구분 | 의미 | 예시 스탯명 |
|------|------|-----------|
| 아웃게임 | 유닛 육성/도감으로 영구 보유 | `add_atk`, `mul_atk_1`, `mul_max_hp_1` |
| 인게임 | 택틱·버프로 전투 중 적용, 종료 시 초기화 | `mul_atk_2`, `mul_max_hp_2`, `buff_mul_atk_1` |

**핵심 스탯 공식 요약:**

| 스탯 분류 | 타입 예시 | 계산 방식 |
|----------|----------|----------|
| 기본 공격력 | `add_atk` | 합산 후 최종 공격력에 가산 |
| 공격력 증가 | `mul_atk_1` (아웃게임) | `(1 + sum(mul_atk_1)) * (1 + sum(buff_mul_atk_1)) * (1 + sum(mul_atk_2))` 순차 곱 |
| 공격력 증폭 | `mul_atk_3` (인게임, 유닛 버프) | 최종 공격력에 추가 곱연산 |
| 속성 위력 | `mul_increase_fire_damage` 등 | 해당 속성 스킬 피해에 곱연산 |
| 속성 피해 보너스 | `fire_weight` 등 | 위력 증가에 우선 곱연산 |
| 받는 피해 증가 | `mul_increase_chaos_damage_taken` 등 | 공격자 속성 기준으로 최종 피해에 곱 |

**공격 빈도 스탯:**
- 공속 계열: `add_atk_speed` (초기값), `mul_atk_speed` (증가), `mul_atk_speed_1` (증폭)
- 쿨타임 계열: `add_skill_recharge_speed` (초기값), `mul_skill_recharge_speed` (증가), `mul_skill_recharge_speed_1` (증폭)

**자원 획득 스탯:**
- `add_base_gas_amount` — 게임 시작 시 기본 미네랄
- `add_wave_reward_gas_amount` — 웨이브 클리어 시 미네랄 추가
- `add_gas_interest_rate` — 아머리 진입 시 이자율 (아웃게임 전용)
- `add_limit_interest_value` — 이자 상한

### 전설 유닛 사용성 논의 — 문제와 해결 방향 (2026.03, 상태: 미적용)

> 출처: [[raw/gdd/02_System/TCA_전설유닛_논의_맥락_종합정리]]

**사용성 낮음의 원인:**
- 기존 전설 유닛은 시너지 없이 단독 소환 시 거의 효용이 없는 구조 (기본 성능 0~20%)
- 소환 덱 독립 루트를 구성하려면 전설 4슬롯 필요 → 비현실적
- 무속성 조합 덱이 중반에 너무 빨리 완성되어 이후 할 게 없음

**확정된 해결 방향:**

| 방향 | 내용 |
|------|------|
| 기본:시너지 50:50 | 단독 소환 시에도 명확한 효능감 제공 |
| 10점 무속성 전설 이중 효과 | ① 하위 유닛 조합 시너지 ② 범용 버퍼 (속성 전환 후에도 유지) |
| 속성 전환 시스템 | 무속성 유닛 전체를 특정 속성으로 전환 — 속성 시너지 수혜 대상 등록 |
| 무속성 덱 → 속성 덱 전환 | 무속성 덱은 속성 덱의 하위 호환으로 자연스러운 전환 동선 제공 |
| 소환 유틸리티화 | 포어맨·오퍼레이터: 독립 루트 아닌 자유 슬롯 유틸리티 포지션 |

**기각된 방향:**
- 인챈트·각성·변이로 무속성 덱 더 깊게 파기 → 방향이 틀림
- 소환 덱 독립 3번째 루트화 → 전설 4슬롯 필요 + 엔지니어 완성 난이도 낮음
- 광부 → 상위 유닛 2갈래 분기 → 광부 비중 과도

**10점 전설 4종 기본 루트 (2×2 대칭 구조):**

|------|------|-----------|----------|--------|
| 공속 | 고양이 사부 | 클리버 공속 증폭 | 전체 공속↑ | 공격형 |
| 공속 | 캡틴 | 실드 추가 방패 | 공격 시 보호막 획득 | 생존형 |

| 쿨타임 | 메카 | 로켓 유도 미사일 | 피격 시 반격 | 생존형 |

**덱 전환 흐름 예시 (공속 고양이 사부 루트):**

| Phase | 상태 | 미네랄 용도 |

| 역할 | 연사형 디버퍼 — 약화를 광범위하게 퍼뜨리고 약해진 적을 처형 |
| 메타 포지션 | 고스트(한방형)의 시너지 파트너, 카오스 속성 조합 필수 유닛 |
| 외형 컨셉 | 스파이더맨의 미스테리오 (어항 헬멧, 연기/안개) |

**스킬 구성:**

```
미스테리오
├─ [저주의 투사체] — Base Skill
│   ├─ Lv1: 다수 투사체 발사 → 적중 시 피해 + [약화] 부여 (공격력 -20%, 카오스 피해 +30%, 5초)
│   └─ Lv2: HP ≤ 30% 일반 몬스터 → 즉결 처형 (보스·엘리트 제외)
├─ [약화 전이] — Sub Skill 1
│   └─ Lv1: [약화] 걸린 적 사망 → 주변 적에게 [약화] 전이
└─ [환영] — Sub Skill 2
    └─ Lv1: [저주의 투사체]를 사용하는 환영(소환수) 소환
```

**핵심 시너지 루프:**
- 환영 + 미스테리오 동시 연사 → 저주 투사체 ×2 (약화 부여 + HP 감소) → 일반 몬스터 HP 30% 도달 시 즉결 처형 → 약화 전이로 주변 확산 → 고스트 강타(카오스 증폭 피해, 약화 대상 +30~50%)

**고스트 대비 포지션:**

| 항목 | 고스트 | 미스테리오 |
|------|--------|-----------|
| 공격 빈도 | 5초 쿨 한방 | 잦은 연사 |
| 약화 부여 | 없음 (수혜자) | 직접 부여 |
| 대보스 성능 | 높음 (damage_weight 1.5) | 중간 |
| 대다수 성능 | 낮음 (단일 타겟) | 높음 (광역 약화 + 즉결 처형) |

**구현 방식:** 신규 코드 없이 기존 CSV 데이터 조합만으로 구현 가능. InstantDeath value_1=0.3 신규 행 추가만 필요.

**미확정 항목:** 환영 소환 트리거·지속시간, 투사체 개수, 타겟팅 규칙, 쿨타임, 약화 전이 범위 (모두 `미확정`).

### 개별 유닛 기획서 — 채굴맨 (2026.04, 상태: 초안)

> 출처: [[raw/gdd/02_System/채굴맨_기획서]]

| 항목 | 내용 |
|------|------|
| 등급·속성 | 전설 (Legendary), 무속성 |
| 점수 | 10점 |
| 계열 | 쿨타임 |
| 합성 재료 | 엔지니어 + 엔지니어 |
| 역할 | 2단계 전환형 경제 유닛 (경제 구간 → DPS 전환) |
| 메타 포지션 | 10점 전설의 "경제 루트" (기존 4종 전투형 기본 루트와 차별화) |
| 외형 컨셉 | 채굴 로봇 (기계/로봇, 드릴 장비 장착) |

**10점 전설 원칙 예외 적용:**
- 하위 유닛 시너지 없음, 범용 버퍼 없음
- 2단계 변신 메커니즘 자체가 핵심 가치인 특수 포지션

**스킬 구성:**

```
채굴맨 (채굴 모드)
├─ [채굴] — Base Skill (Passive)
│   ├─ Lv1: 미네랄 획득 시 30% 확률 추가 미네랄 + [채굴] 스택 1 적립
│   └─ Lv2: 스택 2개씩 적립 + 추가 미네랄 확률 50%
└─ [드릴 다운] — Sub Skill
    └─ Lv1: 미네랄 10 소모 → 영구 변신 (드릴 모드 전환, 되돌릴 수 없음)

채굴맨 (드릴 모드, 변신 후)
├─ [드릴 브레이커] — Base Skill
│   ├─ Lv1: 3초 쿨타임마다 드릴을 땅속으로 발사 → 적 위치에서 솟아올라 범위 피해
│   └─ Lv2: 솟아오른 후 인접 적 1체에 관통 피해 추가
└─ [채굴 역량] — Sub Skill (Passive)
    ├─ Lv1: 보유 [채굴] 스택 × 5% 공격력 증가 (최대 20스택 = +100%)
    └─ Lv2: 스택당 7% (최대 20스택 = +140%)
```

**전략적 딜레마 — "언제 변신할 것인가":**

| 선택 | 이득 | 리스크 |
|------|------|--------|
| 빠른 변신 (스택 5~8) | 빠른 DPS 투입, 위기 대응 | 낮은 스택 = 공격력 증가 미미 (+25~40%) |
| 늦은 변신 (스택 15~20) | 높은 스택 = 폭발적 DPS (+75~100%) | 경제 구간 DPS 공백 길어짐 |
| 변신 안 함 | 미네랄 최대 확보 | DPS 기여 없음, 1슬롯 낭비 |

**구현 과제:** 유닛 내 모드 전환 + 스킬 교체 시스템(변신 시스템) 프로토타입 확립 필요. 택틱 카드는 같은 `unit_group_id` 공유로 변신 전·후 호환.
