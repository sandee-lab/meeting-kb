# M5 7/11 플레이 테스트 가이드

> **요약:** 7월 11일 플레이 테스트용 가이드. 유닛 승급 삭제, 초월 방식 변경, 택틱 획득 방식 설명.
> **현재 상태:** 미적용

7/11 플레이 테스트 가이드

# 플레이 목표

* 5챕터 까지 클리어 하는 것을 목표로 플레이
* 아웃 게임에서 얻은 재화들로 골드 성장, 장비 장착 등을 활용하면서 플레이

# 인게임 주요 변경 사항

## 유닛 승급 삭제

* + 기본 유닛의 등급 구분 및 승급 제거

## 유닛 초월 방식 변경

* + 자동 방식에서 > 커맨드센터 오브젝트로 이동해서 초월 하는 방식으로 변경
  + 초월 유닛 마다 필요한 유닛 조합식을 고려하도록 변경.

## 택틱 획득 방식 정리

* + 레벨업 시 > 스탯 획득
  + 오브젝트 발견 > 유닛 획득

## 유닛 스킬 전체적인 수정 및 폴리싱

* + 칼냥이 > 전방으로 자동 공격

# 유닛 획득 UI 설명

## 가챠 획득

* + 일정 시간마다 랜덤한 위치에 생성
  + 맵에 주사위 인디게이터로 표시

![](data:image/png;base64...)

* + 오브젝트 주변에 체류하는 것으로 랜덤하게 유닛 획득

## 선택 획득

* + 게임 시작 시 보스 처치 시 총 3번의 선택 기회
  + 맵에 택틱 박스 인디게이터로 표시

![](data:image/png;base64...)

* + 오브젝트 주변에 체류하는 것으로 유닛 선택 획득

## 유닛 초월

* + 맵에 배치된 커맨드센터에서 유닛 초월 가능
  + 맵에 깃발 인디게이터로 표시
  + 초월 가능한 유닛이 있을 경우, 인디게이터에 녹색 태그 추가
