# 공허의 균열 하단 탭 UX 개선 제안

> **요약:** 공허의 균열 랭킹 진입 시 하단 탭이 교체되어 현재 위치를 혼동하는 문제를 해결하기 위해, 랭킹 전용 버튼을 제거하고 도전/상점/리그/전체 4개 탭 구조로 통합하는 UX 개선안. 뎁스 감소와 탭 한 번 접근으로 직관성을 높인다.
> **현재 상태:** 미적용

**공허의 균열 — 하단 탭 UX 개선 제안**

문서 유형: UI/UX 개선 제안 | 작성: 윤호민 | 상태: 검토 요청

# 1. 문제

공허의 균열 진입 후 현재 UX 흐름이 직관적이지 않아 유저가 탭 위치를 혼동하는 상황이 발생한다.

**현재 구조**

|  |  |
| --- | --- |
| **진입 경로** | **내용** |
| 공허의 균열 진입 | 하단 탭: 도전 (기본) / 상점 |
| 도전 탭 내 [랭킹] 버튼 터치 | 랭킹 화면으로 전환 |
| 랭킹 화면 하단 탭 | 리그 / 전체 |

**문제점**

* 랭킹 진입 시 하단 탭이 교체되어 현재 위치를 인지하기 어려움
* 도전 탭 → 랭킹 버튼 → 리그/전체 탭으로 이어지는 중첩 구조가 불필요한 뎁스를 만들어냄
* 도전 화면으로 돌아오려면 상단 백버튼 또는 탭 재진입이 필요해 흐름이 끊김

# 2. 개선안

랭킹 전용 버튼을 제거하고, 리그와 전체를 하단 탭으로 승격하여 탭 4개 구조로 통일한다.

**변경 후 구조**

|  |  |
| --- | --- |
| **하단 탭** | **설명** |
| 도전 | 기존과 동일. 현재 시즌 도전 준비 및 입장 |
| 상점 | 기존과 동일. 공허의 정수 교환소 |
| 리그 | 기존 랭킹 > 리그 탭 승격. 현재 시즌 리그 랭킹 |
| 전체 | 기존 랭킹 > 전체 탭 승격. 이전 시즌 기반 전체 랭킹 |

**기대 효과**

* 유저가 도전/상점/리그/전체 중 어디 있는지 항상 명확하게 인지 가능
* 랭킹 확인을 위한 버튼 탭 → 하단탭 전환 없이 탭 한 번으로 바로 접근
* 뎁스 감소로 백버튼 사용 빈도 줄어듦

# 3. 작업 범위

|  |  |
| --- | --- |
| **구분** | **내용** |
| 제거 | 도전 탭 내 [랭킹] 버튼 UI |
| 추가 | 하단 탭에 리그 / 전체 탭 항목 추가 |
| 변경 | 기존 랭킹 화면(리그/전체)을 하단 탭 전환으로 연결 |
| 유지 | 랭킹 화면 내부 콘텐츠 변경 없음 |

# 4. 비고

* 탭 아이콘 및 레이블 시안은 별도 UI 작업 시 확정
* 하단 탭 4개 기준 아이콘 사이즈/간격 조정 필요 여부 개발팀 확인 요청
