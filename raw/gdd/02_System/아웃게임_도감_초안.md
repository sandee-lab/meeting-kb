# 아웃게임 도감 시스템 (초안)

> **요약:** 택틱과 몬스터 수집 도감 시스템 초안. 최초 스폰·선택 시 수집 처리하고 슬롯별 보상을 지급하며, 택틱은 그룹당 1슬롯에서 레벨별 다중 보상을 처리한다. 미완성 상태(작성중).
> **현재 상태:** 이후 버전으로 대체됨

**M6\_아웃게임\_도감**

# 개요

* [도감]은 택틱과 몬스터 정보를 전달한다.
* 유저는 몬스터 최초 스폰 몬스터를 수집할 수 있다.
  + 최초 스폰 시, 인게임에서도 표시해준다. (콩콩던전 참고)
* 유저는 인게임에 등장한 택틱 선택 시 해당 카드를 수집할 수 있다.
* 각 수집 항목은 최초 획득 시 일정 보상을 지급한다.

# 구성

* 수집 항목
  + 몬스터
    - 몬스터 1마리 당 1개 슬롯으로 표현한다.
  + 택틱 그룹
    - 택틱 그룹 1개 당 1개 슬롯으로 표현한다.
* 수집 보상
  + 수집 항목 마다 보상이 배정된다.

# 동작

## 컨텐츠 해제

* + Contents\_Type = Collection

## 진입

* + 메인 화면에 아이콘 배치

## 수집

* + 택틱
    - 도감 수집 항목으로 지정된 택틱 그룹에 한해 수집 여부를 검사한다.
    - 인게임에서 해당 그룹 Lv.1 카드 최초 선택 시 수집 처리한다.
    - 각 그룹의 Lv.2~Lv.Max 카드 최초 선택 시마다 추가 획득 처리해준다.
      * (논의 필요) 인게임 택틱 정보는 클라이언트에서만 확인 가능
  + 몬스터
    - 도감 수집 항목으로 지정된 몬스터 유닛에 한해 수집 여부를 검사한다.
    - 인게임에서 몬스터 별 첫 스폰 시 수집 처리한다.
    - 최초 스폰 시, 인게임에서도 몬스터 정보를 표시한다. (콩콩던전 참고)

## 수집 보상 획득

| ※ 슬롯 상태 구분  - 미획득 : 아직 보유하지 않은 상태  - 최초 획득 : 획득 조건을 만족하여 보상을 얻을 수 있는 상태.  - 보유 중 : 획득 보상 획득까지 완료한 상태 |
| --- |

* + - 최초 수집 시 해당 슬롯을 ‘최초 획득’상태로 출력한다.
    - 슬롯 선택 시 배정된 보상을 지급한다.
      * 공용 보상 연출 출력
    - 보상 획득 후 슬롯은 ‘보유 중’ 상태로 출력된다.
    - ‘보유 중’ 상태 슬롯을 선택하면 [상세보기] 팝업을 출력한다.

![](data:image/png;base64...)

* + - 예외 규칙 (택틱)
      * 택틱은 1개 그룹 당 1개 슬롯을 차지하므로, 1개 슬롯에서 여러 번 보상을 지급해야 한다. (1개 그룹당 레벨 나뉘어 있음)
      * 최초 획득(Lv.1 획득) 시는 몬스터와 동일하게 처리하되, Lv.2 이후로는 ‘보유 중’ 상태 슬롯에 레드닷만 표시해 준다.
      * 리스트에서 진행도를 인지할 수 있도록 택틱 슬롯 하단에는 진행도를 표기한다.
        + 현재 수집한 레벨 / 해당 그룹의 최대 레벨

![](data:image/png;base64...)

# UI 구상안

## 진입 아이콘

* + 메인 챕터 이미지 하단에 배치한다. (전용 바로가기)
    - 아이콘 컨셉
      * 포켓몬 도감 ![](data:image/png;base64...)

![](data:image/png;base64...)

## 도감 UI 구성

* + 도감 UI는 도감 종류에 따라 화면을 분리한다.
    - 도감 종류
      * 택틱
      * 몬스터
  + 각 도감 화면은 기준에 따라 섹션을 구분한다.
    - 택틱
      * 등급 별
    - 몬스터
      * 종족 별

![](data:image/png;base64...)![](data:image/png;base64...)

## 슬롯 상태 구분

* + 미획득
    - 아직 보유하지 않은 상태
    - 슬롯 선택 시 토스트 메시지 출력
      * “아직 수집하지 않은 몬스터입니다.”
      * “아직 수집하지 않은 택틱입니다.”
  + 최초 획득
    - 획득 조건을 만족하여 보상을 얻을 수 있는 상태
    - 슬롯 선택 시 배정된 보상 지급
      * 공용 보상 연출
  + 보유 중
    - 획득 보상 획득까지 완료한 상태
    - 슬롯 선택 시 [상세보기] 팝업 출력

## 슬롯 표시 정보

* + 몬스터
    - 이름
    - 아이콘
    - 종류 (일반/보스)
  + 택틱
    - 이름
    - 아이콘
    - 진행도
      * 택틱은 그룹 단위로 묶어 준다.

## 상세보기 팝업 정보

* + 몬스터
    - 몬스터 이름
    - 등급
    - 태그 (갯수 늘어날 수 있음)
    - 설명
    - 스킬 (해당 시)

![](data:image/png;base64...)

* + 택틱
    - 택틱 이름
    - 등급
    - 설명 (전용 로컬 키 사용)
    - 레벨 별 설명
      * 미획득 레벨은 잠금 처리 한다.

# 레드닷

* 최초 획득
  + 진입 아이콘
  + 해당 그룹 아이콘
  + 해당 슬롯 아이콘 (최초 획득 상태로 노출)
* 택틱 신규 레벨 획득
  + 진입 아이콘
  + 해당 그룹 아이콘
  + 해당 슬롯 아이콘
    - 보유 중 상태 슬롯에 레드닷 표시
