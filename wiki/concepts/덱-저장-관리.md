# 덱 저장 및 관리

> 출처: [[raw/gdd/02_System/덱_저장_시스템_기획서]], [[raw/gdd/02_System/덱_평점_계산_공식]], [[raw/gdd/02_System/달성_임무_조건_기획서]], [[raw/gdd/02_System/인게임_벤치유닛]] *(대체됨)*, [[raw/meetings/[1027] 던전 및 컨텐츠 제안]]

## 개요

게임 클리어 시 덱을 저장하여 던전/무한모드/PVP 등 다양한 콘텐츠에서 활용하는 시스템이다. 인게임 요소는 하드카피, 아웃게임 요소는 소프트카피로 저장하며, 스테이지 보정치로 클리어 난이도에 따른 성능 차이를 제공한다.

## 상세 내용

### 덱 저장 시스템

> 출처: [[raw/gdd/02_System/덱_저장_시스템_기획서]]

**하이브리드 저장 방식:**
- **하드 카피 (인게임 요소)**: 유닛 8개 + 배치 위치 + 획득 택틱 전체(레벨+스페셜 포함) + 소유 유물 + 클리어 스테이지 정보(스테이지 버프 포함)
- **소프트 카피 (아웃게임 요소)**: 유닛 레벨, 지휘관 레벨, 장착 장비, 기타 아웃게임 성장 요소 (현재 유저 상태 반영)

**해금 조건:**
- 5스테이지 클리어 시 해금 (ContentsUnlock 테이블 - DeckSave 타입)
- 최근 플레이한 덱 자동 저장 (결과 화면 진입 시)
- 던전 해금 조건과 스테이지 수 일치

**저장 타이밍:**
- 게임 클리어 시에만 저장 가능 (패배 시 저장 불가)
- 클리어 → 결과 화면 → [다음] 버튼 → 덱 저장 UI → 저장

**스테이지 보정치 (챕터 보정):**
- 공식: `final_atk = 인게임 공격력 × (1 + clear_stage × 0.05)`
- 공식: `final_hp = 인게임 체력 × (1 + clear_stage × 0.05)`
- 1챕터: +5% / 5챕터: +25% / 10챕터: +50% / 20챕터: +100%
- 독립 곱연산으로 적용
- 상수 테이블 `ConstantDatas@deck_stage_buff`(= 5%)로 제어

**모의전:**
- 도전 스테이지 -1~-4까지 덱 구성 가능 (`ConstantDatas@deck_save_stage_gap = 4`)
- 스태미나 소모 없음, 스테이지 보상 없음 (인게임 골드만 지급)
- 5챕터 이상 스테이지에서만 가능
- 게임 종료 후 덱 저장 가능

**탐사대 슬롯:**
- 초기 슬롯 20개 (`ConstantDatas@deck_max_default`)
- 자물쇠(정예 탐사대) 설정 기능 — 잠긴 덱은 해산 불가
- 해산(삭제) 시 골드 지급 (`sell_deck_default` + 평점 × `sell_deck_rate`)

**데이터 변조 방지:**
- 저장 시 덱 해시값을 서버에 함께 저장하여 검증
- 삭제 시 서버에서 해당 해시값도 삭제

**활용 콘텐츠:**
- 무한모드: 저장된 덱으로 끝없는 웨이브 도전
- 던전 4종: 골드 던전(추격) / 유닛 던전(호위) / 장비 던전(보스) / 지휘관 던전(생존)
- 약탈 PVP: 저장된 덱으로 다른 유저와 전투 (추후 길드전으로 확장)

**덱 이름 규칙:**
- 최소 1자 / 최대 8자(한글) 또는 16자(영문)
- 허용 문자: 한글, 영문, 숫자, `-`, `_`, 스페이스
- 기본 이름: "이름없는 탐사대"

### 덱 평점 계산 공식

> 출처: [[raw/gdd/02_System/덱_평점_계산_공식]]

**총 평점 = 유닛 점수 × (택틱 점수 + 유물 점수 + 시너지 점수) × 스테이지 보정**

**1. 유닛 점수 (등급별):**

| 등급 | 점수 |
|------|------|
| 일반 | 4 |
| 고급 | 6 |
| 희귀 | 10 |
| 에픽 | 18 |
| 전설 | 40 |

**2. 택틱 점수 (등급별):**

| 등급 | 점수 |
|------|------|
| 일반 | 3 |
| 고급 | 5 |
| 희귀 | 8 |
| 에픽 | 15 |
| 전설 | 30 |
| 신화 | 50 |

**3. 유물 점수 (등급별):**

| 등급 | 점수 |
|------|------|
| 일반 | 2 |
| 고급 | 3 |
| 희귀 | 5 |
| 에픽 | 9 |
| 전설 | 18 |

**4. 시너지 점수:** 속성 시너지 발동 종 수 × 5 (속성 시너지만 카운트, 형태/빈도 시너지 제외)

**5. 스테이지 보정:** 20 + 클리어 스테이지 수 (곱연산)
- 10스테이지: ×30 / 20스테이지: ×40 / 30스테이지: ×50

**계산 예시 (30스테이지 덱):** 약 16,933점 / **계산 예시 (15스테이지 덱):** 약 1,932점

**달성 임무에서의 활용:**
- 덱 평점 달성이 초보자 미션/프로필 언락 조건 중 하나로 활용됨 (17종 달성 조건 중 하나)
- 달성 조건은 초기화되지 않는 누적 방식으로 처리

### 벤치 유닛 시스템 *(대체됨)*

> 출처: [[raw/gdd/02_System/인게임_벤치유닛]]

초기 설계에서 검토되었으나 대체된 시스템:
- 몬스터 처치 경험치로 레벨업하여 벤치 유닛 소환
- 벤치 유닛은 인벤토리에 보관되며 보유 효과 부여 (Clear_Reward_Consumable / Clear_Reward_Box)
- 승급 가능, 포메이션 배치와 리롤은 불가능

### 회의록 기반 논의

[1027] 던전 및 컨텐츠 제안에서:
- 덱 강화: 특정 진행도 이상에서 출현, 게임 종료 후 남은 미네랄로 덱 강화 가능
- 덱 저장: 유물, 택틱, 유닛 조합을 저장하여 다른 콘텐츠에서 사용
