# M5 아웃게임 우편

> **요약:** 운영 툴 시스템 우편 및 콘텐츠 보상 우편 발송 시스템 설계.
> **현재 상태:** 미적용

M5\_아웃게임\_우편

**M5\_아웃게임\_우편**

# 개요

* 운영 툴을 통해 시스템 우편을 발송할 수 있다.
* 일부 콘텐츠 보상은 우편을 통해 발송된다.
  + 시스템 우편
    - 운영 툴을 통해서 우편을 보낼 수 있다.
    - 테일즈 오브 크라운 참조하여 작성.
  + 콘텐츠 우편
    - 패키지 상품 구매 시 우편을 통해서 발송한다.
    - 일부 콘텐츠 보상도 우편을 통해 발송할 수 있다.

## 콘텐츠 우편

* + 대상
    - 패키지 상품
      * product\_type = package
    - 콘텐츠 개발에 따라 발송 대상이 추가될 수 있다.

### 패키지 상품

* + - 상품 결제 후 팝업 출력
      * 발송 완료 안내
        + 제목 : 구매 완료
        + 내용 : {상품명} 구매를 완료했습니다. 우편함을 확인해주세요.
      * 구매 취소 시 안내
        + 제목 : 구매 실패
        + 내용 : 결제가 정상적으로 완료되지 않았습니다.
    - 우편 발송
      * 우편 내용 안내
        + 제목 : {상품명} 구매 보상
        + 내용 : {상품명}(을)를 구매해주셔서 감사합니다. 구매한 상품 확인 후 우편을 수령해주세요.

## 우편 관리

### 우편 목록

* + - 발송된 우편은 유저의 우편함에 저장된다.
    - 최근 발송된 우편이 상단에 위치한다.

### 우편 열람 및 수령

* + - 우편함에 저장된 우편은 열람할 수 있다.
    - 보상이 포함된 우편은 [열람 및 수령] 버튼 선택 시 열람 및 보상 지급 처리한다.
    - 열람한 우편은 목록 최 하단으로 이동한다.

### 우편 삭제

* + - 열람 우편은 삭제할 수 있다. (일괄 삭제 기능)
      * 단, 영구 설정된 우편은 삭제 불가능하다.

### 편의 기능

* + - 일괄 삭제
      * 보관된 우편 중 열람한 우편을 모두 삭제한다.
        + 미열람 우편만 있는 경우에는 일괄 삭제 불가능
    - 일괄 받기
      * 보관된 우편 중 미열람 우편을 전부 열람 처리한다.
      * 보상이 할당된 경우 보상을 한 번에 획득한다.
