# 조각시스템 & 인벤토리

> **요약:** 재화 가치를 쪼개어 제공하는 조각(Shard) 타입과 인벤토리 시스템 기획이다. BoxData 테이블에 type(Box/Shard)과 open_timing(Auto/Manual), required_shard_amount 컬럼을 추가하여 상자 개봉과 조각 교환을 관리한다. 인벤토리는 아이템/상자/조각 3개 탭으로 구성되며, 상자는 Random/Select 방식으로 개봉하고 조각은 일정 수량 모아 교환한다. 이벤트 최종 보상으로 조건부 개봉 상자(수취인 불명 택배)도 지원한다.
> **현재 상태:** 미적용

M8\_아웃게임\_조각시스템&인벤토리

#

# 개요

* 재화 가치를 쪼개어 제공하는 조각(Shard) 타입을 추가한다.
  + 보상 지급 빈도가 높은 콘텐츠에 배치할 낮은 가치의 아이템 타입이 필요
    - 예: 일일 조별 랭킹 보상 **(지급하는 위치는 현재 미정)**
* 조각 교환 및 추후 아이템 관리를 위해 인벤토리를 추가한다.

## 테이블 변경

### 신규 컬럼

#### type

* + - * 적용 대상
        + 모두
      * 목적
        + 이하 컬럼 처리 및 인벤토리 카테고리 구분 시 활용
      * 타입
        + Box : 상자
        + Shard : 조각

#### open\_timing

* + - * 적용 대상
        + type=Box
      * 목적
        + 상자 개봉 타이밍 관리
      * 타입
        + Auto : 지급 시점에 바로 개봉 (현재의 순찰 골드와 동일)
        + Manual : 상자로 지급. 유저가 인벤토리에서 개봉 **(UI 필요)**

#### required\_shard\_amount

* + - * 적용 대상
        + type=Shard
      * 목적
        + 1회 교환을 위해 모아야 하는 조각 수량 설정
      * 타입
        + Integer

### 기존 컬럼 처리 수정

#### box\_type

* + - 변경 내용
      * 컬럼명 변경
      * type에 따른 처리 정리
    - 컬럼명 변경
      * box\_type ▶ select\_type
    - type=Box
      * Random : pool\_id 에서 랜덤 선택
      * Select : pool\_id 에서 직접 선택 **(UI 필요)**
      * SweepGold : 유지 (pool데이터의 select\_sweep\_reward\_hour 참조 중)
      * SweepExp : 유지 (pool데이터의 select\_sweep\_reward\_hour 참조 중)
    - type=Shard
      * Random : 교환 시 pool\_id 에서 랜덤 선택 **(UI 필요)**
      * Select : 교환 시 pool\_id 에서 직접 선택 **(UI 필요)**
