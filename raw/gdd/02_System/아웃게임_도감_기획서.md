# 아웃게임 도감

> **요약:** 몬스터/택틱/덱 3종 도감으로 구성된 아웃게임 수집 시스템 기획이다. 몬스터 도감은 인게임 조우 시 자동 등록, 택틱 도감은 획득 시 등록, 덱 도감은 특정 유닛 조합으로 웨이브 진입 시 등록된다. 발견 보상(1회)/레벨 보상/반복 보상 3종 보상 구조를 갖추며, 덱 도감은 경험치 축적과 레벨업이 가능하다. 인게임에서는 미발견 택틱/몬스터 등장 시 도감봇이 알림을 제공한다.
> **현재 상태:** 미적용

**M8\_아웃게임\_도감**

![](data:image/png;base64...)

포켓몬을 만나면 자동으로 기록해주는 하이테크한 도구란다!

# 개요

* 플레이 중 몬스터 조우, 택틱 등장, 특정 덱 클리어 활동이 자동으로 기록된다.
* 각 도감은 고유한 수집 목표와 보상 구조를 가진다.
* 수집/발견을 통해 성장 동기와 반복 플레이 동기를 제공한다.

# 구성 요소

## 카테고리

* + 도감을 기능 별로 구분
  + 예 : 몬스터 도감, 택틱 도감, 덱 도감

## 세부 정보

* + 카테고리 별로 항목 구성 및 조건 상이

## 도감 경험치 및 레벨

* + 일부 도감(덱 도감)은 경험치 축적 및 레벨 업 가능

## 보상

* + 각 도감 별로 발견 및 레벨 달성 보상 지급 가능

# 도감 등록 조건

## 공통 규칙

* + 모든 등록은 챕터 클리어 시점에 판정된다.
    - 클라에서 기록한 플레이 정보를 서버로 전달하여 등록 여부를 결정
    - 강제 종료 같은 비정상 종료 시 플레이 정보 확인 불가 (서버 확인 마친 내용)

## 카테고리 별 조건

| **카테고리** | **대상** | **등록 조건** | **판정 시점** | **예외 처리** |
| --- | --- | --- | --- | --- |
| 몬스터 도감 | 몬스터 | 인게임에서 몬스터 조우(spawn) & 해당 챕터 종료(정상 종료) | 전투 종료 | 강제 종료 시 등록 불가 |
| 택틱 도감 | 택틱 | 해당 택틱 획득 & 해당 챕터 종료 | 전투 종료 | 강제 종료 시 등록 불가 |
| 덱 도감 | 덱(유닛+택틱 조합) | 특정 덱 구성 & 웨이브 시작(Armory > 전투 진입 타이밍) | **아머리 종료** | 계정 생성 부터 체크  반복 누적 가능 |

### 특정 덱 플레이 조건은 모드에 따라 체크 방식이 다를 수 있다.

### 던전의 경우 1회 아머리 > 1회 전투로 스테이지가 종료될 수 있어, 덱 완성 후 아머리 종료 & 전투 시작을 조건으로 하고자 합니다.

## 신규 조건

* + 추후 업적으로의 활용을 고려하여 신규 조건을 개발한다.
    - 특정 택틱 n회 사용
      * condition : Check\_Play\_Tactic
      * value1 : tactic id (StageModeTacticPoolData)
      * value2 : count
    - 특정 덱 구성 n회
      * condition : Check\_Play\_Deck
      * value1
        + deck id (CollectionAchievementGroupData)
      * value2 : count (웨이브 수)
        + 예 : 새퍼 8마리 로 웨이브 1회 플레이 > 새퍼 프로필 해제

# 도감 보상

| **보상 유형** | **설명** | **수령 조건** | **지급 시점** |
| --- | --- | --- | --- |
| 발견 보상 | 각 항목 최초 발견 시 지급 | 발견 미수령 → 슬롯 터치 | 즉시 |
| 레벨 보상 | 도감 레벨업 달성 시 지급 | 덱 도감 레벨 상승 → 보상 슬롯 터치 | 즉시 |
| 반복 보상 | 특정 덱 플레이 누적 달성 시 지급 | 게이지 만충 시 → 슬롯 터치 | 즉시 |

* 보상은 1회성 지급이며, 중복 수령 불가
* 반복 보상(덱 도감)은 누적 횟수 기반

## 진입

* + 메인 화면에서 아이콘 선택
  + 기본 탭 : 덱 도감
    - 반복 보상 수령 가능한 도감을 기본 탭으로 해주자.

## 도감 선택

* + 카테고리 별로 탭이 부여된다.
  + 모든 카테고리가 콘텐츠 언락 시 해제된다.
  + 각 도감 목록에는 모든 수집 항목이 표시된다.

## 슬롯 상호작용

* + 조건 불만족: 토스트 메시지 출력
  + 미발견 상태: 미발견 상세 보기 팝업 출력
  + 발견(미수령) 상태: 터치 시 보상 지급
  + 수령 완료 상태: 상세 보기 팝업 출력
    - 단, 덱 도감은 목록 디자인에 따라 상세 보기 팝업이 없을 수 있다.

## 보상 획득

* + 조건 만족 시 유저가 수동으로 획득 가능
    - 발견
      * 만족 시, 목록에서 해당 슬롯 선택 시 보상 수령 가능
    - 특정 덱 플레이
      * 최초 발견 시, 보상 슬롯 선택하여 보상 수령 가능
        + 기본 레벨을 0, 최초 발견 후 레벨 1로 올려준다.
      * 반복 플레이 횟수 만족 시, 보상 슬롯 선택하여 보상 수령 가능
      * 덱 도감 레벨 상승 시, 보상 슬롯 선택하여 보상 수령 가능
