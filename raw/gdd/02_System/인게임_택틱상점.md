# 인게임 택틱 상점

> **요약:** 인게임 아머리에서 가스로 택틱을 구매·리롤하는 상점 시스템. 웨이브별 상점 갱신과 택틱 카드 등급별 비용을 정의한다.
> **현재 상태:** 이후 버전으로 대체됨

M6\_택틱 상점

목표

* 유닛 셋팅이 완성되어 가는 시점에 인게임 재화의 사용 가치가 약해지는 문제를 보완하고, 전략 선택의 깊이를 보완하기 위한 장치를 추가한다.

개요

* 특정 웨이브 이후에 택틱 상점 기능이 언락된다.
* 택틱 상점은 전략 페이즈 (아머리)에서 자유롭게 오픈할 수 있다.
* 택틱 상점 기능
  + 택틱 진열
    - 상점 잠금 해제 시 랜덤하게 결정된 택틱이 진열된다.
    - 진열된 택틱은, 매 웨이브 종료 시 마다 리스트가 갱신된다.
  + 택틱 구매
    - 제시된 택틱을 재화를 소모해서 구매할 수 있다.
    - 구매한 택틱은 구매 완료 상태로 진열된다.
  + 택틱 리롤
    - 재화를 소모해서 진열된 택틱 리스트를 리롤할 수 있다.
    - 리롤 시 구매 완료 상태의 택틱도 구매 가능한 택틱으로 변경된다.

# 시스템 동작 구조

## 상점 잠금 해제

* + - 데이터시트에서 입력한 웨이브 클리어 시 잠금 해제된다.

## 상점 메뉴 접근

* + 아머리에서 상점 메뉴 버튼이 배치된다.
    - 잠금 상태
      * 잠금 상태일 때는 메뉴 버튼이 잠금 상태로 표시된다.
      * 잠금 상태의 버튼 터치 시 잠금 해제 조건을 출력한다.
        + “n 웨이브 클리어 시 상점이 오픈됩니다”
    - 잠금 해제 상태
      * 잠금 해제 시 메뉴 버튼이 활성 상태로 표시된다.
      * 버튼 터치 시 상점 UI를 오픈한다.

## 택틱 상점 시스템 처리

### 택틱 상품 진열

* + - 갱신 조건
      * 상점이 잠금 해제되었을 때,
      * 플레이어가 목록 갱신을 시도했을 때,
      * 웨이브가 변경되었을 때,
    - 갱신 결과
      * 지정한 Pool에서 택틱을 새롭게 뽑는다.

### 택틱 상품 구매

* + - 진열된 택틱 상품은 설정된 재화를 지불하고, 획득할 수 있다.
    - 구매 완료 시 해당 상품은 ‘구매 완료’ 상태로 전환된다.
    - ‘구매 완료’ 상태의 택틱 상품은 중복 구매할 수 없다.

### 택틱 상품 갱신

* + - 비용을 지불하고, 상품 목록을 추가로 갱신할 수 있다.
    - 목록 갱신 시 ‘택틱 상품 진열’ 처리와 동일하게 처리한다.
