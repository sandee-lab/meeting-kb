새 원본 문서를 wiki에 통합한다.

## 입력
$ARGUMENTS — (선택) raw/ 하위의 파일 경로. 생략하면 자동으로 새 파일을 탐지한다.

## 새 파일 자동 탐지
경로가 주어지지 않으면:
1. wiki/index.md에 등록된 원본 목록을 읽는다
2. raw/ 하위 전체 파일을 스캔한다
3. index.md에 없는 파일 = 새 원본으로 판단하여 ingest 대상으로 삼는다
4. 새 파일이 없으면 "새 원본이 없습니다"로 종료한다

## 절차
1. 대상 파일을 읽는다
2. 내용을 분석하여 관련 wiki/ 문서를 파악한다 (index.md 참조)
3. 다음을 업데이트한다:
   - **index.md** — 새 항목 추가 (필수)
   - **concepts/** — 관련 주제 1~5개 업데이트 또는 신규 생성
   - **decisions/** — 새 결정사항이 있으면 해당 문서 업데이트
   - **timeline/** — 해당 시기 업데이트
   - **people/** — 새 참석자가 있으면 업데이트
   - **기존 문서 백링크** — 새 문서를 참조해야 하는 기존 문서에 링크 추가
4. **log.md**에 ingest 항목 추가 (필수)

## 목표
1건의 원본이 5~15개 wiki 페이지에 영향을 미치도록 한다.
