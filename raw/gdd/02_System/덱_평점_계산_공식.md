# 덱 평점 계산 공식

> **요약:** 덱 평점 산출 공식을 정의하는 문서로, 총 평점 = 유닛 점수 * (택틱 점수 + 유물 점수 + 시너지 점수) / 100 * 스테이지 보정으로 계산한다. 유닛은 등급별 4~40점, 택틱은 3~50점, 유물은 2~18점으로 합산하며, 시너지는 속성 시너지 발동 종수 * 5로 산출한다. 스테이지 보정은 20 + 클리어 스테이지 수를 곱연산하여 높은 스테이지 덱일수록 큰 점수 차이를 만든다.
> **현재 상태:** 미적용

# **덱 평점 계산**

## **계산 공식**

****총 평점 = 유닛 점수 \*(택틱 점수+유물 점수+시너지 점수)~~/100~~ \* 스테이지 보정

## **1. 유닛 점수**

배치된 유닛의 등급별 점수를 합산 (score컬럼 데이터로 제어)

| **등급** | **점수** |
| --- | --- |
| 일반 | 4 |
| 고급 | 6 |
| 희귀 | 10 |
| 에픽 | 18 |
| 전설 | 40 |

## **2. 택틱 점수**

획득한 택틱의 등급별 점수를 합산

| **등급** | **점수** |
| --- | --- |
| 일반 | 3 |
| 고급 | 5 |
| 희귀 | 8 |
| 에픽 | 15 |
| 전설 | 30 |
| 신화 | 50 |

## **3. 유물 점수**

획득한 유물의 등급별 점수를 합산 (score컬럼 데이터로 제어)

| **등급** | **점수** |
| --- | --- |
| 일반 | 2 |
| 고급 | 3 |
| 희귀 | 5 |
| 에픽 | 9 |
| 전설 | 18 |

**축복 (UnitItemEffectData Score 컬럼)**

* ~~등급 있음: 1/2/3/4/5점~~
* ~~등급 없음: 2점~~

## **4. 시너지 점수 (현재 적용안되고 있음)**

발동한 속성 시너지만 계산

시너지 점수 = 속성 시너지 발동 종 수 × 5

**예시**

* 화염 6종 달성 = 6 × 5 = 30점
* 냉기 4종 + 전기 3종 = (4+3) × 5 = 35점

## **5. 스테이지 보정**

****스테이지 보정 = 20 + 클리어 스테이지 수

**예시**

* 10스테이지 클리어 = 30배
* 20스테이지 클리어 = 40배
* 30스테이지 클리어 = 50배

## **계산 예시**

### **30스테이지 덱**

****유닛: 전설1(40) + 에픽2(36) + 희귀3(30) + 고급2(12) = 118점

택틱: 신화2(100) + 전설1(30) + 일반6(18) + 고급3(15) + 희귀2(16) + 에픽1(15) = 194점

유물: 전설1(18) + 에픽2(18) + 희귀3(15) + 고급2(6) + 축복3(6) = 63점

시너지: 화염6종(30점)

소계: 118점 x (194+63+30)/100 = 338.66

보정: ×50

총점: 16,933점

### **15스테이지 덱**

****유닛: 희귀1(10) + 고급4(24) + 일반3(12) = 46점

택틱: 전설1(30) + 일반8(24) + 고급4(20) + 희귀2(16) = 90점

유물: 희귀1(5) + 고급2(6) + 일반2(4) = 15점

시너지: 화염3종(15점)

소계: 46점 x (90+15+15)/100 = 55.2점

보정: ×35

총점: 1,932점



## **주의사항**

* 스테이지 보정은 마지막에 곱연산
* 시너지는 속성 시너지만 카운트 (형태, 빈도 시너지 제외)
