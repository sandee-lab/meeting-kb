# 덱 저장 시스템

> **요약:** 게임 클리어 시 덱을 저장하여 던전/무한모드/PVP 등 다양한 콘텐츠에서 활용하는 시스템 기획이다. 인게임 요소는 하드카피(유닛8개+배치+택틱+유물), 아웃게임 요소는 소프트카피(유닛레벨/지휘관/장비)로 저장하며, 스테이지 보정치(클리어 챕터 x 5%)로 1스테이지 차이에도 명확한 성능 차이를 제공한다. 모의전으로 도전 스테이지 -4까지 덱 구성이 가능하고, 해시 검증으로 데이터 변조를 방지한다.
> **현재 상태:** 미적용

덱 저장 시스템

## **목표**

1. 게임 클리어 시 덱을 저장하여 다양한 콘텐츠에서 활용할 수 있도록 한다.
2. 높은 스테이지 클리어 덱일수록 더 강력한 성능을 제공하여 저장 동기를 부여한다.

## **방향**

* 하이브리드 저장 방식 (하드 카피 + 소프트 카피)
  + 인게임 요소는 덱에 저장 (하드 카피)
  + 아웃게임 성장 요소는 현재 유저 상태 반영 (소프트 카피)
* **스테이지 보정치**로 1스테이지 차이에도 명확한 성능 차이 제공
* 자신의 도전 스테이지보다 낮은 단계에서 **모의전**을 통해 덱을 구성할 수 있다.
* **모의전**은 스태미나가 소모되지 않고 보상을 주지 않지만 덱을 저장할 수 있다.

### **덱 저장 조건**

* 덱 저장 콘텐츠의 해금 조건
  + 최초 덱 사용 콘텐츠인 던전 해금 조건과 스테이지 수를 맞춥니다.
    - 현재 던전은 5스테이지 클리어 후 열립니다.
    - 최근 플레이한 덱 자동 저장 (결과 화면 진입 시)
    - 던전 해금 후 저장한 덱으로 플레이 가능
  + ContentsUnlock 테이블 참조
    - 콘텐츠 타입: DeckSave
    - 해금 조건: Stage\_Clear
    - 해금 값1 : 5 (5스테이지 클리어 시 해금)
* 스테이지 저장 조건(최저 스테이지 제어 -X까지 **ConstantDatas에서 deck\_save\_stage\_gap = 4)**
  + 최고 클리어 스테이지 또는 모의전 스테이지 (-1,-2,-3,-4까지 가능)
    - 예시) 33 스테이지 도전 중,
      1. 33 스테이지를 클리어 후 저장 가능
      2. 32,31,30,29 모의전 가능 모의전 클리어 후 저장
    - 이유
      1. 너무 낮은 스테이지 반복 파밍 방지
      2. 진행도에 맞는 도전 유도

## **덱 저장 방식**

### 하드 카피 (인게임 요소)

* 인게임에서 획득/결정된 요소를 덱에 저장
* 저장 내용
  + 유닛 8개 + 배치 위치 (배치 그대로)
  + 획득한 택틱 전체(레벨 + 스페셜 포함)
  + 소유한 유물
  + 클리어한 스테이지 정보 (스테이지 버프 포함)

### 소프트 카피 (아웃게임 요소)

* 유저의 현재 아웃게임 성장 상태 반영
* 반영 내용
  + 유닛 레벨
  + 지휘관 레벨
  + 장착한 장비
  + 기타 아웃게임 성장 요소

### **챕터 보정치**

* 공격력 & 체력 보정 = 1 + (클리어 챕터 × deck\_stage\_buff(5%))
* 계산 예시
  + 1챕터: +5%
  + 5챕터: +25%
  + 10챕터 +50%
  + 15챕터: +75%
  + 20챕터: +100%
* 특징
  + 1스테이지만 올라가도 5%씩 명확한 차이
  + ConstantDatas에서 deck\_stage\_buff 값 제어

### **덱 이름 규칙**

글자 수 제한

* 최소: 1자
* 최대: 8자 (한글 8자, 영문 16자)

허용 문자

* 한글, 영문, 숫자
* 특수문자: - \_ 만 허용
* 중복 허용
* 스페이스 허용

금지 사항

* 욕설/비속어
* 공백만으로 이루어진 이름

기본 이름

* 입력 안하면 자동: "이름없는 탐사대" (**Local\_Deck\_Default\_Name**)

덱 이름 수정

* 언제든지 변경 가능
* 비용 없음

## **모의전**

* 진입 과정: 도전 스테이지보다 아래 단계를 선택합니다. (-1~-4)
* 5챕터 이상인 스테이지부터 가능합니다.
* 스태미나가 소모되지 않습니다
* 인게임은 일반 게임과 같습니다.
* 게임이 끝나면 스테이지에 해당하는 보상을 지급하지 않습니다.
* 인게임에서 획득한 골드만 지급합니다. (개발 편의에 따라 아예 지급하지 않을 수 있습니다.)
* 게임 결과 창에서는 보상을 지급하지 않습니다.
* 일반 게임과 인게임에서 차이는 스테미나 소모 여부와 보상 여부입니다.
* 게임이 종료되면 일반 게임과 마찬가지로 다음 버튼을 눌러 덱 저장을 할 수 있습니다.

## **활용 콘텐츠**

### **무한모드**

저장된 덱으로 끝없이 이어지는 웨이브 도전

높은 스테이지 덱일수록 더 멀리 진행 가능

### **던전 (4종)**

골드 던전 (추격) 저장된 덱으로 골드 획득 도전

유닛 던전 (호위) 저장된 덱으로 유닛 획득 도전

장비 던전 (보스) 저장된 덱으로 장비 획득 도전

지휘관 던전 (생존) 저장된 덱으로 지휘관 육성 재화 획득 도전

### **약탈 (PVP)**

저장된 덱으로 다른 사람과 전투

추후에 길드전으로 확장

### **계산 방식**

final\_atk = 인게임 공격력 × (1 + clear\_stage × 0.05[deck\_stage\_buff])

final\_hp = 인게임 체력 × (1 + clear\_stage × 0.05[deck\_stage\_buff])

소수점 처리: 반올림

## **고려사항**

### 스테이지 보정치 밸런싱

5% 단위 증가로 1스테이지 차이 체감

20스테이지 = 2배 성능 (공격력/체력)

콘텐츠별 권장 스테이지 설정 필요

예시

던전 1단계: 5스테이지 덱 권장

던전 2단계: 10스테이지 덱 권장

던전 3단계: 15스테이지 덱 권장

무한모드: 제한 없음
