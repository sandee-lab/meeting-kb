# 스페셜 스폰 패턴

> **요약:** 스페셜 스폰 시 사용할 스폰 패턴을 지정하는 시스템으로, 프리팹 기반 좌표 리스트와 스킬 부여 기능을 포함한다. None(랜덤), PrefabPivot(상대좌표), MapPivot(고정좌표) 3종의 스폰 타입을 지원하며, 스폰 스킬 패턴, 고정 방향 이동, 돌진, 텔레포트, 이동 스킬, 소환 위치 제어 등 다양한 특별 패턴을 구현한다.
> **현재 상태:** 미적용

# 연출하고 싶은 상황

* 황금 박쥐 떼가 화면 끝에서 반대 방향으로 날아간다.
  + 중간에 유저와 부딪치거나 죽을 때, 독장판을 터트려서 주변에 피해를 준다.
* 하늘에서 몬스터가 랜덤하게 떨어지면서 피해를 주면서 등장한다.
* 맵의 가장자리에서 몬스터들이 일렬로 돌진하면서 지나간다.
  + 중간에 부딪히거나 피격 시 피해를 준다.

# 스페셜 스폰 패턴

* 스페셜 스폰할 떄, 사용할 스폰 패턴을 지정할 수 있게 한다.
* 패턴을 지정하지 않으면, 기본 규칙대로 랜덤 스폰한다.
* 패턴은 프리팹화해서 관리한다.
  + 프리팹에서 몬스터를 스폰할 좌표를 리스트로 등록해둔다.
    - 지휘관 기준 상대좌 표일지
    - 맵 기준 월드 좌표일지 구분할 수 있어야 한다.
  + 프리팹에서 스폰할 몬스터들에게 add\_skill 해줄 스킬 id를 지정할 수 있게 한다.
    - 해당 스킬은 몬스터가 스폰될 때 최우선 발동된다.
  + 랜덤
    - 동시에 다 스폰할지
    - min ~ max 간격으로 n 마리씩 랜덤하게 스폰할지
* 데이터시트
  + StageWaveData
    - special\_spawn\_type
      * None
        + 일반 스폰 처럼 범위 내에서 랜덤 스폰
      * PrefabPivot
        + 지정한 프리팹의 상대 좌표 활용
      * MapPivot
        + 각 맵에 지정해둔 고정 좌표 활용
    - map\_pivot\_id
      * MapPivot 타입일 때, 사용할 id 값
    - fence\_prefab\_path → special\_spawn\_prefab\_path
      * PrefabPivot 타입일 때 사용할 프리팹 경로
    - fence\_prefab\_name → special\_spawn\_prefab\_name
      * PrefabPivot 타입일 때 사용할 프리팹 이름름

# 특별 패턴

## [완]스폰 스킬 패턴

* + 몬스터 스폰 시에 스킬을 한번 쓰고 스폰한다.
    - passive\_spawn 으로 발동하는 스킬
    - 스폰 시 자기 자신은 hide 처리하고, 스킬 쓰고나면, unhide 처리
  + 예)
    - 하늘에서 메테오가 떨어져서 쾅! 하는 폭발과 함께 피해를 준다.
    - 해당 위치에 몬스터가 스폰됨

## [완]고정 방향으로 지나가는 패턴

* + 스폰 시점에 정해진 방향으로 이동한다.
    - 가로 방향
    - 세로 방향
    - 지휘관 방향
  + 맵 바운드를 지나서 일정 거리 까지 이동한다. 이동 후에는 디스폰 처리한다.
    - UnitData@is\_out\_bound 값 추가
    - 값이 TRUE이면, 맵 바운드 밖에서 태어나고, 맵 바운드 밖으로 나갈 수 있다.
    - 이 때, 자폭 스킬을 지정해서 (SelfDestructActionSkillObject) 고정 방향으로 이동하고, 중간에 부딪치면 자폭하게 만든다.
    - 해당 스킬을 가진 유닛이 맵 바운드 밖으로 이동하면 스스로 디스폰처리한다.
  + 이동 중일 때, 패시브 스킬들은 정상 처리되도록 한다.
    - passvie\_move
    - passive\_damaged
    - passive\_dead
      * 등의 조건으로 패시브 스킬이 발동될 수 있다.

## [완]돌진 스킬

* + 시전 모션 - 대쉬 모션으로 구성한다.
    - Dash\_Start → Dash\_Loop
  + 스킬 발동 시 돌진 방향으로 워닝 마크를 표시해준다.
  + 시작 모션이 끝날 때, Dash\_Loop로 넘어간다.
    - 시작 모션이 끝나는 지점에 AttackBox01 트리거를 넣어서 어택박스를 생성한다. (어택박스는 duration -1 이고, 스킬 종료 시 떼어준다.)
  + Dash\_Loop는 Looping 모션으로 스킬에서 지정한 거리만큼 이동을 완료하면 **LoopEnd** 처리한다.
    - 스킬 오브젝트에서 목표 거리, 이동 속도를 지정한다.

## [완]텔레포트 스킬

* + 시전 모션이 있어야 한다. (가능하면 start-loop-end)
  + 시전 중에 목표 좌표 기준으로 워닝 마크를 표시해준다.
  + 시전 모션 중 특정 타이밍에 텔레포트한다.
  + 이동 하면서 나머지 모션을 실행하면서 어택박스를 소환해서 피해를 준다.

## [완]이동 스킬

* + 이동 모션을 그대로 쓴다.
  + 스킬 발동 할 때, 이동 목표를 지정한다.
    - 타겟 반대 방향으로 n 거리 만큼 도망가는 이동 스킬
  + 이동 목표에 도달할 때 까지 이동 스킬 상태로 처리한다.
  + 이동 스킬 중에도 패시브는 발동된다.

## [완]소환 위치와 타이밍을 어택박스로 제어하는 소환 스킬

![](data:image/png;base64...)

* + 어택박스에 attackbox spawn listener로 어택박스가 아니라 소환수 스폰 이벤트를 발동할 수 있게
  + 여기에서 기능을 제어하면, 원하는 타이밍 원하는 위치에 소환수를 원하는 만큼 소환 시킬 수 있음.
    - 예)
      * 슬라임이 독 구슬을 랜덤한 위치에 파라볼라로 발사함
      * 파라볼라가 despawn 되는 타이밍에 소환수 스폰
