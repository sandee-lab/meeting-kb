# 던전 및 콘텐츠 모드

> 출처: [[raw/gdd/03_Content/던전_인게임_기획서]], [[raw/gdd/03_Content/던전_인게임_초안]] *(대체됨)*, [[raw/gdd/03_Content/골드_던전_인게임_폴리싱]], [[raw/gdd/03_Content/유닛_던전_방벽디펜스_폴리싱]], [[raw/gdd/03_Content/유닛_던전_속성선택_기획서]] *(대체됨)*, [[raw/gdd/03_Content/유닛_던전_인게임_기획서]] *(대체됨)*, [[raw/gdd/03_Content/지휘관_던전_기획서]], [[raw/gdd/03_Content/무한_모드_기획서]], [[raw/gdd/03_Content/무한_모드_개발명세서]], [[raw/gdd/03_Content/보스모드_기획서]], [[raw/gdd/03_Content/보스레이드_컨셉]], [[raw/meetings/[1027] 던전 및 컨텐츠 제안]], [[raw/meetings/[1214] 신규 컨텐츠 리스트]]

## 개요

스테이지 모드 외에 유저의 장기 플레이를 유지하기 위한 다양한 콘텐츠 모드가 설계되었다. 크게 4종 던전(골드/유닛/장비/지휘관), 무한 모드(공허의 균열), 보스 모드의 3축으로 구성되며, 챕터에서 파밍한 덱을 활용하여 각 콘텐츠에 도전하는 구조이다. 성장 루프는 "성장 → 챕터 클리어 → 덱 파밍 → 던전/무한모드 → 랭킹전"으로 순환한다.

## 상세 내용

### 공통 시스템

**진입 구조:**
- 탐사 탭(일일 던전/골드 던전 등 숙제형)과 원정 탭(공허의 균열 등 시즌제/고난이도)으로 탭 분리
- 모든 던전 공통 UI: 목표 진행도, 남은 시간(1분), 일시정지

**데이터 구조 (DungeonWaveData):**
- StageWaveData와 유사한 형태로 던전 전용 테이블 신규 제작
- 컬럼: wave_index, wave_type, event_id, end_time_sec, monster_living_count_max, spawn_interval, spawn_count, spawn_monster_ids, special_spawn_ids, interaction_ids, atk_weight, max_hp_weight 등

### 골드 던전 (황금 행성)

> 출처: [[raw/gdd/03_Content/골드_던전_인게임_폴리싱]], [[raw/gdd/03_Content/던전_인게임_기획서]]

**목표:** 1분(DungeonStageData@limit_seconds) 내 황금 돼지 50마리 처치 (목표 킬 수는 DungeonStageData@goal_value로 제어)

**맵:** 사각형 맵, 8개 스폰 포인트

**몬스터:**

| 종류 | 특성 |
|------|------|
| 일반 황금 돼지 | 플레이어 감지(범위 5) 시 도망, 공격 안 함, 처치 시 +1 카운트 |
| 대형 황금 돼지(돼지킹) | 출현 확률 5%(데이터 제어), 체력 일반의 20배, 이속 50%, 처치 시 +10 카운트, 맵 중앙 고정 스폰 |

**스폰 시스템:**
- 초기 스폰: 20마리 (중앙 5칸 이내 랜덤)
- 10초마다 추가 스폰, 동시 최대 50마리
- 50마리 달성 시 추가 스폰 중지, 처치되면 재개

**행동 패턴 (일반 돼지):**
1. 기본: Idle(50%) / 배회(50%)
2. 감지: 플레이어가 범위 5 이내 진입
3. 도망: 플레이어 반대 방향으로 최대 속도
4. 경계 충돌: 벽 부딪히면 90도 방향 전환
5. 벗어남: 플레이어와 범위 10 이상 멀어지면 1로 복귀

**승리/패배:**
- 승리: 제한 시간 내 50마리 처치
- 패배: 1분 내 미달성

**결과 화면:**
- 승리: 보상 리스트 출력, [다음 레벨] 버튼(다음 스테이지 바로 시작)
- 패배: 팁 제공(유닛/장비/훈련 추천), 골드 던전은 무조건 유닛·장비·훈련 팁 노출(초보자/소과금 대상)

**폴리싱 항목 (미적용):**
- 돼지 스폰 위치: 기존 맵 전체 → 중앙 5칸 이내 랜덤으로 변경
- 감지 조건: 기존 맞으면 도망 → 5칸 이내 있거나 맞으면 도망으로 변경
- 아이들 상태: 기존 제자리/소량 이동 → 중앙을 향해 이동 (모서리 박힘 현상 감소)
- 웨이브 스킵: 몬스터 전멸 시 리젠 대기 없이 즉시 다음 웨이브 스폰
- 코인 드랍 연출: 기존 조각상 UI 채우기 → 노란색/갈색 +score 텍스트 출력으로 변경
- 돼지 점수: UnitData@score로 제어 (일반 1점, 돼지킹 10점)

> 출처: [[raw/gdd/03_Content/골드_던전_인게임_폴리싱]]

### 유닛 던전 (방벽 디펜스)

> 출처: [[raw/gdd/03_Content/유닛_던전_방벽디펜스_폴리싱]], [[raw/gdd/03_Content/유닛_던전_인게임_기획서]] *(대체됨)*

**목표:** 60초(DungeonStageData@limit_seconds) 동안 방벽 체력 0이 되지 않도록 방어

**기본 규칙:**
- 승리: 제한 시간 60초 생존
- 패배: 방벽 체력(시작 100, DungeonStageData@goal_value) 0 도달 또는 지휘관 사망

**전장 구역:**
- 아군 구역: 방벽(HP 보유, 파괴 시 패배)
- 전투 구역: 중앙 개활지, 장애물 없음
- 스폰 구역: 화면 밖 위쪽 끝, 안개로 가려짐

**몬스터 3종:**

| 몬스터 | 특성 | 벽 피해(UnitData@score) |
|--------|------|------------------------|
| 저글링 | 물량형, 아래로 직진, 방벽 부딪히면 자폭 | 2점 |
| 폭탄벌레 | 아군 유닛 접촉 시 즉시 자폭(반경 3.5m), 저글링에게 즉사급 피해 | 10점 |
| 코뿔소 | 돌진형, 아래로 돌격, 유닛 충돌 시 중간 피해 | 25점 |

**시간대별 스폰 확률:**

| 시간 | 스폰 주기 | 저글링 : 폭탄벌레 : 코뿔소 |
|------|---------|--------------------------|
| 0~10초 | 1.0초 | 90 : 10 : 0 |
| 10~50초 | 0.5초 | 80 : 15 : 5 |
| 50~60초 | 0.3초 | 60 : 30 : 10 |

**배경:** 산성 늪지대 — 녹색/보라색 걸쭉한 액체, 독성 연기, 아군 진입 불가(시스템 차단), 몬스터는 통과 가능

**폴리싱 항목 (미적용):**
- 방벽 파괴 패배 시 결과 화면: TIMEOUT → MISSION FAILED(Local_Text_Mission_Failed)로 변경
- 다른 던전들도 동일하게 변경 예정

> 출처: [[raw/gdd/03_Content/유닛_던전_방벽디펜스_폴리싱]]

### 지휘관 던전 (보스전)

> 출처: [[raw/gdd/03_Content/지휘관_던전_기획서]]

**목표:** 맵 상단에 고정된 보스를 처치

**특성 (권장 덱):**
- 단일 타격, 다단히트, 보스 추가 피해에 유리한 전기 속성 시너지 조합

**보스 패턴:**

| 타입 | 구분 | 설명 |
|------|------|------|
| 액션 | 기본 공격 | 유저 방향으로 미사일 발사 |
| 액션 | 구슬 소환 | 맵에 속성 구슬 오브젝트 생성 (랜덤 위치 6~7개 중 선택) |
| 액션 | 구슬 흡수 | 맵 오브젝트를 보스가 획득 |
| 액션 | 방사 | 좌우 방향 발사체 연속 발사 (넓은 범위 견제) |
| 패시브 | 광폭화 | 전투 시작 20초 후 공속/쿨타임 가속 버프 |
| 패시브 | 장판 | 일정 시간마다 가로/세로 화면 장판 생성 (구슬 획득 방해) |

**오브젝트 효과:**

| 구슬 | 유저 획득 시 | 보스 획득 시 |
|------|------------|------------|
| 불꽃(1001) | 아군 불꽃 위력 증가 | 불꽃 받는 피해 감소 |
| 냉기(1002) | 아군 냉기 위력 증가 | 냉기 받는 피해 감소 |
| 전기(1003) | 아군 전기 위력 증가 | 전기 받는 피해 감소 |
| 카오스(1004) | 아군 카오스 위력 증가 | 카오스 받는 피해 감소 |
| 무속성(1005) | 아군 체력 회복 | 보스 체력 회복 |

**스킬 타이밍 (Check_BattleTime 기반):**
- 5초 후: 생성 스킬
- 15/25/35/45/55초: 흡수 스킬 → 흡수 후 생성 스킬

**폴리싱 이슈:**
- 기본 공격: 주먹 발사 → 미사일 발사로 변경
- 추가 공격: 하늘에서 주먹 내려찍는 느낌(메테오)으로 리소스 변경

> 출처: [[raw/gdd/03_Content/지휘관_던전_기획서]]

### 초안 버전 던전 (이전 설계)

> 출처: [[raw/gdd/03_Content/던전_인게임_초안]] *(대체됨)*

초안에서 설계된 4종 던전의 원형. 현재 기획서로 대체됨.

- **골드 던전:** 원형 맵, 스폰 방식(10초마다 15마리, 30마리 미만 시 즉시 20마리 추가) — 현재 기획서에서 사각형 맵으로 변경, 스폰 규칙 단순화
- **유닛 던전 (구버전):** 좌→우 진행형, 체크포인트 A-B-C 돌파, 문지기 좀비 처치 조건 — 현재 방벽 디펜스로 대체
- **장비 던전:** 원형 맵, 3방향 장비 쟁탈전(검/벨트/갑옷), 검→갑옷→벨트→검 상성 관계 — 미정 상태
- **지휘관 던전 (구버전):** 기차 탑승 1분 생존, 추격 벌레 격퇴 — 현재 보스전 방식으로 대체
- **관리 페이즈:** 진입 전 덱 구성 단계, 시작 가스 200, 유닛 리롤 10회, 택틱 3개 + 리롤 3회 (현재 기획서에서 제거 여부 미확인)

**속성 선택 던전 리뉴얼 초안 (미적용):**
- CRT 모니터 타일맵 컨셉의 보스전
- 4x3 그리드 타일, 속성 버프(동조 +50% / 비동조 +5%)
- 보스 패턴: 가로/세로 공격, 내려찍기, 화면 꺼짐(NO SIGNAL)

> 출처: [[raw/gdd/03_Content/유닛_던전_속성선택_기획서]] *(대체됨)*

### 무한 모드 (공허의 균열)

> 출처: [[raw/gdd/03_Content/무한_모드_기획서]], [[raw/gdd/03_Content/무한_모드_개발명세서]]

**정의:** 끊임없이 몰려오는 적을 물리치며 전장 오브젝트(거점)를 점령해 실시간 성장, 한계 시간까지 버티는 고밀도 무한 서바이벌 경쟁 콘텐츠

**해금 조건:** 메인 챕터 15 클리어 (ContentsUnlockData@contents_type = InfiniteMode)

**탭 구조:**
- 탐사 탭(Local_Dungeon_Tab_Normal): 일일 던전 등 숙제형, 디폴트
- 원정 탭(Local_Dungeon_Tab_Hard): 공허의 균열 등 시즌제 고난이도

**난이도:**
- 레벨 1~10, 시즌마다 1레벨로 초기화
- 점수 배율: x1.0~x20.0 (데이터 제어)
- 클리어 마크: 3분/4분/5분 달성
- 5분 생존 시 다음 단계 해금
- 전투력 색으로 난이도 가이드(초록 쉬움 / 빨강 어려움)

**달성 보상:**
- 각 난이도(Level 1~10)별 생존 시간(1분/2분/3분) 도달 시 1회성 보상
- 매 시즌(1주일)마다 초기화 → 매주 재획득 가능
- 자동 클리어(스킵) 실행 시 해당 단계 3분 보상까지 일괄 지급

**랭킹 구조:**
- 일간 랭킹과 시즌 랭킹 이원화
- 리그(League): 브론즈/실버/골드/플래티넘/다이아 (InfiniteRank 구조)
- 시즌 종료 시 정산 후 보상 지급

**보상 구조:**
- 입장 티켓 보유 시 참여 가능
- 달성 보상(단계별 생존 시간 마일스톤), 시즌 랭킹 보상
- 교환소(균열의 정수 소모 전용 상점)

### 보스 모드

> 출처: [[raw/gdd/03_Content/보스모드_기획서]]

**정의:** 기존 덱으로 다양한 보스를 연속 처치하며 딜량을 겨루는 주간 경쟁 콘텐츠

**기본 구성:**
- 도전 횟수: 일일 1회 + 광고 1회 충전 (최대 2회/일, 이월 없음)
- 한 판 시간: 고정 60초
- 시즌 주기: 1주일 (매주 초기화)
- 랭킹 점수: 시즌 내 전체 도전 누적 딜량 합산
- 맵: 일반 스테이지 대비 1.5배 넓은 전용 맵

**핵심 규칙:**
- 60초 내 보스 연속 처치 (웨이브 단위 진행)
- 처치 즉시 또는 웨이브 시간 초과 시 다음 보스 스폰
- 웨이브마다 보스 능력치 배율 상승
- 랜덤 보스(가중치 풀) + 특정 웨이브 확정 보스(CSV 지정)
- 시간 초과 시 해당 웨이브 딜량만 손실 (추가 페널티 없음)

**리그 배정:**

| 리그 | 배정 기준 | 정원 |
|------|---------|------|
| 브론즈 | 전체 하위 60~100% | 최대 50명 |
| 실버 | 전체 30~60% | 최대 50명 |
| 골드 | 전체 10~30% | 최대 50명 |
| 플래티넘 | 전체 상위 0~10% | 최대 50명 |
| 다이아 | 전체 상위 100명(Ordinal) | 최대 100명 |

**보상 구조:**
- 주간 도전 횟수 보상 (누적 도전 횟수 마일스톤)
- 달성 보상 (판 내 누적 처치 수 마일스톤, 매 판 초기화)
- 랭킹 보상 (리그 내 순위별, 시즌 종료 후 우편)

> 출처: [[raw/gdd/03_Content/보스모드_기획서]]

### 보스 레이드 (균열의 심장)

> 출처: [[raw/gdd/03_Content/보스레이드_컨셉]]

**정의:** 60초간 최대 DPS를 경쟁하는 주간 기록 경쟁 콘텐츠. 보스 HP 고갈 불가(무한 HP), 60초 누적 피해량이 점수

**설계 비중:** 아웃게임 성장 70% + 인게임 조작 30% 이하

**콘텐츠 포지셔닝:**
- 보스 레이드: "얼마나 강하게 때리는가" — 낮은 복잡도, 즉각적 성장 체감 → 초반 해금
- 공허의 균열: "얼마나 오래 버티는가" — 높은 복잡도, 전략적 깊이 → 5~7일차 이후 해금

**기본 규칙:**
- 입장 횟수: 무제한 (티켓/재화 소모 없음)
- 시즌 종료 30분 전 입장 제한
- 최고 기록 1개만 랭킹 반영, 매주 월요일 정산

**티어 구조:**

| 티어 | 달성 시점 | 특이사항 |
|------|---------|---------|
| 브론즈 | 1일차 | 첫 플레이 즉시 달성 가능 |
| 실버 | 1일차 | 브론즈~실버 단순 구간 |
| 골드 | 2~3일차 | 1차 패턴 추가 |
| 다이아 | 7일차 | 첫 시즌 성실 참여 유저 |
| 플래티넘 | 30일차 | 2차 패턴(덱 지식 요구) |
| 챌린저 | 상위권 성장 유저 | 유일한 순위 경쟁 구간 |

**보상 3레이어:**
1. 일일 구간 보상 (당일 최고 기록 구간별, 매일 초기화)
2. 영구 달성 보상 (티어 첫 돌파 시 단 1회, 영구)
3. 주간 랭킹 보상 (챌린저 티어 진입 유저만 참여, 매주)

**확정 보스 패턴:**

| 패턴명 | 성격 | 실패 페널티 |
|--------|------|------------|
| 광역 폭발 | 페널티 회피 | 5초간 공격력 50% 감소 |
| 돌진 충격파 | 페널티 회피 | 부대 전체 3초 이동 불가 |
| 속성 버프 오브젝트 | 보너스 획득 (플래티넘 이상) | 성공 시 해당 속성 DPS 30초간 증폭 |

### 회의록 기반 초기 논의 (2025년 10~12월)

[1027] 던전 및 컨텐츠 제안에서 던전 모드의 성장 루프가 정의되었다:
- 성장 → 챕터 클리어 → 덱 파밍 → 고점 빌드 깎기 → 랭킹전
- 각 던전에 유용한 덱으로 플레이, 처치 미션 수행
- 무한 모드는 최종 컨텐츠로 보스+웨이브, 여러 속성 모드 중 선택, 5~10분 플레이

[1214] 신규 컨텐츠 리스트에서 엔드 컨텐츠 방향이 제시되었다:
- 덱 2개 이상을 활용한 엔드 컨텐츠 (태그 시스템)
- 덱 빌딩 고점을 깎는 무한 모드
- 슬롯 강화 시스템
