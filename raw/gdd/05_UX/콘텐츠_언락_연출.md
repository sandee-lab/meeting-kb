# 아웃게임 콘텐츠 언락 연출

> **요약:** 신규 콘텐츠 해제 시 순차 연출과 팝업 처리 설계. ContentsUnlockData 기반 시퀀스와 로비 복귀 후 연출 규칙을 정의한다.
> **현재 상태:** 이후 버전으로 대체됨

**M7\_아웃게임\_콘텐츠언락연출**

![](data:image/gif;base64...)

[레퍼런스]

# 개요

* 신규 콘텐츠 해제 시점에 **연출을 적용**한다.
  + 연출을 통해 신규 콘텐츠 해제를 인지시킨다.
* 동시에 여러 콘텐츠가 해제될 때, 정해진 **순서대로** (시퀀스) 연출을 출력한다.
* 일부 콘텐츠는 잠겨있을 때 **아이콘을 숨겨** 피로도를 낮춘다.

# 처리

## 콘텐츠 해제 연출 여부 [기존 규칙 유지]

* + ContentsUnlockData@ui\_popup
    - TRUE : 콘텐츠 해제 연출 적용
    - FALSE : 콘텐츠 해제 연출 미적용 (해제 시 아무 표시 없음)

## 동시 해제 시 순서

* + ContentsUnlockData@order
    - 신규 컬럼 추가
    - 조건이 동일한 경우(=동시에 해제된 경우), order 순으로 출력

## 연출 종료 후 강제 팝업 여부

* + ContentsUnlockData@show\_popup\_on\_unlock
    - TRUE
      * 해당 콘텐츠 해제 연출 끝나고 해당 ui 강제 오픈 (eg.출석부)
      * 뒤에 대기 중인 콘텐츠 해제 연출이 있을 경우, 강제 오픈한 ui 종료하고 로비 복귀했을 때 이어서 출력합니다.
    - FALSE
      * 콘텐츠 해제 연출 끝나고 강제 오픈하지 않음

# 연출 시퀀스

## 출력 시점

* + 조건 만족 후 최초로 로비 복귀했을 때

## 시안

![](data:image/png;base64...)

## 진행 순서

* + 조건 만족 후 로비 복귀

![](data:image/png;base64...)

* + 딤드 화면 출력

![](data:image/png;base64...)

* + - 중앙에 해제할 콘텐츠 아이콘 **흑백** 출력
    - 아이콘 상단에 자물쇠 아이콘 출력
    - 하단에 ‘탭하여 닫기’ 출력
      * 터치 시 연출 종료
  + 해제 연출

![](data:image/gif;base64...)

* + - 1) 자물쇠 흔들거리다 풀어지는 연출
      * 레퍼런스의 경우, 자물쇠 깨진 후 흰색 동심원 연출이 추가됨
    - 2) 콘텐츠 아이콘 **컬러** 출력
      * 아이콘이 커졌다 작아지는 식의 스케일 효과
      * 이후 반짝이는 효과 적용
    - 3) 타이틀 출력
      * 아이콘 상단 - “새 콘텐츠 해제”
      * 시안에 적용할 액션 결정 필요
        + 레퍼런스와 동일하게 가운데서 스케일
        + 혹은 일방향으로 라인과 텍스트 출력 (마스킹)
    - 4) 콘텐츠 이름 출력
      * 아이콘 하단 - “[콘텐츠 이름]”
      * 레퍼런스와 같이 텍스트 작은 사이즈 > 큰 사이즈로 스케일 효과
  + 아이콘 플라이

![](data:image/gif;base64...)

* + - 아이콘이 지정 위치로 플라이
      * 레퍼런스와 같이, 날아가는 반대 방향으로 조금 이동했다가 목표 지점으로 플라이하다가 소멸
      * **타임라인에서 피봇 설정**만 하고, 코드에서 연출 출력할 때 피봇 위치 지정
    - 목표 지점에서 위치 강조 표현
      * 레퍼런스는 흰색 동심원 연출
  + 연출 종료

##

## 기타 처리

* + 잠겼을 때 아이콘 표시 여부
    - 잠금 상태 아이콘 출력
      * 상단 메뉴
        + (현재 기준 해당 콘텐츠 없음)
      * 하단 메뉴
        + 상점
        + 유닛
        + 장비
        + 골드 성장
      * 햄버거 (三) 메뉴 에 속하는 콘텐츠
        + 설정
        + (추가될 수 있음)
    - 아이콘 미출력
      * 메인 화면 양쪽 배치되는 콘텐츠
        + 출석부, 퀘스트, 우편함, 친구 등…

* + 메인 화면에 노출되지 않는 콘텐츠 (=2뎁스 이상인 콘텐츠)
    - 콘텐츠 언락 연출
      * 뎁스 최상위 (메인 화면에서의 진입 아이콘) 아이콘으로 연출을 진행한다.
        + 예 : 유닛 소환(소환 > 유닛 소환 탭으로 구분됨) 해제 시 연출을 하는 경우, 메인 화면 [소환] 아이콘으로 플라이
      * 해당 콘텐츠 아이콘에 [N]마크 부착
        + 해당 콘텐츠 UI 1회 확인 후 [N]마크 삭제

![](data:image/png;base64...)

* + 인게임 콘텐츠
    - 콘텐츠 언락 연출
      * 아웃게임에서 진행하지 않는다.
      * 아머리 들어갔을 때, 언락 연출을 진행한다.
        + ui\_popup은 False 처리

    - 순서대로 연출 시퀀스를 진행한다.
