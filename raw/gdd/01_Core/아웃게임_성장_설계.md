# 아웃게임 성장 설계

> **요약:** 장비(40%), 유닛(30%), 지휘관(20%), 골드성장(10%)의 4축 성장 비중을 정의하고, 초기 빠른 성장에서 후기 도전까지의 챕터별 진행 속도를 설계한 문서. 각 콘텐츠별 레벨/승급 계수와 과금 포지션을 수치로 명시한다.
> **현재 상태:** 이후 버전으로 대체됨

## M6\_아웃 게임\_ 성장

### 목표

* 초기 빠른 성장 : D3에 10챕터, D9에 20챕터로 초반 9일간 하루 평균 2챕터씩 빠른 진행
* 중기 안정화 : D21에 30챕터로 중반 12일간 하루 평균 0.8챕터로 진행으로 플레이 확장
* 후기 도전 : D45에 40챕터, D95에 50쳅터로 장기간에 걸친 엔드게임 경험 제공
* 리텐션 설계 : 초반 성취감 전달 후 전략 실현의 기쁨 그후 단계별 콘텐츠로 장기플레이 유도

### 성장 포지션 (추가 시점 기획 필요)

* **장비(40%)** **:** 메인 성장축이지만 과도하게 의존하지 않아 다른 시스템들이 의미있는 균형점
* **유닛(30%)** **:** 부대 전투에서 조합과 전략이 핵심이므로 전술적 깊이 강화를위해 높은 비중
* **지휘관(20%)** **:** 플레이 스타일 변화의 핵심 역할과 차별화 요소로서 적절한 비중
* **골드 성장(10%) :** 즉시 만족도 전달과 보조 역할로 적정 수준 유지
* Unlock 시점 : (추가되는 콘텐츠 역시)
* 지휘관 강화 재화 (던전) 해금 시점 역시 챕터 10 이후

### 과금 포지션

* **지휘관 :** 매우 높음 (신규 출시 시 강력한 동기)
* **장비 :** 높음 (가챠 확률, 재료 관련)
* **유닛 :** 중간 (각성 재료, 유닛 조각 등)
* **골드 성장 :** 낮음 (골드 구매, 특별 성장)

### 콘텐츠 별 성장도

* 성장 체감을 느끼게하고, 포지션에 맞는 비율을 설정
  + 최고 레벨(단계) : 50챕터 기준 마지막 성장 레벨
  + 성장도 : 1레벨 강화 했을 때 능력치 강화 정도

* 장비
  + 공격력 20 (무기, 목걸이, 장갑의 합)
  + 체력 100 (갑옷, 헬멧, 장화의 합)
  + 장비 강화
    - 최고 레벨 : 70
    - 성장도 : 15%
  + 장비 승급
    - 승급할 때마다 장비의 강화 최고 레벨이 증가
    - 최고 단계 : 7 (신화 0)
    - 일반, 고급, 희귀, 에픽, 전설까지 100% (누적 성장)
    - 전설1, 신화 달성시 50%
    - 예시1) 일반에서 고급 승급 시 1 >> 2 (100%)
    - 예시2) 고급에서 희귀 승급 시 2 >> 3 (1의 100% 증가)
    - 장비 레벨업 스탯 \* 승급 계수
* 지휘관
  + 지휘관 기본 스탯
    - 공격력 20
    - 체력 100
  + 지휘관 강화
    - 최고 레벨 100
    - 성장도 : 10%
  + 지휘관 승급
    - 최고 단계 : 5
    - 50%씩 누적 성장
    - 예시1) 일반에서 고급 승급 시 1 >> 1.5 (50%)
    - 예시2) 고급에서 희귀 승급 시 1.5 >> 2 (10의 50% 증가)
    - 지휘관 레벨업 스탯 \* 승급 계수
* 유닛 레벨업 (30%까지만 D95)
  + 유닛 조각을 일정 수를 모아 레벨 업을 진행
  + 3,6,9,12,15레벨 도달 시 스탯 버프 혹은 스킬 해금을 얻을 수 있다.
  + 유닛 강화
    - 최고 레벨 15
    - 성장도 : 5%
  + 스킬 해금 (전체 유닛 버프 스탯 추가?)
    - 3레벨 : 공격력 스탯 10% 버프
    - 6레벨 : 패시브 스킬 해금 혹은 강화
    - 9레벨 : 공격 속도 스탯 10% 버프
    - 12 레벨 : 패시브 스킬 해금 혹은 강화
    - 15레벨 : 공격력 & 공격 속도 10% 버프 추가
* 골드 성장
  + 골드 성장
    - 지휘관과 장비 스탯의 10% 정도로 할당
  + 특별 성장
    - 유용한 옵션등으로 모두 해금 시 30% 정도 스탯 상승량을 가짐
