> **논의가 필요한 이슈**
> 
- **지휘관**
    - 현재 지휘관의 외형은 대체적으로 좋지 않은 피드백
    - 더 하찮고 귀엽거나 멋졌으면 하는 의견 다수
    - 다만 방향으로 보면 지휘관이 주인공이라기 보다는 지켜주는 대상에 가까우므로 좀 더 귀여운 마스코트 처럼 변경해보자
    - 지휘관에 대한 유저 인지 문제는 스토리와 연출로 자연스럽게 만들 필요가 있다.
- **난이도 및 플레이 타임**
    - 현재 10챕터 수준으로 어렵게 잡혀있기 때문에 실제 1챕터에 적용될 밸런스를 보고 다시 판단
- **웨이브 리셋**
    - 매번 새로운 웨이브로 시작하는 것이 게임의 연속성을 잃게 만들어 성장 체감이 덜 한 방식이다.
    - 웨이브를 이어서 할지, 스폰 레이트로 조정할지에 대해 차주에 좀 더 논의 필요
- **참치캔**
    - 현재는 획득처가 제한적이라 게임에서 구하기 어려운게 아쉽다
    - 처음부터 나오지 않고, 특정 챕터 이후부터는 구할 수 있게 하되 더 많이 얻을수 있도록 조정이 필요하다.
- **아이템 관련 의견**
    - 몬스터 숫자가 많아져서 드랍아이템이 너무 많아진 느낌. 드랍아이템을 많이 줄이거나 아예 없애고 룰렛에서 더 많은 획득과 확실한 효과 체감이 되었으면 좋겠다.
    - 별 먹었을 때 무적 효과가 반드시 추가 되었으면 좋겠다.
    - 기본 회복을 더 할 수 있어야 한다.
- **택틱 관련**
    - 택틱에 시각적 변화가 더 필요하다.
    - 전설 유닛과 조합되어 전설 유닛의 효과를 증대시킬 수 있는 택틱이 필요하다.
- **미네랄**
    - 다른 로그라이크 게임처럼 자석을 이용해 한번에 획득하는 재미나, 유닛을 잡으면서 얻는 재미가 다시 생기면 좋겠다.
    - 미네랄을 이용해 게임내에 새로운 개념을 넣기 보다는 기존 요소를 교체하는 방식으로 고민해보자
        - 택틱 경험치로 활용하고, 누적된 경우에만 스테이지 종료 시 택틱을 얻게 하는 방법
        - 골드 주머니를 삭제하고 골드 획득대신 미네랄로 교체
- **유닛 리롤**
    - 리롤 기능을 다시 살리는 것을 고려.
    - 단, 판매와 비중에 대한 고민이 필요하며 UI상에서도 분리 되는 방향으로 고민해야 함
- **체력**
    - 유닛이 돌파할 때 내 유닛들이 죽는게 갑작스럽게 느껴진다.
    - 유닛과 지휘관을 통합체력으로 관리하는 것을 생각해 볼 수 있다. 다만 현재 이정도 변화를 주기에 어려운 시점임을 감안해야 함
    - 유닛이 충돌대미지를 줄때와 내 유닛이 피격당할 때의 명확한 시각적 피드백이 추가되는 것이 우선이다.
