# [1205] 카드 머지 시퀀스 정리

생성자: JEONG HAE RYONG
생성 일시: December 5, 2024 3:14 PM

- 카드 이동 및 스케일 값
    
    
    | move | 우측에서 좌측 빈슬롯까지 이동 |
    | --- | --- |
    | easing | ease out expo |
    | duration | 0.4 |
    | start delay | 0 |
    
    | scale | from 0.9 > 1 |
    | --- | --- |
    | easing | ease in out cubic |
    | duration | 0.2 |
    | start delay | 0.3 |
- 카드 머지 이동 및 스케일, 로테이션 값
    - 두 카드가 만나는 시점에 작아짐
        
        
        | scale | to 0.8 |
        | --- | --- |
        | easing | ease in out cubic |
        | duration | 0.1 |
        | start delay | 0 |
    - 각각 왼쪽과 오른쪽으로 같은 값으로 이동하면서 사이즈가 조금 커짐
        
        
        | scale | to 1.2 |
        | --- | --- |
        | easing | ease in out cubic |
        | duration | 0.2 |
        | start delay | 0.1 |
        
        | move | x -60 |
        | --- | --- |
        | easing | ease in out cubic |
        | duration | 0.3 |
        | start delay | 0.1 |
    - 두 카드가 가운데로 합쳐짐
        
        
        | move | x 0 (두 카드의 중앙값으로) |
        | --- | --- |
        | easing | ease out cubic |
        | duration | 0.15 |
        | start delay | 0.3 |
    - 카드 슬롯쪽으로 돌아가며 머지 완료
        
        
        | **scale** | to 1 |
        | --- | --- |
        | easing | ease out back |
        | duration | 0.2 |
        | start delay | 0.5 |
        
        | **rotation** | 180 |
        | --- | --- |
        | easing | ease in out cubic |
        | duration | 0.2 |
        | start delay | 0.6 |
    - 캐릭터 아이콘 마지막에 처리
        
        
        | **rotation** | -180 |
        | --- | --- |
        | easing | ease in out cubic |
        | duration | 0.2 |
        | start delay | 0.6 |
        
        | scale | from 0 |
        | --- | --- |
        | easing | ease out back |
        | duration | 0.2 |
        | start delay | 0.7 |
