# 인게임 전체 설계

> 출처: [[raw/gdd/01_Core/인게임_전체_설계서]], [[raw/gdd/01_Core/지휘관_성장_경제효과_기획서]], [[raw/gdd/01_Core/지휘관_인게임_경제_시나리오]], [[raw/gdd/01_Core/전략성_강화_기획서]], [[raw/gdd/06_Feedback/M7_인게임_작업리스트]], [[raw/gdd/06_Feedback/V0012_빌드_작업내역]], [[raw/gdd/06_Feedback/전투_템포_개선_피드백]]

## 개요

행파냥(TCA) 인게임의 전투/정비 페이즈 분리, 소환 및 합성 규칙, 경제 시스템, 유물 카지노, 속성 디버프 밸런스 전체를 통합 명세한 설계서이다. 15웨이브 구조가 현재 표준이며, 다수 시스템(소환 가중치, 3특성 특권, 유물 카지노 등)이 미구현 상태이다.

## 기본 구조

### 전투 / 정비 페이즈 분리

- **15웨이브 구조** (1회 10~20분)
- **전투 페이즈**: 적 처치 → 경험치 → 레벨업 → 택틱 카드 선택
- **정비 페이즈**: 웨이브 종료 후 미네랄로 유닛 소환 + 머지(합성) + 덱 설계
- **전환 이력**: 20웨이브 → 15웨이브로 압축 (M7 완료). 순수 전투 시간은 유지하되 아머리 비중 감소

### 빠른 진행 보상 (미구현)

- 웨이브 내 목표 처치 수를 제한 시간보다 빠르게 달성 시 추가 미네랄 보너스
- 예시: 5웨이브 몹을 15초 만에 처치 → +10 미네랄 추가
- 의도: 성장한 유저에게 사냥 동기 유지 + 후반 유물 카지노의 연료 확보

## 소환 및 합성 규칙

### 순수 머지 (완전 랜덤)

- 같은 등급의 같은 유닛 2기 합성 → 상위 등급 완전 무작위 유닛 등장
- 피티(천장) 없음. 완전 랜덤성이 핵심 도박적 재미
- 예시: 일반 불씨냥 + 일반 불씨냥 → 고급 생선냥 등 전혀 다른 유닛 등장

### 소환 및 머지 가중치 (미구현)

- 택틱에서 특정 유닛 특성 카드 선택 시 해당 유닛의 소환/머지 출현 확률 대폭 상승
- 예시: 불씨냥 택틱 선택 → 이후 소환/머지에서 불씨냥 2배 확률 등장
- 의도: 억까(RNG 스트레스) 완화, "은밀한 서포트"로 덱 빌딩 지원

### 3특성 마스터 특권 2종 (미구현)

**특권 1: 각성 머지 (확정 진화)**
- 3특성 완료 유닛 + 동일 유닛 합성 시 무조건 같은 유닛 상위 등급으로 확정 진화
- 예시: 3특성 고급 불씨냥 + 고급 불씨냥 = 확정 희귀 불씨냥

**특권 2: 태생 전설 조합식 프리패스**
- 3특성 완료 에픽 유닛이 태생 전설 재료로 사용될 때 소멸하지 않고 필드에 잔존
- 예시: 에픽 마린냥(3특성) + 에픽 방패냥 → 태생 전설 캡틴냥 소환 + 에픽 마린냥 필드 생존

## 전설 루트 (2-Track)

| 트랙 | 명칭 | 획득 방법 |
|------|------|---------|
| Track A | 태생 전설 (조합식) | 특정 에픽 유닛 고정 조합식으로만 획득 |
| Track B | 각성 전설 (한계 돌파) | 3특성 마스터 → 각성 머지로 에픽→전설 직행 |

### 덱 압축 및 소환 풀 베이스 승급 (미구현)

- 각성 전설 도달 유닛 계열: 소환 풀 및 랜덤 머지 결과 풀에서 영구 제외
- 특정 등급 모든 유닛이 전설 도달 시: 소환 버튼 기본 결과물이 다음 등급으로 자동 승급
- 예시: 일반 4종 전설 달성 → 소환 버튼에서 일반 등급 완전 제거, 고급부터 시작

## 성장 템포와 택틱 제어

### 페이즈별 택틱 가중치 시스템 (미구현)

- Lv.1~10 (초반): 경제 카드 40% 확률 → 초반 스노우볼
- Lv.11~20 (중반): 유닛 특성 카드 위주 → 덱 방향 확립
- Lv.21~30 (후반): 경제 카드 0%, 성장/태그 카드 70% → 스탯 폭발

### 군단 평가 (배치 유닛 평균 등급) 보상 (미구현)

- 필드 배치 유닛 평균 등급을 점수로 환산
- 점수 높을수록 레벨업 택틱에서 영웅/전설 등급 강력 카드 출현 확률 증가
- 예시: 필드 전부 에픽/전설 → 전설 등급 택틱 25% 확률

### 성장/태그 택틱 시스템 (미구현)

- 관통형, 탄환형 등 공격 태그별 % 단위 스탯 폭발 (곱연산)
- 유물 증폭과 맞물려 뱀서류 특유의 왕귀 무쌍 쾌감 완성

## 극후반 경제: 유물 카지노

### 유물 5구 상점 + 리롤 (미구현)

- 중간 보스 처치 시 5구 유물 상점 오픈 (게임 시간 일시 정지)
- 미네랄로 유물 직접 구매, 리롤(비용 점증)으로 원하는 유물 탐색
- 유닛 1기당 최대 2개 유물 장착

### 확률형 유물 강화 (미구현)

- 유닛 장착 유물에 미네랄 투입 → 레벨업 (성공/실패 확률형)
- +10강 돌파 시 특수 기믹 해금

### 유물 특수 옵션 예시 (미구현)

| 유물명 | 특수 효과 |
|--------|---------|
| 초전도 프리즘 | 명중 시 투사체 2개로 분열 |
| 저주받은 뼈목걸이 | 적 처치 시 시체 폭발 → 연쇄 유폭 |
| 거인 학살자의 톱날 | 체력 15% 이하 엘리트/보스 즉사 |
| 황금 고양이상 | 보유 미네랄 양에 비례 데미지 무한 증폭 |
| 겜블러의 주사위 | 매 웨이브 꽝 없이 100% 랜덤 버프 3개 동시 폭발 |

## 전투 피드백 & 밸런스

### 속성 디버프 수치 극대화 (미구현)

- 기존 밋밋한 디버프 폐기 → 맞으면 확 티 나도록 수치 극단적 상향
- 보스 특징에 따른 속성 카운터 설계:
  - 화염: 휴식하는 보스 카운터
  - 얼음: 캐스팅 보스 카운터 (기절)
  - 번개: 빠른 스킬 속도 보스 카운터 (감전)
  - 암흑: 빠른 이동 속도 보스 카운터

### 전투 템포 재설계 (2026년 3월 요청)

> 출처: [[raw/gdd/06_Feedback/전투_템포_개선_피드백]]

- **패배 조건 전환**: 피격 사망 → 광폭화 기반 DPS 체크(타임어택)
- 몬스터 공격력 50% 이상 감소 필요 (광폭화 보기 전 사망 방지)
- 전체 유닛 공속 2배 / 공격력 0.5배: DPS 총량 유지하되 타격 횟수 2배 → 상태이상 스택 노출 확보

### V0012 완료 항목 (2026-03-17)

- 카메라 세팅 변경 (하단 조작패드 영역 확보)
- 미네랄 획득 연출 변경: 아이콘이 패널까지 이동 → 바로 수량 표시 후 획득
- 몬스터 이동 속도 하향: 1.2 → 0.6
- 보스 체력 가중치 하향: 100 → 50
- 보스 전투 중 일반 몬스터 등장 빈도 상향
- 광폭화 부여 조건 변경: 실제 스폰 수 < 계산된 스폰 수일 때 발동

## 유저 플레이 시뮬레이션 (전체 흐름)

| 구간 | 웨이브 | 핵심 행동 |
|------|--------|---------|
| 초반 | W1~3 | 경제 택틱 선택, 빠른 진행 보상으로 미네랄 확보 |
| 중반 초입 | W4~7 | 특정 유닛 특성 카드 집기 → 가중치 어시스트 활성화 |
| 중반 | W8~11 | 3특성 프리패스+각성 머지 발동, 소환 풀 베이스 승급 |
| 후반 | W12~15 | 태그 스탯 택틱 집중, 유물 리롤+확률 강화로 기믹 해금 |

## 이전 설계

### M7 인게임 작업 완료 목록

> 출처: [[raw/gdd/06_Feedback/M7_인게임_작업리스트]] (superseded)

- 웨이브 20 → 15로 압축 완료
- 웨이브 종료 시 잔여 몬스터 남기기 완료 (kill 처리 제거)
- 부대 로테이션 속도 빠르게 조정 완료
- 보스 전투: 패배 조건 룰 보완 (진행 중)
