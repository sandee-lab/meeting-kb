## 사업팀 BM 피드백 (2025, v0.0.6)

> 출처: [[raw/feedback/Minicraft_0.0.6 사업팀 피드백 취합]]

v0.0.6 빌드에 대한 사업팀 9명의 BM 관련 의견.

- **수익화 노출 타이밍**: 광고 제거는 챕터2 클리어 후, 순찰(소탕)은 챕터4 클리어 후 노출 제안 → 자연스러운 콘텐츠 해금 동선에 맞춤
- **중복 소환 매력도가 핵심**: 전설 유닛 첫 획득 허들을 낮춘(명함 3,000다이아) 대신, 중복 소환 시 외형/스킬 이펙트의 압도적 변화로 과금 욕구 자극 필요. 단순 스탯 상승을 넘어 플레이 경험이 바뀌는 수준의 체감 필요
- **패키지 확장**: 유닛 소환 재화 패키지 + 장비 소환 별도 재화 구성으로 패키지 다양화 가능
- **시장 관점**: "출시 가능한 수준이며, 향후 UA/마케팅을 통한 유저 모객이 중요" — 마케팅 싸움으로 귀결될 것으로 전망

## NCsoft 공식 피드백 — 과금 가치 구조 (2026-03-03, Ver.0.0.10)

> 출처: [[raw/feedback/NCsoft_TinyCatAssemble 피드백_20260303]]

- **다이아 소비 동선 불명확**: 멤버십 외 고액 결제의 체감 혜택 낮음, 다이아를 쓸 곳이 마땅치 않음
- **소액 결제 체감 없음**: 한판에서 의미 있는 도움이 느껴지는 시스템 부재
- **결제 체감 구조 개선 제안:**
  - 다이아 소비를 전설 제작/강화 루프와 명확히 연결
  - 결제 시 전설 획득/성장 속도가 가시적으로 빨라지는 구조 설계
  - 1판 단위 체감형 소비 상품 검토
- **전설 중심 BM 구조 제안**: 과금 가치 명확화를 위해 전설 유닛이 "선택지"가 아닌 "명확한 목표"로 재정의 필요
- **유닛 성장 매력도 저하 지적**: 중복 성장 효용 낮아 가챠 동기 약화, 성장할수록 가챠 성장 빈도 감소
  - 개선 제안: 중복 성장으로 주요 능력 해금 구조 / 유닛 성장을 계정 성장과 연계

## BM 및 성장 논의 — ver.0.0.13 (2026-03)

> 출처: [[raw/feedback/TCA BM 논의 자료 _20260408.pdf.md]]
> 작성 주체: 신규사업본부 (퍼블리셔 측 내부 논의 자료)

### 전투력 성장 척도 이슈

- **전투력이 "강함"의 척도로 기능하지 못함**
  - 전투력 산정 공식 안내 없음 → 계정 강함인지? 덱의 강함인지? 불명확
  - 유닛 강화 시 전투력 상승 → 실제 사용하지 않는 유닛의 성장이 전투력에 포함
  - 미장착 장비 승급 시 전투력 상승하나, 미장착 장비 강화는 반영 안 됨 → 기준 모호
  - 각 개별 상승이 %와 숫자로 중복 표시 → 직관성 저하
- **위험물 보관 등급 (신규 추가)**
  - 강화·누적은 되나 부대 버프에 한정
  - 버프 구조가 복잡하고 설명이 과도한 내러티브 몰입 방식이라 이해 어려움

### 장비 역할 강화 필요성

- **핵심 논리**: 유닛과 장비는 성장의 두 축이지만, 게임 시스템 상 유닛은 "카드 덱"이지 소유 체감이 약함
  - 원인 1: 전투 시 보유 유닛 선택 불가 (머지의 랜덤성)
  - 원인 2: 유닛 강화는 부대 강화이지 본체(배틀슈트) 강화가 아님
- **결론**: 플레이할수록 장비가 확실한 성장 자산 (상수) vs 유닛은 변수
- **이슈**: 보석 재화가 유닛·장비 양쪽에 모두 소모 가능 → 초반에는 유닛 중심으로 기울어짐
- **제안**:
  1. 고등급 장비 BM 도입(적용 완료) + 별도 소환 아이템으로 패키지 구성
  2. 장비 세트 효과 도입으로 수집 목표 제공

### 0.0.13 초반 허들 이슈

- 스테이지 5 기준 급격히 상승하는 난이도 (언락·재화량 조정에 따른 영향)
- 장비 미오픈으로 캐릭터 HP 하락 → 캐릭터 사망 가능
- 파밍 콘텐츠 부재 → 해당 스테이지 반복 헤딩하는 불쾌한 경험
- 소환 소모 재화량 증가 + 초반 지급 보상량 조정 → 기존 대비 성장도 하락
- 3가지 타입 원거리 유닛 동시 등장 → 급격한 난이도 원인

### 전설 패키지 2종 신규 계획

- **포지셔닝**: 2만원대 중반의 필수 고가 상품
- **상품 구성**: 동일 전설 유닛 3개 기본 제공 → 구매 즉시 3강 가능
- **전설 강화 스킬 획득 조건 조정**: 6/12강 → 3/6강으로 당겨 전설 강화 니즈 유도
- **예시**: 미야기 패키지 (미야기 즉시 획득 + 3강 + 신규 스킬 / ₩25,000)
- **핫딜 연계**: 전설칩 베이스(20개) 패키지를 조건형 핫딜로 구성 (2/5강 달성 시 한정 노출)
- **전제 조건**: 현재 다이아 직구매로 전설 명함 획득 가능 → **1티어 전설군은 직과금 대신 전설 패키지 구매로 대체** (미결정)

### 멤버십 개편 방안 검토

- **현재 문제**: 오버클럭 일일 보상(전설 칩 1개/일, 20개 모아야 1개 획득)의 매력도 낮음
- **검토 방안 3가지**:
  1. **컨셉 2종 분리** (예: 7,900원 2종) — 부스트 팩(빠른 성장) / 오버클럭 팩(하드코어 전략), 광고 제거 공통
  2. **베이직/프리미엄** (예: 5,900원 / 11,900원) — 광고+기본 / 광고+전투 편의
  3. **현 3개 유지 + 특색 강화** — 가치 낮은 보상 대체 (부스트 팩에 에너지 추가, 오버클럭에 부활석 추가 등)

### 기타 점검 사항

- **상품 슬롯 광고 배치**: 1번째 FREE 슬롯 → RV 구매(1안) 또는 2번째 슬롯 → RV(2안)
- **골드 박스 아이템 도입**: 8k/11k/9k 직접 표시 → 금/은/동 골드 상자로 대체
- **보석/뽑기권 단위 정리**: 보석 100단위, 뽑기권 10단위로 직관화
- **다이아 충전 이벤트 노출**: 로비 → 상점 내 이동, 결제 마일리지 형태 포장
- **목표 일정**: UI/UX 개선 4주차 완료 → 8주차 서비스 준비 (5/8 사전 업로드)

## TCA 개선 제안 (2026-04-08)

> 출처: [[raw/feedback/TCA 개선 제안_240408.pdf.md]]

### BM/아웃게임 피드백

- **부스트팩**: 전투 배속 추가 + 일일 보상(낮은 등급 유닛 조각, 유닛 교체 횟수) 추가
- **오버클럭팩**: 빠른 순찰 무제한 + 일일 보상(전설 유닛 조각, 시작 미네랄, 택틱 교체)
- **충전 이벤트**: 기간 한정 → 상시 적용 구조 전환 요청 + 로비→상점 내 이동
- **S급 장비**: 시각화된 전투 기능이 복잡성 증가 우려 → 일반 장비 발동 옵션 삭제 + 등급별 능력치로 대체 제안
- **장비 소환권**: 개발팀 의도 청취 희망 (카피바라 고! 레퍼런스 — 일반/특수 분리)

### 인게임 피드백

- **신규 유닛 청소부(쓱싹이)**: 유틸리티 중심 → 공격/속성 시너지 중심 재설계 제안. 투사체·AoE 제거 기능은 장비로 이관 제안
- **포어맨 레시피**: 버프 대상 없는 구조 → 터렛 소환 기능 추가 or 대장 터렛에 공격 기능
- **전략 카드**: 등급별 동일 선택지 → 유닛 카드 효과와 어울리는 스탯 구성으로 전략성 강화
- **이자 시스템**: 기본 시스템에서 제외 희망 (전투 전환 연출 길어짐). 아웃게임 BM 상품으로 활용 가능
