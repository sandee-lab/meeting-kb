# 택틱 시스템 확장(다중 효과)

> **요약:** 하나의 택틱에 여러 개의 효과를 지정할 수 있도록 시스템을 확장하는 기획이다. StageModeTacticPoolData를 TacticPool과 TacticEffectPool로 분리하여 작업 복잡도를 완화하고, 스탯 기본 부여를 통한 성장 체감 제공을 용이하게 한다. 다중 효과 부여 시 UI에서는 각 효과 설명을 쉼표로 구분하여 노출한다.
> **현재 상태:** 미적용

택틱 시스템 확장(다중 효과)

# 개요

* 현재는 하나의 택틱에 하나의 효과만 설정할 수 있다.
* 향후에 택틱의 확장성을 고려하면, 하나의 택틱에서 여러 개의 효과를 지정할 수 있어야 다양한 의도를 충족시킬 수 있다.

# 활용 방안

* 하나의 택틱에서 여러 조건으로 효과를 구성하는 경우의 작업 복잡도가 완화된다.
  + 현재는 A라는 조건일 때, A1 효과를 주고, B라는 조건일 때, B1 효과를 주기 위해서는 스킬을 여러개 만들어서 택틱으로 스킬을 부여하고, 스킬이 다시 여러 개의 스킬을 부여해서 각각의 스킬을 발동시키도록 하고 있다.
  + 다중 효과를 연결 할 수 잇게 되면, 하나의 택틱에서 A1, B1 스킬을 각각 부여하므로 작업 복잡도가 완화된다.
* 필요한 경우에 스탯을 기본적으로 부여해서 택틱 선택에 따른 성장 체감을 쉽게 제공할 수 있다.

# 구현 방향

* StageModeTacticPoolData 데이터를 다음과 같이 분리한다.
  + StageModeTacticPoolData
  + StageModeTacticEffectPoolData
* 하나의 택틱이 여러 개의 Effect를 가질 수 있도록 데이터를 분리하고, 시스템을 확장한다.

# 구조 변경에 따른 UI 규칙

* 하나의 택틱에 요러 개의 효과가 부여된 경우 택틱의 설명은 각각의 효과 로컬을 쉼표(,)로 구분하여 노출한다.
