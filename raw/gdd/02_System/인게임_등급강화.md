# 인게임 스테이지 모드 등급 강화

> **요약:** 가스를 소모하여 유닛 등급을 강화하는 인게임 시스템. 이자율 선택과 택틱 연계를 통한 자원 관리 전략 요소를 설계한다.
> **현재 상태:** 이후 버전으로 대체됨

M6\_스테이지 모드\_등급 강화

# 개요

* 가스 재화의 사용처를 통해 자원 관리 요소를 추가한다.
* 등급 별 강화 요소를 통해서 게임의 진행도(초반, 중반, 후반)에 성장 판단을 요구한다.
* 가스 재화에 이자 요소를 도입하여, 가스 소모 시점에 대한 판단을 요구한다.

# 시스템 구조

## 컨텐츠 언락

* + 스테이지 모드에서 기본으로 제공된다.
  + 튜토리얼 등에서 언락을 제어할 수 있도록 고려한다.
  + 언락 이후에는 별도의 제한없이 1웨이브에서부터 기능이 제공된다.

## 기본 규칙

* + 등급 강화 성장 선택지는 4종으로 구성한다.

| id | 성장 선택지 |
| --- | --- |
| 1 | 일반, 고급 강화 |
| 2 | 희귀, 에픽 강화 |
| 3 | 전설 강화 |
| 4 | 가스 이자율+ |

* + 각 선택지 종류 마다 레벨이 존재한다.
  + 인게임 재화인 가스를 소모해서 각각의 레벨을 올릴 수 있다.
    - 이미 최대 레벨인 경우 더이상 강화할 수 없다.
  + 각 달성 레벨에 따라서 데이터시트에서 지정한 택틱을 add 해준다.

## 신규 스탯

### 피해 증폭(Mul\_Atk\_2)

* + - 곱연산 처리하기 위한 \_2 능력치 추가

### 가스 이자율(add\_gas\_interest\_rate)

* + - 규칙
      * 아머리 진입 시 마다 현재 보유한 가스량에 % 만큼의 보너스를 지급한다.
    - 계산 공식
      * 보유량에 이자율을 곱해서 나온 값 만큼 추가로 지급한다.
      * 소숫점 이하 값은 버림 처리 한다.

| 추가 지급량 = 아머리 진입 시 보유량 \* (add\_gas\_interest\_rate ) |
| --- |

* + - 적용 스탯
      * add\_gas\_interest\_rate (신규 스탯)
    - UI 연출
      * 아머리 진입 시 이자로 획득하는 가스 수량이 있을 경우 짧은 UI 연출을 노출한다.

# 필요 데이터시트

## StageModeGradeEnchantListData

* + id
    - 각 성장 별 고유 id
  + local\_name
    - 성장 선택지 명칭
  + icon\_path
    - 아이콘 경로
  + icon\_name
    - 아이콘 이름

## StageModeGradeEnchantDetailData

* + id
    - 성장 선택지 고유 id
  + level
    - 성장 선택지 별 레벨
  + enchant\_fee
    - 레벨업에 필요한 재화
  + add\_tactic\_ids
    - 각 레벨 달성 시 추가로 넣어줄 택틱 ids
      * 배열로 처리 (id;id)
    - 0레벨은 기본 레벨이므로 제외
