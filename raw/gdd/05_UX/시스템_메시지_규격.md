# 시스템 메시지 규격

> **요약:** 인게임·아웃게임 시스템 메시지 타입과 출력 규칙을 정의. 토스트/팝업/컨텐츠언락 등 메시지 유형별 동작과 대상을 명시한다.
> **현재 상태:** 이후 버전으로 대체됨

**M6\_인게임/아웃게임\_시스템메시지**

시스템 메시지

| **타입** | **구성** | **예시** |
| --- | --- | --- |
| 1 | 텍스트 | 지정된 로컬키만 출력한다. |
| 2 | 텍스트 + 아이콘 | 지정된 로컬키와, 아이콘을 함께 출력한다. |

시스템 메시지 규격

* 공통
  + 최소 사이즈 : 300
  + 최대 사이즈 : 500
    - 폰트 입력으로 500 넘어가는 경우 줄넘김, 상하로 사이즈 확장
  + 폰트 사이즈 : 32
  + 좌측 아이콘과 옆에 있는 텍스트 박스는 그리드 형태로 제작해서 중앙정렬

시스템 메시지 출력 규칙

* 조건 만족 시 출력한다.
* 메시지는 2초 간 유지한다.
* 시스템 메시지 출력 중 신규 시스템 메시지 발생 시, 이전 시스템 메시지를 즉시 종료하고 신규 시스템 메시지를 출력한다.

비고1. 컨텐츠 언락

* 개요
  + 컨텐츠 언락 알림은 시스템 메시지와 구분하여 사용한다.
* 출력 규칙
  + 조건 만족 후 최초 아웃게임 복귀 시 출력한다.
    - 로컬키 : Local\_Contents\_Unlock\_Desc
    - 내용 : <color=#FFFBE3>{0}</color> <color=#D9CFB5>해제</color>!
      * {0} 내용은 ContentsUnlockData@local\_name
  + 메시지는 2초 간 유지한다.
    - 컨텐츠 언락 알림이 뜨는 동안에는 **인풋을 막는다. (터치 방지)**
      * 뒤에 불투명 검은 배경을 깔 수 있겠다. (터치 방지 알림)
    - 동시에 여러 콘텐츠가 해제된 경우, 순차적으로 출력한다.
      * ContentsUnlockData@order 참조
    - 다른 연출과 겹칠 경우, 아래 순서로 출력한다.
      * 레벨 업
      * 컨텐츠 언락
      * 시스템 메시지

비고2. 인게임 보상 연출

* 개요
  + 인게임에서 몬스터 킬 수 달성에 따른 보상 획득은 별도 연출을 사용한다.
* 인벤토리 보상 종류
  + 몬스터 킬 달성
    - 로컬키: ?
    - 내용: 몬스터 {0}마리 처치 성공!
    - 아이콘
      * 보상 아이콘 : 아이콘 제작 필요
      * 몬스터 아이콘 : 아이콘 제작 필요
* 연출 순서
  + 조건 만족 시 메시지 창 출력
  + 조건 만족 시 인게임에 오브젝트 드롭
  + 메시지 창 점멸 시점, 드롭된 오브젝트가 인벤토리로 들어가는 연출 출력
  + 인벤토리 아이콘 점멸
* 메시지 창 출력 규칙
  + 조건 만족 시 출력한다.
    - 화면 상단에서 내려오는 연출 (테오크 새 대화 연출 참고)
  + 메시지는 2초 간 유지한 후 상단으로 다시 올라가며 점멸
    - 메시지 뜨는 동안 인풋이 허용된다.
    - 시스템 메시지와 출력 타이밍이 겹칠 경우, 동시 출력한다.

비고3. 룰렛 보상 연출

* 개요
  + 인게임에서 룰렛으로 보상을 획득하는 경우 오브젝트 드롭 연출을 추가한다.
* 연출 순서
  + 룰렛 연출
  + 룰렛 결과 출력 후 인게임에 오브젝트 드롭
  + 오브젝트 지정된 위치로 플라이 연출
