# 2025년 하반기 타임라인 (8월~12월)

> 이 문서는 2025년 8월부터 12월까지의 회의록을 시간순으로 정리한 타임라인이다.
> M7~M10 마일스톤 진행, 알파빌드 완성, 도쿄게임쇼 출품, 전략성 재설계, FGT 준비까지의 흐름을 다룬다.

## 2025년 7월 (M5 마일스톤)

> 한줄 요약: M5 마일스톤 — 인게임 방향 전환(승급 삭제·커맨드센터 초월·관리 페이즈 도입), 플레이테스트 2회 실시.

### M5 방향 전환

- 액션/관리 페이즈 구분 방향 설계. 지휘관(SCV) / 용병(일반 유닛) 분리 개념 논의 — [[raw/gdd/01_Core/M5_next_방향_고민]]
- M5 아웃게임 신규 개발 목표 정리 (상점, 퀘스트, 출석부, 초보자 미션, 도감, 우편, 친구 등) — [[raw/gdd/06_Feedback/M5_]]
- 인게임 개선 방향 및 구현 할일 정리: 택틱 상점 도입, 유닛 인벤토리(관리 페이즈) 구조 확정 — [[raw/gdd/06_Feedback/M5_2_방향_및_할일_정리]]

### M5 플레이테스트

- **(2025-07-11) 플레이테스트**: 유닛 승급 삭제, 커맨드센터 수동 초월 방식 변경, 택틱 획득→스탯 분리, 유닛 획득 오브젝트 방식 테스트. 피드백 포커스: 변경 방식 주관 의견, 유닛 조작 난이도 — [[raw/gdd/06_Feedback/M5_7_11_플레이_테스트_가이드]]
- **(2025-07-24) 플레이테스트**: 20웨이브(10/20 보스) 구성, 관리 페이즈 플레이 흐름, 지휘관 SCV 추가(사망 시 패배), 일반 유닛 부활 메커니즘, 종족 시너지(바이오닉/메카닉/사이오닉), 배치 효과 도입 — [[raw/gdd/06_Feedback/M5_7월_24일_테스트_가이드]]

### 핵심 변화

- **유닛 승급 제거**: 기본 유닛 등급 구분 및 승급 개념 삭제, 직접 초월 조합식으로 단축
- **커맨드센터 초월 방식 도입**: 자동 초월 → 맵 내 오브젝트에서 수동 초월 (완료)
- **관리 페이즈 구조 확정**: 택틱 상점(`wave_type=TacticShop`) + 유닛 인벤토리(`wave_type=Armory`) 이원화
- **지휘관 SCV 포지션 확정**: 전투 미참여, 자원 수집 역할. 사망 시 게임 패배
- **종족 시너지 도입**: 바이오닉/메카닉/사이오닉 3종족, 세트 효과 (2/3/5/8마리 배치 시 활성화)
- **택틱 다양화 기획**: 유니크 택틱, 무기 택틱, 복수 효과 택틱 방향 설계 (아이디어 단계, 미구현)

### 핵심 방향 논의 (0903 · 0904)

- **프로젝트 목표 재정립**: "재미있는 한 판으로 온보딩 → 아웃게임 성장 댑스로 장기 플레이 유도"
- **택틱 역할 재정의**: 택틱은 유닛 조합을 보조하는 역할, 성장 체감의 핵심은 유닛 조합
- **전설 유닛 중요도 강화**: 전설 없어도 충분히 재밌는 구조가 BM과 충돌 → 전설을 덱빌딩·메타 중심에 배치
- **난이도 설계 방향**: 몬스터 체력 증가 대신 **스폰 레이트 조절**로 난이도 상승 → 많은 몬스터를 처치하는 쾌감
- **진형 시스템**: 특정 빌드에만 돌파가 되는 구조 폐기, 어떤 덱이든 스펙 맞으면 몬스터를 밀 수 있어야 함

### 미결 논의 항목 (0912)

- 지휘관 외형 → 귀여운 마스코트로 변경 검토 (주인공보다 지켜주는 대상 포지션)
- 웨이브 리셋 문제: 매 웨이브 초기화가 성장 체감 저해 → 웨이브 연속 vs 스폰 레이트 조절 방향 미확정
- 택틱에 시각적 변화 필요, 전설 유닛과 조합되는 택틱 설계 필요
- 유닛 리롤 재도입 검토 (판매와 역할 분리 전제)
- 체력 통합 논의 (유닛+지휘관 통합 체력) — 구현 복잡도로 대기

### 알파빌드 피드백 (0929)

- 한판 길이가 길게 느껴짐 (경험 반복, 택틱 반복성, 작은 실패 없음)
- 패배 조건: SCV 너무 잘 죽고 유닛은 안 죽는 불균형 → 유닛이 다 죽으면 패배로 변경 제안
- **부대 배치 4슬롯 → 3챕터 이후 8슬롯** 단계적 언락 제안 (초반 실패-극복 사이클 필요)
- 중간 보스 기점 무드·전투 변화 필요 (악마의 거래 재고려)
- 유닛 언락을 챕터 진행에 연동: 1챕터(일반·고급·희귀 일부) → 2챕터(전부) → 3챕터(에픽 일부)
- 전설 유닛 조합 난이도 완화 필요, 사거리 시각화 요청

### 구현 완료 항목

**전투 구조**
- **20웨이브 → 15웨이브** 압축 (순수 전투 시간 유지, 아머리 비중 감소)
- **잔여 몬스터 남기기**: 웨이브 종료 시 kill 처리 않고 다음 웨이브로 이월
- **보스 타임어택** (`end_time_sec`): 보스 전투 제한 시간 설정
- **특수 몬스터 개체 수 제한** (`spawn_monster_select_max`): 패턴 과부하 방지
- 부대 **로테이션 속도** 개선 (감속 처리 완화, ConstantDatas 수치 분리)

**게임 템포**
- 아군 이속 2.8 → **3.2**, 적군 이속 1.2 → **1.4** (+15%)
- 룰렛 빈도 평균 7.2회 → **5.1회** (-30%), 인터랙션 시간 2.5초 → **1초**
- 유닛 판매 효율 상향: 희귀 4→8가스, 영웅 0→16가스
- 등급 강화 최대 레벨 5 → **3** 하향
- 통조림: 킬 보상 제거 → 룰렛 보상으로만 지급

**스킬 시스템 확장**
- 패시브 발동 조건 신규 추가: `passive_move`, `passive_stand`, `passive_hp_recovery`, `passive_dead_ourteam`, `passive_hit_ourteam`, `passive_kill_ourteam`, `passive_dead_enemyteam`
- 스킬 서브 조건 추가: `Check_CasterStat`(스탯 비교), `Check_CasterHpRatio`(체력 비율), `Check_OurteamFormationUnitCnt`(생존 유닛 수), `Check_Random`(확률), `Check_CasterHasSkilleffect`(상태이상 보유 여부)
- **`is_volatility`** 컬럼: 웨이브 클리어 시 버프 초기화 여부 제어
- **`CoolTime_Fix`**: 쿨타임 가속 스탯 무시하는 고정 쿨타임
- 패시브 트리거 타겟 기능: 조건 만족 대상을 타겟으로 가져오는 기능 (`PassiveTriggerTarget`)

**택틱 구조 재설계**
- **레벨업 → 택틱 즉시 지급** 시스템 도입 (기존: 웨이브 클리어 후 → 변경: 레벨업 시 즉시)
- **그룹 택틱 + Mythic 보너스**: 특정 그룹 택틱 만렙 달성 시 Mythic 등급 보너스 카드 선택 UI
- 택틱 개별 확률 제어 컬럼 (`StageModeTacticPoolData@select_weight`)
- 택틱 그룹 최대 레벨 설정 (`StageModeTacticPoolData@max_level`)

**신규 스탯**
- `atk_transition_from_move_speed_weight`: 이속%를 공격력으로 전환
- `atk_transition_from_max_hp_weight`: 체력%를 공격력으로 전환
- `mul_atk_3`, `mul_max_hp_2`: 아웃게임 성장과 분리된 인게임 전용 곱연산 스탯

- 택틱 구조 재설계: **스택형**(3단계 진화, 마지막 스페셜 택틱 보너스)과 **단일형**(즉시 효과)으로 이원화

- 피아 식별 개선: 적 유닛 아웃라인 컬러/톤 조정으로 시인성 향상
- 컨텐츠 루프 설계: 챕터(덱 파밍) → 골드 성장 → 던전(성장) → 무한모드(랭킹) → 업데이트 사이클
- 게임 스토리 컨셉 결정: 스페이스 인베이더 모티브의 **침략 반전** 스토리
