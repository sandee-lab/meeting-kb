# 관리페이즈 인벤토리

> 출처: [[raw/gdd/02_System/M5_관리페이즈_인벤토리]], [[raw/gdd/02_System/M5_인게임_아웃게임_지휘관_v2]]

## 개요

M5 빌드에서 설계된 인게임 관리 페이즈 시스템. 웨이브 종료 후 유닛 편성·합성·판매 및 지휘관 운용이 이루어지는 인벤토리 구조이다. 현재 상태: 미적용 (아이디어 단계, 미구현).

## 슬롯 구조

### 전투 슬롯 (상단)

- 웨이브 진행 시 실제 전투에 참여하는 유닛 편성 슬롯
- 지휘관(SCV)이 부대 0번 위치에 고정 배치 (리더 슬롯)

### 인벤 슬롯 (하단)

- 최대 **15개 (3×5)** 슬롯. 각 슬롯에 유닛 최대 1개 보관
- 전투에 참여하지 않고 저장 대기 중인 유닛 보관

### 유닛 획득 흐름

- 전투 페이즈 중 유닛 획득 가능 (박스·슬롯 연출 유지)
- 전투 슬롯 가득 → 인벤 슬롯에 추가
- 전투 슬롯을 의도적으로 비워두고 플레이하는 것이 권장됨 (논의 완료)
- 전투·인벤 슬롯 모두 가득 찰 경우: **자동 판매** (판매가만큼 가스 획득)

## 유닛 관련 상호작용

### 유닛 합성

- 같은 유닛이 동일 슬롯에 배치될 경우 **grade+1 등급의 랜덤 유닛** 획득
- 합성에 사용된 유닛은 삭제, 결과 유닛은 드롭된 슬롯에 배치
- 영웅(최고 등급) 유닛은 합성 불가 (이미 최대 등급)
- 현재 가장 많이 장착된 유닛 타입의 가중치 증가 기능은 지휘관 옵션으로 이전 예정 (아이디어 단계, 미구현)

### 유닛 판매

- 유닛을 휴지통 아이콘에 드래그 앤 드롭 → 판매
- 올려두면 판매 가격 미리 표시
- 예시 판매가: 일반 유닛 1가스, 희귀 유닛 2가스 (데이터 미확정)

### 유닛 이동

- 슬롯 간 드래그 앤 드롭으로 배치
- 전투 슬롯 ↔ 전투 슬롯 / 전투 슬롯 ↔ 인벤 슬롯 / 인벤 슬롯 ↔ 인벤 슬롯 모두 교체 가능
- 슬롯에 유닛이 이미 있으면 위치 교체

### 유닛 정보 확인

- 유닛 터치 시 스탯(공격력·체력·공속·방어력) 및 패시브 스킬 확인
- 근접·원거리·원소 등 피해 형태 구분

## 전설 합성

- 전설 합성 조건 충족 시 HUD에 소환 버튼 등장
- 버튼 선택 → 전설 합성 창 호출 → 원하는 조합식 선택
- 조합식(RequireUnitIdSet) 데이터는 미확정
- 사용된 유닛은 삭제
- 결과 전설 유닛은 전투 슬롯 우선 배치, 빈 슬롯 없으면 인벤 슬롯에 추가

## 시너지 정보

- 시너지 버튼 선택 시 텍스트 박스 출력
- **단일 시너지**: 전투 슬롯 배치 수에 따라 누적 수치 증가
  - 예: 마린 (2기) 공속 20% / (3기) 공속 40% / (4기) 공속 60% / (5기) 공속 100%
- **역할 시너지**: 탱커/딜러/서포터 조합 효과 (아이디어 단계, 미구현)
  - 3탱커: 전체 HP 증가 + 피해 감소
  - 4딜러: 크리티컬 확률 증가 + 공격속도 증가
  - 2서포터: 아군 회복 + 버프 효과 증가
- **속성 시너지**: 극열/극한/전기 조합 효과 (아이디어 단계, 미구현)
  - 극열 3유닛: 지속 화상 피해 + 얼음 효과 면역
  - 극한 3유닛: 적 행동 속도 감소 + 화염 피해 감소
  - 전기 3유닛: 적에게 충격(기절) 디버프

## 페이즈 전환

- **택틱 상점** 이동 버튼: 관리 페이즈 ↔ 택틱 상점 페이즈 상호 이동 가능
- **전투 시작** 버튼: 선택 시 전투 페이즈로 전환

## 지휘관 (SCV) 인게임 역할

> 출처: [[raw/gdd/02_System/M5_인게임_아웃게임_지휘관_v2]]

지휘관은 SCV 형태로 인게임에 배치된다. 현재 상태: 미적용 (아이디어 단계, 미구현).

### 포지션 및 체력

- 부대 **0번 위치**에 고정 배치 (유저 장착 지휘관 자동 배치)
- 자원 수집 역할 담당 (게임 스토리 포장)
- 지휘관 체력은 인게임 상시 표기
- **지휘관 사망 시 게임 패배** 판정

### 리더 효과

**시작 유닛 고정:**
- 지휘관별 시작 유닛 지정 (`CommanderEvolutionData@start_unit_ids`)
- 아웃게임에서 플레이 시작 방향을 결정하는 역할

**승급 단계별 추가 효과:**
- 승급 레벨 달성 시 효과 해금 (합산 방식)
- (1) 종족·역할 스탯 보너스: 승급 추가 스탯 테이블 활용
- (2) 택틱 선택지 추가: `add_tactic_id` 컬럼으로 특정 택틱 풀에 추가

**택틱 카드 구분 (지휘관 ver.3 기준):**
| 유형 | 설명 |
|------|------|
| 기본 카드 | 시작 유닛 카드 + 전투 시작 시 자동 적용되는 택틱 (`add_tactic_type=Default`) |
| 추가 카드 | 택틱 선택 풀에 추가되는 택틱 (`add_tactic_type=Pool`) |

- 시작 유닛·기본 카드는 현재 승급 단계에 해당하는 것만 적용
- 승급 추가 스탯은 현재 승급 단계 이하 수치 모두 합산

### 아웃게임 지휘관 효과 구조

| 효과 유형 | 설명 |
|---------|------|
| 장착 효과 | `CommanderData`에서 관리. 지휘관 장착 시 적용 (add_atk 포함) |
| 보유 효과 | `CommanderEnchantData`에서 `special_stat_type_1/value_1`으로 관리 |
| 승급 스탯 | `CommanderEvolutionData`에서 승급 단계별 누적 합산 적용 |

- 지휘관 택틱 지원 effect_type: Add_Stat / Add_SkillEffect / Spawn_Unit / Add_Skill (Add_Skill은 개발 진행 중)
