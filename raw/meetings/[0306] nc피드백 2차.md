### 핵심 USP

머지가 핵심 재미, 머지를 잘해서 전설 유닛을 잘 쓰는 것이 재미

→ 개발팀 의견 : 처음 온보딩 과정에서 유저들이 가장 인지하기 쉬운 재미가 머지라는 것에는 동의. 하지만 머지만 남길 경우 채워지지 않는 재미의 부재가 작지 않다고 생각하기 때문에, ‘머지와 덱 빌딩’ 의 적절한 균형을 찾는 것이 우리 프로젝트의 골

(특히 머지의 경우 유저 대상(도쿄 게임쇼)으로 테스트 했을 때, 자동 머지를 넣어 달라거나, 이 행위가 핵심 재미는 아니라는 피드백이 생각보다 다수 있었기 때문에, 우리가 직관적인 재미를 위해 깊이 있는 플레이를 놓치지는 않을지 우려)

### 전설이 메인이 되도록 하는 밸런싱

전설 유닛의 획득 템포 개선, 유닛 능력치 편차 명확하게 제공, 전설 제작 경험 개선

→ 개발팀 의견 : 

- 전설 유닛의 재료를 조금 더 유저들이 인지하기 쉬운 방향으로 개선 예정. 예를들어, 속성별 전설 유닛은 각 속성 별 에픽 유닛 1기를 요구하고, 쉬운 재료 1개정도를 더 추가하는 방향. 유닛 등급별 능력치 편차는 현재보다 조금 더 체감되게 바꾸고, 수준에 대해서는 테스트 후 결정

### 목표가 되는 전설 유닛에 대한 매력도 강화

전설 유닛의 특징/ 컨셉 강화, 속성 내 다양한 역할군 설정,  공격속도와 쿨타임의 체계 일원화 등

→ 개발팀 의견 : 

- 현재는 속성별로 2종(메인딜러, 서포터) 가 존재하고 아직 잘 가다듬어지지 않아 눈에 띄지 않는 문제가 존재함. 추후 각 속성마다 역할을 더 하는 유닛을 최소 2종이상 더 추가해야 한다는 점에서 동의. 단, 개발 기간등의 여유를 생각하면 소프트런칭시점에는 현재 전설 수량에서 밸런스와 특징을 가다듬는 정도로 개선이 현실적

| 속성 | 의도된 특징 |
| --- | --- |
| 무속성 | 전설 유닛 기준으로 하나의 유닛 덱을 강화 하는 방향 |
| 화염 | 최대한 넓은 범위에 상태이상을 걸어서 이동속도 증가 버프를 받고 전장을 빠르게 돌진 |
| 전기 | 단일 공격력 최고, 보스전에서 유리함 |
| 얼음 | 냉기 상태이상으로 PVP등에서 유리 |
| 카오스 | 시간지연, 죽은 위치에 소환수 다수 소환으로 디버프+ 소환 특성 |
- 쿨타임과 공격속도 통합에 대해서 개발팀도 공감. 작업량이 크다보니 적용 시점은 추후 다시 논의
- 추후 신규 시스템을 통해 운빨존많겜과 같이 성장 방식에 변별력을 주는 것도 고려 중. 다만 시간적으로 우선순위를 낮춰서 생각.

### 특성 카드 관련

유닛 카드 삭제, 전술카드 카테고리 분류 등

→ 개발팀 의견 :  

- 앞서 언급한 것과 같이 머지를 살리고 유닛 카드를 모두 삭제할 경우 게임의 깊이가 너무 낮아지고 단순해지는 문제가 우려 됨. 기존에도 게임이 너무 단조롭다는 의견이 많았기에 이 방향에서 조금 더 발전시켜서 테스트를 해봤으면 함.
- 유닛 승계 시스템 :
    - 유닛 카드를 최대치 (3장+)로 모은 유닛의 경우 머지를 할 경우 등급이 상승하며 최대 전설 등급까지 승급이 가능함.
    - 유닛 카드를 최대치 까지 모으지 못한 경우 기존과 같이 랜덤하게 윗 등급의 유닛이 등장함.
    - 이 조치를 통해 내가 원해서 끝까지 강화한 유닛은 강화된 카드를 이용하여 더 높은 등급으로 활용 가능해지고, 원치 않는 경우 합성으로 다른 유닛으로 교체할 수 있으므로 유저 입장에서 조금 더 능동적인 플레이가 가능해 짐.

이 시스템의 경우 1주내로 빠르게 테스트가 가능하므로 빌드 테스트 후 다시 리뷰했으면 함.

### 기타

공격 범위 인지 문제

→ 개발팀 의견 : 

- 단순히 공격 방식의 문제가 아닌, 추가적인 시스템이 필요할 수 있음. 예시로 공격한 곳의 컬라이더 박스를 0.5초에서 1초 정도 잠시 더 남겨 둔다던가, 시각적 피드백을 강화하는 것에 대해 테스트가 필요함
- 출시 전에 적군 유닛의 사이즈나 색감등을 보정해야할 필요성이 있음
