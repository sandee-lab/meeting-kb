# 유닛도감 (총합 레벨 시너지)

> **요약:** 사용하지 않는 유닛의 레벨업 동기를 부여하기 위해, 보유한 모든 유닛의 레벨 총합에 따라 계정 전체 공격력/체력 버프를 자동 적용하는 시스템이다. 별도 포인트 환산 없이 단순 합산으로 구간별 마일스톤 달성 시 버프가 증가하며, 장비 효율 항에 합산(Additive) 방식으로 적용된다. 유닛 목록 화면 상단에 총합 레벨 게이지와 현재 버프 수치를 실시간 표시한다.
> **현재 상태:** 미적용

# **M8\_성장\_유닛도감 (총합 레벨 시너지)**

##

## **1. 개요**

* **목적:** 사용하지 않는 유닛(저등급)의 레벨업 동기 부여 및 재화 소모 유도.
* **방식:** 별도의 포인트 환산 없이 보유한 모든 유닛의 레벨 총합에 따라 즉시 계정 전체 스탯 버프를 적용.

## **2. 시스템 로직**

### **2.1. 기본 규칙**

1. **Total Level (총합 레벨):** 보유 중인 모든 유닛의 현재 레벨을 단순 합산한 값.
2. **구간별 버프:** 총합 레벨이 특정 구간(마일스톤)에 도달할 때마다 공격력/체력 버프 수치가 증가.
3. **적용 방식:** 별도의 '레벨업 버튼' 없이 조건 달성 시 **즉시 자동 적용** (Passive).

### **2.2. 테이블 설계 (UnitLevelSynergyData)**

* 복잡한 공식을 제거하고, 총합 레벨 구간과 버프 수치만 정의합니다.

| **컬럼명** | **설명** | **타입** | **비고** |
| --- | --- | --- | --- |
| req\_total\_level | 필요 총합 레벨 | Int | 100, 200, 300... (구간 설정) |
| buff\_atk\_per | 공격력 증가량(%) | Float | 장비 효율 항목에 합산 |
| buff\_hp\_per | 체력 증가량(%) | Float | 장비 효율 항목에 합산 |

###

### **2.3. 스탯 계산식 (합연산 적용)**

* 도감 버프는 최종 대미지 곱연산이 아닌, 장비/패시브 효율 항에 합산(Additive)되어 적용됩니다.{Final Stat} = {Base Stat} X (1 + {장비옵션}% +{도감버프}%}) X {최종증폭(곱연산 계열)}
* **적용 예시:**
  + 장비 옵션: +50%
  + 유닛 총합 레벨 버프: +10%
  + **결과:** 100(add\_stat) X (1 + 0.5 + 0.1) = 160

## **3. UI/UX 기획**

별도의 팝업이나 메뉴 진입 없이, 유저가 가장 자주 보는 **[유닛 목록]** 화면에서 직관적으로 인지하도록 배치합니다.

###

### **3.1. 유닛 목록 화면 (Unit List)**

* **위치:** 유닛 목록 스크롤 최상단 고정 영역 (Header)
* **표시 정보:**
  1. **현재 총합 레벨:** (예: 현재 총 레벨: 1,450)
  2. **현재 적용 버프:** (예: 계정 버프: 공격력 +14.5% / 체력 +14.5%)
  3. **다음 목표(Progress):** 다음 버프 구간까지 남은 레벨을 게이지 바(Bar)로 표현.

###

### **3.2. 피드백 연출**

* **레벨업 시:** 유닛 레벨업 버튼 터치 시, 상단의 [총합 레벨 게이지]가 실시간으로 차오르는 연출 적용.
* **구간 돌파 시:** 게이지가 가득 차면 상단 헤더가 반짝이는 이펙트와 함께
  + **총합 레벨 효과 강화! (공격력 14% → 15%)** 토스트 메시지 노출.

##

## **4. 데이터 예시 (밸런스 참고용)**

유저에게 끊임없는 목표를 주기 위해 계단식으로 보상을 배치합니다.

| **req\_total\_level** | **buff\_atk\_per** | **buff\_hp\_per** |
| --- | --- | --- |
| 1 | 0.5% | 0.5% |
| 2 | 1.0% | 1.0% |
| 150 | 1.5% | 1.5% |
| 200 | 2.0% | 2.0% |
| ... | ... | ... |

##

## **5. 기대 효과**

1. **직관성:** 모든 유닛을 키우면 무조건 강해진다는 단순한 규칙으로 유저 학습 비용 제거.
2. **접근성:** 유닛 관리 화면에 통합되어 자연스럽게 성장 목표 노출.
3. **개발 효율:** 복잡한 등급별 점수 계산 로직 없이 단순 합산(Sum) 쿼리로 구현 가능.
