# [완료] 6.유닛 필터 정리

* 유닛 정렬 규칙에서 id 순 제거
* 기본 정렬 등급 오름차순으로
* 정렬 : 등급 오름차 / 등급 내림차 / 레벨 오름차 / 레벨 내림차
  + 등급 오름차 : 등급 낮은것부터 > (동일시) 레벨 낮은것부터
  + 등급 내림차 : 등급 높은것부터 > (동일시) 레벨 높은것부터
  + 레벨 오름차 : 레벨 낮은것부터 > (동일시) 등급 낮은것부터
  + 레벨 내림차 : 레벨 높은것부터 > (동일시) 등급 높은것부터

Local\_Btn\_Parts\_Sort\_Grade\_High\_to\_Low 등급 높은 순

Local\_Btn\_Parts\_Sort\_Grade\_Low\_to\_High 등급 낮은 순

Local\_Btn\_Parts\_Sort\_Level\_High\_to\_Low 레벨 높은 순

Local\_Btn\_Parts\_Sort\_Level\_Low\_to\_High 레벨 낮은 순
