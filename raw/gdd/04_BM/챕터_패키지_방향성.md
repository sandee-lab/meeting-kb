# 챕터 클리어 패키지 방향성

> **요약:** 챕터 진행도 연동 패키지의 보상 구조(1000% 효율 기준) 및 컨셉 다양화 방향성 정의
> **현재 상태:** 미적용

# 챕터 클리어 패키지 방향성

> 최종 수정: 2026-04-02

---

## 1. 배경

기존 챕터 클리어 패키지(10개)는 모두 동일한 보상 풀(pool_id=1)을 공유하여, 가격만 1,100~2,000원으로 증가하고 보상은 동일한 구조였다. 후반 챕터일수록 가성비가 하락하여 구매 동기가 약화되는 문제가 있었다.

### 작업 의도

1. **구매 동기 유지**: 후반 챕터에서도 매력적인 보상을 제공하여 구매 전환율 유지
2. **진행도 연동 가치**: 챕터가 올라갈수록 가격과 보상 모두 스케일업하여 성장 체감
3. **컨텐츠 해금 연계**: 장비 해금(Ch4), 고급 장비(Ch14+) 등 시스템 해금 시점에 맞춘 패키지 구성
4. **컨셉 다양화**: 유닛/장비/골고루 컨셉을 순환 배치하여 패키지마다 차별화된 매력 제공
5. **초보자 상품과 분리**: 출발가 3,300원으로 한정 상품(~1,500원)과 가격대 경쟁 회피
6. **일관된 효율 기준**: 전 패키지 1000%(±5%) 효율로 가치 투명성 확보

---

## 2. 설계 원칙

| 원칙 | 설명 |
|------|------|
| 효율 1000% | 과금 가격의 10배 가치 (다이아 환산 기준) |
| 다이아 100% 기본 지급 | 동일 가격의 다이아 직구 수량을 기본 포함 |
| 나머지 900% 아이템 배분 | 유닛 소환권, 장비 열쇠, 골드 티켓 등으로 채움 |
| 각 패키지별 독립 pool_id | 보상 차별화 |
| 출발가 3,300원 | 초보자 패키지 등 한정 상품(~1,500원)과 가격 경쟁 회피 |
| 오차 허용 | 수량 맞추기 어려울 때 1000~1050% 범위 허용 |
| 골드 직접지급 제한 | Ch1~3만 허용, Ch4 이후는 골드 티켓으로 대체 |

## 3. 챕터 배분 (12개)

- **1~5챕터**: 매 챕터 (5개) — 신규 유저 전환 유도
- **8~20챕터**: 3챕터마다 (5개: Ch8, 11, 14, 17, 20) — 정착 유저 대상
- **25~30챕터**: 5챕터마다 (2개: Ch25, 30) — 하드코어 유저 전용

## 4. 가격 구간

| 구간 | 챕터 | 가격 범위 |
|------|------|----------|
| 소과금 | Ch1~8 | 3,300 ~ 8,800원 |
| 중과금 | Ch11~17 | 14,000 ~ 33,000원 |
| 대과금 | Ch20~30 | 44,000 ~ 69,000원 |

## 5. 컨셉 분류

### 초반 (Ch1~5)

| 컨셉 | 챕터 | 구성 | 비고 |
|------|------|------|------|
| 유닛 특화 | Ch1~3 | 소환권 + 행동력 + 골드 | 장비 소환 잠금 상태 |
| 장비 해금 특화 | Ch4~5 | 일반 열쇠 + 강화 스크롤 3종 | 장비 시스템 첫 해금 |

### 중후반 (Ch8~30) 6가지 컨셉 순환

| 컨셉 | 구성 | 배정 |
|------|------|------|
| 골고루A | 유닛 소환권 + 일반 열쇠 + 황금 디스크 | Ch11 |
| 골고루B | 유닛 소환권 + 고급 열쇠 + 황금 디스크 | Ch20 |
| 유닛 중심 | 유닛 소환권 + 골드 티켓 + 전설칩 | Ch14, Ch25 |
| 장비A | 일반 열쇠 + 고급 열쇠 | Ch30 |
| 장비B | 일반 열쇠 + 스크롤 3종 | Ch8 |
| 장비C | 고급 열쇠 + 스크롤 3종 | Ch17 |

순환 패턴: 장비 → 골고루 → 유닛 → 장비 → 골고루 → 유닛 → 장비

### 고급 열쇠 등장 기준

고급 장비 열쇠(id=31, 300dia)는 **Ch17(33,000원) 이상 패키지**부터만 포함.

### 골드 직접지급 제한

골드를 직접 지급하는 것은 **Ch1~3 패키지만 허용**. Ch4 이후에는 골드 티켓(Box 2001~2003)으로 대체.
