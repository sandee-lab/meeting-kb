# 명예의 전당 - 선착순 랭킹

> **요약:** 최종 챕터 보스 클리어 후 엔딩 연출과 선착순 랭킹 시스템을 정의한다. AI 흑막 엔딩(1안)과 슈뢰딩거 엔딩(2안)의 2가지 스토리 분기가 있으며, 각각 1등 유저와 나머지 유저에게 다른 대사와 연출을 제공한다. 1등 유저는 우주 이름 명명 권한을, 나머지 유저는 순위에 따른 차등 보상을 받는 명예의 전당 구조이다.
> **현재 상태:** 미적용

명예의 전당

엔딩

# 1. AI 흑막 엔딩

* 요약 : 완벽한 배송을 위해 행성을 날려버리는 AI의 광기
* 진실 :
  + 최종 보스 : (효율에 미친) AI 매니저
    - 정체 ? :
      * 매일 밀려드는 고객CS로 피로 누적 상태
      * ‘오배송률 1위’의 오명을 씻기 위해 모든 배송지(행성)을 날려버릴 계획을 남몰래 세웠다.
      * 어느 날, 배송 사고(첫 시작)가 터지자 매니저는 자신의 계획을 실행에 옮긴다.
* 전개 :
  + 마지막 챕터의 보스로 등장
  + 등장 시 대사 :
    - 404, 나의 계획에 동참해주어 고마워요. 카마존의 끔찍한 고객 센터에서 오배송의 오명과 고객의 살해 협박을 감당하며 생각했죠. 카마존의 오배송률을 0%로 만드는 유일한 방법은 배송지를 모두 지우는 것이라고!
    - 행성이 사라질 때마다 우주의 복잡한 물리 엔진 연산량이 줄어들어 AI의 처리 속도가 비약적으로 상승했습니다.
    - 404, 당신은 아주 훌륭한 '삭제 도구'였어요. 피날레는 제것입니다.
  + 클리어 후 대사 :
    - 공통
      * AI가 파괴되면서 시스템 메시지 출력
        + [경고: 메인 AI 치명적 오류. 시스템 안정화를 위해 재부팅합니다.]
        + [알림: 재부팅 완료.]

1등 유저는 1-A안

나머지 유저는 1-B안

* + - 1-A
      * 스테이지
        + [알림: 데이터 로딩을 위해 우주 식별 번호를 입력하십시오.]
        + 팝업 출력
        + [알림: 번호 확인 완료.]
        + 로딩 후 메인으로 이동
      * 메인
        + 재부팅된 매니저 등장

오... 당신이 바로 그 '1번째' 로봇이군요.

똑같아 보여도 저는 새 AI입니다. 지난 버전과는 전혀 다른 모델이죠. 앞으로 잘 부탁해요.

아, 츄르는 더 이상 찾지 않아도 돼요. 그건 1번째 로봇이 할 일이 아니니까요. 또 다른 404가 할 겁니다.

사건 해결에 대한 보상이 지급될 테니 우편함을 수시로 확인해보세요. 복지서비스였나? 그게 아니면 사원지원부에서 보내줄 거예요. 클레임은 그쪽으로 하세요.

(v2.01 - 버그 수정 없음)

* + - 1-B
      * 메인
        + 재부팅된 매니저 등장

404, 지난 사건 개요는 확인했어요. 확인 결과, 당신은 지옥 같은 난장판을 {n}번째로 정리했군요. 축하합니다.

당신의 순위만큼의 정밀도로 보상을 지급하죠. 0.000... (잠시 침묵).

정산할 로봇이 너무 많아 시간이 걸릴 것 같아요. 어쨌든 보상을 주긴 줄테니 참고 기다려요.

# 2. 슈뢰딩거 엔딩

* 요약 : 수십 개의 행성을 부수고 얻어낸 츄르 한 봉지
* 전개
  + 공통
    - 마지막 챕터 보스 클리어 후
    - 컷신
      * 마침내 상자를 찾은 404
    - 이후 분기
      * 1등 유저는 2-A안
      * 나머지 유저는 2-B안
  + 2-A
    - 대화
      * 매니저
        + (깜짝 놀라며) 세상에, 진짜로 츄르를 가져왔다고요? 빅 보스께서 기분이 좋으신지 이 우주의 이름을 당신 마음대로 하라십니다.
        + 자, 선택하세요. 이 은하계를 '자네는 해고야' 은하로 부를 건가요, 아니면 본인 이름으로 하실 건가요? 뭐로 바꾸든 세금 고지서는 거기로 발송될 겁니다.
      * 서버명 입력 팝업 출력 (예 : 아무개의 위대한 제국)
        + 반전

입력하면 인증서 출력. 그런데 이름이 묘하게 바뀌어 있다.

[우주 101 - 공식 등록 완료]

등록 명칭: [닉네임]의 우주 101

[시스템 알림] 입력한 것과 이름이 다르다고요? 그야 슈뢰딩거니까요. 이름의 입자가 불안정하게 충돌하여 '가장 하찮은 확률'로 고정되었습니다. 적어도 당신 이름은 들어갔으니 이 정도로 만족하세요. 아, 반품은 안 됩니다.

* + 2-B
    - 대화
      * 매니저
        + (하품하며) 아, 왔어요? 당신은 [n번째]로 도착했네요. 어쩌죠? 빅 보스가 "트렌드가 지났으니 폐기하라"시네요. 두바이 츄르가 대세라던가, 뭐… 그래도 종전이 선언됐으니 다행이죠. 다시 업무로 복귀하세요. 뭘 할지 모르겠다면 지시를 기다려요.
        + 아, 그리고 정산할 게 있어요. 행성 50개 파괴한 비용에 유류비에 주말통신비에… 회사에 300억 미네랄 정도 빚진 셈이네요.
        + 그치만 무슨 무슨 명목으로 추가 지급된 게 있어서… 오히려 받아갈 게 있네요? 운이 좋네요. 이건 우편함으로 보낼게요.
