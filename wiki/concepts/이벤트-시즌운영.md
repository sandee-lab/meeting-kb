# 이벤트 및 시즌 운영

> 출처: [[raw/gdd/03_Content/이벤트_고도화_재화경쟁_기획서]], [[raw/gdd/03_Content/랜덤_플레이미션]], [[raw/gdd/01_Core/게임설계_방향성]], [[raw/meetings/[1208] 이벤트 및 BM]]

## 개요

유저 참여를 유도하고 매출을 강화하기 위한 이벤트 및 시즌 운영 시스템이다. 이벤트 고도화를 통해 계층적 이벤트 구조(상시/주기/일회성)를 갖추고, 재화 경쟁 이벤트로 소과금 유저의 과금 동기를 자극한다. 랜덤 플레이 미션은 반복 플레이에 변화를 주면서 광고 수익을 확보한다.

## 상세 내용

### 이벤트 고도화: 이벤트 허브 시스템

> 출처: [[raw/gdd/03_Content/이벤트_고도화_재화경쟁_기획서]] — 상태: 미적용

**신규 데이터 테이블:**

| 테이블 | 역할 |
|--------|------|
| EventHubData | 이벤트 허브(홈) 정의. hub_id, hub_type 등 |
| EventHubEntryData | 허브 내 개별 이벤트 슬롯. event_id, hub_id, role_type, sequence_group, event_series_id |

**role_type 종류:**

| role_type | 역할 |
|-----------|------|
| ResourceRankPreview | 사전 집계 + 예고 (집계 주체) |
| ResourceRank | 본 경쟁 랭킹 (집계 주체) |
| ResourceRankExchange | 교환소 (참조 전용, 집계 없음) |
| ResourceRankQuest | 마일스톤/미션 보상 (참조 전용) |
| ResourceRankShop | 이벤트 연계 한정 상점 (판매/접근 중심) |

**event_series_id:** 프리뷰~랭킹~교환 3단계를 하나의 시리즈로 묶는 공통 식별자. 예시: 장비 소환 이벤트 `event_series_id=5001`, `hub_id=2001`

**series_type:**
- OneShot: 1회성
- Repeatable: 주기 반복 (주간/월간)
- Seasonal: 시즌 단위 반복

### 재화 소모 경쟁 이벤트 (ResourceRank)

> 출처: [[raw/gdd/03_Content/이벤트_고도화_재화경쟁_기획서]] — 상태: 미적용

**이벤트 3단계 시리즈 구조 (예시: 28일 주기):**

| 단계 | role_type | 기간 | 역할 |
|------|-----------|------|------|
| 1단계 | ResourceRankPreview | Day 3~7 (5일) | 사전 집계 + 예고. 집계 값이 본 이벤트로 이관됨 |
| 2단계 | ResourceRank | Day 7~21 (14일) | 채널 단위 랭킹 경쟁 본편 |
| 3단계 | ResourceRankExchange | Day 21~28 (7일) | 랭킹 결과 기반 보상 수령 및 교환소 운영 |

**집계 이관 규칙:**
- 프리뷰 기간 누적 카운트는 초기화 없이 ResourceRank 이벤트로 그대로 이관
- ResourceRank 종료 후 집계 데이터는 ResourceRankExchange에서 읽기 전용으로 활용
- ResourceRankExchange 종료 시 하위 이벤트 자동 종료 처리

**집계 조건 타입 (ResourceRankSeriesData.condition_type):**

| 조건 타입 | 설명 |
|-----------|------|
| Event_Total_Currency_Expense | 이벤트 소모 - 다이아 누적 소모 |
| Event_Total_Summon_Unit | 이벤트 소모 - 유닛 누적 소환 횟수 |
| Event_Total_Summon_Parts | 이벤트 소모 - 장비 누적 소환 횟수 |

※ 장비 강화/승급 재료 재화는 집계 제외 (시스템상 강화/약화 재화 필터 불가)

**보상 구조:**
- **랭킹 보상**: 채널 단위 순위별 지급, **랭킹 코인** (영구 재화, 이벤트 종료 후 소멸하지 않음)
- **마일스톤 보상**: 누적 집계 구간 달성 시 확정 지급 (ResourceRankQuest 연계)
- **교환소**: ResourceRankExchange 기간 중 미수령 보상 + 랭킹 코인 소비 교환 상점 운영

**레드닷 발동 조건:**
- 교환 가능 상품 존재 시: 이벤트 센터 진입 아이콘, 교환 상점 아이콘
- 수령 가능 마일스톤 보상 존재 시: 이벤트 센터 진입 아이콘, 해당 이벤트 슬롯, 퀘스트 탭

**UI:**
- 장비/유닛/다이아 소모 3종 이벤트 홈은 전용 진입 UI `EventCenterList` 에서 통합 조회
- 교환 상점은 이벤트 종료 후에도 진입 아이콘 유지

### 랜덤 플레이 미션 (전술 실험)

> 출처: [[raw/gdd/03_Content/랜덤_플레이미션]] — 상태: 구현됨

**컨셉:** "카마존 연구소"의 전술 데이터 수집 실험. 메인 태스크(필수 유닛 배치)와 추천 태스크(덱 가이드)로 구성.

**운영 규칙:**
- 1일 최대 3회 실험 제공
- 각 실험당 광고 시청으로 리롤 1회 가능 (리롤 후 버튼 비활성화)
- 서버 초기화: 매일 오전 9시 KST 전체 초기화
- 보상 미수령 시 초기화 시간에 소멸 (우편 미발송)

**성공 판정 기준 (모두 만족 필요):**
1. 메인 태스크 만족 상태로 1웨이브 이상 생존
2. 해당 챕터 정상 종료 (승패 무관, 중도 포기 시 실패)

**보상:** 아웃게임 전용 UI에서 수령 (인게임 보상 없음 — 밸런스 변동 변수 차단)

**태스크 선정 로직 (RandomMissionTaskGroupData 기반):**

1단계: 후보군 설정 — 챕터 클리어 조건 만족한 모든 그룹 리스트

2단계: 필터링 (우선순위 순 제외):
- 1순위 제외: 오늘 보상 수령한 그룹 ID
- 2순위 제외: 현재 화면 표시 중인 그룹 ID (리롤 시 적용)
- 3순위 제외: 오늘 등장했으나 리롤/포기로 거절된 그룹 ID

3단계: 남은 후보 없을 시 역순으로 필터 해제 (3순위 → 2순위 → 1순위)

※ 매끄러운 운영을 위해 기본 그룹은 최소 3개 이상 유지 약속 필요

**확률 보정 시스템 (회차별 감쇄):**

| 구분 | 1회차 | 2회차 | 3회차 |
|------|-------|-------|-------|
| 유닛 교체 가중치 (`Random_Mission_Unit_Change_Weight`) | 0.5 | 0.3 | 0.2 |
| 택틱 spawn_weight 배율 (`Random_Mission_Tactic_Weight_Multiplier`) | 2.0배 | 1.5배 | 1.0배 |

- **유닛 보정 적용 시점**: 진행 중인 실험이 있고 메인 태스크 미완료 상태일 때
- **유닛 보정 해제**: 포메이션+인벤토리에 필수 태스크 유닛 수량 충족 시 (각 유닛별 1회 보정 카운트 후 해제 — 어뷰징 방지)
- **예외**: 전설 등급 유닛은 보정 제외
- **택틱 보정 적용**: 추천 태스크 택틱 ID의 spawn_weight 일시 상향 후 기존 방식대로 등급→택틱 결정

### GDD 기획서 — 무한 모드: 시공간의 균열 (2026년 3월, 미적용)

> 출처: [[raw/gdd/01_Core/무한_모드_컨셉_시공간의_균열]]

**컨텐츠 방향성 4원칙:**
1. End-Game Showroom: 스테이지 클리어 후 저장된 덱 스냅샷으로 극한 검증
2. Time Compression: 5분 타임어택 내 압도적 화력으로 섬멸
3. Volatile Overclock: 유물/택틱으로 일시적 한계 초월 스펙 완성 (휘발성)
4. Ranking Competition: 보상 획득 제한 + 도전 무제한으로 코어 유저 상위 랭킹 유도

**입장 규칙:**
- 입장 재화: 없음 (무제한 입장)
- 덱 로드: 스테이지 클리어 시 저장된 세이브 데이터 선택 (입장 후 덱 변경 불가)
- 심도(Depth): 전투력 구간 맞춤 난이도, 심도 높을수록 점수 배율 기하급수적 증가
- 공허의 계약(Contracts): 추가 제약 조건 자발 선택 → 점수 보너스(+%)

**보상 규칙:**
- 일일 보상: 매일 기본 1회, 광고/멤버십으로 추가 충전
- 시즌 업적 보상: 최고 기록이 특정 구간(10만/50만…) 도달 시 성장 재화 지급 (큐브/다크 매터)
- 랭킹 정산 보상: 매주 일요일 시즌 종료 시 등급별 차등 지급

**시즌 등급 시스템 (실시간 백분위 기반):**

| 등급 | 달성 조건 | 보상 성격 |
|------|---------|---------|
| 챌린저 | Top 3명(절대수치) | 명예: 유니크 테두리/칭호 |
| 마스터 | 0.1%~1% | 명예+성장 |
| 다이아 | 1.1%~10% | 핵심 성장(큐브/다크 매터 대량) |
| 플래티넘 | 10.1%~30% | 일반 성장 |
| 골드/실버 | 30.1%~100% | 참가상 |

**시즌 규칙:**
- 운영 주기: 1주(매주 월요일 09:00 시작)
- 초기화: 랭킹 기록, 업적 보상, 입장 기록
- 보존: 최고 기록은 명예의 전당/내 기록실에 영구 저장

**시즌 랜덤 효과 (매 시즌 전장 변화):**
- 속성 테마: 매주 1속성 선정 → 해당 속성 공격력 +50% 또는 쿨타임 -30%, 역상성 -30%
- 몬스터 변이: 시즌 테마에 맞춰 변이
- 로테이션: 데이터 기반 특정 속성 육성 장려 방향으로 배정

**웜홀 상점 (인게임, 보스 처치 시 발동):**
- 전용 재화: 균열 파편 (1회성 휘발성)
- 판매 품목: 유물(랜덤) / 택틱(랜덤) / 소모품(회복/버프)
- 리롤(비용 점증) / 제거(횟수 제한, 전략적 덱 압축)

**점수 산정 공식:**
```
최종 점수 = 몬스터 처치 점수 × 심도 배율 × (1 + 계약 보너스)
몬스터 처치 점수 = 몬스터 기본 점수 × (1 + 웨이브 가중치)
```
