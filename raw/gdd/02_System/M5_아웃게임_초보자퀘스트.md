# M5 아웃게임 초보자 퀘스트

> **요약:** 초반 단기 목표 제시 및 구매 유도를 위한 초보자 퀘스트+전용 상점(세트) 시스템.
> **현재 상태:** 미적용

M5\_아웃게임\_초보자퀘스트

**M5\_아웃게임\_초보자퀘스트**

# 목표

* 전용 퀘스트를 통해서 초반 단기 목표 제시
* 기간 한정 패키지 상품을 배치하여 구매 유도 강화

# 개요

* 퀘스트와 별도로 동작하는 초보자퀘스트 시스템을 구성한다.
* 초보자퀘스트는 정해진 기간 동안 진행할 수 있다.
* 각 일차별 퀘스트를 완료하면 할당된 보상을 지급한다.
* 각 일차별 기간 한정 패키지 상품이 배치된다.
* 유저는 퀘스트 보상 또는 상품 구성품으로 [토큰](가칭, 각 초보자퀘스트 전용 아이템)을 획득할 수 있다.
* 유저는 보유한 토큰 수량에 따라 추가 보상을 획득할 수 있다.
  + 추가 보상에 고가치 아이템을 배치하여 퀘스트 및 패키지 상품 구매 유도
* 초보자퀘스트는 누적 접속일에 따라 추가 운영 가능하다. (초보자퀘스트1, 초보자퀘스트2 식)

# 구성

## 세트

* + 초보자퀘스트는 퀘스트와 전용 상점 으로 나뉜다.
  + 한 벌이 되는 퀘스트와 전용 상점을 [세트]로 칭한다.

## 퀘스트

* + 개요
    - 달성 / 카운팅 조건으로 이루어진다. (기존 퀘스트에서 사용하는 것과 동일)
    - 콘텐츠 해금 이후 각 퀘스트의 진행 상황을 체크한다.
    - 각 퀘스트 조건을 완료한 경우 퀘스트 보상을 수령할 수 있다.

## 그룹

* + - 각 퀘스트는 하나의 그룹에 소속된다.
    - 그룹 잠금 및 해제
      * 그룹은 일자를 기준으로 하며, 게임 진입 후 해당 일자가 되어야 그룹이 해제된다.
      * 그룹이 해제되기 전에도 각 퀘스트는 동작한다.
        + 챕터 클리어 조건의 경우 해당 일자 이전에 달성할 수 있음
        + 단, 해당 일자가 오기 전까지 그룹은 잠겨진 상태로 표시한다.
    - 그룹 클리어
      * 그룹에 포함된 퀘스트들의 완료 보상을 모두 수령하면, 그룹 클리어 상태가 된다.

## 상점

* + 개요
    - 초보자퀘스트 UI에서 일자별로 해제되는 상품 목록을 가리킨다.
  + 구매
    - 일반 패키지 상품과 달리 무료 판매하거나, 복수 횟수 구매를 허용할 수 있다.

## 진행도 보상

## 각 세트에는 전용 재화가 설정된다.

* + 퀘스트 완수 및 상품 구매 시 전용 재화가 지급된다.

## 각 세트에 설정된 전용 재화 보유량이 일정치 만족한 경우, 진행도 보상을 수령할 수 있다.

# 동작

## 콘텐츠 해금

* + Contents\_Type = Beginner\_Event

## 진입

* + 최초
    - 콘텐츠 해금 시 강제 팝업
  + 기본
    - 메인 화면 좌 또는 우측에 바로가기 아이콘 배치
    - 아이콘 선택 시 팝업 출력

## 세트 결정

* + 조건 만족한 세트를 열어준다.
    - BeginnerEventSetData@unlock\_condition\_type / value1 / value2

## 기간 설정

* + 해금 시점으로부터 정해진 기간동안 진행한다.
    - BeginnerEventSetData@period\_day

## 그룹 해제

* + 해금 시점 당일 1번 그룹 해제
  + 유저의 접속 유무와 관계 없이, 해당 일차가 도래하면 그룹을 해제한다.

## 퀘스트

* + 상태
    - 진행 중
      * 기준
        + 달성 조건 : 달성 전 상태
        + 카운팅 조건 : 목표 횟수 미달 상태
      * 이동
        + 각 조건별로 ‘이동’ 버튼 배치, 선택 시 관련 UI로 이동

BeginnerEventQuestData@open\_ui\_name

* + - 보상 수령 가능
      * 기준
        + 달성 / 카운팅 조건 : 만족
      * 수령
        + 버튼 선택 시, 할당된 보상 수령 처리
        + 보상 수령 가능한 미션은 목록 최 상단으로 이동시킨다.
    - 보상 수령 완료
      * 기준
        + 조건에 할당된 보상 수령 완료
      * 완료 처리
        + UI에서 표현 차등
        + 목록 최 하단으로 이동시킨다.

## 상품

* + 상태
    - 구매 가능
      * 기준 : ShopBeginnerEventPackageData@purchase\_limit 잔여
      * 구매 버튼 배치
      * 버튼 선택 시 구매 팝업 출력
    - 구매 완료
      * 기준 : ShopBeginnerEventPackageData@purchase\_limit 소진
      * 구매 완료 상품은 UI에서 표현 차등
        + 목록 최하단으로 이동시킨다.

## 진행도 보상

* + 상태
    - 진행 중
      * 기준 : 보유 재화량 조건 불만족
      * 표현 : 닫힌 상자 아이콘
        + 선택 시 툴팁 출력 (보상 안내)
    - 보상 수령 가능
      * 기준 : 보유 재화량 조건 만족
      * 표현 : 반짝이는 상자 아이콘 (또는 다른 표현)
        + 선택 시 보상 획득 처리

공용 보상 팝업

* + - 보상 수령 완료
      * 기준 : 보상 수령 완료
      * 표현 : 열린 상자 아이콘
        + 선택 시 툴팁 출력 (보상 안내)

## 종료

* + 기본적으로 설정된 기간 만료 시 종료 처리
  + 조기 종료
    - 기간 이내 모든 퀘스트 보상 획득, 진행도 보상 획득, 모든 상품 구매한 경우
      * 팝업 종료 시 진입 아이콘 제거

# 레드닷

* 퀘스트 보상 수령 가능 시
  + 바로 가기 아이콘
  + 해당 세트 버튼
  + 퀘스트 탭 버튼
  + 해당 그룹(일자) 버튼
  + 해당 퀘스트 보상 받기 버튼
* 진행도 보상 수령 가능 시
  + 바로 가기 아이콘
  + 해당 세트 버튼
  + 해당 진행도 보상
    - 레드닷 대신 다른 표현도 가능
* 무료 상품 구매 가능 시
  + 바로 가기 아이콘
  + 해당 세트 버튼
  + 상점 탭 버튼
  + 해당 그룹(일자) 버튼
  + 구매 버튼
