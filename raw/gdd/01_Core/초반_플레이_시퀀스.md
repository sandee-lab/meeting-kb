# 초반 플레이 시퀀스 설계

## 툴팁

* + 동작
    - 아웃게임에서 보상 슬롯 터치 시 툴팁 출력
  + 내용
    - 이름
    - 설명
  + 대상
    - ~RewardSlot
      * 타입 Consumable
        + 이름 : ConsumableData@local\_name
        + 설명 : ConsumableData@local\_desc
      * 타입 Box
        + 이름 : BoxData@local\_name
        + 설명 : BoxData@local\_desc
      * 타입 Parts
        + 장비 팝업 (현안 유지)
      * 타입 Unit
        + (논의 필요) 해당 유닛 상세 보기 UI
