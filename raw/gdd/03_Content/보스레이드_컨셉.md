# 보스 레이드 — 균열의 심장 컨텐츠 기획 초안

> **요약:** 제한 시간 60초 내 최대 DPS를 경쟁하는 주간 보스 레이드 컨텐츠의 목표, 방향성, 성장/조작 비중 설계 초안. 아웃게임 성장 70% + 인게임 조작 30% 비중으로 설계하며, 일일 구간 보상/영구 달성 보상/주간 랭킹 보상 3레이어 보상 구조와 브론즈~챌린저 6단계 티어 시스템을 포함한다.
> **현재 상태:** 미적용

**보스 레이드**

**균열의 심장**

콘텐츠 기획 초안

v0.1

# **1. 콘텐츠 목표**

| **정해진 시간 안에 최대 DPS를 경쟁하는 주간 기록 경쟁 콘텐츠** |
| --- |

| **목표** | **상세** |
| --- | --- |
| **성장 검증** | 아웃게임에서 쌓아온 전투력(DPS)을 수치로 즉각 확인하는 창구 제공 |
| **접속 루틴 형성** | 일일 구간 보상을 통해 매일 한 번 이상 자연스러운 참여 유도 |
| **장기 성장 동기** | 영구 달성 보상과 챌린저 랭킹 경쟁으로 코어 유저의 지속 성장 욕구 자극 |
| **콘텐츠 진입 허들 최소화** | 무한 모드 대비 단순한 구조로 신규 유저의 첫 전투력 체감 경험을 담당 |

# **2. 의도 및 방향성**

## ● **2.1. 아웃게임 성장 중심 설계**

인게임 조작 비중은 전체 점수의 30% 이하로 제한합니다. 유저가 인게임에서 느끼는 성과는 손 빠르기가 아닌 덱 성장의 결과여야 합니다.

| **구분** | **비중** | **내용** |
| --- | --- | --- |
| **아웃게임 성장** | **70%** | 유닛, 배틀슈트 등 누적 전투력 |
| **인게임 조작** | **30% 이하** | 패턴 회피, 버프 오브젝트 선택 |

## ● **2.2. 콘텐츠 포지셔닝**

보스 레이드는 무한 모드(공허의 균열)와 함께 운영되는 별도 경쟁 콘텐츠입니다. 두 콘텐츠는 서로 다른 역할을 담당합니다.

| **구분** | **보스 레이드 (균열의 심장)** | **무한 모드 (공허의 균열)** |
| --- | --- | --- |
| **핵심 질문** | **얼마나 강하게 때리는가** | **얼마나 오래 버티는가** |
| **복잡도** | 낮음 — 보스만 공격 | 높음 — 이동, 거점, 경제 관리 |
| **성장 체감** | 즉각적 — 점수 숫자로 직결 | 간접적 — 생존 시간으로 확인 |
| **해금 순서** | 먼저 해금 (낮은 챕터 조건) | 이후 해금 (챕터 15 클리어) |

**기획 의도** 보스 레이드에서 내 전투력을 확인한 유저가 무한 모드에서 그 전투력을 전략적으로 활용하는 흐름을 만듭니다.

## ● **2.3. 배틀슈트 던전과의 차별화**

배틀슈트 던전(9챕터 언락)과 보스 레이드는 모두 보스를 상대하는 형태이지만, 디자인 컨셉과 분위기를 명확히 다르게 구성하여 유저가 다른 경험으로 인식하도록 합니다.

| **구분** | **배틀슈트 던전** | **보스 레이드** |
| --- | --- | --- |
| **분위기** | 어둡고 폐쇄적인 파밍 던전 | 웅장하고 개방적인 경기장 |
| **목표** | 보스 처치 (클리어) | DPS 측정 (기록 경쟁) |
| **HUD** | 보스 HP바 중심 | DPS 미터, 점수 카운터 전면 노출 |
| **결과 화면** | 획득 재화 중심 | 점수 및 랭킹 중심 |
| **핵심 감정** | 해치운다 | 얼마나 강한지 보여준다 |

# **3. 콘텐츠 개요**

| **항목** | **내용** |
| --- | --- |
| **콘텐츠명** | 균열의 심장 (보스 레이드) |
| **전투 시간** | 고정 60초 |
| **보스 체력** | 사실상 무한 — HP 고갈 불가 (점수는 누적 피해량으로 산출) |
| **점수 산출** | 60초간 보스에게 가한 누적 피해량 |
| **입장 횟수** | 무제한 (별도 티켓 및 재화 소모 없음) |
| **기록 반영** | 시즌 내 단일 최고 기록 1개만 랭킹 반영 |
| **시즌 주기** | 매주 월요일 정산 (무한 모드와 동일 주기) |

# **4. 아웃게임 동작 구조**

## ● **4.1. 콘텐츠 언락**

| **항목** | **내용** |
| --- | --- |
| **언락 조건** | 배틀슈트 던전(9챕터 클리어) 이후 별도 챕터 조건 달성 시 해금 (구체적 챕터 수치 추후 결정) |
| **해금 순서 의도** | 배틀슈트 던전에서 보스전에 익숙해진 유저가 보스 레이드로 자연스럽게 진입하도록 유도 |
| **성장 던전 순서** | 유닛 소환권 던전(3챕터) → 골드 던전(6챕터) → 배틀슈트 던전(9챕터) → 보스 레이드 |

보스 레이드는 직관적이고 단순한 구조(보스에게 60초간 최대 DPS) 덕분에 게임 이해도가 높지 않은 시점에도 즉각적인 전투력 체감이 가능합니다.

반면 공허의 균열은 전략 오브젝트, 재화 순환, 단계 구조 등 복합적인 인게임 요소로 구성되어 있어 게임 루프에 충분히 익숙해진 이후에 투입해야 콘텐츠의 깊이가 온전히 전달됩니다.

현재 플레이 흐름 분석 결과, 도전 콘텐츠의 투입 시점을 당초 계획보다 앞당길 필요가 있다고 판단했다. 이에 따라 보스 레이드를 먼저 투입하고, 공허의 균열은 유저가 기본 루프에 충분히 익숙해진 5~7일차 이후로 시점을 조정한다.

| **순서** | **콘텐츠** | **투입 시점** | **역할** |
| --- | --- | --- | --- |
| **1순위** | **보스 레이드** | 초반 | 학습 부담 없이 즉각적인 전투력 체감 제공 |
| **2순위** | **공허의 균열** | 5~7일차 이후 | 게임 숙지 후 전략적 깊이를 온전히 체감 |

**기획 의도** 보스 레이드의 가치는 ‘단순하지만 즉각적’이라는 데 있다. 유저가 기본 루프를 체득하기 전에도 ‘내가 얼마나 강한가’를 숫자로 확인할 수 있는 콘텐츠가 먼저 필요하다. 보스 레이드로 성장 동기를 만든 뒤, 전략적 깊이가 있는 공허의 균열로 자연스럽게 이어지는 흐름을 설계한다.

## ● **4.2. 입장 규칙**

| **항목** | **내용** |
| --- | --- |
| **입장 횟수** | 무제한 — 티켓 및 재화 소모 없음 |
| **입장 제한** | 시즌 종료 30분 전 입장 제한 (정산 준비) |
| **덱 구성** | 기존 덱 구성 그대로 참여 (별도 덱 제한 없음) |

**기획 의도** 입장에 재화 비용을 붙이면 상위 유저의 반복 도전 피로도가 급증합니다. 기록 경쟁 콘텐츠는 도전 자체의 허들을 없애고 결과(점수)로만 경쟁하는 구조가 핵심입니다.

## ● **4.3. 보상 규칙**

보상은 3개의 독립 레이어로 구성됩니다. 각 레이어는 서로 다른 유저군과 동기를 타겟합니다.

| **레이어** | **보상명** | **핵심 동기** | **지급 시점** | **초기화** |
| --- | --- | --- | --- | --- |
| **1** | 일일 구간 보상 | 오늘 접속 및 재도전 유도 | 매 판 종료 즉시 | 매일 |
| **2** | 영구 달성 보상 | 한계 도전 동기 | 구간 첫 돌파 즉시 | 없음 (영구) |
| **3** | 주간 랭킹 보상 | 경쟁 및 명예 동기 | 매주 월요일 정산 | 매 시즌 |

**레이어 1 — 일일 구간 보상**

당일 최고 기록이 도달한 구간까지 보상을 수령합니다. 이미 수령한 구간은 당일 재획득 불가이며 매일 초기화됩니다.

| **구간** | **점수 허들 (예시)** | **지급 보상 (예시)** | **비고** |
| --- | --- | --- | --- |
| 구간 1 | 1,000점 이상 | 소모성 재화 | 신규 유저 당일 달성 가능 |
| 구간 2 | 3,000점 이상 | 소모성 재화 | 성장 중인 유저 주요 구간 |
| 구간 3 | 5,000점 이상 | 소모성 재화 + 균열의 정수 | 골드 티어 기준점 |
| 구간 4 | 10,000점 이상 | 균열의 정수 + 큐브 | 코어 유저 도달 구간 |
| 구간 5 | 30,000점 이상 | 큐브 + 다이아 | 플래티넘 이상 유저 전용 |

**동작 예시** 첫 판 3,500점 달성 → 구간 1~3 보상 즉시 수령 / 재도전 후 11,000점 → 구간 4 보상 추가 수령 / 다음 날 초기화 → 구간 1부터 다시 수령 가능

**레이어 2 — 영구 달성 보상**

각 티어 점수 구간을 생애 최초로 돌파했을 때 단 1회 지급합니다. 이후 동일 구간 재돌파 시 미지급이며 초기화되지 않습니다.

| **달성 티어** | **영구 보상 (예시)** | **성격** |
| --- | --- | --- |
| 골드 첫 진입 | 다이아 30 | 성장 중인 유저 첫 성취감 제공 |
| 플래티넘 첫 진입 | 다이아 80 + 전용 칭호 | 상위권 도달 기념 명예 보상 |
| 챌린저 첫 진입 | 다이아 200 + 한정 테두리 | 최상위 달성 기념 고가치 명예 보상 |

**레이어 3 — 주간 랭킹 보상 (챌린저 전용)**

챌린저 티어 진입 유저만 참여하는 순위 경쟁입니다. 브론즈~플래티넘은 랭킹 순위 경쟁 없이 티어 달성 자체가 목표입니다.

| **챌린저 내 순위** | **추가 보상 (예시)** | **비고** |
| --- | --- | --- |
| **1위** | 전용 칭호 + 다이아 대량 | 최상위 명예 보상 |
| 2~3위 | 다이아 |  |
| 4~10위 | 균열의 정수 |  |
| 11위 이하 | 소량 소모성 재화 | 점수 1점 이상 기록 시 수령 가능 |

## ● **4.4. 랭킹 규칙**

| **항목** | **내용** |
| --- | --- |
| **기록 방식** | 시즌 내 단일 최고 기록 1개만 랭킹 반영 |
| **정산 주기** | 매주 월요일 정산 |
| **경쟁 범위** | 챌린저 티어 진입 유저 내 순위 경쟁 (브론즈~플래티넘은 순위 경쟁 없음) |
| **동점자 처리** | 동일 점수 시 먼저 기록한 유저가 상위 순위 점유 (Timestamp 기준) |
| **시즌 종료** | 종료 30분 전 입장 제한 및 정산 중 안내 출력 |

**티어 구조 및 달성 기준**

| **티어** | **진입 기준** | **목표 달성 시점** | **특이사항** |
| --- | --- | --- | --- |
| 브론즈 | 절대 점수 | 1일차 (당일) | 첫 플레이 즉시 달성 가능 |
| 실버 | 절대 점수 | 1일차 | 브론즈~실버 동일 패턴 구간 |
| 골드 | 절대 점수 | 2~3일차 | 1차 패턴 추가 구간 |
| 다이아 | 절대 점수 | 7일차 | 첫 시즌 성실 참여 유저 |
| 플래티넘 | 절대 점수 | 30일차 | 2차 패턴 추가 구간 (덱 지식 요구) |
| 챌린저 | 절대 점수 | 상위권 성장 유저 | 유일한 순위 경쟁 구간 |

# **5. 인게임 동작 구조**

## ● **5.1. 기본 승패 규칙**

| **항목** | **내용** |
| --- | --- |
| **전투 시간** | 고정 60초. 타이머 종료 시 자동 종료 |
| **보스 상태** | HP 고갈 불가 — 죽지 않는 설정 (섹션 6 참조) |
| **승리 조건** | 절대적 승리 없음 — 60초간 누적 피해량(점수)이 곧 기록 |
| **종료 트리거** | 60초 타이머 만료 |
| **재도전** | 즉시 재입장 가능 (무제한) |

## ● **5.2. 보스 패턴 구조**

패턴은 점수 구간(티어) 달성에 따라 단계적으로 추가됩니다. 초반 유저는 단순 회피만 요구하고, 성장할수록 덱 지식이 필요한 패턴이 추가됩니다.

**패턴 설계 원칙**

* 부대는 단일 컨트롤 단위 — 개별 유닛 지정 이동 기믹 사용 불가
* 회피 실패 시 페널티 구조 유지 (회피 성공 보너스가 아닌 실패 페널티)
* 패턴 요구 난이도가 점수 차를 만들지 않도록 설계 (조작 비중 30% 원칙)
* 특정 속성/포지션에 유리한 패턴 금지 — 덱 고착화 방지

**구간별 패턴 활성화**

| **티어** | **활성 패턴** | **요구 능력 및 비고** |
| --- | --- | --- |
| 브론즈 | 회피 패턴 N개 | 범위 보고 이탈 — 수량은 테스트 필요 |
| 실버 | 브론즈 동일 | 단순 회피 구간 유지 |
| **골드** | **기존 + 패턴 추가** | **1차 허들 — 방향 판단 및 이탈 요구 / 실패 시 DPS 페널티** |
| 다이아 | 골드 동일 | 적응 구간 — 새 패턴 없이 숙련도 안정화 |
| **플래티넘** | **기존 + 속성 버프 오브젝트** | **2차 허들 — 덱 조합 이해 요구** |
| 챌린저 | 모든 패턴 활성 | 순수 DPS + 지식 경쟁 |

**확정 패턴 종류**

| **패턴명** | **성격** | **요구 액션** | **실패 페널티 / 성공 보너스** |
| --- | --- | --- | --- |
| 광역 폭발 | 페널티 회피 | 특정 구역 밖으로 부대 전체 이동 | 실패 시 5초간 공격력 50% 감소 |
| 돌진 충격파 | 페널티 회피 | 보스 전방 직선 범위 부대 이탈 | 실패 시 부대 전체 3초 이동 불가 |
| **속성 버프 오브젝트** | **보너스 획득** | **덱 속성에 맞는 오브젝트로 이동 선택** | **성공 시 해당 속성 DPS 30초간 증폭** |

**속성 버프 오브젝트 설계 조건** 오브젝트는 여유 있게 출현하여 반응 속도가 아닌 선택의 문제로 유지. 단일 선택(중복 획득 불가)으로 덱 조합 의사결정 유도. 플래티넘 티어 이상에서만 활성화.

**패턴 추가 자체가 성장 지표** 유저가 새 패턴을 처음 마주쳤을 때 '내가 그 구간까지 왔구나'를 체감합니다. 패턴 추가는 허들인 동시에 성장 피드백 역할을 합니다.

## ● **5.4. 밸런스 적용 규칙**

난이도 단계가 없는 단일 경쟁 구조 특성상, 보스 스킬의 고정 피해량은 충분히 성장한 유저에게 긴장감을 제공하지 못합니다.

이를 대응하기 위해 보스 스킬 피해량에 유저 최대 체력 비례 피해를 추가 적용하여, 성장 수준에 무관하게 적절한 긴장감을 유지합니다.

비례 수치(N%)는 스킬 제작 단계에서 제어하며 별도 구현이 필요하지 않습니다.

**티어 갱신 및 아웃게임 반영**

| **상황** | **처리 방식** |
| --- | --- |
| 전투 중 티어 갱신 | 누적 DPS가 티어 기준점 도달 시 즉시 갱신 + 보스 광폭화 연출 출력 |
| 아웃게임 랭킹 티어 반영 | 전투 종료 후 최종 점수 기준으로 반영 (전투 중 갱신된 티어와 별개) |
| 시즌 초기화 후 | 매 시즌 시작 시 티어 초기화 → 전투 시작 시 브론즈 기준으로 시작 |
