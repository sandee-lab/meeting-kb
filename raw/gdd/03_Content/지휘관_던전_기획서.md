# 지휘관 던전

> **요약:** 지휘관 던전의 보스전 기획서로, 맵 상단에 고정된 보스 몬스터가 스킬 패턴(기본 공격, 구슬 소환/흡수, 방사, 광폭화, 장판)으로 동작한다. 맵 기믹으로 속성 구슬 오브젝트(불꽃/냉기/전기/카오스/무속성)를 배치하며, 유저와 보스가 경쟁적으로 획득하는 시스템이다. 단일 타격, 다단히트, 보스 추가 피해에 유리한 전기 속성 시너지 조합이 권장된다.
> **현재 상태:** 미적용

# 플레이 개요

* 보스 몬스터는 맵의 상단 가장자리에 고정된 위치에 존재
  + 고정형 몬스터
* 스킬 시스템을 기반으로 패턴 동작

| 타입 | 구분 | 설명 |
| --- | --- | --- |
| 액션 | 기본 | Attack   * 주먹에 해당하는 부위를 유저 방향으로 발사체로 발사한다. |
| 액션(> 패시브) | 구슬 소환 | 오브젝트를 생성한다.  구슬을 소환하는 모션 |
| 액션 | 구슬 흡수 | 오브젝트를 흡수한다.  해치가 열리고 1-2초 동안 흡수하는 느낌 |
| 액션 | 방사 | Attack\_2  좌-우 방향으로 발사체를 연속으로 발사. 넓은 범위 견제 |
| 패시브 | 광폭화 | 전투 시작 후 20초 지나면, 한번 화내면서, 자기 자신에게 버프 (공속, 쿨타임 가속) |
| 패시브 | 장판 | 일정 시간마다 세로, 가로 각각 화면을 가로지르는 장판을 생성한다. 오브젝트 획득 시도 시 장판을 통해서 방해하는 의도 |

* 맵 기믹 요소
  + 오브젝트 생성
    - 맵의 특정 위치에 오브젝트 생성 위치를 지정
      * 6~7개의 위치를 지정해둔다.
    - 특정 스킬을 사용하면, 맵 오브젝트를 랜덤 생성한다.
      * 생성할 오브젝트 pool 관리
      * 위치는 지정해둔 위치 중에서 랜덤한 곳에 각각 생성한다.
  + 오브젝트 획득
    - 생성된 오브젝트는 인터렉션해서 획득할 수 있다.
      * 유저 획득 시 오브젝트에 따라서 버프 효과를 얻는다.
  + 오브젝트 회수
    - 보스가 특정 스킬을 사용하면, 맵 오브젝트를 보스가 획득한다.

* + 오브젝트 효과

| 구분 | InteractionObjectData@Id | 유저 획득 시 | 보스 획득 시 |
| --- | --- | --- | --- |
| 불꽃 구슬  Boss\_Object\_Fire | 1001 | 아군 불꽃 위력 +  (1001) | 불꽃 받는 피해 감소  (1006) |
| 냉기 구슬  Boss\_Object\_Ice | 1002 | 아군 냉기 위력 +  (1002) | 냉기 받는 피해 감소  (1007) |
| 전기 구슬  Boss\_Object\_Electric | 1003 | 아군 전기 위력 +  (1003) | 전기 받는 피해 감소  (1008) |
| 카오스 구슬  Boss\_Object\_Chaos | 1004 | 아군 카오스 위력 +  (1004) | 카오스 받는 피해 감소 (1009) |
| 무속성 구슬  Boss\_Object\_Nothing | 1005 | 아군 체력 회복  (1005) | 보스 체력 회복  (1010) |
