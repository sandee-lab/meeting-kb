- 유닛 승급 정보
    - 2마리에서 자동으로 승급할 때 연출을 넣으면 인지 문제는 해결 되겠지만, 일반 스킬 카드와 달리 캐릭터를 선택할 때 해당 캐릭터의 효율을 알 수 없는 문제가 있고 여전히 승급 시 얼마나 강해질지도 알기 어렵다.
    - 따라서 어떤 카드를 전략적으로 선택해야 하는지 명확하지 않기 때문에 택틱에서 선택의 의미를 찾기 힘들다.
- 유닛 승급의 재미
    - 조합식을 가지고 가기에는 직접 컨트롤까지 해야되는 불편함이 존재하고, 어려운 방식이기도 해서 유닛 조합을 통해 초월까지 도달하는건 굳이 필요 없다는 생각은 듬
    - 다만 단순히 같은 유닛을 합쳐서 승급하는 방식이라면 결과에서 랜덤한 유닛을 받는 정도의 의외성이 필요한게 아닌가 싶음. 현재는 택틱으로 유닛을 합친다고 해도 유저 스스로 해냈다는 느낌보다는 정해진 루트를 따라간다는 단조로움이 더 크게 느껴짐
    - 승급 안전지대가 있거나, 승급 버튼을 추가해보는건 어떨까
- 전투 및 몬스터
    - 전체적으로 기본 유닛들의 사정거리가 짧기 때문에 몰려오는 몬스터를 피해 사이드로 돌수밖에 없는 패턴. 사이드로 돌다보면 안쪽에 보석이 쌓이고 몬스터는 가운데에 몰리는 현상이 반복됌
    - 기본 유닛 안에서도 사정거리나 무기 타입에 따른 공격방식을 좀 더 다양화 해주면 어떨까?
    - 특히 지금은 기본 유닛에서 초월 유닛으로 한번에 점프하는 느낌이 강한데, 오히려 중간 단계의 유닛이 필요하다는 느낌이 많이 듬
    - 몬스터끼리의 간격이 너무 넓기 때문에 일관되게 스킬의 영향을 덜 받는 느낌을 받는다.
    - 몬스터가 무리지어 나오는 패턴이 너무 일정하게 느껴진다. 템포의 변화도 필요하다.
    - 프로젝타일을 피하는게 어렵기 때문에 미리 경로를 알려주거나, 혹은 원거리 몬스터는 좀 더 크게 표현해서 확실히 위험을 인지시키는게 좋겠다.
- 유닛
    - 기본 유닛의 사용성이 좀 더 제한적이었으면 좋겠다.
    - 좌우로, 상하로, 유도탄 등 특정한 방향에서 뾰족한 성능을 보이게하고 아쉬운 부분을 유저들이 찾아서 채우는 방향으로 했을 때 좀 더 게임을 만들어 가는 느낌을 받는다
    - 현재는 대부분 만능으로 느껴진다.
        - 예를들면 포탄은 처음부터 바닥에 깔리는 장판 추가
        - 힐냥이는 직선 거리의 긴 프로젝타일, 혹은 즉시 피해
        - 총냥이와 불냥이는 사방에 피해를 주지만 피해량이 약한편
        - 칼냥이는 좌 우 바라보는 방향으로 대미지를 주지만 크게 피해
    - 택틱에서 쿨타임이나 대미지 관련이 많이 나와서 점차 게임의 속도를 끌어 올려주는게 느껴지면 좋겠다
    - 몬스터의 베리에이션이 1-2분 사이에 계속 변화가 필요하다
    - 현재는 이런 특징들이 약하다보니 사방에서 몰려오는 몬스터의 수량으로 찍어누르는 패턴이 반복될수밖에 없어보인다. 공격 특징을 더 명확히 하면 적은 유닛으로도 이리저리 만들어가는 재미를 줄수 있을듯 하고, 초 중반 이후에는 조합을 통해 막강한 모습의 부대를 만드는 재미로 이어지면 좋겠다.
    - 아무래도 이런 단조로운 패턴으로 인해 금방 지루해지는 느낌이다.
- 연출
    - 새로운 유닛이 스폰되거나 승급 되었을 때 연출 필요
    - 레벨업 택틱 뜰 때 슬로우 삭제하고 트랜지션 연출 추가
    - 초월 유닛 등장시 연출 추가
    - 보스 체력바 따로 제공
    - 별 먹었을 때 스크린 연출
    - 아이템 사용 했을 때 아이템 효과 설명
    - 자석 아이템 썼을 때 너무 빠르게 들어온다. 천천히 순차적으로 들어와서 과정을 알수 있게 하자

    - 중간보스를 처치하면 지휘관이 승급해서 더 높은 티어의 유닛을 뽑게 된다거나
