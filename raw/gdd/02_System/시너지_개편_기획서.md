# 시너지 개편

> **요약:** 동일 속성 유닛을 인접 배치할 때 시너지 포인트를 획득하는 속성 배치 시너지 시스템 개편 기획이다. 2마리 이상 인접 시 카운팅하며, 분리된 그룹 중 다수 그룹만 표시한다. 전설 유닛에는 인접 슬롯 아군 대상 패시브 스킬(쿨타임 가속, 속성 공격력 등)을 부여하여 배치 형태에 따른 전략적 선택을 유도한다. 무속성 유닛이 택틱으로 속성을 부여받은 경우에도 해당 속성 시너지에 포함된다.
> **현재 상태:** 미적용

# M8\_시너지 개편

# 목표

* 속성 조합을 메인으로 하므로 동일 속성을 많이 배치하도록 유도
* 시너지를 발현할 때, 배치하는 재미를 직관성있게 적용.

# 개요

* 동일 속성 유닛을 인접 배치하는 경우 시너지 포인트 획득
* 전설 유닛 스킬에 인접한 유닛에게 효과를 부여하는 기능 추가하여, 전설 유닛 소환 시 배치 형태가 달라지도록 유도

# 시스템 구조

### 속성 배치 시너지

* + SetEffectData@Condition\_type 에 Property 추가
    - Fire
    - Ice
    - Electric
    - Chaos
    - Nothing
  + 시너지 카운트 조건
    - **2마리 이상 인접한 경우에 카운팅.**
      * 동일 속성으로 8마리 배치 시 시너지 8
      * 2마리, 2마리 각각 인접해 있다면, 2마리로 카운팅
      * 3마리, 2마리 각각 인접해 있다면, 3마리로 카운팅
    - **무속성 유닛이 택틱으로 특정 속성을 부여 받은 경우 해당 속성 카운팅 시 포함되어야 함.**

### 시너지 효과

* + - 기존의 시스템 그대로 적용
  + 시너지 표현
    - 인접한 유닛끼리 같은 속성으로 묶어 주는 UI 표시를 추가한다.
    - 3유닛, 2유닛 각각 떨어져 있는 경우 시너지 효과를 부여하는 다수의 유닛 그룹(3유닛)에만 표시를 적용한다. 만약 2유닛, 2유닛인 경우 내부에서 처리하는 순서에 따라 한 쪽 유닛 그룹에만 표시를 해준다.

### 전설 유닛 배치 시너지

* + 스킬 시스템으로 포메이션을 타겟으로 스킬 쓸 수 있게 target 조건 추가
    - ~~Ourteam / Formation\_Line (같은 라인에 위치한 아군)~~
      * 라인은 표현하기 어려운 개념이므로 제외
    - Ourteam / Formation\_Near (인접한 슬롯에 배치한 아군)
  + 예)
    - 라이카
      * 패시브 : 인접한 위치의 유닛에게 쿨타임 가속 -20%
    - 헬파이어
      * 패시브 : 같은 라인에 배치한 유닛에게 불속성 공격력 +20%
