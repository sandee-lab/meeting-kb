# 전투력 보정

> **요약:** 스테이지 모드의 요구 전투력 대비 유저 전투력 비율에 따라 5단계(매우 쉬움~매우 어려움) 난이도 보정을 적용하는 시스템이다. 각 구간별로 몬스터 공격력과 체력에 가중치를 곱연산하여, 스펙 차이에 의한 난이도 체감을 강화한다. StageModePowerReviseData 데이터시트로 관리한다.
> **현재 상태:** 미적용

전투력 보정

# 목표

* 아웃게임에서 스펙업을 통해서 올라가는 지표인 전투력이 각 챕터의 클리어를 위한 최소 기준이 되어야 하는데, 인게임에서의 편차가 크기 때문에 어려움이 있다.
* 권장 전투력을 만족했을 때와 만족하지 못했을 때, 스펙 보정을 적용하여 스펙에 의한 난이도 체감을 보다 잘 체감할 수 있도록 하고자 한다.

# 개요

* 스테이지 모드의 요구 전투력 대비 유저의 전투력 비율에 따라서 난이도 보정을 적용한다.
  + 매우 쉬움
    - 유저 전투력 / 요구 전투력 >= 1.50 이상이면, 매우 쉬움
  + 쉬움
    - 유저 전투력 / 요구 전투력 >= 1.25 이상이면, 쉬움
  + 보통
    - 0.75 이상 <= 유저 전투력 / 요구 전투력 < 1.25 미만이면, 보통
  + 어려움
    - 유저 전투력 / 요구 전투력 = 0.75 미만이면, 어려움
  + 매우 어려움
    - 유저 전투력 / 요구 전투력 = 0.5 미만면, 어려움
* 각 구간 별로 몬스터에게 가중치를 적용한다.

| 단계 | 몬스터 공격력 | 몬스터 체력 |
| --- | --- | --- |
| 매우 쉬움 | 0.75 | 0.75 |
| 쉬움 | 0.75 | 1 |
| 보통 | 1 | 1 |
| 어려움 | 1 | 1.2 |
| 매우 어려움 | 1.2 | 1.5 |

# 필요 데이터시트

* StageModePowerReviseData
  + difficulty\_type
    - Very\_Easy
    - Easy
    - Normal
    - Hard
    - Very\_Hard
  + condition\_ratio\_min
    - (유저 전투력 / 요구 전투력) 값과 비교할 최소 값
    - 해당 값 이상 조건을 만족하는지 비교한다.
  + condition\_ratio\_max
    - (유저 전투력 / 요구 전투력) 값과 비교할 최대 값
    - 해당 값 미만 조건을 만족하는지 비교한다.
  + monster\_atk\_weight
    - 각 난이도 구간에 따라 몬스터의 최종 공격력에 곱연산한다.
  + monster\_hp\_weight
    - 각 난이도 구간에 따라 몬스터의 최종 생명력에 곱연산한다.

| difficulty\_type | condition\_ratio\_min | condition\_ratio\_max | monster\_atk\_weight | monster\_hp\_weight |
| --- | --- | --- | --- | --- |
| Very\_Easy | 1.5 | 999 | 0.75 | 0.75 |
| Easy | 1.25 | 1.5 | 0.75 | 1 |
| Normal | 0.75 | 1.25 | 1 | 1 |
| Hard | 0.5 | 0.75 | 1 | 1.2 |
| Very\_Hard | 0 | 0.5 | 1.2 | 1.5 |
