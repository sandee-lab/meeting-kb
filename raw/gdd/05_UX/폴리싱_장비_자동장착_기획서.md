# 장비 자동 장착

> **요약:** 장비 자동 장착 버튼을 추가하여 등급-티어-강화레벨-ID 순서로 보유 및 장착 장비 전체를 정렬한 뒤, 각 부위별 최상위 장비를 자동으로 장착(교체)하는 편의 기능이다. 세트 효과는 무시하며, 이미 최상위 장비 장착 중일 때는 토스트 메시지를 출력한다.
> **현재 상태:** 미적용

![](data:image/png;base64...)

* 신규 기능
  + 장비 자동 장착 버튼 배치
    - 개요
      * 자동 장착은 등급, 티어, 강화 레벨 기준으로 가장 높은 장비로 장착(교체)해준다.
      * 세트 효과는 무시한다.
    - 처리
      * 검사 대상 : 보유한 장비 + 장착 장비 전체
      * 검사 절차 :
        + 1단계 : 각 장착 부위 별로 아래 순서대로 정렬

등급 높은 것 > 낮은 것

티어 높은 것 > 낮은 것

강화 레벨 높은 것 > 낮은 것

ID 낮은 것 > 높은 것

* + - * + 2단계 : 위 순서대로 하여 각 장착 부위별 최상위 장비 1개 선정
        + 3단계 : 선정 결과가 현재 장착 장비와 다를 경우 → 교체 (자동 장착)
    - 예외 처리
      * 이미 최상위 장비 장착 중일 때
        + 토스트 메시지 출력

"이미 추천 장비 장착 중입니다"
