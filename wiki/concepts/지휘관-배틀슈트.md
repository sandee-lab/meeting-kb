# 지휘관 및 배틀슈트

> 출처: [[raw/gdd/02_System/배틀슈트_시스템_명세서]], [[raw/gdd/02_System/지휘관_배틀슈트_기획서]], [[raw/gdd/02_System/지휘관_부활_시스템]] *(대체됨)*, [[raw/gdd/02_System/드론_장비_시스템_기획서]], [[raw/gdd/01_Core/지휘관_성장_경제효과_기획서]], [[raw/gdd/01_Core/지휘관_인게임_경제_시나리오]]

## 개요

지휘관(SCV)은 플레이어가 직접 조종하는 핵심 캐릭터이며, 배틀슈트는 지휘관의 아웃게임 성장 축이다. 지휘관 성장이 인게임 경제(이자, 유닛 소환 비용, 웨이브 밸런스)에 미치는 효과가 시나리오별로 설계되어 있다. 드론/장비 시스템은 배틀슈트와 연계된 부가 성장 요소이다.

## 상세 내용

### 배틀슈트 시스템

> 출처: [[raw/gdd/02_System/배틀슈트_시스템_명세서]], [[raw/gdd/02_System/지휘관_배틀슈트_기획서]]

챕터 10 이후 해금되는 중후반 핵심 성장 콘텐츠. 유닛 성장의 한계를 보완하여 덱의 완성도를 높이는 것이 목적이다.

**기획 의도:**
- 최종 성장: 챕터 10 이후 해금되는 중후반 핵심 성장 콘텐츠
- 결핍 유도: 배틀 패스만으로는 승급 재화가 미세하게 부족하도록 설계 → 3개월 차(시즌 3)부터 과금 강유도
- 과금 한도 확장: 5성 졸업 이후에도 큐브 소모(잠재력 리롤)로 지속 과금 유도

**레벨업 시스템:**
- 소모 재화: 에너지 쉘(경험치) + 골드
- 승급 상태에 따른 레벨 제한:

| 등급(성) | 레벨 범위 |
|---------|----------|
| 0성 | Lv. 1~20 |
| 1성 | Lv. 21~40 |
| 2성 | Lv. 41~60 |
| 3성 | Lv. 61~80 |
| 4성 | Lv. 81~100 |
| 5성 (졸업) | Lv. 101~120 (MAX) |

**승급 시스템 (총 210조각 필요):**
- 고유 조각 기반 승급 (0~5성)
- 주요 획득처: 배틀 패스(시즌 당 유료 30개), 상점 패키지

| 등급 | 필요 조각 | 누적 | 승급 보상 | BM 의도 |
|------|-----------|------|-----------|---------|
| 0성 해금 | 10 | 10 | 슈트 해금, 기본 무기 | S1 획득 |
| 1성 | 20 | 30 | 잠재력 1슬롯, 무기 강화 | S1 깔끔 소진 |
| 2성 | 30 | 60 | 잠재력 2슬롯, 무기 강화 | S2 안착 |
| 3성 | **40** | 100 | 잠재력 3슬롯, 무기 진화 | **S3 결핍 발생 (10개 부족)** |
| 4성 | 50 | 150 | 잠재력 4슬롯, 무기 강화 | S5 완성 |
| 5성 | 60 | **210** | 잠재력 5슬롯, 무기 최종형 | **S7 졸업** |

- 배틀 패스 유료 유저: 시즌 당 30개 획득 (무료 5개 + 유료 10개 + 미션 완료 15개)
- 공용 조각 변환: 5성 달성 후 잉여 조각을 공용 조각으로 전환 가능
- 공용 조각 교환 비율: 20개 → 10개 (2:1), 최소 10개 단위
- 승급에 따른 외형 진화: Mk.1(0~1성) → Mk.2(2~3성) → Mk.3(4~5성)

**잠재력 시스템 (엔드 콘텐츠):**
- 5성까지 순차 해금되는 5개 슬롯 (0성: 0슬롯, 1성: 1슬롯, 2~5성: 슬롯 +1씩)
- 소모 재화: 카오스 큐브
- 랜덤 옵션 부여 로직 3단계:
  1. 등급 결정: D~SSS 확률표(RandomPotentialProb)에 따라 등급 결정
  2. 종류 결정: 슈트별 전용 옵션 풀(RandomPotentialPoolGroup)에서 가중치 랜덤 선택
  3. 수치 결정: 5% 확률(`ConstantDatas@MaxProb`)로 최대값 당첨 체크 → 당첨 시 Max 수치 즉시 부여, 미당첨 시 [Min~Max 미만] 균등 결정 (Tick 단위)
- 승급(합성) 시 기존 옵션 100% 계승
- 잠금 기능: 2성(슬롯 2개)부터 사용 가능 (0~1성은 잠금 버튼 비활성화)
- 잠금 개수별 리롤 비용: 0개=큐브 1개 / 1개=2개 / 2개=5개 / 3개=10개 / 4개=20개
- 슈트별 전용 옵션 풀: 화염 슈트→POOL_FIRE / 냉기 슈트→POOL_ICE 등 속성별 분리
- 잠재력 등급 뱃지: D(회색) / C(녹색) / B(파랑) / A(보라) / S(금) / SS(붉은색) / SSS(무지개빛)

**데이터 테이블 구조 (4개):**
- `CommanderData`: potential_type 컬럼 추가 (지휘관별 잠재력 풀 연결)
- `RandomPotentialManager`: 슬롯 해금 조건, 연결 풀 ID, 확률표 ID, 잠금별 비용
- `RandomPotentialProb`: D~SSS 등급 확률 (만분율)
- `RandomPotentialPoolGroup`: 옵션 종류별 등급별 Min/Max/Tick/Weight

**5월 테스트에서는 미노출 예정**, 고과금 유저 전용 성장 요소로 활용 계획 (Ncsoft 제안)

### 드론/장비 시스템

> 출처: [[raw/gdd/02_System/드론_장비_시스템_기획서]]

기존 RPG 장비를 공격/폭격/방어 3종 역할군의 드론 시스템으로 전환.

**장비 타입 변경:**

| 기존 | 변경 |
|------|------|
| Parts_1 | AttackDrone (공격 드론) |
| Parts_2 | BombardDrone (폭격 드론) |
| Parts_3 | DefenceDrone (방어 드론) |
| Parts_4~6 | 미사용 |

**챕터별 슬롯 해금:**

| 챕터 조건 | 해금 슬롯 | 총 드론 수 |
|---------|---------|-----------|
| 튜토리얼 | 공격1/폭격1/방어1 | 3기 |
| Chapter 5 | 공격2 (Parts_AttackDrone_Slot) | 4기 |
| Chapter 10 | 폭격2 (Parts_BombardDrone_Slot) | 5기 |
| Chapter 15 | 방어2 (Parts_DefenceDrone_Slot) | 6기 |

**드론 세부 타입 (sub_type, PartsData@sub_type 컬럼):**
- 공격형: 래피드(Rapid) / 소닉(Sonic) / 헤비(Heavy)
- 폭격형: 버스터(Buster) / 네이팜(Napalm) / 썬더(Thunder)
- 방어형: 가디언(Guardian) / 리페어(Repair) / 쇼크(Shock)
- 동일 sub_type은 중복 장착 불가 (타입 내 최대 1종 운용)

**성장 구조:**
- 레벨업: 골드 + 드론 스크롤(타입별) 소모, 깡공격력/깡체력 상승
- 승급: 동일 ID + 동일 등급 2개 합성 → 등급 상승 + Max Level +10 + 잠재력 해금
- 드론은 공격력/체력 모두 오를 수 있음 (기존 부위별 스탯 분리 방식과 다름)

**장착 규칙 (2026-01-22 확정):**
- 슬롯 1개 개방 시: 새 장비 장착 시 기존 장비 자동 교체
- 슬롯 2개 개방 시: 타입 충돌 검사 → 동일 타입이면 자동 교체 (토스트 메시지 출력) / 다른 타입이면 슬롯 선택 팝업
- 상위 등급 교체 시 강화 레벨 이전: 교체 장비가 강화 0 상태이고 더 높은 등급이면 기존 강화 계승 (`Local_Toast_Parts_Enchant_Transferred`)
- 추천 태그: 현재 슬롯에 장착했을 때 전투력이 상승하는 장비에만 표시

### 지휘관 성장 경제 효과

지휘관 성장이 인게임 경제에 미치는 시나리오별 분석:
- 웨이브별 이자/투자 밸런스 설계
- 성장 격차에 따른 경제 시나리오
- 유닛 소환 비용과 지휘관 레벨의 관계

### 지휘관 인게임 경제 시나리오

- 인게임 경제에서 지휘관 성장의 위치
- 웨이브 진행에 따른 이자 시스템
- 유닛 소환과 밸런스의 관계
- 성장 격차 보정 메커니즘

### 지휘관 부활 시스템 *(대체됨)*

> 출처: [[raw/gdd/02_System/지휘관_부활_시스템]] (M7, 이후 버전으로 대체됨)

초기 설계에서 검토되었으나 대체된 시스템:
- 지휘관 사망 시 재화 소모 또는 광고 시청으로 부활
- 전투 당 부활 횟수 제한 (`stage_revive_limit_per_battle`)
- 일일 광고 시청 부활 횟수 제한 (`stage_revive_ads_daily_limit_per_battle`)
- 부활 시 주변 넉백 + n초 무적 스킬 자동 부여
- ContentsUnlockData - Stage_Revive 타입으로 해금 시점 제어
