# 미니크래프트 게임 플레이 아이디어

> **요약:** 오토 슈터 장르 기반 코어루프 아이디어. 시작 가챠-웨이브 진행-부대/기지 관리 사이클 설계.
> **현재 상태:** 미적용

**미니크래프트 게임 플레이 아이디어**

**목표**

* 프로젝트 방향성 (오토 슈터) 장르 틀과 핵심 게임 플레이 유지하는 방향에서 아이디어 도입
  + 부대와 기지 관리를 통해 전략성 높이는 방향으로

**플레이 시나리오**

* 시작
  + 최초 슬롯 머신 경험 (무작위성)
  + 시작 유닛 **3종** 결정
    - 3종인 이유
      * [메카닉]과 [바이오닉] 중 유리한 계열 설정하고, 한정적인 유닛 슬롯 관리를 위해 3종으로 설정
    - 리롤 횟수 제공
      * 사용하지 않은 리롤 횟수는 자원으로 지급 (기회 비용 개념)
      * 지휘관에 따라 리롤 횟수가 추가되는 등으로 연계
* 웨이브 진행
  + 플레이 목표는 **빠르게** 웨이브 클리어
    - 플레이 중 자원(가스)과 유닛 획득 가능
      * 슬롯 연출 선호도가 높으니, 유닛 획득 방식(박스, 슬롯 연출)은 유지함
    - 자원은 기본적으로 몬스터 사망 시 드롭.
      * 자원 상자로 드롭되는 경우에는 이동하여 획득 (이동 유도)
    - 웨이브 클리어 타임에 따라 획득 자원 보너스 N% 지급 (생존 < 빠르게)
* 웨이브 종료 (부대 및 기지 관리)
  + 성장 라인 선택
    - (1) 부대 업그레이드 [현재 조합으로 진행하고 싶은 경우]
      * 유닛 판매 및 초월 (슬롯 관리)
      * 부대 업그레이드 (스탯 업그레이드)
    - (2) 기지 업그레이드 [유닛 정리 + 새 유닛 뽑고 싶은 경우]
      * 다음 웨이브부터 보유 유닛 수 증가
      * 다음 웨이브부터 상위 유닛 획득 가능
      * 다음 웨이브부터 상위 초월 유닛 해금
        + 기지에 신규 자원 생산하는 역할을 주는 것도 좋겠으나, 생산한 자원 활용처 고민.

* 추가 조합 규칙
  + 현재 제작된 시너지 구조 활용하면서, 이해하기 쉬운 방향으로.
  + (1) 역할 시너지 : 탱커/딜러/서포터
    - 3탱커: 전체 HP 증가 + 피해 감소
    - 4딜러: 크리티컬 확률 증가 + 공격속도 증가
    - 2서포터: 아군 회복 + 버프 효과 증가
  + (2) 속성 시너지 : 극열/극한/전기 등..
    - 극열 3유닛: 지속 화상 피해 + 얼음 효과 면역
    - 극한 3유닛: 적 행동 속도 감소 + 화염 피해 감소
    - 전기 3유닛: 적에게 충격(기절..) 디버프
* 기타
  + 화면 구도 변경
    - 회피 어려움 개선
      * 적 이동 방식을 일방향으로 테스트
        + 자원 상자 드롭하여 최소한의 이동 유도
        + 단, 전선 형성하게 한다면 적 사망 시 드롭되는 아이템과 경험치에 한해 자동 수집되도록 하여야 함.
