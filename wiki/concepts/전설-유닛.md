# 전설 유닛 설계

> 출처: [[raw/meetings/[1022] M8 전략성 개선 회의]], [[raw/meetings/[260305]TCA_2월 빌드 피드백에 대한 개선 방향]], [[raw/meetings/[0319] 3월 2-3주차 작업 리스트]], [[raw/gdd/02_System/등급별_유닛_리스트]], [[raw/gdd/02_System/TCA_등급별_유닛_리스트]], [[raw/gdd/02_System/전설유닛_설계_논의_맥락]], [[raw/gdd/02_System/TCA_전설유닛_논의_맥락_종합정리]], [[raw/gdd/02_System/미스테리오_기획서]], [[raw/gdd/02_System/채굴맨_기획서]]

## 개요

전설 유닛은 3계층 구조로 구성된다. 기본 성능 50% + 시너지 성능 50% 원칙(2026.03 확정)에 따라 단독 소환 시에도 효능감을 제공하고, 조합 완성 시 극대화된다.

| 계층 | 점수 | 포지션 | 조합 난이도 |
|------|:----:|--------|-----------|
| 10점 무속성 | 10 | 기본 루트 / 소환 옵션 | 에픽1 + 고급1 |
| 14점 속성 진입 | 14 | 속성 시너지 시작 | 에픽1 + 고급1 |
| 24점 덱 완성 | 24 | 덱 압축 완성 | 에픽2 |

## 1. 전설 유닛 14종 목록 (2026.03~04 확정)

> 출처: [[raw/gdd/02_System/TCA_등급별_유닛_리스트]], [[raw/gdd/02_System/등급별_유닛_리스트]]

| 이름 | 속성 | 계열 | 포지션 | 점수 |
|------|------|------|--------|:----:|
| 고양이 사부 (신규) | 무속성 | 공속 | 기본 루트 공격형 — 클리버 시너지 + 전체 공속↑ | 10 |
| 캡틴 | 무속성 | 공속 | 기본 루트 생존형 — 실드 시너지 + 공격 시 보호막 | 10 |
| 포어맨 | 무속성 | 공속 | 소환 옵션 — 엔지니어 시너지 + [소환수] 공속↑ | 10 |
| 시즈 | 무속성 | 쿨타임 | 기본 루트 화력형 — 캐논 시너지 + 전체 쿨타임↓ | 10 |
| 메카 | 무속성 | 쿨타임 | 기본 루트 생존형 — 로켓 시너지 + 피격 시 반격 | 10 |
| 오퍼레이터 | 무속성 | 쿨타임 | 소환 옵션 — 엔지니어 시너지 + [소환수] 소환 쿨타임↓ | 10 |
| 스카우트 | 전기 | 공속 | 속성 시너지 시작 | 14 |
| 섀도 | 냉기 | 공속 | 속성 시너지 시작 | 14 |
| 간호순 | 화염 | 쿨타임 | 속성 시너지 시작 | 14 |
| 미스테리오 | 암흑 | 쿨타임 | 속성 시너지 시작 | 14 |
| 볼트 | 전기 | 공속 | 덱 완성 | 24 |
| 프로스트 | 냉기 | 공속 | 덱 완성 | 24 |
| 헬파이어 | 화염 | 쿨타임 | 덱 완성 | 24 |
| 고스트 | 암흑 | 쿨타임 | 덱 완성 | 24 |
| 엣지 (변경 예정) | 전기 | 공속 | 재설계 예정 | TBD |

전체: 일반 4 + 고급 6 + 희귀 6 + 영웅 5 + 전설 14 = **35종**
일반~영웅 등급 추가 계획 없음. 전설은 장기적 50종까지 확장 가능.

## 2. 전설 조합식 (2026.03 확정)

> 출처: [[raw/gdd/02_System/밸런스_변경사항]]

**1티어 (에픽1 + 고급1):**

| 속성 | 전설 유닛 |
|------|----------|
| 불꽃 | 간호순 |
| 냉기 | 섀도 |
| 전기 | 스카우트 |
| 카오스 | 미스테리오 (추가 예정) |
| 무속성 | 고양이 사부, 캡틴, 포어맨, 시즈, 메카, 오퍼레이터 |

**2티어 (에픽2):**

| 속성 | 전설 유닛 |
|------|----------|
| 불꽃 | 헬파이어 |
| 냉기 | 빙결술사 |
| 전기 | 볼트 |
| 카오스 | 고스트 |

1전설 = 1조합식 1:1 대응. 현재 풀(고급6 + 희귀6 + 영웅5)로 가능한 조합 161가지 → 전설 50종까지 대응 가능.

## 3. 전설 유닛 설계 원칙 (2026.03.23 확정)

> 출처: [[raw/gdd/02_System/전설유닛_설계_논의_맥락]]

**핵심 원칙:**
- **기본:시너지 = 50:50** — 단독 소환 시에도 명확한 효능감 제공, 조합 완성 시 극대화
- **10점 무속성 이중 효과**: ① 하위 유닛 조합 시너지 ② 범용 버퍼 (속성 전환 후에도 유지)
- 간호순·포어맨은 이 원칙에 미달 → To-Be 설계 완료 (간호순: 화염 주사기 투척 + 아군 소량 회복, 포어맨: 자체 터렛 다수 소환)

**속성 전환 시스템 (확정):**
- 게임 내 희귀 확률로 "무속성 → 특정 속성 전환" 전략 카드 등장
- 선택 시 모든 무속성 유닛이 해당 속성으로 전환 (속성 디버프 미부여, 시너지 수혜만)
- 한번 선택 시 재등장 없음

**기각된 아이디어:**
- 인챈트/각성/변이로 무속성 덱 강화
- 소환 덱 독립 3번째 루트화 (전설 4슬롯 필요)
- 광부 → 상위 유닛 2갈래 분기

## 4. 전설 계승/진화 이원화 (M8 전략성 개선, 2025.10.22)

> 출처: [[raw/meetings/[1022] M8 전략성 개선 회의]]

- 전설 유닛이 재료 유닛의 공격 형태와 속성을 계승받는 방식은 유지하되, 단조로움 보완
- **[계승]**: 재료 유닛 공격 형태/속성 그대로 계승. 초기 학습 단계에서 위주로 사용
- **[진화]**: 카오스 계열 유닛 포함 레시피에서 새로운 형태로 변형
- 특정 택틱으로 유닛 무기 타입/형태 변화 → 히든 조합식 활용 가능

## 5. 전설 유닛 사용성 논의 — 문제와 해결 방향 (2026.03)

> 출처: [[raw/gdd/02_System/TCA_전설유닛_논의_맥락_종합정리]]

**사용성 낮음의 원인:**
- 기존 전설 유닛은 시너지 없이 단독 소환 시 거의 효용 없는 구조 (기본 성능 0~20%)
- 소환 덱 독립 루트 구성 시 전설 4슬롯 필요 → 비현실적
- 무속성 조합 덱이 중반에 너무 빨리 완성 → 이후 할 것이 없음

**확정된 해결 방향:**

| 방향 | 내용 |
|------|------|
| 기본:시너지 50:50 | 단독 소환 시에도 명확한 효능감 제공 |
| 10점 무속성 이중 효과 | ① 하위 유닛 조합 시너지 ② 범용 버퍼 |
| 속성 전환 시스템 | 무속성 전체 → 특정 속성 전환으로 시너지 편입 |
| 무속성 → 속성 전환 동선 | 무속성 덱은 속성 덱의 하위 호환으로 자연스러운 전환 |
| 소환 유틸리티화 | 포어맨·오퍼레이터: 독립 루트 아닌 자유 슬롯 유틸리티 |

**10점 전설 4종 기본 루트 (2×2 대칭):**

| 계열 | 유닛 | 하위 시너지 | 범용 버프 | 스타일 |
|------|------|-----------|----------|--------|
| 공속 | 고양이 사부 | 클리버 공속 증폭 | 전체 공속↑ | 공격형 |
| 공속 | 캡틴 | 실드 추가 방패 | 공격 시 보호막 | 생존형 |
| 쿨타임 | 시즈 | 캐논 추가 포탄 | 전체 쿨타임↓ | 화력형 |
| 쿨타임 | 메카 | 로켓 유도 미사일 | 피격 시 반격 | 생존형 |

**미해결 과제 (2026.03.23 기준):**
- 캡틴 루트 8슬롯 시나리오 상세 미작성
- 쿨타임 계열 덱 전환 구조 (시즈·메카 루트) 미착수
- 엣지 → 전기 속성 전설 재설계 방향 확정, 수치 미정

## 6. 개별 유닛 기획서 — 미스테리오 (2026.03, 상태: 초안)

> 출처: [[raw/gdd/02_System/미스테리오_기획서]]

| 항목 | 내용 |
|------|------|
| 등급·속성 | 전설 (Legendary), 카오스(암흑) |
| 점수 | 14점 |
| 역할 | 연사형 디버퍼 — 약화를 광범위하게 퍼뜨리고 약해진 적을 처형 |
| 메타 포지션 | 고스트(한방형)의 시너지 파트너, 카오스 속성 조합 필수 유닛 |

**스킬 구성:**

```
미스테리오
├─ [저주의 투사체] — Base Skill
│   ├─ Lv1: 다수 투사체 → 적중 시 피해 + [약화] 부여 (공격력 -20%, 카오스 피해 +30%)
│   └─ Lv2: HP ≤ 30% 일반 몬스터 → 즉결 처형 (보스·엘리트 제외)
├─ [약화 전이] — Sub Skill 1
│   └─ [약화] 걸린 적 사망 → 주변 적에게 [약화] 전이
└─ [환영] — Sub Skill 2
    └─ [저주의 투사체]를 사용하는 환영(소환수) 소환
```

**핵심 시너지 루프:**
환영 + 미스테리오 동시 연사 → 약화 부여 + HP 감소 → 일반 몬스터 HP 30% 도달 시 즉결 처형 → 약화 전이 확산 → 고스트 강타 (약화 대상 +30~50%)

| 항목 | 고스트 | 미스테리오 |
|------|--------|-----------|
| 공격 빈도 | 5초 쿨 한방 | 잦은 연사 |
| 약화 부여 | 없음 (수혜자) | 직접 부여 |
| 대보스 성능 | 높음 | 중간 |
| 대다수 성능 | 낮음 (단일) | 높음 (광역 약화) |

구현: 신규 코드 없이 기존 CSV 조합만으로 가능. `InstantDeath value_1=0.3` 신규 행 추가만 필요.

## 7. 개별 유닛 기획서 — 채굴맨 (2026.04, 상태: 초안)

> 출처: [[raw/gdd/02_System/채굴맨_기획서]]

| 항목 | 내용 |
|------|------|
| 등급·속성 | 전설 (Legendary), 무속성 |
| 점수 | 10점 |
| 합성 재료 | 엔지니어 + 엔지니어 |
| 역할 | 2단계 전환형 경제 유닛 (경제 구간 → DPS 전환) |
| 메타 포지션 | 10점 전설의 "경제 루트" — 기존 4종 전투형과 차별화 |

**스킬 구성:**

```
채굴맨 (채굴 모드)
├─ [채굴] — Base Skill (Passive)
│   ├─ Lv1: 미네랄 획득 시 30% 확률 추가 미네랄 + [채굴] 스택 1 적립
│   └─ Lv2: 스택 2개씩 적립 + 추가 미네랄 확률 50%
└─ [드릴 다운] — Sub Skill
    └─ 미네랄 10 소모 → 영구 변신 (드릴 모드, 되돌릴 수 없음)

채굴맨 (드릴 모드)
├─ [드릴 브레이커] — Base Skill
│   ├─ Lv1: 3초 쿨타임마다 드릴을 땅속으로 발사 → 적 위치에서 솟아올라 범위 피해
│   └─ Lv2: 솟아오른 후 인접 적 1체에 관통 피해 추가
└─ [채굴 역량] — Sub Skill (Passive)
    ├─ Lv1: 보유 [채굴] 스택 × 5% 공격력 증가 (최대 20스택 = +100%)
    └─ Lv2: 스택당 7% (최대 +140%)
```

**전략적 딜레마 — "언제 변신할 것인가":**

| 선택 | 이득 | 리스크 |
|------|------|--------|
| 빠른 변신 (스택 5~8) | 빠른 DPS 투입 | 낮은 스택 = 공격력 미미 (+25~40%) |
| 늦은 변신 (스택 15~20) | 폭발적 DPS (+75~100%) | 경제 구간 DPS 공백 길어짐 |
| 변신 안 함 | 미네랄 최대 확보 | DPS 기여 없음, 1슬롯 낭비 |

구현 과제: 유닛 내 모드 전환 + 스킬 교체 시스템 프로토타입 확립 필요.

## 8. 전설 유닛 결정 흐름 — 4단계 변천 (2025.02~2026.03)

> 출처: [[wiki/decisions/유닛-밸런스]]

| 단계 | 시기 | 핵심 변화 | 남은 문제 |
|------|------|----------|---------|
| 1 | 2025.02~07 | "강한 유닛"으로 도입, 확률 12% 설정 | 설계 원칙 없음 |
| 2 | 2025.09~11 | 조합식·시너지 체계 정립, 계승/진화 이원화 | 기본 성능 0~20%, 시너지 없으면 "꽝" |
| 3 | 2026.01~02 | 역할 재배치, 속성 빌드 체계 확정 | 전설 접근성 허들, 등급 간 체감 부족 |
| 4 | 2026.03 | 기본:시너지 50:50, 14종 확정 | 승계 시스템 미검증, 일부 루트 미작성 |

**피드백 기반 4축 방향 (2026.03):**

| 축 | 방향 | 상태 |
|----|------|------|
| 목표 명확화 | 전설 = 핵심 목표, 하위 유닛 = 전설로 가는 과정 | 원칙 확정 |
| 단독 효능감 | 기본 50% 원칙 + 포어맨·간호순 To-Be 설계 | 구현 필요 |
| 접근성 개선 | 레시피 단순화 (에픽1+재료1), 초반 2종 체험 루트 | 확정 |
| 투자 보존 | 유닛 승계 시스템 (머지 시 카드 소멸 해결) | 테스트 미완 |
