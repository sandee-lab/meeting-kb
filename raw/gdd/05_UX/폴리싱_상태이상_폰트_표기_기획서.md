# 폴리싱 - 상태이상 폰트 표기 추가

> **요약:** 상태이상으로 인한 체력 감소/증가/즉시 처치를 WorldUI에 시각적으로 표현하기 위한 폰트 타입 추가 기획이다. 체력 회복(InstantHeal, DotHeal)은 +999K 형태로 상승 후 소멸, 체력 감소(StatEffect_Fire, InstantDamage, DotDamage)는 기존 DamageSlot과 동일 방식, 즉시 처치(InstantDeath)는 해골 아이콘으로 표현한다. 아군과 일반 몬스터 모두 대상이다.
> **현재 상태:** 미적용

**폴리싱\_상태이상 폰트 표기 추가**

# 개요

* 상태이상 체력 감소 및 증가를 표현하기 위해 WorldUI에 타입을 추가한다.

# 현재 UI 표기

* 아군
  + 체력 바
    - 배틀 슈트 (지휘관) : 상시
    - 전투 유닛 : 피격 시
  + 피격 시 대미지 **-999.99**
    - 아래 방향으로 하강 후 소멸

**![](data:image/png;base64...)**

* 일반 몬스터
  + 피격 시 체력 바 표시
  + 피격 시 대미지 **999.99**
    - 위 방향으로 상승 후 소멸

![](data:image/png;base64...)

# 추가 항목 (상태 이상)

## 체력 회복 (=힐)

* + 대상
    - 아군, 일반 몬스터
  + 타입
    - InstantHeal
    - DotHeal
  + 표시
    - 체력 회복 했을 때,
      * **+999K** (회복값) / 위 방향으로 상승 후 소멸한다.

## 체력 감소 (=피해)

* + 대상
    - 아군, 일반 몬스터
  + 타입
    - StatEffect\_Fire
    - InstantDamage
    - DotDamage
  + 표시
    - 체력 감소했을 때, 현재의 DamageSlot 표현 방식과 동일하게 표현해준다.

## 즉시 처치

* + 대상
    - 일반 몬스터
  + 타입
    - InstantDeath
  + 표시
    - 사망 시,
      * (이미지) / 위치 이동 연출 없음. 출력 후 소멸
        + 이미지 사용 시, DamageSlot과 구분되는 전용 슬롯 생성해서 사용한다.
        + 신규 아이콘 필요 (WorldUI) (예: ic\_instant\_death)
