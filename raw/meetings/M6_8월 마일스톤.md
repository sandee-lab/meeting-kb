5. ## (진행 필요)신규 택틱 추가하기

   1. ### 스킬 기능 추가

      1. #### (완료)아이템 획득

         1. 스킬 기능으로 아이템을 드랍 처리한다.  
            1. Drop\_Item 용 스킬프리팹 분리  
            2. spawn\_id \= 스폰할 아이템 id (stagemodedropitemdata)  
            3. 스폰 위치 \= target\_pivot  
         2. 예)  
            1. 새퍼가 터랫 3회 설치 시 마다 가스 1개 발견

   2. ### 스킬 추가 (Add\_Skill)

      1. 특정 유닛 혹은 특정 그룹을 대상으로 스킬을 추가해준다.  
      2. 사용할 대상 기준

         1. #### (완료)지휘관에게 스킬 부여

            1. 의도  
               1. 지휘관 유닛에게 스킬을 부여한다.  
            2. 데이터 설정  
               1. tactic\_target\_team \= FALSE  
               2. is\_commander \= TRUE  
               3. effect\_type \= Add\_Skill  
               4. effect\_value \= skill\_id

         2. #### (완료)랜덤\_선택 유닛에게 스킬 부여

            1. 의도  
               1. 현재 배치되어 있는 에픽, 전설 등급의 유닛 중 랜덤한 대상을 선택해서 공용 스킬을 부여한다.  
            2. 데이터 설정  
               1. unlock\_condition\_type\_1  
                  1.  \= Spawn\_Unit\_Epic\_Random (추가)  
                  2.  \= Spawn\_Unit\_Legendary\_Random (추가)  
               2. tactic\_target\_team \= Ourteam  
               3. tactic\_target\_grade \= Epic  
               4. effect\_type \= Add\_Skill  
               5. effect\_value \= skill\_id

         3. #### (완료)특정 유닛 지정 스킬 부여

            1. 의도  
               1. 현재 배치되어 있는 에픽, 전설 유닛 중 특정 유닛을 대상으로 지정하여 스킬을 부여한다.  
            2. 데이터 설정  
               1. unlock\_condition\_type\_1  
                  1.  \= Spawn\_Unit\_Fixed  
               2. unlock\_condition\_value1\_1 \= **unit\_id**  
               3. effect\_type \= Add\_Skill  
               4. effect\_value \= skill\_id

         4. #### (테스트 필요)지휘관 제외 & 아군 전체

            1. 의도  
               1. 지휘관 유닛을 제외한 나머지 유닛을 대상으로 스킬을 부여한다.  
            2. 데이터 설정  
               1. tactic\_target\_team \= Ourteam  
               2. is\_commander \= FALSE

         5. #### (테스트 필요)지휘관 포함 & 아군 전체 (예 : 흡혈 공격 추가)

            1. 의도  
               1. 지휘관을 포함한 전체 유닛을 대상으로 스킬을 부여한다.  
            2. 데이터설정  
               1. tactic\_target\_team \= Ourteam  
               2. is\_commander \= TRUE
