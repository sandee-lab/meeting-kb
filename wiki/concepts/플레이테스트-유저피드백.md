## 개요

미니크래프트는 초기 내부 테스트(2025.06)부터 소프트런칭 직전(2026.03)까지 사내 테스트, 정기 피드백 세션, 도쿄 게임쇼 현장 설문, FGT(Focus Group Test), NC 피드백, 로드컴플릿 외부 리뷰 등 다양한 채널을 통해 유저 피드백을 수집해왔다. 반복적으로 지적된 핵심 이슈는 시인성 부족, 랜덤성 과다, 전투 반복감/템포, 연출 딜레이, 보스 연출 임팩트 부족, 택틱 다양성 결여, 그리고 전설 유닛 사용성 문제였다. 전투 자체의 재미(몬스터 처치 쾌감, 유닛 조합)는 전 빌드에 걸쳐 긍정적 평가를 받았다.

## 상세 내용

### M5 플레이 테스트 설문 설계 (2025.07)

콴틱 파운드리 모델과 바틀의 플레이어 유형론을 기반으로 10개 문항의 설문을 설계하였다.
- 1부(프로필): 플레이어 동기 프로필 구축 (리커트 척도), 플레이어 유형 분류 (성취가/탐험가/사교가/킬러)
- 2부(경험 감사): 자유 연상, 긍정/부정 감정 최고점 탐색, 감정 곡선 그리기, 재플레이 의향
- 3부(레시피 검증): 핵심 판타지 인식 확인, 특정 시스템 평가, 최우선 개선점 도출
- 테스트 가이드: 최소 3회 플레이 권장, 1챕터 클리어 시 리셋 후 재플레이

### M5 플레이 테스트 결과 (2025.07) — 사내 11명 응답

#### 긍정적 피드백
- 높은 등급 초월유닛(고스트 등) 조합 후 몬스터를 쓸어버리는 쾌감
- 아슬아슬하게 웨이브를 버텨낸 뒤의 긴장감과 안도감
- 같은 유닛을 많이 모아 통일된 분대로 플레이하는 만족감
- 원하던 전설 유닛을 만들었을 때의 성취감
- 머지 결과로 원하는 유닛이 등장했을 때 운이 좋다는 느낌

#### 부정적 피드백
- 초월유닛 조합법을 바로 확인할 수 없는 불편함
- 유닛들이 너무 많이 겹쳐져서 내 캐릭터를 분간하기 어려움
- 레벨업 시 슬롯 연출로 전투의 맥이 끊기는 느낌
- 장착된 유닛이 죽었을 때 인벤토리 부족으로 자동 판매되는 문제
- 몬스터를 뚫고 지나가지 못하고 뒤로만 도망쳐야 하는 답답함
- 장판 지속 시간 때문에 길을 뚫어도 뚫린 것처럼 느껴지지 않음
- 게임이 복잡하여 피로도가 높다는 의견

#### 경쟁력 평가
- 아트 스타일과 귀여운 캐릭터에 대한 호평
- 오토배틀러와 달리 직접 조종하는 차별점이 경쟁 우위
- 그러나 같은 장르 게임과의 차별점을 찾기 어렵다는 의견도 존재
- 유닛 개수 한계로 다양한 조합이 나올 수 없다는 우려

### Alpha 1 빌드 피드백 (2025.01)

#### 전반적 감상
- 아트 스타일이 트렌디하고 고유한 색채가 있으며 젊은 유저에게 어필 가능
- 감각적 즐거움과 다수 몬스터 처리 재미가 잘 구현됨
- 게임 코어에 한 가지 새로운 요소 추가 시 완성도가 크게 향상될 것으로 예상
- 시스템 드로우에 게임 흐름이 결정되므로 유저가 운에 개입할 수 있는 요소 필요

#### 주요 이슈
- **속도감 문제**: 2배속 요구, 머지 연출 속도, 카드 쿨타임 후 생산 딜레이 등에 대한 답답함이 많음. 단, 전체 템포를 일률적으로 높이기보다 1-2-3성급 버프 효과를 늘려 순간적 템포 변화를 크게 주는 방향 제안
- **전장 시인성**: 기지와 유닛 간격이 멀어 한눈에 안 들어오는 문제. 유닛 생산 위치를 비행기 앞부분으로 이동하고 유닛-비행기 간 최대 이격도 설정 필요
- **밸런스**: 생산속도 증가에 올인해야만 클리어가 가능한 구조. 광폭화 타이머 표시 필요

### 도쿄 게임쇼 설문 결과 (2025.09)

일본어 설문 평균 4.39점, 영문 설문 평균 4.25점 (5점 만점 추정).

#### 연령대별 인사이트
- **몬스터를 빠르게 쓰러뜨리는 쾌감**: 모든 연령대에서 가장 인기 (20대 17회, 30대 13회, 40대 7회)
- **유닛을 합쳐 새로운 랭크를 만드는 재미**: 젊은 연령대에서 특히 선호 (20대 16회, 30대 11회)
- **맵의 미네랄 채굴 및 룰렛 보상**: 20대에서 두드러짐 (7회)
- **전략 카드 덱 구성**: 30대에서 상대적으로 높은 선호 (8회)

#### 낮은 점수(3점) 유저 인사이트
- 긍정: 적 처치 및 성장 시 즐거움, 핵 앤 슬래시의 재미, 화려한 이모트
- 부정: 성능 문제(렉), 모바일 조작 어려움, 초반 난이도 진입 장벽
- 개선 요청: 튜토리얼 및 설명 부족, 타격감/상쾌함 강화 필요

### 알파빌드 피드백 (2025.09)

#### 핵심 지적
- **한 판이 길게 느껴짐**: 경험 반복, 웨이브 변화 부족, 택틱 반복성, 작은 실패가 존재하지 않음
- **체력/난이도 이슈**: 일반 유닛은 안 죽고 SCV는 너무 잘 죽음. 포지션 고정으로 시인성 높이고 유닛이 잘 죽게 변경 제안
- **진행도 체감 부족**: 재미의 고점이 3챕터 이내에 너무 빨리 도달. 부대 배치 최초 4슬롯 → 3챕터 이후 8슬롯으로 확장 제안
- **전투 가시성**: 사정거리 표기 필요 (최대/최저 사거리 표현)
- **인게임 텐션**: 중간 보스 기점으로 무드 변화 필요, 악마의 거래 재고려, 1.5배속(광고), 일괄합성(광고)
- **유닛 언락**: 가챠 형태 수집 시 전설 카드를 얻지 못하면 재미 급감. 전설 유닛 1-2마리 시작 보유 제안

### FGT 커스텀 설문 설계 (2025.12)

10개 문항으로 구성된 FGT 설문:
1. 이 게임만의 독특한 매력 유무 (5단계)
2. 한 판에서 가장 재미있는 요소 (중복 선택: 전투/유닛 합성/조합 전략/전술 카드/랜덤 결과)
3. 게임을 멈추고 싶다고 느낀 순간 (서술형)
4. 게임오버 시 가장 먼저 든 생각 (억울/아깝다/궁금/지루)
5. 한 번 더 쓰고 싶거나 부족하다고 느낀 자원 (선택지 바꾸기/부활/소환 재화/랜덤 재시도)
6. 성장 선택지 다양성 (5단계)
7. 재플레이 의지 (5단계)
8. 재플레이 동기에 가장 큰 영향을 준 부분 (서술형)
9. 가장 큰 개선점 (서술형)
10. 친구에게 추천 시 소개 방법 (서술형)

### 내부 플레이 테스트 피드백 시계열 (2025.06~12)

> 출처: raw/feedback/ 폴더 내 12개 피드백 문서

내부 테스터들이 빌드마다 작성한 피드백 기록으로, 6월부터 12월까지 게임의 진화 과정과 반복 이슈를 추적할 수 있다.

#### 6월 11일 — 초기 프로토타입 (46항목)
- 카드 선택 중복 문제, 게임 템포 느림, 보스 체력 과다 (챕터1 기준)
- 유닛 합성/포지셔닝 전략 부족, 유닛 군집화 과다
- 사망 피드백 불명확, 필드 아이템 아이콘 가독성 낮음
- **운 경험 부족**: "운이 좋다는 경험을 인게임에서 자주 마주칠 기회가 있는 것이 좋겠음"
- **유닛 승급 의미 불명확**: "승급별 추가 효과가 명확해야 함 — 스킬이 쌓이는 느낌이 빠르게 다가오지 않음"
- **유닛 처분 불편**: 슬롯 꽉 찼을 때 유닛을 죽여야만 처분 가능 → 가스 소비 처분 요청
- **전장 진행상황 인지 어려움**: 전장의 진행 상태를 한눈에 파악하기 어려움
- 제안: 속도 향상, 탱커/딜러 고정 배치, 초월 유닛 시각 임팩트 강화, 보스 처치 후 보상 연출(도파민)

#### 7월 4일 — 템포/패턴 중심 (27항목)
- 전반적 게임 속도 느림, 한판이 너무 김
- 보스 패턴 회피 메커닉 필요 (바닥 공격이 플레이어 타겟 기준 생성 → 회피 불가)
- 피격 피드백(그래픽/진동) 부재, 공격 패턴 단조로움
- 몬스터 밀집 3-4분 후 혼란, 스킬 구분 어려움
- **운빨 부재 직접 지적**: "운빨좋많겜+로그라이크를 추구한다고 했는데 로그라이크는 좀 있는데 운빨은 없습니다"
- 유닛 컨트롤 답답함: 유닛이 한발짝 늦게 따라와 적 군체 회피 안 됨
- 제안: 지휘관 캐릭터 도입, 보스 바닥 패턴 사전 표시

#### 7월 11일 — 구조적 제안 (핵심 디자인 변경) ⚠️ 7월 방향성 확정에 직접 반영됨
- **웨이브제 전환 제안**: 시간 기반 → 웨이브 기반 (브로테이토 스타일) + 정비 페이즈 → 실제 채택
- **유닛 제한 제안**: 유닛 2종 + 지휘관으로 전략적 선명도 확보 → 부대 시너지 구조로 발전

### NC 피드백 정리 (2026.03)

#### 명확한 문제
- **전설 유닛 사용성**: 캐주얼 유저들은 직관적으로 높은 등급 유닛을 쓰고 싶어 하지만, 전설 유닛의 빌드업 과정이 너무 길다. 전설 유닛이 나오는 시점에 기존 유닛을 잡아먹고 나오기 때문에 덱 파워가 오히려 낮아지는 체감. 전설 유닛 자체의 파워 강화가 필요하며, 낮은 등급 파워를 내리고 승급 조건을 쉽게 하여 빠르게 전설로 넘어가게 하는 방안 제안
- **계승 전설 아이디어**: 같은 유닛을 모아 전설로 승급하고, 강화 카드 효과를 모두 받을 수 있는 시스템
- **스테이지 단조로움**: 어떤 방식으로든 해결 필요

#### 불확실한 사항
- 슬롯 제한: 4개 시작 가능하나 너무 적으면 조합 단조로움 문제. 순차적 개방은 가능
- 카드 메타 구조 전환: 가능하면 좋겠지만 이미 시도했으며 밸런스 측면에서 어려움

- **진행 바/남은 시간 가시성**: 전투 집중 중 상단 UI 인지 불가 → 갑자기 웨이브가 끝나는 느낌
- **전설 유닛 속성 표기 미흡**: 속성 아이콘 터치 시 설명/미리보기 필요 (테오크처럼 예시 애니메이션)
- **등급 색상 혼란**: 웨이브 클리어 업그레이드 화면에서 베이지/녹색/파란색 등 등급 구분 어려움
- **소환 화면 UX**: 보석으로 추가 선택 불가, 자동 다음화면 전환 없음, 박스 드롭 정보 부족
- **속성 마킹 사라지는 버그**: 번개 유닛끼리 연결 시 번개 아이콘 사라짐
- **채집 속도**: 전투화면 다이아 채집 2초를 선형 → 로그 스케일로 개선 제안
- **스테이지 난이도**: 개체수 밀도 변화, 속도 다양성 필요
- **화면 전환 피로도**: 웨이브 10회 + LV업 팝업으로 1회 플레이가 너무 길게 느껴짐
- **카드 도감 개선**: 일반/희귀/영웅/전설 고정 탭 + 확보순 자동 소팅
- **소환 연출 도파민 부족**: 팡 터지는 느낌 없음
- **업그레이드 UI**: 탕탕/존망겜과 동일한 UI — 차별화 필요
- **본체 강화 콘텐츠 필요**: 전투 시작마다 제로에서 시작 → 본체 누적 성장 요소 필요
- **레벨업 슬로우 모션**: 이펙트가 아닌 렉처럼 느껴져 제거 요청
- **이어하기 기능 필요**: 실패 후 처음부터 재시작 강요

#### 공통 결론
유저는 전투 기본 재미는 긍정적이나, 잦은 화면 전환과 흐름 끊김(연출 딜레이), UI 불친절함으로 인해 몰입이 방해받고 있음. FGT 유저 분석(2026.01)의 내용과 일치.
