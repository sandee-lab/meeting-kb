# 아웃게임 메뉴

> **요약:** 로비 화면의 아이콘 배치 규칙과 햄버거 메뉴 구조를 정의한다. 좌측에는 종료 일자가 있는 기간제 콘텐츠(멤버십, 배틀패스 등), 우측에는 상시 열람 빈도가 높은 콘텐츠(퀘스트, 랭킹)를 배치한다. 햄버거 메뉴는 3열 F형 시선 동선 기준으로 우편-친구-퀘스트/출석부-도감-랭킹/설정 순서로 정렬하며, 중앙 배너 영역과 순찰 게이지 표현도 포함한다.
> **현재 상태:** 미적용

### 좌측 바로가기

* + 종료 일자가 세팅되는 콘텐츠를 **배치**하는 것을 기본으로 한다.
  + 좌측 바로가기 아이콘은 햄버거 메뉴 내부에 포함되지 않는다.
* 우측 바로가기
  + 상시로 열린 콘텐츠 중 열람 빈도가 높은 콘텐츠를 우측에 배치한다.
  + 우측 바로가기 아이콘은 햄버거 메뉴 내부에도 포함된다. (cf. 테오크)

### 정렬 규칙

* + - 햄버거 메뉴 내부는 3열로 구성된다.
    - F형 시선 동선을 기준으로, 유저가 가장 자주 찾는 기능을 배치한다.
      * 위→좌측→우측 순으로 배치
      * 1행은 게임 플레이 루프 핵심: 우편(각종 보상), 퀘스트(보상), 친구(스태미나)
      * 2행은 조회/기록 중심: 랭킹, 도감
      * 옵션은 맨 끝/맨 아래가 UI 표준 → 최하단 오른쪽

## 중앙 배너

* 첫 구매 이벤트, 특별 패키지 등의 구성을 고려하여 배너 위치를 비위둔다.
* 배너는 개발 시, 전용 테이블에서 관리한다.
* 배틀패스는 중앙에 표시하는 것을 고려한다.
  + 관리 항목
    - 이미지
    - 텍스트
    - 순서
    - 종료 기준
      * 없음
      * 상품 구매 (상품 구매 시 제거, 해당 상품 ID 연결)
      * 이벤트 종료 (이벤트 종료 시 제거, 해당 이벤트 ID 연결)
    - 클릭 시 이동할 곳 (UI)
