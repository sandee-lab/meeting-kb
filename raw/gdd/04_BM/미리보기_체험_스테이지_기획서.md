# 미리보기 체험 스테이지 기획서

> **요약:** 패키지 판매 상품(전설 유닛, 장비)의 강점을 유저가 직접 체험할 수 있는 전용 쇼케이스 스테이지 시스템이다. 상점 패키지 상세 화면에서 진입하며, 기획자가 사전 세팅한 고정 덱으로 3웨이브를 플레이한다. 체험 콘텐츠 특성에 맞춰 적 웨이브를 의도적으로 설계하여 상품 가치를 극대화하고, 체험 완료 후 구매 유도 화면을 노출한다. 실제 스펙보다 강화된 수치 적용은 금지하며 보상은 미지급된다.
> **현재 상태:** 미적용

# **미리보기 체험 스테이지 기획서**

## **1. 개요**

### **1.1 컨셉**

패키지 판매 상품(전설 유닛, 장비)을 위한 전용 체험 스테이지. 해당 콘텐츠의 강점이 극대화되는 상황을 설계하여, 유저가 "이 유닛/장비가 얼마나 멋진지"를 직접 느끼게 한다.

핵심은 **쇼케이스**다. 유저 덱에 끼워넣는 게 아니라, 해당 콘텐츠가 주인공인 무대를 제공한다.

### **1.2 기대 효과**

* 상품 가치의 직관적 전달 (텍스트 설명 → 체감)
* 구매 전 불확실성 제거
* 콘텐츠 자체가 마케팅 소재로 기능

## **2. 체험 스테이지 구조**

### **2.1 기본 설정**

| **항목** | **내용** |
| --- | --- |
| 진입 조건 | 상점 패키지 상세 화면에서 [체험하기] 선택 |
| 스테이지 길이 | 3웨이브 (1~2분 내외) |
| 덱 구성 | 사전 세팅된 고정 덱 (체험 콘텐츠 포함) |
| 난이도 | 체험 콘텐츠가 빛나는 수준으로 튜닝 |
| 보상 | 없음 |

### **2.2 덱 구성 원칙**

유저 덱을 쓰지 않는다. 체험 스테이지 전용으로 **기획자가 짜놓은 덱**을 제공한다.

**구성 의도**

* 체험 콘텐츠가 핵심 딜러/핵심 역할
* 나머지 유닛은 보조 역할 (서포터, 탱커 등)
* 시너지 태그가 체험 콘텐츠와 맞물리도록 설계

## **3. 스테이지 연출 설계**

### **3.1 적 웨이브 설계 원칙**

체험 콘텐츠의 강점이 드러나는 상황을 **의도적으로** 만든다.

| **콘텐츠 특성** | **웨이브 설계** |
| --- | --- |
| 광역 공격 | 밀집 소형몹 대량 출현 |
| 단일딜 | 고체력 보스 단독 출현 |
| 속성 특화 | 해당 속성에 약한 적 위주 |
| 생존기 보유 | 강력한 즉사기 패턴 → 생존기로 회피 |
| 버프/디버프 | 장기전 상황에서 누적 효과 체감 |

## **5. 콘텐츠별 체험 스테이지 예시**

### **예시 1: 전설 유닛 "심연의 냥이" (단일 타겟 암살자)**

**콘셉트**: 보스를 순삭하는 쾌감

| **웨이브** | **적 구성** | **의도** |
| --- | --- | --- |
| 1 | 중형몹 3기 | 워밍업, 기본 공격 체감 |
| 2 | 엘리트 1기 | 고딜 스킬 발동, 빠른 처리 |
| 3 | **보스** (암살 대상 속성) | 궁극기로 보스 체력 40% 즉삭 연출 |

**연출 포인트**

* 3웨이브 보스전에서 궁극기 발동 시 암전 + 슬래시 이펙트

### **예시 2: 전설 장비 (범위 공격 강화)**

**콘셉트**: 화면을 뒤덮는 번개의 쾌감

| **웨이브** | **적 구성** | **의도** |
| --- | --- | --- |
| 1 | 소형몹 20기 밀집 | 범위 공격 한 방에 쓸림 |
| 2 | 소형몹 30기 + 중형 5기 | 연쇄 번개 효과 체감 |
| 3 | 대량 러시 | 화면 가득 번개, "100 HITS" 카운터 |

**연출 포인트**

* 카메라 줌아웃 및 번개 이펙트 풀스크린 연출
* 히트 수 누적 카운터 대형 표시

### **예시 3: 전설 유닛 "수호자 냥이" (탱커/생존 특화)**

**콘셉트**: 절대 안 죽는다

| **웨이브** | **적 구성** | **의도** |
| --- | --- | --- |
| 1 | 일반 공격 다수 | 기본 방어력 체감 |
| 2 | 고데미지 엘리트 | 피격 시 보호막 발동 |
| 3 | 보스 즉사기 패턴 | 무적기로 회피 → "IMMUNE" 연출 |

**연출 포인트**

* 보스 즉사기 발동 직전 경고 → 무적 발동 → 화려한 피격 이펙트 → 생존

## **6. 제한 및 운영**

### **주의사항**

**과장 금지**

* 체험 스테이지가 유리하게 설계되어 있다는 건 괜찮음
* 그러나 **실제 스펙보다 강화된 수치 적용은 절대 금지**
* 유저가 구매 후 "체험이랑 다르잖아"라고 느끼면 신뢰 붕괴

**보상 미지급**

* 체험 스테이지 클리어 보상 없음
* 퀘스트/업적 진행 미반영
* 순수하게 "느껴보기"만을 위한 공간
