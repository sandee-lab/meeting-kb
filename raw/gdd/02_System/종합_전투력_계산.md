# 종합 전투력 계산 설계

> **요약:** 공격력·생존력 기반 유저 종합 전투력 계산 공식을 설계한 문서. 치명타 계수, 공속 계수를 포함한 수식을 정의한다.
> **현재 상태:** 이후 버전으로 대체됨

종합 전투력

# 방향

* 실제 전투 공식을 최대한 활용하여 전투력을 계산한다.
* 전투에 간접적으로 영향을 주는 능력치들은 별도의 가중치를 반영해서 소급 적용 한다.
* 아웃게임에서 적극적으로 쓰지 않는 능력치들은 계산에서 제외한다. (추후에 추가 가능)

## 종합 공격력

### 의도

* + - 피해량 증가에 직접적으로 영향을 주는 주요 능력치를 기준으로 공격력을 계산한다.
    - 특정 상황에만 적용되는 추가 피해는 제외하고, 상시 적용되는 능력치를 기준으로 적용한다.

### 대상 능력치

* + - 계산 공격력
      * = (공격력 합산) \* (1 + 공격력%합산) \* (1 + 공격력%합산) ….
    - 치확 공격력
      * = 치확 \* (계산 공격력 \* (기본 치피 \* ( 1 + 치피 증가))- 계산 공격력)
    - 공속 계수
      * = (1 + mul\_atk\_speed) \* (1 + mul\_atk\_speed\_1)
    - 쿨속 계수
      * = (1 + mul\_skill\_recharge\_speed) \* (1 + mul\_skill\_recharge\_speed\_1)

### 종합 공격력 계산

* + - = (계산 공격력 + 치확 공격력) \* 공속 계수 \* 쿨속 계수

## 종합 생존력

### 의도

* + - 생존 시간에 영향을 주는 주요 능력치를 기준으로 생존력을 계산한다.
    - 생존 시간은 여러 상황에 따라서 효율이 달라지므로 가중치 값을 적용해서 보정할 수 있도록 설계한다.

### 대상 능력치

* + - 계산 생존력
      * ((생존력 합산) \* (1 + 생존력%합산)\* (1 + 생존력%합산)... \* 생존 가중치)
      * 생존 가중치 = 0.2 (constantdatas)
    - 회피 계수
      * (1 + (회피율%합산\*회피율 가중치))
    - 저항 계수
      * (1+ (피해 저항%합산\*저항 가중치치))
    - 회복 계수
      * (1 + (회복강화% 합산\*회복 가중치))
    - 이동 계수
      * (1+(mul\_move\_speed\* weight)) \* (1+(mul\_move\_speed\_1 \*weight))
      * 이동 가중치 = 0.1 (constantdatas)

### 종합 생존력 계산

* + - = 계산 생존력 \* 회피 계수 \* 저항 계수 \* 회복 계수 \* 이동 계수

## 유닛 계수

### 의도

* + - 유닛 레벨에 따라서 유닛의 계수가 증가하는 것은 직접적으로 공격력과 생존력에 영향을 주므로 전투력에 포함할 수 있어야 한다.
    - 기본 제공되는 유닛의 레벨 합산에 따른 계수 성장율을 고려해서 전투력에 반영한다.

### 유닛 계수 계산

* + - = 기본 계수 + (( 유닛 레벨 합산 \* 레벨 성장율 ) / 기준 마리수)
      * 기본 계수 = 1
      * 레벨 성장율 = 0.05
      * 기준 마리 수 = 25

## 최종 전투력 계산

### 공식

| 종합 전투력 = (종합 공격력 + 종합 생존력) \* (유닛 계수) |
| --- |

### 갱신

* + - 아웃게임 성장을 통해서 유저의 능력치가 갱신되는 시점마다 전투력을 갱신한다.

### 변경 알리미

* + - 갱신 시 항상 변경 전 전투력 값을 별도로 남겨서 전투력의 변동을 알 수 있도록 한다.

# 권장 전투력

## 대상 컨텐츠

* + 각 컨텐츠 별로 권장 난이도를 가이드하기 위해서 권장 전투력 수치를 제공한다.

## 권장 전투력 계산

* + 권장 전투력은 데이터시트를 통해서 값을 직접 입력한다.

| ***Note)***  유저 전투력 초기 값을 기준으로 스테이지 모드 1챕터의 전투력을 설정한다. 이후 난이도의 전투력은 각 단계 별 난이도 증가율 %에 따라서 동일하게 권장 전투력 수치를 계산한다. |
| --- |

## 난이도 가이드

* + 내 전투력, 권장 전투력 값을 계산한 결과에 따라서 난이도를 가이드 한다.

| 계산 결과 | 난이도 가이드 |
| --- | --- |
| 내 전투력 / 권장 전투력 >= 1.25 | **쉬움** |
| 내 전투력 / 권장 전투력 <= 0.75 | **어려움** |
| 0.75 < 내 전투력 / 권장 전투력 < 1.25 | **보통** |
