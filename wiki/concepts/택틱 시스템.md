# 택틱 시스템

> 출처: [[raw/[0305] 신규 마일스톤[m1]]], [[raw/[0820] 퀄리티업 이슈]], [[raw/[0829] M7 1주차 테스크 정리]], [[raw/[0919] M7 4주차 테스크 정리]], [[raw/[1013] M8 추가 이슈 회의록]], [[raw/[1022] M8 전략성 개선 회의]], [[raw/[1125] 튜토리얼 및 택틱 정리]], [[raw/[0319] 3월 2-3주차 작업 리스트]]

## 개요

택틱은 인게임에서 레벨업 시 선택할 수 있는 전략 카드 시스템으로, 유닛 강화/스킬 획득/버프 등의 효과를 제공한다. 초기에는 로그라이크 스타일의 드로우 시스템으로 출발했으나, 개발 과정에서 덱 빌딩 방향성 제공, 시각적 변화 강조, 전설 유닛과의 연계 등으로 역할이 지속적으로 재정의되었다. 택틱은 유닛 조합을 보조하는 역할에 집중하되, 시각적으로 직관적인 성장 체감을 제공하는 방향으로 설계되고 있다.

## 상세 내용

### 초기 설계 (2025년 3월)

M1 마일스톤에서 택틱의 기본 구조가 정의되었다.

- 몬스터 처치 시 경험치를 획득해 레벨업하면 택틱 드로우 기능 사용
- **주요 택틱 유형:**
  - 평타 강화
  - 신규 스킬 획득 / 강화
  - 유닛 소환 / 강화
  - 버프
- **히든 택틱:**
  - 궁극기 혹은 시너지 스킬 (2~3개) — 계열 확인용
  - 부대 전투 강화 — 유닛 수에 따른 공격력 강화
  - 특수 유닛 강화 — 단일 유닛 중복 강화 시 궁극기 생성

3월 2-3주차 작업 리스트에서는 **택틱 스마트 밴 시스템**이 추가되었다.
- 레벨업 단계에서 택틱 리롤 시, 한번 리롤한 택틱은 리롤 결과로 나오지 않음
- 등급 내 모든 택틱이 밴되면 해당 등급 내 택틱 밴이 모두 풀림
- 리롤 횟수가 쌓일수록 원하는 카드를 찾을 확률이 선형적으로 상승

또한 **택틱 등급 확률 동적 제어** 시스템도 설계되었다.
- 포메이션에 배치한 유닛의 등급 평균이 높아질수록 높은 등급 카드의 출현 확률 증가
- 높은 등급 유닛의 스킬 카드를 모으기 어려운 문제를 개선하기 위한 목적

### 택틱 최적화 논의 (2025년 8월)

[0820] 퀄리티업 이슈에서 택틱 시스템의 대규모 개선안이 논의되었다.

- **덱 빌딩 연동:** 유저의 현재 덱을 분석하여 적절한 택틱을 확률 상향하여 제공
  - 매직 유닛이 많으면 관련 택틱 확률업
  - 진영 선택 시 해당 위치 확률 증가
  - 전략성 카드에만 적용
- **컨셉별 택틱 추가:**
  - 잡덱: 서로 다른 유닛 1마리당 대미지 5% 증가
  - 넉백: 넉백 1마리당 쉴드 10 생성
  - 원유닛: 같은 유닛 버프 효율 50% 증가
  - 마덱, 물덱 등
- **UX 개선:**
  - 택틱 레벨 표기 (기존에 한번 선택한 택틱 인지 어려운 문제 해결)
  - 택틱 추천 마크 표시
  - 택틱으로 아이템 획득 기능 (패널티 존재하는 아이템, 판매 가능)

### M7 스프린트 개선 (2025년 8~9월)

[0829] M7 1주차에서는 택틱 관련 핵심 코멘트가 정리되었다.

1. 택틱에 시각적 변화를 추가한다
2. 탄환수 증가 같은 밸런스가 어려운 요소는 배제한다
3. 덱 빌딩에 기준이 되는 스킬과 연관 시너지를 셋트 형태로 정리한다 (추후 아웃게임 컨텐츠로 연결)
4. 택틱 리롤 기능을 다시 살린다 (사용 재화 관련 테스트 필요)
5. 게임 시작 시 지휘관이 가진 택틱 기준으로 연출이 필수

피드백에서는 택틱 관련 추가 의견이 나왔다.
- 좋은 택틱 획득 시 연출 강화 필요
- 마법/피지컬 유닛 구분 표현 부족 — 유닛카드에 속성 표기 또는 택틱 페이지에서 적용 유닛 표시
- 공격력 강화 경로 복잡 — 단순화 제안
- 단일 유닛 시너지 표현 강화 (예: 방패병 5개 시 방패 크기 증가)
- 다양한 유닛 조합 시너지 부재 문제

[0919] M7 4주차에서는 택틱 획득 방식과 진화 시스템이 논의되었다.

- **택틱 획득 방식 변경:** 일반적인 로그라이크와 동일하게 접근
  - 경험치(가스) 획득 시 레벨업 누적
  - 레벨업 보상은 웨이브 종료 후 처리
- **택틱 진화 시스템 (신규):**
  - 특정 전설 유닛과 연계되는 택틱으로, 해당 유닛에만 적용
  - 기존 전설 유닛 획득 후 스킬 제공 방식은 빌드업 도파민과 대치되어 재미 반감
- **덱 방향성 루트 설정:**
  - 스탯 택틱을 3~5회 강화 시 특정 전설 유닛 전용 택틱 획득 가능
  - 스피드 덱, 화염 덱, 체력 덱 등 다양한 컨셉 제공
  - 덱의 주축이 되는 택틱과 전설 유닛을 쉽고 재밌게 고를 수 있게 구성

### M8 전략성 개선 (2025년 10월)

[1013] M8 추가 이슈에서 택틱 리롤 기능이 우선순위 높은 항목으로 지정되었고, 신규 택틱 추가도 계획에 포함되었다.

[1022] M8 전략성 개선 회의에서 택틱 구조가 대폭 재설계되었다.

- **택틱 종류를 [스택]과 [단일] 두 가지로 구성:**
  - **[스택]:** 3단계로 진화 (일반 → 고급 → 레어), 마지막 단계에서 스페셜 택틱 추가 획득
    - 스페셜 택틱은 새 카드가 아니라 기존 성능을 과감히 올려주는 형태 (즉시성)
    - 스택형 택틱 선택은 리스크 감수, 이후 리턴이 큰 구조로 도파민 제공
  - **[단일]:** 눈에 띄는 즉각적 효과
    - 공격 형태에 변화를 주는 택틱 추가 필요
    - 지휘관 및 단일 유닛 대상 택틱 고려
    - 전설 유닛 획득 후 관련 택틱 등장
- **배치 연동 아이디어:**
  - 배치 위치 관련 택틱이 사라지면서 전략적 선택이 줄어든 문제
  - 배치된 유닛 조합에 따라 지휘관이 스킬을 상속받는 방식 제안
  - 예: 불속성 유닛 4마리 연속 배치 시 지휘관 근처에 불의 고리 생성
  - 예: 방패병 2마리 + 불속성 2마리 유닛이 붙어있으면 방패에 불 효과 추가
- **특정 택틱으로 유닛의 무기 타입/형태를 변화시키고, 해당 유닛을 히든 조합식으로 활용** 가능하게 하는 아이디어도 제시됨

### 튜토리얼 및 택틱 종합 정리 (2025년 11월)

[1125] 튜토리얼 및 택틱 정리에서 택틱의 역할과 구체적 효과가 상세하게 정리되었다.

- **튜토리얼에서의 택틱:** 첫 레벨업 시 3가지 택틱 중 선택 (확실하게 체감되는 스킬로 픽스)
- **한판 재미 사이클:** 웨이브 한번에 택틱 2~3개 선택, 특정 이벤트에 여러개 동시 지급 후 몬스터 몰려오게 하여 체감
- **택틱 효과 상세:**
  - 범위 증가: 속성별 적용도 가능, 좀 더 적극적으로 체감되게 적용
  - 공속 및 쿨타임 증가: 통합 후 최대 2~3배 이상 속도 증가로 체감
  - 투사체 증가, 사이즈 증가: 즉시 눈에 보이는 효과
  - 타격 시 추가 효과: 얼음 파편, 불꽃 대미지, 중력장 효과, 기절 등 (속성별 적용, 상태이상 확산)
  - 속성 강화: 속성 + 스탯 조합 (얼음+체력, 불+치명타, 암흑+이동속도 등)
  - 효과 인지: 버프 효과는 전투화면에 표시, 평점 표기

- **유닛 무기 진화(EVO) 시스템:** 기본 무기에서 진화 무기로 업그레이드되는 체계가 정리됨
  - 테슬라 캐논 → 반물질 캐논
  - 추적 폭죽 → 어벤저
  - 바운서 → 재퍼
  - 공습 I → 파괴자 I (외 다수)
  - 부메랑 발사기 → 스핀 블레이드
  - 펀치 발사기 → 라이트닝 펀치
  - 헬기 폭탄 → 뉴클리어 스트라이크
  - 윙윙 덫 → 매시브 라이트닝

- **전략성 보완 방향:**
  - 유닛이 곧 스킬이라면, 진화 시 스킬 형태를 이어받아야 전략적으로 느낄 수 있음
  - 랜덤 요소가 있더라도, 목표하는 전략 완성을 위한 의도된 플레이는 유지되어야 함
  - 기본 유닛 강화가 전설 효과로 이어지는 구조 필요

### 2026년 3월 추가 시스템

[0319] 3월 작업 리스트에서는 택틱의 스마트 밴과 등급 동적 제어가 구체화되었다.

- **무속성 유닛을 특정 속성으로 바꾸는 택틱**을 통해 속성 시너지를 강화하는 방향이 추가됨
- 전술 카드 선택 시 렉 걸리던 문제 개선 완료
