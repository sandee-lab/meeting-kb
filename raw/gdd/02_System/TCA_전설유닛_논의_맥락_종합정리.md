# TCA 전설 유닛 사용성 개선안 — 논의 맥락 종합 정리

이 문서는 전설 유닛 사용성 개선안(v0.1~v0.5)을 논의하며 확정된 결정사항, 논의 과정의 핵심 논리, 미해결 과제를 정리한 맥락 전달용 문서입니다.

---

## 1. 프로젝트 개요

- **게임:** TCA (Tiny Cat Assemble) — 머지 + 로그라이크 + 오토배틀 하이브리드 캐주얼 모바일 게임
- **퍼블리셔:** NC
- **개발사:** Super Rabbit Games
- **부대 슬롯:** 8칸
- **핵심 메커니즘:** 같은 유닛 × 2 머지 → 상위 등급 (일반→고급→희귀→영웅→전설)
- **속성:** 무속성, 전기, 화염, 냉기, 카오스(암흑)
- **공격 빈도 계열:** 공격 속도(공속), 쿨타임
- **속성-계열 매칭:** 전기/냉기 = 공속, 화염/카오스 = 쿨타임

## 2. 유닛 구성 (확정)

### 2.1. 등급별 유닛 수

| 등급 | 종수 | 추가 계획 |
|------|:----:|----------|
| 일반 (Common) | 4 | 없음 |
| 고급 (Uncommon) | 6 | 없음 |
| 희귀 (Rare) | 6 | 없음 |
| 영웅 (Epic) | 5 | 없음 |
| 전설 (Legendary) | 14 (엣지 변경 예정 포함) | 장기적으로 50종까지 확장 가능 |

**일반~영웅 등급의 유닛 추가 계획은 없음. 현재 구성이 최종.**

### 2.2. 유닛 리스트

**일반:** 클리버 주니어(무속성/공속), 피스톨 주니어(전기/공속), 파이어 주니어(화염/쿨타임), 피즈 주니어(냉기/공속)

**고급:** 클리버(무속성/공속), 슬러그(전기/공속), 파이어(화염/쿨타임), 피즈(냉기/공속), 펀치(카오스/쿨타임), 캐논(무속성/쿨타임)

**희귀:** 라이플(전기/공속), 플레임(화염/쿨타임), 쿠나이(냉기/공속), 벤시(카오스/쿨타임), 로켓(무속성/쿨타임), 쉴드(무속성/공속)

**영웅:** 트루퍼(전기/공속), 라이더(화염/쿨타임), 아이시클(냉기/공속), 어보이드(카오스/쿨타임), 엔지니어(무속성/쿨타임)

- 엔지니어는 원래 고급이었으나 **영웅으로 등급 변경 확정**
- 변경 이유: 전설 유닛 조합식의 다양성 확보

### 2.3. 엔지니어 등급 변경 영향 (시뮬레이션 30만회)

| 목표 | 변경 전 (고급7/영웅4) | 변경 후 (고급6/영웅5) | 변화 비율 |
|------|---------------------|---------------------|----------|
| 아무 영웅 도달 | 17.8회 | 17.2회 | 3.4% 빨라짐 |
| 특정 영웅 획득 | 39.4회 | 45.3회 | 15.0% 느려짐 |
| 영웅 적중 확률 | 25% | 20% | 20.0% 감소 |

### 2.4. 전설 조합식

- 1전설 = 1조합식 (1:1 대응)
- 유형: 에픽+희귀, 같은 에픽×2, 희귀+희귀, 에픽+고급×2 등 혼합 운영
- 현재 풀(고급6+희귀6+영웅5)로 가능한 조합 경우의 수: 161가지
- **전설 50종까지 충분히 대응 가능**

## 3. 전설 유닛 설계 원칙 (확정)

### 3.1. 기본 성능 50% + 시너지 성능 50%

- 단독 소환 시에도 효능감이 있어야 함
- 조합 완성 시 성능이 극대화되는 구조
- 이 원칙에 미달하는 유닛: 간호순, 포어맨 → To-Be 설계 완료

### 3.2. 10점 전설의 이중 효과 구조

모든 10점(무속성) 전설은 두 가지 효과를 동시에 갖는다:
1. **하위 유닛 조합 효과** — 특정 하위 유닛과 함께할 때 시너지 발동
2. **범용 버퍼 효과** — 아군 전체에 적용되는 버프 (속성 전환 후에도 유지)

---

## 4. 10점 전설 유닛 역할 (확정)

### 4.1. 기본 루트 — 4종

첫 전설이 곧 플레이스타일 선택. 공격/생존 × 공속/쿨타임 2×2 대칭 구조.

| 계열 | 유닛 | 하위 유닛 | 범용 버프 | 스타일 |
|------|------|----------|----------|--------|
| 공속 | **고양이 사부** (신규) | 클리버 공속 증폭 | 아군 전체 공속↑ | 공격형 |
| 공속 | **캡틴** | 실드 추가 방패 | 아군 전체 생존↑ (공격 시 보호막 획득) | 생존형 |
| 쿨타임 | **시즈** | 캐논 추가 포탄 | 아군 전체 쿨타임↓ | 화력형 |
| 쿨타임 | **메카** | 로켓 유도 미사일 | 아군 전체 피격 시 반격 | 생존형 |

**대칭:**
- 공속: 공격이 방어를 만듦 (캡틴 — 때리면 보호막)
- 쿨타임: 방어가 공격을 만듦 (메카 — 맞으면 반격)

### 4.2. 소환 옵션 — 2종

기본 루트와 달리 **독립 루트가 아닌, 자유 슬롯에 투입하는 유틸리티**.

| 유닛 | 계열 | 하위 시너지 | 범용 시너지 | 자체 DPS |
|------|------|-----------|-----------|---------|
| **포어맨** | 공속 | 엔지니어 터렛 강화 | [소환수] 전체 공속↑ | 자체 터렛 |
| **오퍼레이터** | 쿨타임 | 엔지니어 터렛 강화 | [소환수] 소환 쿨타임↓ | 자체 로보 |

- 소환수 없는 덱에서도 자체 DPS 기여 (기본 성능 50%)
- 소환수 많을수록 시너지 극대화
- **카오스 덱(소환수 기믹)의 정석 파트너**로 오퍼레이터 활용
- 포어맨은 계열 불일치(공속)이지만 소환수 시너지가 강력하면 감수 가치 있음

### 4.3. 고양이 사부 / 엣지 변경 이력

- **기존:** 엣지가 공속 공격형 기본 루트 담당
- **변경:** 고양이 사부가 엣지의 역할을 대체 (신규 유닛)
- **엣지:** 추후 전기 속성 전설 유닛으로 재설계 예정 (메커니즘/시너지/점수 미정)
- **변경 이유:** 엣지를 전기 속성 전설 포지션으로 활용하기 위해

### 4.4. 소환 덱 논의 경과

소환 덱을 독립 루트로 만들려는 시도가 있었으나 아래 이유로 기각:
- 엔지니어가 고급(→영웅) 등급이라 덱 완성 난이도가 너무 낮음
- 소환수 등급 체계를 새로 만들면 배보다 배꼽이 커짐
- 전설 4슬롯(포어맨+오퍼레이터+공속버퍼+쿨타임버퍼) 필요 → 비현실적

**최종 결론:** 포어맨/오퍼레이터는 자유 슬롯 유틸리티로 포지셔닝. 소환수 재미는 유지하되 독립 루트가 아닌 옵션으로.

## 5. 속성 전환 시스템 (확정)

### 5.1. 메커니즘

- 게임 내 전략 카드 중 희귀 확률로 "무속성 → 특정 속성 전환" 카드 등장
- 유저가 선택하면 **모든 무속성 유닛이 해당 속성으로 전환**
- 한번 선택 시 속성 전환 카드는 더 이상 등장하지 않음

### 5.2. 전환의 의미

| 구분 | 전환 전 | 전환 후 |
|------|---------|---------|
| 자체 능력 | 유지 | 유지 (동일) |
| 속성 시너지 호환 | 못 받음 | **받을 수 있음** |
| 속성 디버프 부여 | 못 함 | 못 함 (동일) |

**핵심:** 속성 전환 = "속성 팀 소속 등록". 냉기 스킬을 얻는 게 아니라 냉기 버프/디버프의 수혜 대상이 되는 것.

### 5.3. 전환 후 덱 동기

전환 후에도 클리버 등 하위 유닛은 디버프를 거는 능력이 없으므로, "버프/디버프를 거는 유닛이 필요하다" → 클리버를 냉기 전용 유닛으로 교체하는 동기가 자연 발생.

---

## 6. 덱 전환 흐름 (확정 — 공속 고양이 사부 루트)

| Phase | 덱 상태 | 미네랄 용도 |
|-------|---------|-----------|
| 1. 초반 | 고양이 사부 + 클리버5 + 광부 + (빈 슬롯) | 유닛 구매/리롤 |
| 2. 중반 | 고양이 사부 + 클리버5 + 드릴 + 청소부 | 유틸 조합 |
| 3. 속성 전환 | 전체 냉기화 | 비축 (광부 수급) |
| 4. 후반 | 냉기 사부 + 섀도(14점) + 냉기 딜러들 + 드릴 + 청소부 | 냉기 유닛 구매 |
| 5. 완성 | + 프로스트(24점) 합류 | 조합 비용 |
| 6. 최종 압축 | 드릴/청소부 → 딜러로 교체. 풀 DPS | 딜러 교체 (잉여 소모) |

**미네랄 잉여 구간이 없음.** 매 구간마다 소모처 존재.

### 6.1. 무속성 덱 → 속성 덱 전환의 배경 논의

- **문제:** 무속성 조합 덱(클리버7+엣지)이 중반에 너무 빨리 완성되어 할 게 없음
- **등급 부스트:** 존재하지만 재미 없음
- **검토한 방향:** 전설 유닛 강화 장치(인챈트/변이/각성) → 기각. 무속성 덱을 더 깊게 파는 건 방향이 틀림
- **채택한 방향:** 무속성 덱은 속성 덱의 하위 호환. 자연스러운 전환 동선을 제공하는 게 맞음
- 전환 후에도 기본 루트 전설(고양이 사부)의 범용 버프가 유지되므로 교체 불필요

### 6.2. 최종 압축 딜레마

| 선택 | 이득 | 리스크 |
|------|------|--------|
| 청소부 유지 | 생존 안정 | DPS 1슬롯 손해 |
| 청소부 제거 | DPS↑ | 보스 패턴 전멸 |
| 드릴 유지 | 미네랄 소량 수급 | DPS 1슬롯 손해 |
| 드릴 제거 | DPS↑ | 미네랄 파이프 차단 |

콘텐츠별 최적: 공허의 균열(무한 모드) → 청소부 유지, 보스 레이드 → 풀 압축.

## 7. 속성 유닛 구조 (확정)

속성 유닛은 14점-24점 쌍으로 존재:

| 속성 | 계열 | 14점 (속성 시너지 시작) | 24점 (덱 완성) |
|------|------|----------------------|--------------|
| 전기 | 공속 | 스카우트 | 볼트 |
| 냉기 | 공속 | 섀도 | 프로스트 |
| 화염 | 쿨타임 | 간호순 | 헬파이어 |
| 암흑/카오스 | 쿨타임 | 미스테리오 | 고스트 |

### 7.1. 빙결술사 = 프로스트

초기 논의에서 빙결술사와 프로스트가 별도 유닛으로 오인됨. 동일 유닛 확인 후 빙결술사 제거. 프로스트 = 24점 냉기 공속 (v0.3에서 수정).

---

## 8. 신규 유닛 (확정 — 3종)

초기에 5종(고양이 사부, 입자 가속기, 슈뢰딩거 박스, 청소부, 광부)을 제안했으나, 역할 재설계 과정에서 조정:

- **고양이 사부:** 엣지의 기본 루트 역할 대체 → 전설 리스트에 포함 (신규이나 기본 루트 유닛)
- **입자 가속기:** 제거. 시즈가 쿨타임 버퍼 역할을 대체.
- **슈뢰딩거 박스:** 유지 (랜덤 소환기, 와일드카드 유틸)
- **청소부:** 유지 (생존 도우미, 2페이즈 루프)
- **광부 → 드릴 머신:** 유지 (경제 경유 유닛, 1갈래 머지)

### 8.1. 슈뢰딩거 박스 상세

- 웨이브 시작 시 빈 슬롯에 랜덤 유닛 소환 (웨이브 종료 시 소멸)
- 전설 유닛 소환 가능 (체험 기회), 중복 등장 허용
- **Lucky Streak 성장:** 미네랄 투자로 박스 4단계 업그레이드 (나무→철제→보라→수정)
- **소환 전 암시 연출:** 박스 외형으로 등급 힌트 (평범→흔들림→빛→황금)

### 8.2. 청소부 상세

- Phase 1: 적 발사체/장판 제거, 카운터 +1
- Phase 2: 카운터 100 도달 → 2~3초 광역 분출
- 리셋: 분출 종료 → 0 → 청소 복귀
- 보스 즉사기 등 제거 불가 오브젝트는 카운터 미증가

### 8.3. 광부 → 드릴 머신 상세

| 단계 | 자원 | DPS | 전략 |
|------|------|-----|------|
| 광부 | 강함 (추가 미네랄) | 약함 (패시브만) | 경제 구간 |
| 드릴 머신 | 약함 (소량 유지) | 강함 (관통 공격) | 전투 전환 |

- 전환 타이밍이 매판 달라지는 전략 판단
- 머지 경로 1갈래 고정 (광부 → 드릴 머신)
- 광부 DNA 계승: 드릴도 미네랄 소량 드랍 유지

## 9. 기본 효과 추가 필요 유닛 (확정)

### 9.1. 간호순

| As-Is | To-Be |
|-------|-------|
| 힐만 제공, 기본 공격 없음 | **화염 주사기 투척** — 적 [화염] 부여 + 아군 소량 회복 동시 |

## 10. 미스테리오 기본 메커니즘 (확정)

- 구현 방식: 적 외형을 소환수로 소환 (아군 식별 위해 보라색 마법진 + 머리 위 아이콘)
