# 유닛 던전 방벽 디펜스 폴리싱

> **요약:** 유닛 던전 방벽 디펜스의 폴리싱 기획서로, 60초간 외계 곤충 군단을 막는 극한 디펜스를 상세 정의한다. 저글링(물량형), 폭탄벌레(유폭 유도), 코뿔소(돌진형) 3종 몬스터의 스킬/행동 패턴, 방벽 피해 시스템(UnitData@score), 시간대별 스폰 확률 변화, 산성 늪지대 배경 디자인, 리소스 제작 리스트를 포함한다.
> **현재 상태:** 미적용

유닛 던전 : 방벽 디펜스

## 1. 기획 개요

### 1.1. 기획 의도

* **목표:** 60초 동안 쉴 새 없이 쏟아지는 외계 곤충 군단을 막아내는 극한의 디펜스.
* **핵심 재미:**
  + **난전:** 화면을 가득 메운 적들을 보며 느끼는 압박감과 해소감.
  + **전략적 유폭:** 적의 자폭병(가스백)을 역이용해 대량 살상을 노리는 지능적 플레이.
  + **멀티태스킹:** 회피(공중), 방어(폭탄벌레), 공격(딜러)을 동시에 수행하는 피지컬.

### 1.2. 기본 규칙

* **승리 조건:** 제한 시간 60초 [DungeonStageData@limit\_seconds] 동안 방벽 체력이 0이 되지 않도록 방어.
* **패배 조건:** 방벽 시작 체력 100[DungeonStageData@goal\_value]방벽 체력이 0이 되는 순간 즉시 패배.(또는 지휘관 사망)

### **A. 저글링 - 물량형 잡몹**

![](data:image/png;base64...)

저글링 형태.

* **기본 행동 :** 스폰 후 아래방향으로 직진 이동
* **특성 :** 아군 유닛과도 충돌하고, 벽과도 충돌함 (폭발이 아군 유닛에게만 피해를 줌)
* **공격:** 방벽에 도달하면 자폭하여 2 피해를 입힘.(벽에게 주는 피해는 SkillData참조)
* **대응:** 아군 딜러의 **광역 공격**으로 빠르게 녹여야 함. 놓치면 방벽이 금방 무너짐.
* **행동 패턴:**
  + 스폰 후 왼쪽으로 일직선 이동
  + 벽에 부딪힐 때까지 이동 벽에 도착하면 폭발
  + 스턴 및 상태 이상 적용 (넉백 포함)

**스킬: 몸통 박치기**

* **발동 조건:** 유닛 혹은 방벽에 부딪힘
* **실행:**
  1. **벽에게 피해 입힘 혹은 유닛에게 피해를 입힘**
  2. **이펙트:**벽에 부딪혀서 사망 이펙트 재생 (좌표: 본인 위치).
  3. **소멸**
* **이펙트**
  + 벽에 부딪혀서 사망 이펙트
  + 그냥 사망
* 자폭 성공과 실패가 다르게 표시되어 유저에게 피드백

### **B. 폭탄벌레 - 움직이는 폭탄**

![](data:image/png;base64...)

등 뒤에 녹색 형광 가스 주머니를 단 뚱뚱한 딱정벌레.

* **기본 행동 :** 스폰 후 아래 방향으로 직진 이동
* **특성:**
  + 아군 유닛과 **충돌**하면 즉시 자폭.
  + **자폭:** 반경 **3.5m** 내의 모든 유닛(아군+적군)에게 큰 피해.
  + 아군에게는 아프지만, 저글링에게는 **즉사급 피해**를 줌.
* **대응:** 폭탄벌레 유닛을 노려서 **유폭**을 유도.

**스킬: 화학 폭발**

* **발동 조건 :**
  + 플레이어와 충돌
  + HP가 0이하가되어 사망
* **실행 로직 :**
  + 범위 감지
  + **대상별 피해 적용:** 감지된 대상을 확인
    - 아군 유닛이면
    - 적이면 즉사 처리
  + **이펙트:** 자폭 이펙트 재생
  + **소멸**
* **데이터 참조**
  + 화학 폭발 스킬 데이터 참조

### **C. 코뿔소 느낌의 거대 몬스터 (박치기 공룡)**

**![](data:image/png;base64...)**

* **공격 패턴:**
  1. **돌진:** 스폰되자마자 아래로 돌격
  2. **충돌:** 유닛에 부딪히면 중간 피해, 벽에게 부딪히면 큰 피해
* **대응:** 딜로 녹이거나, 폭탄 벌레 유폭을 이용하거나, 아군 유닛으로 앞을 막양함
  + *기획 의도:* 폭탄 벌레를 이용하거나, 유저가 몸으로 막아 벽의 체력이 대량으로 감소하는 것을 막아야함

  + Wall 레이어와 충돌 (UnitData@score만큼 벽에게 피해)

### **벽에게 가하는 피해**

* UnitData@score로 제어
  + 저글링이 부딪히면 score 2점으로 벽에 부딪힐 때마다 2피해
  + 폭탄벌레가 부딪히면 score 10점으로 벽에 부딪힐때마다 10피해
  + 코뿔소가 부딪히면 score 25점으로 벽에 부딪힐때마다 25피해

### 외계 행성의 황량한 전장과 기괴한 곤충 군단

![](data:image/png;base64...)

A. 배경 디자인

### **[산성 늪지대]**

**"우리는 녹아버리지만, 놈들은 저기서 태어났다."**

* **비주얼:**
  + 바닥이 푹 꺼져 있고, **녹색이나 보라색의 걸쭉한 액체**가 보글거리며 끓고 있습니다.
  + 위로 독성 연기가 피어오릅니다.
* **이동 논리 (Why?):**
  + **아군:** 닿으면 슈트가 녹아내리기 때문에 진입 불가능 (시스템적 차단).
  + **적군:** 놈들은 산성액에 내성이 있거나, 체액 자체가 산성이라서 늪지대를 수영하듯 편안하게 건너옴.
* **연출:**
  + 몬스터가 늪지대를 지날 때 발이 살짝 잠기며 액체가 튀는 이펙트 추가.
* **구조물:**
  + 좌측 (아군): [플라즈마 방벽]. 투명한 푸른색 에너지 쉴드가 육각형 벌집 모양으로 웅웅거리며 켜져 있음. 뒤쪽에는 방어 포탑이나 기지 구조물이 살짝 보임.
  + 우측 (적군): [하이브 입구]. 땅이 갈라지거나 거대한 곤충 둥지의 입구. 붉은색 빛과 연기가 새어 나옴.
* **시인성:** '위험 지역'임이 색깔(녹색/보라색)로 명확히 드러납니다.
* **개연성:** 가스백(독) 몬스터와 테마가 일치하여, "아, 쟤네는 독 안 통하니까 그냥 오는구나"라고 유저가 1초 만에 이해합니다.
* **요약:** 아군이 밟으면 녹을 것 같은 위험한 느낌이어야 합니다. 몬스터들은 그 위를 아무렇지 않게 첨벙거리고 건너옵니다.

B. 몬스터 디자인 (Creatures)

* 공통: 외골격(갑각)을 가진 곤충형 외계 생명체. 징그럽지만 타격감 있게 터지는 질감.
* 1. 저글링 (일반):
  + 외형: 저글링, 개미, 사마귀 등을 섞은 소형 곤충. 날카로운 앞발과 턱 강조.
  + 특징: 떼를 지어 빠르게 기어가는 모션. 죽을 때 녹색 체액을 튀기며 조각남.
* 2. 폭탄벌레:
  + 외형: 뚱뚱하고 둔해 보이는 딱정벌레/진드기 형태.
  + 핵심 포인트: 등 뒤에 거대하고 투명한 녹색 가스 주머니가 달려 있음. 주머니 안에서 액체가 출렁거리거나 맥동(Pump)하는 애니메이션. (딱 봐도 터트리고 싶게 생겨야 함)
* 3. 코뿔소 :
  + 외형: 울트라 리스크 외형
  + 특징: 박치기 잘할것 처럼 생겨야함.

## **4. 인게임 플레이 시나리오**

### **[00:00 ~ 00:10] 탐색전**

* **상황:** "WARNING" 사이렌 소리와 함께 저글링이 산발적으로 등장.
* **플레이:** 유저는 유닛을 중앙에 배치하고 딜러들의 사격으로 저글링 잡음.
* **목적:** 조작감 익히기 및 기본 화력 점검.

### **[00:10 ~ 00:50] 난전**

* **상황:** 3종류 몬스터가 섞여서 나옵니다.
  1. **저글링**: 쉴 새 없이 쏟아져 방벽으로 쇄도.
  2. **폭탄벌레**: 저글링 무리 속에 섞여서 천천히 전진.
  3. **코뿔소**: 가끔씩 나와서 난이도 올림
* **핵심 플레이:**
  1. **코뿔소 막기:** 벽을 향해 달려드는 코뿔소 길막
  2. **유폭 유도:** 폭탄벌레 노리고 유폭 유도.
  3. **잔당 처리:** 유폭으로 정리되지 않은 저글링들을 딜러가 마무리.

### **[00:50 ~ 01:00] 최후의 발악 (Last Stand)**

* **플레이:** 더 이상 정교한 컨트롤은 불가능. 폭탄벌레를 보이는 족족 터트려 **연쇄 폭발**을 일으켜야만 생존 가능.
* **엔딩:** 0초가 되면 승리 연출 후 결과화면 출력

## 6. 연출

1. **게임 시작**
   * **게임 시작** → 로딩 (맵 로드) → 카메라 페이드 인 (좌측 방벽부터 우측 스폰 지역까지 훑음) → "DEFENSE!" 문구 출력 → 타이머 시작 (1:00)
2. **게임 종료**
   * **승리 시** → 1분 경과 → 남은 몬스터 일제 소멸 → "DEFENSE SUCCESS" 연출 → 결과 화면 (보상 획득)
   * **패배 시** → 방벽 HP 0 도달 → 방벽 붕괴 연출 → "DEFENSE FAILED" 연출 → 결과 화면
