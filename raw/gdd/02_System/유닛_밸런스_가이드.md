### 소환 1회 기대 가치
```
= 0.40×300 + 0.30×500 + 0.20×700 + 0.09×1,000 + 0.01×30,000
= 120 + 150 + 140 + 90 + 300
= 800원

소환 비용: 100다이아 × 7원 = 700원
기대 가치율: 800 / 700 = 114.3%
```

- 전설 1%로 확률 2배 증가 → 필요 소환 절반으로 감소

- Lv.15 시점: 비전설 전원 **Lv.14 수렴** (격차 1레벨)
- 영웅 9%로 상향되어 영웅 조각 공급량 증가
