# 아웃게임 소환 확률 고정

> **요약:** 초반 소환 결과를 고정하여 신규 유저 경험을 보장하는 시스템. ShopDrawFixedResultData로 고정 결과를 설정하고 1회 소환 버튼을 제거한다.
> **현재 상태:** 이후 버전으로 대체됨

**M7\_아웃게임\_소환 확률 고정**

# 개요

* 초반 플레이 경험 완성을 위해 초반 유닛 소환 결과를 고정하여 제공한다.
* 유닛 소환 1회 버튼을 OFF 처리한다.

# 데이터시트 변경

## 소환 결과 고정

### ShopDrawData 수정 [[링크]](https://docs.google.com/spreadsheets/d/1C8IoWcVPT4VTWCiiwfY40fgjd965RaYJmDFQK2kXSo8/edit?gid=1037543009#gid=1037543009&range=A1)

* + 컬럼 추가
    - fixed\_result\_count
      * -1 : 해당 사항 없음
      * 1 이상 : 해당하는 횟수만큼, 각 횟수 결과를 정해진 내용으로 지급
        + 결과는 별도 테이블에서 지정한다. (ShopDrawFixedResultData)

### ShopDrawFixedResultData 추가 [[링크]](https://docs.google.com/spreadsheets/d/1C8IoWcVPT4VTWCiiwfY40fgjd965RaYJmDFQK2kXSo8/edit?gid=1052618526#gid=1052618526&range=A1)

* 구성
  + product\_id
    - 결과 지정할 draw\_id (서버 처리를 위해 product\_id 로 사용 중)
    - ShopDrawData@product\_id
  + order
    - 결과 지정할 draw\_id의 order
    - ShopDrawData@order
  + fixed\_result\_step
    - 소환 회차
  + result\_type
    - * Unit : UnitData
      * Parts : PartsData
  + result\_ids
    - ; 사용해서 리스트로 입력
    - 각 타입 별로 참조 데이터를 지정한다.
      * Normal : PartsData
      * Unit : UnitData
