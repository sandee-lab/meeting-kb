# 인게임 택틱 최종

> **요약:** 인게임 택틱 시스템의 최종 개선안으로, 공격 속도 체감 정상화(기본 쿨타임 제거, 곱연산 스탯 추가), 공격 형태 계열 제거, 유닛 전설 변신(Unit_Transform_To_Legendary) 택틱 추가, 신화 카드 랜덤 등급 선택 방식 도입을 핵심으로 한다. 택틱 등급별 선택 확률 테이블과 신화 카드 오브젝트(Special_Tactic_Draw) 시스템도 포함된다.
> **현재 상태:** 미적용

# 기존의 목표를 상기하고, 현재 버전의 문제와 해결 방향에 대해서 정리한다.

# 목표

* 전략성이 드러나는 플레이
  + 나의 선택이 누적되어 원하는 결과를 만들어내는 것.
* 도파민이 느껴지는 플레이
  + 나의 선택 혹은 운에 의해서 평소와 다른 쾌감이 느껴지는 상황을 구현하는 것.

# 이슈

* 공격 속도, 쿨타임 등의 수치가 체감되지 않는 문제
  + 밸류를 높이더라도 공격 속도가 체감되기 힘든 구조인 것은 맞음.
    - 기본 쿨타임의 존재
    - 기본 모션의 속도가 너무 빠른 문제
    - 곱연산 부족
* 공격 형태 계열이 선택지로 잘 동작하지 않는 문제
  + 공격 범위 증가, 소환수 증가 등의 제한적인 경우를 제외하면 ‘눈에 보이는 효과’에 어울리는 공격 형태를 제공하기 어려운 문제가 있음.
  + 계열이라는거는 유닛이 각 계열에 고르게 분포가 되어야 밸런스적으로 문제없이 적용할 수 있는데, 유닛이 그러한 설계를 기반으로 만든 것이 아니기 때문에 예외 케이스가 계속 발생하는 등의 적용 단계에서 어려움이 있음.
  + 공격 형태를 통해서 하고 싶은 것이 결과적으로 범위 증가 등 눈에 보이는 공격 변화라면, 계열이 아니라 유닛의 스킬 변화를 보다 적극적으로 사용하는 것이 훨씬 좋은 방향이라고 생각이 됨.
* 유닛 스킬 카드를 적극적으로 선택하기 힘든 문제
  + 유닛의 스킬 카드를 골라서 해당 유닛이 더 잘 싸우는 상황 자체는 재미있다고 생각함.
  + 다만, 낮은 등급의 유닛을 끝까지 사용하기 어렵고, 전설 조합으로 사라지는 경험을 하게되면, 유닛 스킬 카드를 선택하는 것이 망설여지게 되어 결과적으로 전체적인 택틱 선택지가 단조로워지는 문제가 있음.
* 스페셜 카드
  + 스택 카드를 최대 레벨까지 달성해야만 스페셜 카드를 획득할 수 있어 중간 과정에서의 기대감이 떨어짐.
  + 또한, 각 스탯에 따라서 스페셜 카드의 Pool이 제한되므로 스페셜 카드의 조합이 다양하기 어렵고, 스페셜 카드 자체의 종류가 많더라도 게임에서 제한적으로 제공되므로 전체적인 택틱의 다양성에는 영향을 주지 못하는 것도 아쉬운 부분임.

# 해결 방향

* 공격 빈도 스탯 체감
  + 공격 속도 유닛의 기본 쿨타임을 없애서 공격 속도 스탯의 밸류를 정상화한다.
  + 공격 속도 유닛의 기본 모션 속도를 0.5배 수준으로 낮춰 스탯 차이에 의한 체감이 극대화 될 수 있도록 한다.
  + 곱연산 스탯을 추가해서 특정 상황에 수치를 크게 뻥튀기 해주는 경험을 제공한다.
* 유닛 전설 변신
  + 공격 형태 보다는 유닛의 스킬에 변화를 주는 것이 보다 직관적이고 큰 변화를 줄 수 있는 가능성을 갖는다.
  + 공격 형태는 게임의 구성에 잘 맞지 않으므로 제거하되, 유닛 쪽에서 필요한 재미를 충족 시켜줄 수 있도록 룰 개선을 시도해본다.
    - 대상 유닛을 전설 유닛으로 변신 시켜주는 택틱 선택지를 추가하여, 원하는 유닛을 더 성장 시키고, 마지막까지 성장이 유지될 수 있도록 새로운 시스템을 추가한다.
* 스페셜 카드 선택 방식 변경
  + 일반 택틱 드로우 시 운에 따라 스페셜 택틱이 선택지에 포함될 수 있도록 규칙을 변경한다.
  + 스페셜 카드는 결과가 정해지지 않은 상태로 노출되어, 스페셜 카드를 선택하는 것 자체가 약간의 리스크를 동반하도록 한다.
  + 운이 좋게 좋은 것이 등장하는 기회를 넣어서 도파민을 느낄 수 있게 (ex: 번뜩임)

## 신화 카드

### 신화 카드 랜덤 제공

* + - 신화 카드(스페셜 카드)도 일반 택틱 선택지에 랜덤하게 제공된다.
      * ***일반 등급 카드 3레벨 달성 시 보너스 카드 제공하는 시스템은 제거한다.***

### 택틱 등급 선택 확률 추가

* + - 택틱을 선택할 때, 개별 선택을 하기 전에 등급을 우선 선택하도록 규칙을 추가한다.
    - 데이터시트에서 지정한 등급 별 확률에 따라서 등급을 우선 선택하고, 해당 등급 안에서 개별 택틱의 selectrate 값에 따라서 상대 확률 처리한다.
      * StageModeTacticGradeSelectData
        + 기본 확률 레벨은 1로 설정한다.
        + 아웃게임에서 확률 레벨을 증가 시킬 수 있도록 고려한다.

### 신화 등급 선택 시 처리

* + - 등급 선택 시 신화 등급이 선택된 경우 개별 택틱을 결정하지 않은 ‘미결정’ 상태로 택틱 카드를 제공한다.
      * UI에서는 뒷면 상태로 표시한다.
      * **미결정 상태 처리를 위한 택틱 ID : 99999**
    - 뒷면 상태의 신화 등급 카드 선택 시 신화 등급 카드 3개를 랜덤 선택해서 UI에 제시한다. (기존의 보너스 카드 선택 UI 연출)
      * 유저는 제시된 3개의 신화 택틱 중에서 하나를 선택할 수 있다.

## 신화 카드 오브젝트(Special\_Tactic\_Draw)

* + 상호작용 오브젝트 결과 타입 추가
    - result\_reward\_type = Special\_Tactic\_Draw
  + 아웃게임에서 컨텐츠 잠금 해제 시 인게임에 적용한다.
    - 스페셜 상점
      * 7 웨이브 시작 시 [스페셜 카드]가 확정 등장합니다.
  + 잠금 해제 시 처리
    - 지정한 웨이브 시작 시 상호작용 오브젝트 생성 시점에 항상 신화 카드 상점 오브젝트로 대체해서 생성한다 (게임 중 1회).
  + 신화 카드 선택 UI
    - 뒷면 카드 선택 시 뜨는 신화 카드 선택 UI를 동일하게 사용한다.

# 연관 스탯/성장

## 택틱 카드 확률 레벨

* + 레벨에 따라서 등급 별 확률이 결정됨.
  + 특별 성장에서 레벨을 성장 시킬 수 있음.
  + 최종 레벨에 따라서 등급 확률이 결정됨.
  + 레벨은 스탯으로 처리
    - add\_tactic\_draw\_level

## 스페셜 상점 오브젝트 언락

* + 특별 성장에 신규 효과 타입 추가
    - special\_effect\_type == Special\_Tactic\_Draw
      * 스탯명은 소문자로 입력 (special\_tactic\_draw)
    - special\_effect\_value == 1
  + 해당 타입 값이 1이면, 인게임에서 오브젝트 등장 처리

폴리싱 리스트

* 전설 변신 카드 노출 빈도 제어를 위한 unlock\_condition\_type 추가
  + Check\_Skill\_Target\_Unit\_Group
    - 조건
      * unlock\_condition\_type\_1 == Check\_Unit\_Group
      * effect\_type == Add\_Skill or Upgrade\_Skill
    - 비교
      * unlock\_condition\_value1\_2 지정한 unit\_group\_id 와 unlock\_condition\_value1\_1 지정한 unit\_group\_id 가 같으면 Count
    - 처리
      * Count 값이 unlock\_condition\_value2\_2 에서 지정한 값 이상이면, 해당 택틱을 Pool에 추가한다.
  + 조건 로컬
    - {unlock\_condition\_value1\_2 } 스킬 카드 {unlock\_condition\_value2\_2 }개 보유
* 전설 유닛 변신 시 이펙트 붙이기
  + 인게임 중에 배치했을 때, 빛기둥 같은거 붙여주자
  + Unit\_Transform\_Effect
* 유닛 변신 택틱 등장 시 연출 뎁스 분리해주기
* 신화 카드 오브젝트 리소스 / 아이콘
