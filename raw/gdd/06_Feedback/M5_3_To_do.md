# M5_3 방향 및 구성 정리

> **요약:** 전략 깊이 유지 + 액션 재미 포커싱 방향, 슬롯 배치 효과와 어빌리티 등 구성 정리.
> **현재 상태:** 미적용

방향

1. 전략의 깊이는 충분함.
   1. 일부 전략 요소는 보다 캐주얼한 방향으로 정리하여 복잡도를 완화.
2. 액션의 재미에 포커싱.
   1. 한 웨이브의 전투 템포를 보다 긴장감있게 구성하는 방향으로 진행
   2. 유닛의 메카닉을 다양하게 확보하여 목표가 되는 컨셉 플레이 다양화
3. 정보 표현
   1. 일부 유지되는 기능들은 필요한 정보를 추가
4. 플레이 사이클
   1. 한 판의 전체적인 플레이 흐름은 앞에 내용들 중심으로 진행하면서 고민

spawn\_interval

# 플레이 사이클 변화

## 레벨업 O

* 1. 적 처치 > 경험치 획득 > 레벨업 > 택틱 팝업
  2. 가스 > 유닛 소환

## 레벨업 x

* 1. 적 처치 > 미네랄 획득 > 유닛 소환
  2. 가스 > 택틱 상점
     1. 액션 중에 택틱 상점 방문
     2. 액션 클리어 보상에서 가스로 추가 구매

## 레벨업 O

* 1. 적 처치 > 경험치 획득 > 레벨업 > 택틱 팝업
  2. 가스 > 택틱 상점
     1. 액션 중에 택틱 상점 방문
     2. 액션 클리어 보상에서 가스로 추가 구매
  3. 미네랄 > 유닛 소환

폴리싱 이슈들

1. EPic 등급은 합성 못하니까 합성 가능 상태 표시 안하기
2. 랜덤 소환으로 Epic 등급 나오면 연출 같이 붙여주기 (합성 연출이 아니라 생성 연출)
3. 빈 슬롯 없어서 소환 버튼 disable 처리 됐을 때, 슬롯 빈자리 생기면 disable 해제 시켜주기
4. 유닛 소환, 합성 등으로 조합식 만족했을 때, 실시간 갱신 안되는 문제
5. 유닛 조합, 합성, 판매 등으로 조합식 만족 해제됐을 때, 실시간 갱신 안되는 문제
6. 웨이브 시간 표시 ms 초 단위까지 표시 00:29 > 29:99
7. 보스 웨이브 시간 -1 표시 대신 > 보스 처치 로 표시
