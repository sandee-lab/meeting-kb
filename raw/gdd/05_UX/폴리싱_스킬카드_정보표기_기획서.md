# 폴리싱 - 스킬 카드 정보 표기

> **요약:** 유닛의 사용 가능 스킬을 한눈에 볼 수 있도록 택틱 기반 스킬 정보를 인게임/아웃게임 UI에 추가하는 기획이다. UnitEnchantData의 강화 스킬과 StageModeTacticPoolData의 add_skill/upgrade_skill 정보를 스킬 슬롯으로 표시하며, 등급별 스킬 카드 보유 제한(ConstantDatas)과 인게임 보유 카드 수 집계 기능을 포함한다. 잠금 상태는 인게임(택틱 획득 여부)과 아웃게임(강화 레벨)에서 각각 다르게 처리한다.
> **현재 상태:** 미적용

# 목표

* **유닛의 사용 가능 스킬을 한 눈에 볼 수 있게 한다.**

# 개요

* [스킬 카드로 발동되는 유닛 스킬 정보]와 [인게임 스킬 카드 획득 현황]을 인게임과 아웃게임에 추가한다.
  + 스킬 카드로 발동되는 유닛 스킬 정보 → 인게임, 아웃게임
  + 인게임 스킬 카드 획득 현황 → 인게임

# 처리

## 유닛 스킬 정보 개선

* + 기존 정보의 문제점
    - 기존 유닛의 스킬 정보는 UnitData 만을 참조한다.
    - 문제? : 택틱으로 발동되는 스킬 정보를 알 수 없다.
  + 개선 포인트
    - 택틱으로 발동되는 스킬 정보를 추가한다.
    - 해당 정보는 어디에? :
      * UnitEnchantData
      * StageModeTacticPoolData
    - 단, 개선 시에도 UnitID를 기준으로 검색할 수 있도록 한다.

| 현재 | 1. UnitData@linked\_base\_skill\_id  2. UnitData@linked\_sub\_skil\_ids |
| --- | --- |
| 변경 | 1. UnitData@linked\_base\_skill\_id  2. UnitData@linked\_sub\_skil\_ids  3. **[추가: 이하 강화 스킬] UnitEnchantData**@special\_enchant\_type=add\_tactic  4. **[추가: 이하 add\_skill]** **StageModeTacticPoolData**@unlock\_condition\_type\_1=Check\_Unit\_Group 이고, unlock\_condition\_value1\_1=(해당 유닛 ID) 일 때, effect\_type=add\_skill인 스킬  5. **[추가: 이하 upgrade\_skill]**  **StageModeTacticPoolData**@unlock\_condition\_type\_1=Check\_Unit\_Group 이고, unlock\_condition\_value1\_1=(해당 유닛 ID) 일 때, effect\_type=upgrade\_skill인 스킬 레벨 정보 |

## 인게임 스킬 카드 획득 현황

* + 기존 방식
    - 현황을 알 수 없다.
    - 문제? :
      * 일부 택틱 카드는 '특정 유닛 스킬 카드 n장 수집'을 조건으로 한다.
      * 위 조건 만족 여부를 유저가 알 수 없다.
  + 개선 포인트
    - 각 유닛 별로 스킬 택틱 카드 수량을 집계한다.
    - 어떻게?
      * 집계 대상
        + 보유 카드 중 StageModeTacticPoolData@ check\_unit\_group 인 것
        + (AND) 그 중에서 add\_skill 또는 upgrade\_skill 인 것
      * 유닛 강화에서 주는 카드는?
        + 강화 스킬 카드는 시작 택틱으로 적용되면 집계에서 제외된다.
        + 강화 스킬 카드가 시작 택틱으로 적용되지 않으면 유저 보유 카드에 포함되므로 문제 없음.
      * 안내 방식
        + 집계된 카드 갯수는 인게임 UI에 표시한다.
        + 어디에? :

툴팁

최대 획득 수량도 안내한다.

포메이션

인벤토리

## 각 유닛 별 스킬 카드 보유 제한

* + 기존 방식
    - 제한 없음
  + 신규 처리
    - 각 유닛은 등급을 기준으로 스킬 카드 보유 수량 제한을 둔다.
      * 등급별 기준? : ConstantDatas에서 지정한다.
        + Common\_Unit\_Tactic\_Max
        + Uncommon\_Unit\_Tactic\_Max
        + Rare\_Unit\_Tactic\_Max
        + Epic\_Unit\_Tactic\_Max
        + Legendary\_Unit\_Tactic\_Max
    - 해당 유닛의 스킬 카드가 수량 제한 도달 시, 해당 유닛의 스킬 카드는 더 이상 등장하지 않는다.

## 스킬 레벨 정보

### 목표

* + - 일부 택틱은 유닛 스킬의 레벨을 올려준다.
    - 해당 정보를 UI에 표현한다.

### 위치

* + - 인게임 : UnitDetailUI
    - 아웃게임 : SkillSlotInfoPopUp

### 공통

* + - 설명 슬롯
      * 스킬 설명 슬롯은 레벨 별로 할당된다.
        + 스킬 레벨이 1만 있는 경우에도, 1레벨 슬롯으로 할당한다.
      * 스킬 설명 슬롯에는 SkillData@local\_desc 를 표시한다.
      * 잠금 해제 조건은 가능한 방식으로 표시한다.
        + 필수 아님

### 인게임

* + - 잠금 상태

        + 잠금

* + - * + 해제
