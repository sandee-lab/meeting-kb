## 축약본

> **목적:** 카오스 속성의 두 번째 전설 유닛으로, 고스트(한방형)와 대비되는 연사형 디버퍼 역할을 수행하여 카오스 속성 조합의 전략적 깊이를 확장한다.
> **핵심 제한사항:** 즉결 처형은 일반 몬스터에만 적용 (보스/엘리트 제외), 신규 코드 구현 없이 기존 시스템 조합으로 구현
> **비전 연결:** "유닛 기획서 시리즈 착수" — 전설 유닛 개별 설계의 첫 번째 사례. 속성별 시너지 구조의 카오스 축 완성.

## 2. 달성 목표

### 플레이어 경험 목표
- 약화를 뿌리고 → 체력이 깎이면 즉결 처형 → 사망 시 약화 전이라는 **연쇄 처형 루프**의 쾌감 제공
- 고스트와 함께 편성 시 카오스 증폭 피해로 "속성 세트를 모았다"는 성취감

### BM 목표
- 카오스 속성 두 번째 전설 유닛으로, 고스트 보유 유저의 **시너지 완성 욕구** 자극
- 뽑기 풀 추가를 통한 매출 기여

### 운영 목표
- 기존 시스템(InstantDeath, Passive_Kill, 약화 이펙트) 재활용으로 개발 비용 최소화
- 향후 카오스 속성 유닛 추가 시 약화 시너지 확장의 기반

### 개발 목표
- 신규 코드 구현 없이 기존 CSV 데이터 조합만으로 구현

## 3. 수행 방향

### 핵심 설계 원칙
1. **고스트와 역할 분담** — 고스트(5초 쿨, 한방 강타) vs 미스테리오(연사, 약화 + 처형)
2. **약화 중심 시너지** — 3개 스킬 모두 약화 이펙트를 축으로 연결
3. **기존 시스템 재활용** — 신규 이펙트 타입이나 스킬 카테고리 없이 구현

### 제약 사항 / 금지 사항
- 즉결 처형은 보스/엘리트에게 적용하지 않음 (밸런스 보호)
- 환영 분신이 아닌 "허상"으로 차별화 (닌자 컨셉과 혼동 방지)

### 우선순위
| 우선순위 | 항목 | 이유 |
|---------|------|------|
| P0 (필수) | 저주의 투사체 (Base Skill) | 유닛의 핵심 기능 |
| P0 (필수) | 약화 전이 (Sub Skill 1) | 카오스 시너지의 핵심 |
| P1 (중요) | 환영 소환 (Sub Skill 2) | 연사량 증폭, 세부 스펙 확정 필요 |
| P1 (중요) | 즉결 처형 (Lv2 해금) | 파워 스파이크, 레벨업 동기 |

## 4. 개요

### 유닛 기본 정보

| 항목 | 값 |
|------|-----|
| 등급 | Legendary |
| 속성 | Chaos |
| 외형 컨셉 | 스파이더맨의 미스테리오 (환영술사, 어항 헬멧, 연기/안개) |
| 기능적 역할 | 연사형 디버퍼 — 약화를 광범위하게 퍼뜨리고 약해진 적을 처형 |
| 메타 포지션 | 고스트(한방형)의 시너지 파트너, 카오스 속성 조합의 필수 유닛 |

### 컨셉 점검

| 체크 항목 | 판정 |
|----------|------|
| 외형-메커니즘 연결성 | 환영술사 → 저주 투사체 + 환영 소환 + 처형 = 자연스러움 |
| 속성 내 어울림 | 카오스: 고스트(유령) + 미스테리오(환영술사) = 초자연/오컬트 테마 일관 |

### 스킬 구성 요약

```
미스테리오
├─ [저주의 투사체] — Base Skill (linked_base_skill_id)
│   ├─ Lv1: 여러 개 투사체 → 피해 + [약화] 부여
│   └─ Lv2: 적중 시 HP ≤ 30% 일반 몬스터 → 즉시 처치
├─ [약화 전이] — Sub Skill 1 (linked_sub_skill_ids)
│   └─ Lv1: [약화] 걸린 적 사망 → 주변 적에게 [약화] 전이
└─ [환영] — Sub Skill 2 (linked_sub_skill_ids)
    └─ Lv1: [저주의 투사체]를 사용하는 환영 소환
```

### 핵심 시너지 루프

```
환영 + 미스테리오 동시 연사
  → 저주의 투사체 x2 (약화 부여 + HP 감소)
  → 일반 몬스터 HP 30% 도달 → 즉결 처형
  → 약화 전이 → 주변 적에게 약화 확산
  → 고스트 강타 → 카오스 증폭 피해 (약화 대상에게 +30~50%)
  → 반복
```

## 5. 시스템 동작 흐름

### 전체 흐름

```
[전투 시작] → [환영 소환] → [미스테리오 + 환영 연사]
  → [저주 투사체 적중] → [약화 부여] + [데미지]
  → [HP ≤ 30%?] ─Yes→ [즉결 처형 (일반몬만)]
                  ─No→ [계속 공격]
  → [적 사망] → [약화 걸려있었나?] ─Yes→ [약화 전이]
                                     ─No→ [종료]
```

### 스킬별 상세 흐름

#### [저주의 투사체] Lv1
1. 쿨타임 완료 시 발동 (Action_Skill)
2. 다수의 투사체를 대상 방향으로 발사
3. 적중 시: 데미지 + 약화 이펙트(SkillEffectData id=1) 부여
4. 약화 효과: 공격력 -20%, 카오스 피해 추가 +30%, 지속 5초

#### [저주의 투사체] Lv2 (레벨업 해금)
1. Lv1 효과에 추가
2. 적중 시 대상의 현재 HP가 최대 HP의 30% 이하인 경우:
   - **일반 몬스터**: 즉시 처치 (InstantDeath)
   - **보스/엘리트**: 효과 없음 (normal_monster_weight로 분기)

#### [약화 전이] Lv1
1. Passive_Kill 카테고리 — 적 사망 이벤트 시 트리거
2. 조건: 사망한 적에게 약화 이펙트가 걸려있었는지 확인 (Check_TargetHasSkilleffect, value=1)
3. 조건 충족 시: 사망 위치 주변 적에게 약화 이펙트 부여

#### [환영] Lv1
1. 환영(소환수) 소환 — [저주의 투사체] Lv1과 동일한 스킬 사용
2. 환영은 미스테리오의 스탯 기반으로 축소된 능력치
3. [TODO] 소환 트리거, 지속 시간, 재소환 조건 미확정

### 예외 처리 / 엣지 케이스

| 상황 | 처리 방식 |
|------|----------|
| 즉결 처형 대상이 보스/엘리트 | 효과 없음 — 일반 데미지만 적용 |
| 약화 전이 시 주변에 적이 없음 | 전이 불발, 이펙트만 소멸 |
| 환영이 즉결 처형 트리거 | 가능 — 환영의 저주 투사체도 동일 InstantDeath 판정 |
| 약화가 이미 걸린 적에게 재적중 | 기존 약화 지속시간 갱신 (is_stackable=TRUE 기준) |
| 환영 사망 시 | [TODO] 재소환 조건 미확정 |

## 7. 밸런스 계획

> ⚠️ 세부 수치는 확정값이 아니며, 테스트 후 조정될 수 있습니다.

### 설계 기준
- 고스트(5초 쿨, damage_weight 1.5)와 총 DPS가 유사하되, 분배 방식이 다름
- 즉결 처형 30%는 일반 몬스터 한정이므로, 보스전에서는 순수 DPS로 기여

### 고스트 대비 포지션

| 항목 | 고스트 | 미스테리오 |
|------|--------|-----------|
| 공격 빈도 | 5초 쿨 한방 | 잦은 연사 |
| 타겟팅 | 가장 먼 적 1체 | [TODO] |
| 약화 부여 | 없음 (약화 증폭만) | 직접 약화 부여 |
| 대보스 성능 | 높음 (damage_weight 1.5) | 중간 (다수 투사체 분산) |
| 대다수 성능 | 낮음 (단일 타겟) | 높음 (광역 약화 + 즉결 처형) |
| 시너지 역할 | 카오스 증폭 피해 수혜자 | 약화 공급자 |

### 검증 기준
- 무과금: 미스테리오 단독으로도 일반 몬스터 처리 효율이 체감됨
- 소과금: 고스트 + 미스테리오 조합으로 카오스 시너지 완성 시 확실한 파워 스파이크
- 고과금: 약화 전이 + 즉결 처형 루프로 웨이브 클리어 속도 눈에 띄게 향상
