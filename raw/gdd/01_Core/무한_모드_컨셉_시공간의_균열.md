# 무한 모드 컨셉 - 시공간의 균열

> **요약:** 스테이지 모드 클리어 후 저장된 덱 스냅샷을 불러와 5분 타임어택으로 몬스터를 섬멸하는 엔드게임 콘텐츠 기획이다. 심도와 공허의 계약으로 난이도를 조절하고, 주간 시즌제 랭킹(챌린저~골드/실버)으로 경쟁을 유도한다. 웜홀 상점에서 유물/택틱을 획득하여 일시적 초월 성장을 경험하며, 시즌마다 속성 테마와 몬스터 변이가 적용되어 메타 고착화를 방지한다.
> **현재 상태:** 미적용

# **무한 모드: 시공간의 균열 (Rift of Time)**

## **1. 컨텐츠 방향성 (Direction)**

* **End-Game Showroom (최종 점검의 장):** 스테이지 모드 클리어 후 저장된 \*\*[세이브 데이터(덱 스냅샷)]\*\*를 불러와, 내 덱의 성능을 극한까지 검증하는 최종 컨텐츠입니다.
* **Time Compression (시간 압축):** 지루한 버티기가 아닌, **[5분 타임어택]** 내에 압도적인 화력으로 적을 섬멸하며 성장하는 속도감 있는 플레이를 지향합니다.
* **Volatile Overclock (휘발성 초월 성장):** 저장된 덱 스펙에 만족하지 않고, 컨텐츠 플레이 과정에서 획득하는 \*\*[유물/택틱]\*\*을 통해 일시적으로 한계를 초월한 스펙을 완성, 매 판마다 극적인 플레이 스타일을 경험합니다.
* **Ranking Competition (경쟁 유도):** 보상 획득 횟수는 제한하되 도전 기회는 무제한으로 제공하여, 코어 유저층이 상위 랭킹(명예)을 목표로 끊임없이 최적의 빌드를 연구하도록 유도합니다.

## **2. 아웃 게임 기본 규칙 (Out-Game Rules)**

### **2.1. 입장 규칙**

* **입장 재화:** 제한 없음 (무제한 입장 가능).
  + *의도:* 컨트롤 최적화 및 고득점 갱신을 위한 반복 도전(Retry)을 적극 장려합니다.
* **덱 로드 (Loadout):** 스테이지 모드 클리어 시 저장된 **[세이브 데이터]** 중 하나를 선택하여 입장합니다.
  + 입장 시 선택한 덱 구성(유닛, 스킬)은 변경 불가하며, 인게임 플레이를 통해서 추가 성장이 가능합니다.
* **난이도 설정:**
  + **심도 (Depth):** 유저의 전투력 구간에 맞는 기본 난이도(몬스터 스펙)를 선택합니다. 심도가 높을수록 점수 배율이 기하급수적으로 증가합니다.
  + **공허의 계약 (Contracts):** '이동속도 증가', '내 체력 반감' 등 추가 제약 조건을 유저가 직접 선택하여 추가 점수 보너스(+%)를 획득합니다.

### **2.2. 보상 규칙**

* **일일 보상 (Daily):**
  + **획득 제한:** 매일 기본 1회 획득 (도전 횟수는 무제한이나 보상 수령은 1회).
  + **충전 기회:** 광고 시청(RV) 또는 멤버십 가입을 통해 일일 보상 획득 기회를 추가 충전할 수 있습니다.
* **시즌 업적 보상 (Seasonal Milestone):**
  + 매주 시즌 초기화 시 갱신됩니다.
  + 누적 점수가 아닌 \*\*[최고 기록 점수]\*\*가 특정 구간(예: 10만, 50만...)에 도달할 때마다 성장 재화(다크 매터, 큐브 등)를 1회씩 지급합니다.
* **랭킹 정산 보상:**
  + 매주 일요일 시즌 종료 시, 아래의 랭킹 등급(Tier)에 따라 차등 지급합니다.

### **2.3. 랭킹 규칙**

* **운영 방식:** **[전체 통합 랭킹 (Global Ranking)]**
  + 별도의 하위 리그(조 편성) 없이 시즌 참여자 전체를 하나의 랭킹 보드에서 경쟁시킵니다.
* 점수 산정 공식:

| **몬스터 처치 점수** = 몬스터 기본 점수 \* ( 1 + 웨이브 가중치)  **최종 점수** = 몬스터 처치 점수 \* 심도 배율 \* ( 1 + 계약 보너스) |
| --- |

*정렬 기준:* 점수 내림차순 > 달성 시간 선착순.

* **시즌 등급 (Tier) 시스템:**
  + 매 시즌 \*\*실시간 순위 백분위(%)\*\*에 따라 등급이 즉시 변경됩니다.

| **등급 명칭** | **달성 조건 (상위 %)** | **주요 보상 성격** |
| --- | --- | --- |
| **챌린저 (Challenger)** | **Top 3명** (절대수치) | **[명예]** 유니크 테두리, 칭호 |
| **마스터 (Master)** | **0.1% ~ 1%** | **[명예+성장]** 최상위권 안착 유도 |
| **다이아 (Diamond)** | **1.1% ~ 10%** | **[핵심 성장]** 큐브, 다크 매터 대량 |
| **플래티넘 (Platinum)** | **10.1% ~ 30%** | **[일반 성장]** 숙련 유저 보상 |
| **골드 / 실버** | **30.1% ~ 100%** | **[참가상]** 일반 유저 보상 |

### **2.4. 시즌 규칙 (Season Rules)**

무한 모드 <시공간의 균열>은 주 단위 시즌제로 운영되며, 매 시즌 변화하는 전장 환경을 제공하여 특정 메타의 고착화를 방지합니다.

#### **A. 시즌 기본 규칙 (Basic Rules)**

* **운영 주기:** 1주 (Weekly Cycle)
  + 시작: 매주 월요일 09:00 (서버 시간 기준)
  + 종료: 매주 월요일 00:00 ~ 09:00 (정산 및 휴식기)
* **초기화 항목 (Reset):**
  + **랭킹 기록:** 이전 시즌의 랭킹 및 티어 등급이 초기화됩니다.
  + **업적 보상:** 시즌 업적 달성 보상(최고 점수 보상) 획득 정보가 초기화되어, 매주 다시 보상을 획득할 수 있습니다.
  + **입장 기록:** 지난 시즌에 사용한 덱 로드 정보가 초기화됩니다.
* **기록 보존 (History):**
  + 내가 달성한 최고 기록은 [명예의 전당] 또는 \*\*[내 기록실]\*\*에 영구 저장됩니다.

#### **B. 시즌 랜덤 효과 규칙 (Season Mutators)**

매 시즌마다 전장에 \*\*[행성 환경 효과(Planetary Environment)]\*\*가 무작위로 부여됩니다. 유저는 해당 효과를 분석하여 가장 유리한 \*\*[세이브 데이터(덱)]\*\*를 선택해야 고득점을 노릴 수 있습니다.

1. **속성 테마 (Main Theme):**
   * 매주 하나의 \*\*주요 속성(화염, 냉기, 전격, 독, 무속성 등)\*\*이 테마로 선정됩니다.
   * **Buff**: 해당 속성을 가진 유닛/스킬의 공격력 +50% 또는 쿨타임 -30%.
   * **Debuff**: 역상성 속성의 유닛 공격력 -30% (예: 화염 시즌 시 냉기 유닛 페널티).
2. **몬스터 변이 (Enemy Mutation):**
   * 등장하는 몬스터들이 시즌 테마에 맞춰 변이합니다.
3. **로테이션 규칙:**
   * 속성 테마는 순환(Rotation)하되, 유저들의 덱 보유 현황 데이터(Data-Driven)를 기반으로 특정 속성 육성을 장려하는 방향으로 배정합니다.

## **3. 인게임 기본 규칙 (In-Game Rules)**

### **3.1. 웨이브 규칙 (타임어택 섬멸)**

* **진행 방식:**
  + 각 웨이브는 정해진 \*\*[목표 처치 수]\*\*를 갖습니다.
  + 목표 달성 시 **즉시 다음 웨이브**로 레벨업하며, 난이도가 상승하고 몬스터가 리필(Refill)됩니다.
* **타임라인 (총 5분):**
  + **0~4분:** 일반 웨이브 진행. 매 분(1, 2, 3, 4분) 경과 시점에 **[보스 몬스터]** 난입.
  + **4~5분 (피버 타임):** 몬스터 스폰량 폭증

### **3.2. 승리 및 패배 규칙**

* **승리 조건:** 제한 시간 **5분 생존**에 성공. (종료 시점의 점수로 랭킹 산정)
* **패배 조건:** 지휘관(플레이어 캐릭터)의 **체력이 0이 되어 사망**. (사망 시점까지의 점수를 기록으로 인정).

### **3.3. 상점 규칙 (웜홀 상점)**

* **발동 조건:** 시간 경과에 따라 등장하는 **[보스 몬스터]** 처치 시 즉시 발동 (게임 시간 일시 정지).
* **전용 재화:** **균열 파편** (무한 모드 내에서 몬스터 처치 시 획득하는 일회성 휘발성 재화).
* **판매 품목**
  + **유물 (Artifact) :** 기본 유물 중 랜덤 구성, **(전용 유물 추가 고려 가능)**
  + **택틱 (Tactic) :** 기본 택틱 중 랜덤 구성, **(전용 택틱 추가 고려 가능)**
  + **소모품 (Consumable):** 체력 회복, 능력치 버프
* **부가 기능:**
  + **리롤 (Reroll):** 재화를 소모하여 판매 목록 갱신 (비용 점증).
  + **제거 (Remove):** 재화를 소모하여 보유 중인 유물/택틱 삭제 (전략적 덱 압축, 횟수 제한).
