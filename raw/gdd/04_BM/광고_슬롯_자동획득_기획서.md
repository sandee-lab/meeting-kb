# 광고 슬롯 자석 (자동 획득)

> **요약:** 인게임 플레이에 영향을 주는 광고 슬롯으로, 광고 시청 시 아이템 획득 범위가 크게 증가하는 자석 효과가 적용된다. 하루 5회 시청 제한이 있으며, 광고 제거 멤버십 구매 시 강력 자석 효과가 영구 활성화된다. AdsSlotData에 Add_Collection_Range 타입을 추가하고, 멤버십 상품 숏컷을 인게임 팝업에 연결한다.
> **현재 상태:** 미적용

![](data:image/png;base64...)![](data:image/png;base64...)

# 개요

* 인게임 플레이에 영향을 주는 광고 슬롯을 추가한다.

# 시스템 구조

## 광고 슬롯 추가

* + 광고 슬롯에 자동 획득 타입을 추가한다.

## 시청 횟수 제한

* + 해당 광고는 하루에 5회 시청 횟수를 제한한다.

## 시청 효과

* + 광고 시청한 게임 중에 효과가 적용된다.
  + 효과 적용 시 아이템 획득 범위가 크게 증가한다.
    - 화면 내 자동 획득으로 할지
    - 기존의 택틱 처럼 반지름을 늘려줄지 구현 고민 해본다.

## 상품 연결

* + 광고 제거 멤버쉽 상품 숏컷을 연결해준다.
  + 게임 중에 상품을 구매할 수 있다.

## 상품 구매 시 효과

* + 상품 구매 시 강력 자석 효과가 영구 활성화된다.

튜토리얼 예외 처리 고려려

* 튜토리얼에서 강력 자석 효과를 무료로 획득할 수 있는 예외 처리가 있을 수 있다.

# UI 규칙

## 인게임에 버튼 배치

### 조건

* + - 광고 시청 가능
    - 남은 시청 횟수가 0이 아닐 때,

### 버튼 터치 시

* + - 게임을 일시 정지한다.
    - 팝업 UI를 오픈한다.
      * 광고 시청 버튼을 터치해서 효과를 활성화 할 수 있다.
      * 멤버쉽 상품 숏컷 버튼을 터치해서 멤버쉽 상품을 구매할 수 있다.

## 버튼 제거

### 조건

* + - 광고 시청으로 효과 적용 중일 때,
    - 멤버쉽 구매로 효과 영구 적용 중일 때,

# 필요 데이터시트

## AdsSlotData

* + ads\_type
    - Add\_Collection\_Range
  + count
    - 5

## ConstantsData

* + add\_Collection\_range
    - 획득 거리 증가 값
    - (화면으로 처리하는 경우 필요 없음)
  + add\_collection\_range\_linked\_product\_id
    - 연결할 상품 id (60002)
