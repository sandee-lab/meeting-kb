        - 행성 도감과 그 행성의 짧막한 이야기가 담긴다면 재밌을 것
        - 초반에 자동으로 넘어가는 스토리는 터치로 변경 해줄 것
    - **UI/UX**
        - 미션, 도감 보상 일괄 수령
        - 장비 추천 장착
        - 아머리에서 합성 가능한 유닛 표기

    - **게임 플레이**
        - 속성 배치나 유물 사용법 등이 튜토리얼이 반드시 필요함

        - 터치 이동방식 추가
        - 진행도를 수동으로 저장할 수 있어서 해당시점에서 시작 할 수 있으면 좋겠다
        - 전체적으로 랜덤성이 너무 강하다
            - 유닛 머지 시 속성은 유지했으면
        - 특정 타이밍에는 체력을 회복할 기회가 주어졌으면. 미네랄을 소모하더라도.
    - **시인성 개선**
        - 메인 기체가 피격 받을 때 화면에 빨간색 플래쉬 추가
        - 메인 기체 피격음 추가
        - 메인 기체에 강조 표시 필요
        - 우리편의 투사체는 끄거나 간소화 하는 옵션 추가
