# 리롤(새로고침) 시스템

> **요약:** 인게임에서 유닛·택틱을 새로고침하는 리롤 시스템 초안. 유닛 드래그 판매 UX와 콘텐츠 언락 조건을 정의한다.
> **현재 상태:** 이후 버전으로 대체됨

새로고침 시스템

## 개요

* 새로고침 기능을 추가하여 플레이어가 원하는 방향의 빌드를 구성할 수 있다.
* 유닛 새로고침 기능을 추가하고 판매 UI/UX를 개선한다.
* 택틱 새로고침 기능을 추가해 택틱에서 원하는 방향으로 나아갈 수 있도록 한다.
* 새로고침은 매판 횟수 제한으로 골드 특별 성장으로 획득할 수 있다. (황금 디스크)

## 유닛 새로고침 시스템

![](data:image/png;base64...)

### 해금 시점

### **ContentsUnlock 추가**

* 유닛 판매 (Unit\_Sell)
* 유닛 리롤 (Unit\_Reroll)
* 택틱 리롤 (Tactic\_Reroll)

### 기능 요구 사항:

* 관리 페이즈에서 유닛을 새로고침 할 수 있는 기능
* 새로고침 영역은 드래그 중에만 등장합니다.
* 유닛을 새로고침 영역에 드래그하여 놓으면 새로고침을 실행 합니다.
* 새로고침 시 동일 등급 내에서 랜덤하게 다른 유닛으로 교체 (같은 유닛이 등장하지 않음)
* 새로고침 횟수가 부족한 경우 새로고침이 일어나지 않으며 유닛은 다시 제자리로 갑니다. (드래그 전 위치)

### 판매 UX 변경

* 기존 : 인벤토리 마지막 슬롯 차지
* 변경 후 : 인벤토리에서 삭제, 드래그하면 뽑기 버튼 위쪽에 등장
* 유닛을 판매 영역에 놓으면 해당 유닛을 판매합니다.
* 유닛을 드래그 하는 시점에 판매 영역에 가격이 표시됩니다. (기존과 같음)

### 새로고침 제한:

* 판당 제한 횟수: 기본 5회(콘텐츠 언락으로 해금) (특별 성장으로 10까지 가능)
* 비용: 무료
* 새로고침 가능 횟수는 UI에 표시 (예: “새로고침 7/10”)

### 데이터 처리:

* 현재 판의 남은 새로고침 횟수 추적
* 새로고침 사용 시 카운트 차감
* 게임 종료 시 리셋

## 택틱 새로고침 시스템

### 기능 요구 사항:

* 레벨 업 시 택틱 3택 1선택창에서 새로고침 가능
* 새로고침 시 새로운 3개의 택틱 새로 뽑기
* 새로고침 UI는 택틱 선택창 하단에 배치
* 직전에 나온 택틱은 제외 (중복 방지)

![](data:image/png;base64...)

### 새로고침 제한:

* 판당 제한 횟수: 기본 4회 (특별 성장으로 8까지 가능)
* 비용: 무료
* 새로고침 가능 횟수는 UI에 표시 (예: “새로고침 7/8”)

### 데이터 처리:

* 유닛 새로고침과 별도의 카운터로 관리
* 현재 판의 남은 새로고침 횟수 추적
* 게임 종료 시 리셋

UX : 스와이프를 통해서 새로고침 진행 (버튼 터치일 경우 실수가 잦음)

## 황금 디스크 (골드 특별 성장)

### 새로운 성장 항목 추가:

유닛 새로고침 횟수 증가

* 항목명: “유닛 새로고침 +5회”
* 효과: 판당 유닛 새로고침 가능 횟수 +5
* 최대 10회

택틱 새로고침 횟수 증가

* 항목명: “택틱 새로고침 +4회”
* 효과: 판당 택틱 새로고침 가능 횟수 +4
* 광고 보고 충전으로 +2
* 최대 10회

**새로운 스탯 추가**

add\_unit\_reroll : 인게임에서 유닛 새로 고침 갯수 추가

add\_tactic\_reroll : 인게임에서 택틱 새로 고침 갯수 추가

ConstantDatas@start\_unit\_reroll: 5

ConstantDatas@start\_tactic\_reroll: 4

ConstantDatas@ads\_tactic\_reroll\_charge: 2

## UI 상태 표시

**관리 페이즈 화면:**

* 유닛 리롤 잔여 횟수: "유닛 리롤 7/10"
* 택틱 리롤 잔여 횟수: "택틱 리롤 8/10"
* 두 수치는 별도로 관리되며 서로 영향 없음

**비활성화 상태:**

* 리롤 가능: 활성화 상태
* 리롤 불가 (0회 남음): 비활성화 + 횟수 표시 "0/10"
* 비활성화 시 버튼 터치해도 반응 없음
* 비활성화 시 아이콘 회색으로 변경

**택틱 리롤의 경우**

택틱 리롤이 해금 된 후 모든 리롤 사용 시 광고를 보고 횟수를 충전할 수 있습니다. ConstantDatas@ads\_tactic\_reroll\_charge 참조해서 횟수 충전 (1회 가능)
