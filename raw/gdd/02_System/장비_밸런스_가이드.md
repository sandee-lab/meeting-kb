## 2. 소환 시스템

### 2-1. 이중 소환 구조

| 항목 | 일반 소환 | 스페셜 소환 |
|:----:|:--------:|:---------:|
| 비용 | **100 다이아** | **300 다이아** |
| 일반(Common) | 86% | 62% |
| 고급(Uncommon) | 11.5% | 25% |
| 희귀(Rare) | 2% | 9% |
| 영웅(일반 Epic) | 0.5% | 3.4% |
| 영웅(S급 Epic) | — | 0.6% |
| 일반영웅 천장 | **80회** | **10회** |
| S급영웅 천장 | — | **60회** |

> S급 영웅은 스페셜 소환에만 존재. 두 천장 카운터는 **별도 카운팅**.

### 2-2. Pity 적용 유효확률 (스페셜 소환)

| 구분 | 기본 확률 | 천장 | E[주기] | **유효확률** | 배율 |
|:----:|:-------:|:----:|:-------:|:-----------:|:----:|
| 일반 영웅 | 3.4% | 10회 | 8.60회 | **11.63%** | 3.4x |
| S급 영웅 | 0.6% | 60회 | 50.51회 | **1.98%** | 3.3x |

```
E[일반영웅 주기] = Σk=1..9 k×0.034×0.966^(k-1) + 10×0.966^9 = 8.60
E[S급영웅 주기] = Σk=1..59 k×0.006×0.994^(k-1) + 60×0.994^59 = 50.51
```

> **MC 검증 (100K x 1000회)**: 천장이 에픽 출현의 **~71%** 를 차지.
> 천장 발동 시 하위 등급 드로우를 대체하므로, 실제 Common/Uncommon/Rare 비율은 기본 설계보다 5~8% 하락.

### 2-3. 소환 1회 기대 가치

등급 기준 가치 (Common = 70 dia, 진화비용 기반 3배 체계):

| 등급 | 기준 가치 | 환산 근거 |
|:----:|:--------:|:--------:|
| Common | 70 dia | 기준값 |
| Uncommon | 210 dia | Common × 3 |
| Rare | 630 dia | Common × 9 |
| Epic | 1,890 dia | Common × 27 |
| S급 Epic | 1,890 dia | 일반 Epic과 동일 |

**일반 소환 기대 가치:**
```
= 0.86×70 + 0.115×210 + 0.02×630 + 0.005×1890
= 60.2 + 24.15 + 12.6 + 9.45
= 106.4 dia  (비용 100 dia → +6.4%)
```

**스페셜 소환 기대 가치 (Monte Carlo 100K×1000회):**
```
= 375.1 dia  (비용 300 dia → +25.0%)
```

> 기대 가치는 풀 크기(N)와 무관 — 드랍 등급 확률만 영향.
> 천장 발동 시 하위 등급을 대체하므로, 단순 유효확률 합산(409.8 dia)보다 실측이 낮음.

### 5-2. Ancient 1개 제작 비용 (역추적)

> 소비아이템 없음(순수 장비), 티어 승급 재료는 해당 등급 t0 기준

| 단계 | 계산식 | Common 누적 |
|------|--------|:-----------:|
| Common t0 기준 | 1 | **1** |
| → Uncommon | 1 + 2×1 | **3** |
| → Rare | 3 + 2×3 | **9** |
| → Epic t0 | 9 + 2×9 | **27** |
| → Epic t1 | 27 + 1×27 | **54** |
| → Epic t2 | 54 + 2×27 | **108** |
| → Legendary t0 | 108 + 1×108 | **216** |
| → Legendary t1 | 216 + 1×216 | **432** |
| → Legendary t2 | 432 + 2×216 | **864** |
| → Legendary t3 | 864 + 3×216 | **1,512** |
| → Mythic t0 | 1,512 + 1×1,512 | **3,024** |

> **Ancient 1개 = Common 42,336개 = Epic 환산 1,568개** (소비아이템 없음, 순수 장비 기준)

## 8. 소견

### 이중 소환 설계 포인트

| 항목 | 분석 |
|------|------|
| 일반 소환 기대가치 | 106 dia (비용 100) → +6.4% 흑자, 진입장벽 낮음 |
| 스페셜 소환 기대가치 | **375 dia** (비용 300) → **+25.0%** 흑자 (MC 실측) |
| 천장 기여도 | Epic 출현의 **~71%가 천장 발동**, 핵심 가치 드라이버 |
| S급 Ancient 비용 | **2.19억 dia** — 일반 Ancient(3,788~3,965만)의 **5.5~5.8배**, 진성 과금 콘텐츠 |

> ⚠️ 위 비용은 **소비아이템 없음(순수 장비)** 기준. 소비아이템 활용 시 비용 대폭 단축됨.

### 진화 트리 변경 영향 (구버전 TARGET=288 → 신버전 TARGET=1,568)

| 시나리오 | 구버전 (288) | 신버전 (1,568) | 증가율 |
|:--------:|:----------:|:------------:|:-----:|
| 일반소환 → 일반 Ancient | **686만** | **3,788만** | **5.5x** |
| 스페셜소환 → 일반 Ancient | **748만** | **3,965만** | **5.3x** |
| 스페셜소환 → S급 Ancient | **3,870만** | **2.19억** | **5.7x** |
| 일반소환 → 일반 Legend t3 | **76만** | **135만** | **1.8x** |

- **Legend t3**: 신버전에서도 135만 dia — 중기 목표로 적절한 수준
- **Ancient**: 전 시나리오 **5.3~5.7배** 증가 — 티어 승급 수량 증가(1→2→3)와 등급 전환 재료 티어 상향이 복합 작용
- 분산이 낮아(5%~95% 범위 ±1~2%) 유저 간 비용 편차가 극히 적음
