# V.0.0.12 빌드 작업 내역

> **요약:** V.0.0.12 빌드에 포함된 인게임 개선 작업 완료 내역. 카메라 세팅 변경, 지휘관 범위 표시, 미네랄 획득 연출 변경, 스마트 밴 시스템, 등급별 밸런스 변경, 조합식 정리, 유닛 사용성 개선, 택틱 등급 확률 동적 제어, 광폭화 부여 조건 변경 등 인게임 및 아웃게임 작업 내역을 포함한다.

## [완]미네랄 획득 연출 변경

* + 기존 : 미네랄 아이콘이 보유 수량 패널까지 이동 후 획득
  + 변경 : 미네랄 아이콘과 수량 출력 후 바로 획득
  + 기획의도: 화면의 복잡성을 낮춰 시각적 피로도를 낮추고 상황을 더 잘 인지시킴

## [완]스마트 밴 시스템

* + 유닛 스마트 밴
    - 같은 섹션에서 유닛 리롤 시 한번 리롤한 유닛은 리롤 결과로 나오지 않음
    - 같은 등급내 모든 유닛이 모두 밴 될 경우, 밴이 모두 풀림
  + 택틱 스마트 밴
    - 레벨업 단계에서 택틱 리롤 시 한번 리롤한 택틱은 리롤 결과로 나오지 않음
    - 등급내 모든 택틱이 밴되었다면 해당 등급 내 택틱 밴은 모두 풀림
  + 리롤 횟수가 쌓일수록 원하는 카드를 찾을 확률이 선형적으로 상승

## [완]인게임 밸런스 조정

* + 기본 템포
    - 몬스터 이동 속도 하향 조정 : 1.2 > 0.6
    - 보스 등급 몬스터 체력 가중치 하향 조정 : 100 > 50
    - 보스 전투 중 일반 몬스터 등장 빈도 상향
  + 웨이브 밸런스
    - 상위 등급 유닛을 요구하도록 웨이브 상승에 따른 난이도 스케일 상승

## [완]택틱 등급 확률 동적 제어

* + 의도
    - 높은 등급 유닛의 스킬 카드를 모으기 어려운 문제 개선
  + 규칙
    - 포메이션에 배치한 유닛의 등급 평균이 높아질 수록 높은 등급 카드의 출현 확률이 증가하도록 시스템 추가

## [완] 물리 속성 체인 off

* + 물리 속성 > 무속성으로 변경
    - 물리 속성은 기존처럼 무속성으로 범용 적인 서포트 유닛 포지션으로 활용.
    - 무속성 유닛을 특정 속성으로 바꾸는 택틱을 통해서 속성 시너지 강화

## [완] 광폭화 부여 조건 변경

* + 의도
    - 기본 이속이 감소함에 따라서 광폭화를 통한 긴장감 부여를 보다 점진적으로 체감할 수 있도록 타이밍 변경.
  + 기존
    - 몬스터 스폰 시 더이상 스폰할 수 없을 때,
  + 변경
    - 몬스터 스폰 시 실제 스폰 수가 계산된 스폰 수 미만일 때, 광폭화 부여.
    - 예) 데이터 상으로 10마리를 스폰해야 하는데, 남은 자리가 10자리가 안되는 경우 광폭화 부여

# 아웃게임

## 신규 컨텐츠

### 명예의 전당

* + - 개요 및 목적
      * 컨셉: 특정 챕터 최초 클리어 유저를 기리고, 전 채널 유저와 보상을 나누는 골든벨 및 선착순 랭킹 시스템
      * 목표: 선착순 경쟁을 통한 과금 유도 및 서버 전체의 성장 동력 순환
      * 경험 구분:
        + 상위 유저: 전 채널에 이름 공표, 영구 기록 등재 및 골든벨 결제를 통한 명예 획득
        + 일반 유저: 상위 유저의 성과로 인한 무료 보상 획득 및 성장을 돕는 특가 패키지 이용 가능.
    - 핵심 구성
      * 명예의 전당
        + 성격: 이타적, 공유형 보상
        + 진행 개요:

최초 클리어 발생

전 채널 브로드캐스팅과 동시에 전 유저 대상 [기본 보상]과 [특가 상품] 해제

골든벨 패키지

최초 클리어 유저에게 일정 시간 전용 상품 노출

과금 시 전 채널 2차 브로드캐스팅 및 전 유저에게 우편으로 추가 선물 지급

영속성

최초 클리어 유저의 정보는 해당 챕터 명예의 전당 UI에 영구 게시

* + - * 선착순 랭킹
        + 성격: 경쟁형, 개인 보상
        + 진행 개요:

각 챕터 클리어 순서에 따른 선착순 산정

상위 30위까지 UI 게시

별도 보상은 없으며 명예 보상으로 갈음함

### 플레이 미션

* + - 개요 및 목적
      * 컨셉: 카마존 연구소의 데이터 수집을 위해 특정 덱 조합을 시험하는 시스템.
      * 목적: 덱 가이드, 전략 재미 강화, 광고 시청(리롤)을 통한 수익화.
    - 핵심 규칙 및 판정
      * 실험 구성: 메인 태스크(지정 유닛 배치 필수) + 추천 태스크(플레이 가이드/팁 성격)
      * 성공 조건: 메인 조건 만족 후 1웨이브 이상 생존 및 챕터 정상 종료(승패 무관)
      * 보상: 아웃게임 보상(다이아) 지급 (인게임 밸런스 영향 차단, 미수령 보상은 매일 소멸)
      * 횟수 및 리롤: 일 3회 제공. 광고 시청을 통해 총 3회 리롤 가능.
    - 태스크 생성 및 보정 규칙
      * 생성 방식: 유저 진행도(챕터) 기반 그룹 추출 → 유저 보유 유닛 검사(진행 가능한 그룹 선별) → 당일 완료/노출/거절 그룹 제외 후 무작위 선정
      * 확률 보정: 미션 관련 유닛/택틱의 등장 확률을 일시 상향하여 클리어 지원 (초회차에 보정치 최대. 03-17 기준 테스트를 위해 유닛의 경우 등장 확률을 100%로 적용)
