# 로컬 푸쉬

> **요약:** 특정 조건 만족 시 로컬 푸시를 발송하는 시스템으로, 푸시 종류를 최소화하고 우선순위를 부여한다. 최대 행동력 달성, 최대 순찰 보상 달성, 일일 초기화의 3종 푸시를 정의하며, 하나의 푸시 발송 후 지정 시간 내에는 추가 발송을 하지 않는다. 이벤트 및 시즌 콘텐츠 추가에 따라 확장 가능하다.
> **현재 상태:** 미적용

로컬 푸쉬

# 개요

* 특정 조건 만족 시 로컬 푸쉬를 발송한다.
* 푸쉬의 종류는 최소화한다.
* 푸쉬 우선순위가 존재한다.
* 하나의 푸쉬가 발송된 이후에 지정한 시간 내에는 푸쉬를 발송하지 않는다.
* 이벤트, 시즌 컨텐트 추가에 따라 로컬 푸쉬는 추가될 수 있다.

# 푸쉬 종류

## 최대 행동력 달성

* + 조건 : 보유 행동력이 최대 행동력에 도달한 경우
  + 제목 : 행동력이 가득찼습니다.
  + 내용 : 행동력이 가득찼습니다. 접속해서 사용하세요!

## 최대 순찰 보상 달성

* + 조건 : 순찰 보상이 최대치에 도달한 경우
  + 제목 : 순찰 보상이 가득찼습니다.
  + 내용 : 순찰 보상이 가득찼습니다. 접속해서 수령하세요!

## 일일 초기화

* + 조건 : 일일 초기화 시간
  + 제목 : 새로운 하루
  + 내용 : 새로운 하루가 시작되었습니다!
