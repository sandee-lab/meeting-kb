# M5 인게임 택틱 상점

> **요약:** 액션 페이즈 종료 후 등급별 택틱을 무료/유료로 선택하는 인게임 상점 시스템.
> **현재 상태:** 미적용

M5\_인게임\_택틱상점

**M5\_인게임\_택틱상점**

# 개요

* 특정 액션 페이즈 종료 후 택틱 상점을 출력한다.
* 1번째 택틱은 무료, 2번째 부터는 재화를 소모해서 택틱을 추가 구매할 수 있다.
* 진행 웨이브 단계에 따라 택틱 등급이 결정된다.

![](data:image/png;base64...)

# 구성

* UI
  + TacticShopUI(가칭, 신규 UI)
    - Initial\_Offer
      * 최초 택틱 선택 창
      * 웨이브 정보
        + 웨이브 단계 숫자
        + 택틱 등급 확률 (상위 2개만)

표기

[등급] [확률] [추가확률]

추가 확률 : 아웃게임에서 얻은 경우 표시

예 : 전설 10% (+5%)

* + - * 선택 버튼 필요
        + 무료로 표기
      * 넘어가기 버튼 배치치
    - Repeat\_Offer
      * 최초 이후 택틱 선택 창
      * 웨이브 정보는 Initial\_Offer와 동일
      * 선택 버튼 필요
        + 비용 표기
      * 넘어가기 버튼 배치

# 동작

## 진입

* + StageWaveData@wave\_type=Tactic\_Shop 인 경우
  + 액션 페이즈 종료 시 택틱 상점 호출

## 카드 세팅

* + UI 결정
    - UI 호출 후 첫 세팅 : Initial\_Offer
    - 첫 세팅 이후 : Repeat\_Offer
  + 드로우 규칙
    - UI 진입 시점의 웨이브 단계 점검
    - 해당 웨이브의 택틱 3종 등급 결정
    - 해당 웨이브에 지정된 풀에서 등급으로 필터링
    - 랜덤하게 3개의 택틱을 제공
    - 리롤 제공 X
  + 유저에게 2개의 선택지를 제공한다.
    - 택틱 선택
    - 넘어가기

##

## **[선택지1]** 택틱 선택

* + 카드 선택 후 [선택(2회차부턴 구매)] 터치 시 비용 차감 후 해당 택틱 적용
    - 비용
      * UI 호출 후 첫 선택 : 무료
      * 2회 : ConstantDatas에서 상수 값 설정
        + tactic\_shop\_cost\_default

기본 판매가 (가스 수량)

* + - * 예외 처리
        + 비용 부족 시 토스트 메시지 출력

“{0}이 부족합니다”

Local\_Toast\_Need\_Item

## **[선택지2]** 넘어가기

* + [넘어가기] 선택 시 다음 화면 (유닛 인벤토리)로 이동할 수 있다.

# 데이터시트

* TacticShopLevelData
  + wave\_step : 웨이브 단계
  + pool\_id : 현 웨이브에에서 사용할 택틱 풀 아이디
  + common\_rate : 현 레벨에서 택틱 3종의 등급으로 결정될 확률 (전 등급 합계 10000)
    - StageModeTacticPoolData에서 설정된 Grade 카드 호출
  + uncommon\_rate
  + rare\_rate
  + epic\_rate
  + legendary\_rate
  + mythic\_rate
  + ancient\_rate
