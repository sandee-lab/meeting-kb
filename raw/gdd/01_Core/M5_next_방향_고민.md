# M5 next 방향 고민

> **요약:** 액션/전략 페이즈 구분, 지휘관과 용병 분리, 스탯 시너지 중심 성장 방향 설계.
> **현재 상태:** 미적용

# 방향

* 액션 / 전략 페이즈를 구분하여 게임 플레이 요소를 명확하게 구분
* 지휘관, 용병으로 유닛을 구분하여 컨트롤 대상을 명확하게 하고, 회피 난이도 문제 제거
* 개별 유닛의 스킬을 강화하기 보다는 스탯 시너지 위주로 성장 요소를 정리하여, 화려함보다는 전략적인 조합의 재미를 강조

# 액션 페이즈

## 플레이

* + 한 웨이브 단위로 몰려오는 적들을 모두 처치하는 것을 목표로 한다.
  + 전략적인 판단 요소보다는 조작을 통한 액션의 재미를 비중있게 경험한다.
  + 웨이브 한번의 시간을 타이트하게 해서 몰입도를 높이고, 랜덤 가챠 요소를 통해서 강한 자극과 텐션을 관리한다.
  + 지휘관과 용병 개념을 정리하여, 컨트롤 대상을 명확하게 하고, 회피 난이도 문제를 해소한다.
  + 전체적인 전략의 목표에 있어서도 지휘관의 성장과 용병 조합의 시너지가 중요하도록 설계하여, 단일 유닛의 화려한 스킬 활용보다는 전체적인 지휘관과 용병의 구성을 잘 조합하는 것이 중요하도록 한다.

## 구성 요소 - 지휘관

### 초월 유닛 > 지휘관

* + - 지휘관은 아웃게임에서 선택한다.
    - 게임 시작 시 지휘관으로 시작하며, 게임 시작 시에는 기본 공격 능력만 갖는다.
    - 게임 플레이를 통해서 지휘관 고유의 스킬을 획득하고 성장하게 된다.
    - 지휘관에 따라서 지휘관 자체의 강함 비중이 높은 경우도 있고, 용병과의 조합에 따라서 강해지는 경우도 있어 다양한 전략 플레이가 가능하도록 한다.

| 지휘관 자체의 강함이 중요한 경우는 필요한 스탯 시너지 기준으로 용병을 조합하는 것이 중요하고, 용병과의 조합이 중요한 경우는 효율을 낼 수 있는 용병 조합을 높은 등급으로 압축하는 것이 중요하다.  리더 효과 같은 개념을 도입해서 보다 직관적으로 조합을 유도하는 것도 좋을 것 같다. (속성과 종족 등으로 다양성 확보)   * 불 속성 공격력 강화 * 불 속성 공격 속도 강화 * 바이오닉 공격력 강화 |
| --- |

### 조합 시너지

* + - 지휘관 + 용병 조합의 시너지를 어떤 방향으로 설계할 것인지 고려해야 한다.

| 스탯 | 스탯을 기반으로 하는 경우 지휘관에게 핵심 스탯을 하나씩 고려해야할 수 있다.  예) 라이더 - 이동속도 / 이동속도가 높을 수록 공격력이 상승한다.  예) 제다이 - 치명타 확률 / 치명타 적중 시 쿨타임을 즉시 초기화한다. |
| --- | --- |
| 속성 | 속성으로 지휘관과 유닛을 자연스럽게 묶어 주면, 조합의 다양성을 자연스럽게 추구할 수 있다.  현재 구성에서는 일반 용병의 종류가 속성에 1:1로 대응하기 어려운 부분은 해결해야 한다. |

### 지휘관 교체

* + - 지휘관과 용병 조합에 실패한 경우에 게임 중에 1번 정도는 지휘관을 변경하여, 전략을 크게 수정할 수 있는 기회를 제공한다.
    - 이러한 요소로 전체적인 지휘관의 활용 빈도를 높여 아웃게임에서의 지휘관 획득이 게임에 좀더 큰 영향력을 가질 수 있도록 한다.
    - 지휘관 교체는 액션 페이즈에서 랜덤하게 기회를 제공하는 것으로 하여 액션 페이즈에서의 단조로움을 해소해주는 역할을 한다.
    - 지휘관 교체는 상호작용 오브젝트를 통해서 제공하며, 일정 시간 안에 상호작용하지 않으면, 기회는 소실된다.

## 구성 요소 - 용병

### 일반 유닛 > 용병

* + - 일반 유닛은 용병 개념으로 피격 대상에서 제외한다.
    - 컨트롤 유닛을 명확하게 하고, 전체적인 회피 난이도 문제를 해결한다.

### 용병 구성

#### 등급

* + - * 1성 > 2성 > 3성 승급 구조로 되돌아간다.
      * 승급에 따라서 유닛의 스킬이 성장한다.
    - 시너지
      * 개별 유닛에 따라서 소환 시 패시브 스탯을 부여한다.
      * 유닛의 고유 시너지는 승급에 따라서 성장하지는 않도록 한다.
        + 마린 : 공속
        + 파뱃 : 공격력
        + 탱크 : 체력
        + 칼냥 : 치확
        + 대포 : 효과 적용
        + 힐냥 : 쿨타임

| 시너지를 개별 스탯으로 할지 속성으로 할지 고려할 필요가 있겠다.  개별 스킬 보다는 지휘관과의 시너지를 고려하면, 속성을 기반으로 덱을 조합하는 것이 여러모로 유리할 수 있다. |
| --- |

#### 무기

* + - * 초월 유닛을 지휘관으로 포지셔닝하여, 일반 유닛만 남아 있는 경우 결과적으로 사용하는 유닛의 조합 수가 적어 매번 비슷한 상황을 플레이하게 될 가능성이 높다.
      * 이러한 상황을 고려했을 때, 같은 유닛 조합을 하더라도 결과가 달라질 수 있는 요소로 무기 교체 시스템을 고려해볼 수 있겠다.

| 유닛 승급 | 스킬 레벨 업을 통한 성장 |
| --- | --- |
| 무기 교체 | 스킬 교체를 통해 스타일 변경 |

* + - * 예)
        + 칼냥이

불의 검 > 불의 검기+ 화염 효과

라이트세이버 > 전기 검기 + 감전 효과

* + - * + 방패냥

가시 방패 > 발사체 피격 시 발사체 반사 효과

회전 방패 > 방패를 던져서 공격

* + - * + 총냥이

권총 > 한발 한발 강한 공격 1회 관통

샷건 > 3발씩 사격

기관총 > 빠른 연속 사격

저격총 > 긴 재장전 속도 + 단일 대상에 강한 공격

### 용병 획득

#### 레벨업을 통한 용병 선택 획득

* + - * 레벨업 시 1성~ 3성 유닛 랜덤하게 선택지 제공
      * 선택지 중에서 원하는 유닛을 소환

| **Note) 선택지 다양화**  레벨업을 통한 택틱 선택지에는 기본적으로 용병 획득과 용병의 사용성 강화 두 가지로 구분하여, 상대적으로 용병 획득이 즉시적인 성장으로 동작하지 않을 때, 전체적인 사용성 강화 선택을 통해서 게임 플레이에서 성장감을 지속적으로 제공할 수 있도록 한다.   * 칼냥이 스킬 강화 * 칼냥이 스킬 추가 * 등등 개별 용병의 사용성 확장 |
| --- |

#### 탐험을 통한 용병 랜덤 획득

* + - * 1성 유닛 획득
        + 유닛 / 꽝 / 꽝 -> 랜덤 유닛 1성 소환
      * 2성 유닛 획득
        + 유닛 / 유닛 / 꽝 -> 랜덤 유닛 2성 소환
      * 3성 유닛 선택 획득
        + 유닛 / 유닛 / 유닛 -> 유닛 3성 선택 소환 ( 택틱 선택)

#### 유닛 획득 시 처리

* + - * 빈자리가 있으면, 소환하고, 없으면 인벤토리로 획득
      * 승급 시 항상 좋은 것은 아니므로 자동 승급하지는 않는다.

# 전략 페이즈 - 유닛 관리

## 인벤토리 정리

### 랜덤 승급

* + - 보유 중인 잉여 유닛을 승급 시도한다.
    - 승급 시 해당 등급의 유닛이 랜덤하게 결정된다.

### 판매 - 재구매

* + - 보유 중인 유닛을 판매하여, 재화로 환전한다.
    - 보유 재화로 상점 목록에서 원하는 유닛을 구매할 수 있다.

| 구매를 통한 유닛 획득은 안정적인 선택이므로 랜덤 승급 보다는 밸런스적인 고점은 낮도록 고려 해야 한다. |
| --- |

# 보스 처치 보너스 페이즈

* 보스 처치 등 한판에서 2-3번의 기회로 제공되는 전략 페이즈 이다.
* 일반적인 전략 페이즈를 대체한다.
* 보너스 페이즈는 제공되는 택틱을 구매하는 것만 할 수 있다.
* 보스 처치 시 몇 개의 택틱은 랜덤하게 제공되며, 추가적으로 구매할 수 있는 리스트를 제공한다.
* 전략 페이즈 등에서 자원을 잘 관리했다면, 여기에서 플레이에 도움이 되는 택틱을 구매하여 안정적으로 성장할 수 있는 기회를 갖게 된다.

| 일반적으로 상점에서 원하는 용병을 찾기 위해서 리롤 비용을 소모하는 것은 리스크가 있는 선택이므로, 현재 충분한 성장을 한 상황이라면, 재화를 아꼈다가 여기에서 투자하는 것이 보다 안정적인 선택이 될 것이다. |
| --- |

* 전체적으로는 레벨업 택틱 선택지와 유사한 개념이지만, 유닛 획득 이외에 특수한 선택지를 제공하여 차이를 둔다.

## 전략성 택틱

* + - 적 처치 시 경험치 획득량 증가 - 초반이라면, 더 빠른 성장을 제공
    - 상점 품목 1개 추가 - 전략 페이즈에서의 리스크 줄이기
    - 매 페이즈 마다 상점 리롤 무료 1회 - 전략 페이즈에서의 리스크 줄이기
    - 특정 유닛 등장 확률 높이기 - 운 제어하기
    - 웨이브 시작 시 - 랜덤 가챠 생성 확률

## 액션성 택틱

* + - 특정 유닛 혹은 전체 유닛의 성능을 드라마틱하게 바꿔주거나 새로운 효과를 부여하여 전체적인 플레이 텐션에 큰 영향을 주는 택틱 효과 구현
    - 예)
      * 지휘관 공격 시 주변 적에게 범위 피해 추가
      * 랜덤한 지역에 메테오 지속 소환
      * 지휘관 체력 30% 미만일 때, 아군 전체 공격력 50% 증가
      * 지휘관 궁극기 쿨타임 -50% 감소
