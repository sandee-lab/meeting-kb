# M5 7월 24일 테스트 가이드

> **요약:** 7월 24일 테스트용 가이드. 20웨이브 구성, 가스 오브젝트, 관리 페이즈 플레이 흐름 설명.
> **현재 상태:** 미적용

7월 24일 테스트 가이드

# 기본 진행 흐름

* 한 판은 20웨이브로 구성됩니다.
  + 10, 20웨이브는 보스 전투로 구성됩니다.
  + 웨이브 마다 [가스 오브젝트]를 찾아서 수집할 수 있습니다.

![](data:image/png;base64...)

* + 몬스터 처치 시 경험치를 수집하고, 레벨업하면 유닛을 랜덤하게 획득할 수 있습니다.
* 1 웨이브는 30초를 생존하면 클리어 됩니다.

![](data:image/png;base64...)

* 웨이브 클리어 모다 택틱 카드를 보상으로 1회 지급합니다. (3중 1택)
  + 택틱을 선택하고, 확인 버튼을 눌려아 합니다.
* 웨이브 클리어 마다 [관리 페이즈]가 진행됩니다.
  + 관리 페이즈에서 유닛을 관리하고, 전략을 수정할 수 있습니다.
    - 유닛 소환

![](data:image/png;base64...)

* + - * 게임에서 획득한 가스 3개를 소모해서 유닛을 랜덤 소환합니다.
    - 유닛 합성
      * 동일한 유닛을 드래그&드랍으로 합치면, 랜덤 승급합니다.
    - 유닛 판매

![](data:image/png;base64...)

* + - * 잉여 유닛을 쓰레기통으로 드래그&드랍하면, 유닛을 판매합니다.

| 등급 | 획득 가스 |
| --- | --- |
| 일반 | 2 |
| 고급 | 3 |
| 희귀 | 4 |
| 영웅 | 5 |
| 전설 | 5 |

* + - 유닛 배치
      * 보관함에 배치된 유닛을 포메이션 슬롯에 드래그&드랍하여, 배치할 수 있습니다.
    - 전설 유닛 소환

![](data:image/png;base64...)

* + - * 관리 페이즈의 [UNIT] 버튼을 눌러서 조합식을 확인할 수 있습니다.
      * 조합을 만족하는 경우 재료를 소모해서 전설 유닛을 소환할 수 있습니다.
  + 관리 페이즈에서 할 일이 끝나면, [NEXT WAVE] 버튼을 눌러서 다음 웨이브를 시작할 수 있습니다.

![](data:image/png;base64...)

# 주요 변경 요소

* 지휘관 유닛 (SCV)가 추가되었습니다.
  + 지휘관 유닛이 체력바는 상시 표시되며, 지휘관 유닛이 사망하면, 게임에서 패배합니다.

![](data:image/png;base64...)

* 사망한 일반 유닛은 웨이브 클리어 시 **부활**합니다.
  + 사망 시 인벤토리로 되돌아가며, 웨이브를 클리어 했다면, 관리 페이즈에서 유닛을 다시 사용할 수 있게 됩니다.
* 종족 시너지가 추가되었습니다.

![](data:image/png;base64...)

* + 종족은 바이오닉, 메카닉, 사이오닉으로 구성됩니다.
  + 같은 종족 배치 수에 따라서 세트 효과를 부여받게 됩니다.
  + 세트 효과는 2,3,5,8 마리 배치 시에 활성화 됩니다.
* 일부 유닛에게 배치 효과가 추가되었습니다.

![](data:image/png;base64...)

* + 유닛 배치 시 아군 전체에 버프 효과가 부여됩니다.
  + 버프 효과를 보유한 유닛 배치 시 관리 페이즈 UI에 아이콘이 표시됩니다.
* 일부 유닛의 스킬 구성이 변경되었습니다.
