# 머지/카드 시스템

> 출처: [[raw/meetings/[1029] 기획 방향 초안]], [[raw/meetings/[1205] 카드 머지 시퀀스 정리]], [[raw/meetings/[1216] 머지 룰 개선]], [[raw/meetings/[1224] 승급 연출]], [[raw/meetings/[0208] 운빨 메모]], [[raw/meetings/[0306] nc피드백 2차]], [[raw/meetings/[0315] 코어 루프 개선안]], [[raw/feedback/NCsoft_TinyCatAssemble 피드백_20260303]], [[raw/gdd/01_Core/인게임_전체_설계서]], [[raw/gdd/01_Core/전략성_강화_기획서]], [[raw/gdd/02_System/소환_확률_고정]] *(대체됨)*, [[raw/gdd/02_System/인게임_등급강화]] *(대체됨)*, [[raw/gdd/02_System/스페셜_스폰_패턴_기획서]]

## 개요

머지/카드 시스템은 본 게임의 핵심 코어 루프로, 같은 등급의 유닛 카드를 합쳐 상위 등급으로 승급시키는 메커니즘이다. 초기 기획(2024.10)부터 럭가챠+머지를 인게임 코어로 설정했으며, 이후 머지 룰, 연출, 승급 시스템 등이 지속적으로 개선되어왔다. NC 피드백과 유저 데이터 분석을 거치면서 "머지와 덱 빌딩의 균형"을 찾는 것이 프로젝트의 핵심 과제로 자리잡았다.

## 상세 내용

### 초기 기획 (2024.10.29)

- 인게임 룰로 럭가챠와 머지를 핵심 코어로 설정
- 연속된 같은 유닛 카드가 머지와 함께 승급됨
- 머지 단계는 최초 2단계이며, 유닛 강화가 될 때마다 확장 (최대 5단계)
- 감각적인 부분을 잘 살리는 것이 무엇보다 중요하다고 강조
- 성장 체계: 일회성 성장(특수 카드 뽑기)과 영구 성장(전체 유닛 파워 증가)으로 이원화

### 머지 시퀀스 연출 (2024.12.05)

카드 머지 시 세부 애니메이션 시퀀스가 정의됨:

1. **카드 이동**: 우측에서 좌측 빈슬롯까지 이동 (ease out expo, duration 0.4)
2. **두 카드 만남**: 만나는 시점에 스케일 0.8로 축소 (duration 0.1)
3. **좌우 분리**: 각각 왼쪽/오른쪽으로 이동하면서 스케일 1.2로 확대 (duration 0.2~0.3)
4. **합체**: 두 카드가 가운데로 합쳐짐 (ease out cubic, duration 0.15)
5. **머지 완료**: 카드 슬롯으로 돌아가며 180도 회전, 스케일 1로 복귀 (ease out back)
6. **캐릭터 아이콘**: 마지막에 -180도 회전하며 스케일 0에서 등장 (ease out back, duration 0.2)

### 머지 룰 개선 (2024.12.16)

- 유닛마다 개별 생산 속도를 보유
- 생산 시간 완료 시 좌측부터 전장으로 출격
- 유닛 카드는 좌측/우측 어디든 한 칸씩 이동 가능하며, 모든 카드는 단 한번만 이동 가능
- 같은 성급의 두 카드가 마주하면 머지됨
- 머지가 되면 새로운 카드이므로 다시 이동 가능

### 승급 연출 참고 (2024.12.24)

- 머지 이펙트의 레퍼런스로 Lucky Defense의 노말 클리어 전략 영상을 참조

### 운빨과 머지의 관계 (2025.02.08)

- 게임 한판에서 신화를 뽑아보는 한 사이클의 경험이 가장 중요
- 기본 플레이는 운으로 뽑는 것이지만, 재미는 빌드업에서 온다
- 원하는 결과를 얻기까지 다양한 "운"의 루트가 존재해야 함 (POE의 아이템 제작 시스템과 유사한 결)
- 리스크와 운의 비율을 다양하게 조합하고, 단계적으로 쪼개서 유저가 원하는 타이밍에 원하는 플레이를 할 수 있게 해야 함
- 신화를 많이 가질수록 더 자주 신화를 경험하는 선순환 사이클 진입
- 한 스테이지에 웨이브를 많이 넣고 최대한 신화까지 빌드업하는 과정이 필요
- 랜덤 뽑기를 하나로 두되, 게임 내에서 성능을 올려가면서 확률을 증가시키는 방안 제안
- 마딜, 방깍 덱 등 빌드 타입을 크게 2~3개로 나누는 방향

### NC 피드백 대응 (2026.03.06)

**핵심 USP에 대한 논의:**
- NC 측은 머지가 핵심 재미이며, 머지를 잘해서 전설 유닛을 잘 쓰는 것이 재미라고 평가
- 개발팀은 온보딩 과정에서 유저들이 가장 인지하기 쉬운 재미가 머지라는 것에 동의하되, 머지만 남길 경우 채워지지 않는 재미의 부재가 크다고 판단
- 도쿄 게임쇼 유저 테스트에서 자동 머지 요청이나, 머지가 핵심 재미는 아니라는 피드백이 다수 있었음
- 목표: "머지와 덱 빌딩"의 적절한 균형을 찾는 것

**유닛 승계 시스템 제안:**
- 유닛 카드를 최대치(3장+)로 모은 유닛의 경우, 머지 시 등급이 상승하며 최대 전설 등급까지 승급 가능
- 최대치까지 모으지 못한 경우 기존과 같이 랜덤하게 상위 등급 유닛 등장
- 이를 통해 유저가 능동적 플레이와 랜덤 합성 중 선택 가능
- 1주 내로 빠르게 테스트 후 리뷰 예정

**특성 카드 관련:**
- NC 측은 유닛 카드 삭제, 전술카드 카테고리 분류 등을 제안
- 개발팀은 유닛 카드를 모두 삭제할 경우 게임의 깊이가 너무 낮아지고 단순해지는 문제를 우려

### 코어 루프 개선안 — 승급 전설 시스템 (2026.03.15)

**핵심 문제 진단 — "지연된 선택":**
- 유닛 = 스킬 구조에서, 커먼/언커먼 유닛의 스킬카드를 선택해도 머지하면 다른 유닛으로 변경되어 투자가 무의미해짐
- 유저 피드백 233건 분석 결과: 개선 요청 1위는 과도한 랜덤성(55%), 리플레이 동기 1위는 유닛 조합 실험(60%)
- Random Dice와의 차이: 랜덤 다이스는 머지해도 같은 유닛이 강화되지만, 본 게임은 완전히 다른 유닛으로 변경됨

**해결안 — 승급 전설 시스템:**
- 스킬카드를 일정 수 적립하면 해당 유닛을 "승급 전설"로 전환 가능
- 승급 전설은 기존 스킬/속성을 유지하고 스탯만 전설급으로 상승
- 커먼은 승급 대상에서 제외 (원패턴 최적화 방지)
- 한 판에 1마리만 승급 전설 보유 가능
- 파워 서열: 커먼(15) → 언커먼(25) → 레어(45) → 에픽(60) → 승급 전설(75) → 조합 전설(100)

**등급 체계 조정 (선행 작업):**
- 언커먼: 7마리 → 6마리로 축소 ("답답한 구간" 완화)
- 에픽: 4마리 → 5마리로 확대 (패시브/버퍼 강화, 전설 대용 가능)
- 커먼(4마리), 레어(7마리), 전설(15마리)은 유지

**카드 재활용 상점:**
- 풀강화된 스킬카드 1세트를 전설 유닛 강화 카드 또는 전설 스탯 카드로 교환
- 교환 대상은 유저가 직접 선택 (랜덤 아님)
- 중간 보스 처치 시 보물상자 드랍 또는 맵 오브젝트를 통해 교환 기회 등장

**구현 우선순위:**
- P0: 등급 체계 조정, 승급 시스템 코어
- P1: 승급 가능 UI 표시 + 연출, 승급 전설을 조합 재료로 사용 허용
- P2: 카드 재활용 상점, 맵 오브젝트 승급 기회

**추가 방향 결정:**
- 머지해서 상위 유닛 및 전설로 플레이하는 게임으로 방향 확정
- 무속성 유닛 링크 제거하고 패시브 효과 부여 검토
- 속성 링크 제거, 패시브 링크로 변경
- 전설 유닛 조합식 변경

- **소환/머지 가중치**: 택틱에서 특정 유닛 특성 카드 선택 시, 해당 유닛의 소환/머지 출현 확률 대폭 상승 (은밀한 서포트)
- **3특성 마스터 특권 2종**:
  - 각성 머지: 3특성 완료 유닛 합성 시 무조건 같은 유닛 상위 등급으로 확정 진화
  - 프리패스: 3특성 에픽 유닛이 태생 전설 재료로 사용될 때 소멸하지 않고 필드에 잔존
- **전설 투트랙**: Track A 태생 전설(조합식) + Track B 각성 전설(3특성 마스터 직행)
- **유물 카지노**: 극후반에 미네랄을 소모하여 유물 상점 리롤 + 확률 강화

**스페셜 스폰 패턴:**
- 프리팹 기반 몬스터 웨이브 배치
- 좌표 기반 스폰 제어, 스킬 부여 가능
- 웨이브 리듬감 설계를 위한 데이터 드리븐 구조

### NCsoft 공식 피드백 — 머지 정체성 이슈 (2026-03-03, Ver.0.0.10)

> 출처: [[raw/feedback/NCsoft_TinyCatAssemble 피드백_20260303]]

**머지 vs 덱빌딩 정체성 충돌 (핵심 이슈):**
- 두 시스템(머지 + 로그라이크 덱빌딩)이 혼재하여 어느 쪽에도 집중하기 어려움
- 머지에 집중하면 전투 유닛 수 감소로 오히려 약해지는 역체감
- 머지 없이 초반 유닛으로 진행하는 것이 오히려 더 쉬운 상황
- 스킬 카드 효과가 배치 유닛에 귀속되어, 머지 시 기존 투자가 무의미해짐

**카드 메타 구조 전환 제안:**
- 카드 메타를 '유닛 귀속' → '캐릭터/덱 귀속 능력치 강화' 중심으로 전환
- 고등급 이상(일정 등급 이상)에서만 유닛별 특화 카드 등장
- 자원 수급량, 이동 속도 등 범용 버프 카드 비중 확대

**덱 배치 슬롯 축소 제안:**
- 8슬롯 → 3~4개 수준으로 축소 (조합 특색 명확화, 커뮤 공유 용이)
- 슬롯 축소 시 각 유닛 역할 명확화, 전투 연출/전황 파악 깔끔해질 것 기대

**전설 레시피 패턴 없음 (테스터 지적):**
- 조합 재료의 속성/공격 방식이 결과 전설과 전혀 다른 경우 다수
- 예: 불꽃(영웅)+냉각(희귀) → 물리(전설), 물리(희귀)+물리(희귀)+카오스(고급) → 전기(전설)
- 속성, 공격 방식, 역할을 계승하는 형태로 레시피 조정 필요

### GDD 기획서 — 인게임 전체 설계서 상세 (2026년 3월) 추가분

> 출처: [[raw/gdd/01_Core/인게임_전체_설계서]]

**15웨이브 경제 템포 구조:**
- 전투 페이즈: 적 처치 → 경험치 → 레벨업 → 택틱 카드 선택
- 정비 페이즈: 웨이브 종료 후 미네랄로 유닛 소환 및 머지 설계
- 빠른 진행 보상 (미구현): 목표 처치 수를 제한 시간보다 빠르게 달성 시 추가 미네랄 지급

**덱 압축 & 소환 풀 승급 (미구현):**
- 각성 전설 도달 유닛 계열은 소환 풀에서 영구 제외 → 덱 압축 효과
- 특정 등급 모든 유닛이 전설 도달 시 소환 풀 기본 등급 자동 승급
- 예시: 일반 4종 전설 달성 시 소환 버튼 기본 결과물이 고급부터 시작

**군단 평가 보상 (미구현):**
- 배치 유닛의 평균 등급 점수에 따라 레벨업 택틱에서 영웅·전설 등급 강력 카드 출현 확률 증가
- 예시: 필드 전부 에픽/전설 채우면 전설 등급 택틱 25% 확률 등장

**극후반 유물 카지노 경제 (미구현):**
- 중간 보스 처치 시 5구 유물 상점 오픈, 미네랄로 원하는 유물 직접 구매
- 리롤 비용 점증, 확률형 유물 강화(10강 돌파 시 특수 기믹 해금)
- 유물 특수 옵션 예시: 투사체 분열, 시체 폭발, 조건부 즉사, 쿨타임 0초 스팸

### GDD 기획서 — 전략성 강화 (2026년 3월)

> 출처: [[raw/gdd/01_Core/전략성_강화_기획서]]

**태그 시스템:**
- 공격 형태 태그: 소환, 지속, 범위, 연사, 넉백, 지원, 멀티타겟
- 속성 태그: 불꽃/냉각/전기/카오스/무속성 (각 유닛당 속성 1개 + 공격 형태 복수)
- 유닛 조합 직관성: 재료 유닛의 계열이 전설 유닛으로 이어지도록 구성

**택틱 재구성 (전략성_강화_기획서 기준):**
- 스택 카드: 최대 레벨 5→3으로 압축, 최대 레벨 달성 시 스페셜 카드 선택 (A/B/C 플레이스타일 분기)
- 단일 카드: 누적 불가, 1장만, 전설 등급 고정
  - 지휘관 스킬 부여(Add_Skill), 공격 형태 강화, 속성 부여/변경
- 범용 스탯 스택: 공격력·공격속도·쿨타임가속·치명확률
- 속성 강화 스택: 특정 속성 강화

**2월 빌드 피드백 이후 개선된 랜덤 제어:**
- 리롤 Pool 보정(스마트 밴): 택틱 리롤 시 제시된 카드 밴 처리 → 리롤 시 같은 카드 재등장 방지
  - 등급별 최대 필요 횟수: 일반 3회, 고급 6회, 희귀 5회, 영웅 3회
- 운명의 갈림길: 기존 분실물(랜덤 유물) 대체 → 유저가 플레이 방향을 직접 선택
  - 선택지: 주력 속성 확률 증가, 이자율 증가, 미네랄 일괄 지급, 배치 슬롯 제한+전투력 상승

**공용 스탯 배치 (등급별):**
- 일반: 공격력 증가 (All 공통)
- 고급: 공격 속도(클리버/슬러그/피즈) 또는 쿨타임(파이어/캐논/엔지니어) 계열
- 희귀: 속성별 위력 증가 (불꽃/냉기/전기/카오스/물리)
- 영웅: 치명 피해 증가 (공통)

**전설 유닛 밸런스 기준 변경:**
- 기존: 기본 성능 0~20% / 시너지 성능 80~100%
- 변경: 기본 성능 50% / 시너지 성능 50%
- 의도: 전설 유닛 소환만으로도 기본 만족감 → 시너지 완성 시 극대화

**V0012 빌드 구현 현황 (2026-03-17):**
- 스마트 밴 시스템 구현 완료 (유닛·택틱 동일 적용)
- 등급별 기준 밸런스 변경: 희귀 4→5, 영웅 8→12, 전설 10~16→16~24
- 조합식 정리: 에픽 포함 2마리 기준으로 변경
- 택틱 등급 확률 동적 제어: 배치 유닛 등급 평균 기반 카드 출현 확률 연동
