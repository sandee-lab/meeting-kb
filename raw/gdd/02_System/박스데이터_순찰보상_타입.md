# BoxData 순찰 보상 타입 추가

> **요약:** BoxData에 SweepGold/SweepExp 타입을 추가하여 n시간 순찰 보상(골드, 경험치)을 상자로 지급하는 기능. BoxPoolData에 select_sweep_reward_hour 컬럼을 추가하여 시간 기반 보상을 계산한다.
> **현재 상태:** 이후 버전으로 대체됨

**M6\_기타\_BoxData타입추가**

**개요**

* n시간 순찰 보상(골드, 경험치)을 보상으로 지급할 수 있도록 상자 타입 추가

![](data:image/png;base64...)![](data:image/png;base64...)

**테이블 수정**

* BoxData ([링크](https://docs.google.com/spreadsheets/d/1C8IoWcVPT4VTWCiiwfY40fgjd965RaYJmDFQK2kXSo8/edit?gid=394635915#gid=394635915&range=A1))
  + box\_type 추가
    - SweepGold
      * 골드 순찰 보상 상자
      * 해당 타입인 경우 BoxPoolData에서 입력한 순찰 시간 (select\_sweep\_reward\_hour) 만큼 골드 보상을 받는다.
        + 순찰 보상 기준 = 내 현재 스테이지
    - SweepExp
      * 계정 경험치 순찰 보상 상자
      * 해당 타입인 경우 BoxPoolData에서 입력한 순찰 시간 (select\_sweep\_reward\_hour) 만큼 계정 경험치 보상을 받는다.
        + 순찰 보상 기준 = 내 현재 스테이지
* BoxPoolData ([링크](https://docs.google.com/spreadsheets/d/1C8IoWcVPT4VTWCiiwfY40fgjd965RaYJmDFQK2kXSo8/edit?gid=347541572#gid=347541572&range=A1))
  + 컬럼 추가
    - select\_sweep\_reward\_hour
      * BoxData@box\_type= SweepGold, SweepExp 인 경우 참조
        + 해당 시간 만큼 계산하여 보상 지급
  + 기타
    - BoxData@box\_type= SweepGold, SweepExp 인 경우 select\_ratio는 10000으로 입력한다.
    - BoxData@box\_type= SweepGold, SweepExp 인 경우 select\_reward\_amount는 -1로 입력한다.
      * 해당 컬럼은 참조하지 않는다.
