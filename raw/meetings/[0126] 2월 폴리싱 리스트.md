| 게임 플레이 | **미네랄 수집 반복, 스테이지 반복요소
  •** 미네랄 외에 인 게임 수집요소 다양화 필요
  • 특별 스폰 타입 : 부대로 나오거나 돌진하기, 원형으로 감싸기 등 특별 스폰타입을 더 잘 활용하는 방향으로 가자.
  • 스테이지 형식 다양화 : 현실적으로 당장 할 수 없지만 소프트런칭 전 후로 반드시 고려해야 함. 스테이지 자체를 다양화 하는것은 덱을 저장하는 게임의 룰을 해치는 부분이므로 5챕터 단위로 특별 챕터가 등장해 경험을 환기 시켜주자
  • 웨이브 내에 스폰 간격 리듬감 만들기
  • 적을 처치한 위치에서 조금 떨어져서 스폰되도록, 혹은 내 주변에 너무 가까이 스폰되는 빈도를 줄이자 |
|  | **조작 반응성 개선
  •** 터치 이동 추가  :  버추어패드를 오랜 시간 이끄는게 피로할 수 있음
  • 상하 카메라 바운딩 제한하지 않기 : 카메라 바운딩이 제한되면 부대의 위치가 항상 화면 가운데에 위치하지 못하고, 그러다보면 손에 가려지는 구간이 많아짐. 좌우는 현재와 같이 유지하되, 상하 바운딩을 제거하서 엄지손가락에 유닛이 가리지 않게 하자
  • 이동 조작을 급격하게 할 수록 가운데로 모이는 속도 빠르게 해보자 : 둥둥 뜨는듯한 컨트롤 느낌은, 즉각성이 떨어지기 때문으로 보임
  • 기본속도가 올라가면 유닛 조작 반응이 좀 더 빠릿한 느낌을 받음. 기본 속도를 1.2배 하거나 처음에 광고 시청으로 1.2배 할 수 있도록 변경 |
|  | **랜덤 요소 완화
  •** 특정 속성 유닛 획득 확률 증가 : 운명의 갈림길에서 선택지로 등장하게 하자 
  • 원하는 전술 카드를 획득할 수 있는 기회 제공  : 전술카드 리롤 시 한번 돌린 카드는 해당 스탭에서는 더이상 나오지 않게 밴 처리. 다음 기회에서는 다시 나올 수 있음
  • 유닛 승급 시스템 도입 : 유닛 스킬 카드를 획득 하는것이 머지를 하는 행위와 충돌되되는 문제는 여전함. 하지만 유닛별 스킬 강화를 하지 않는다면 전술카드의 효용성이 매우 떨어지는 문제도 해결되지 않음. 따라서, 유닛 스킬카드를 3개까지 모으면 엘리트 부대원으로 명명하고 해당 유닛은 같이 머지 했을 때 새로운 유닛으로 바뀌는 것이 아니라 유닛종류는 유지되고 등급만 올리는 시스템을 도입함. 이를 통해 내가 원하는 유닛을 전설까지 머지해서 쓸 수 있고, 원치 않는 경우 일반 머지를 통해 새로운 유닛으로 받을 수 있는 선택지를 유저에게 제공해본다. 
  **•** 전설 유닛 재료 단순화 :  조합식이 너무 복잡해지면 유저들이 기억하지 못하고 지나쳐서 실제로 전설유닛을 만들어야 하는 타이밍을 지나치게 되는 경우가 많음. 에픽유닛1 + 낮은등급 1 정도를 요구하게 되면 머지를 통해 무지성 에픽을 만들던 유저들도 최종 유닛등급에서 만나게 되므로 전설 유닛을 놓칠 확률이 낮아짐. 여기에 추가로 5웨이브 정도에서 확정적으로 에픽 유닛을 가져갈 수 있는 전술카드 기회를 제공하면 확률을 더 압축할 수 있음
  • 리롤, 판매 튜토리얼 개선 :해당 기능을 잘 모르고 넘어가는 유저들이 많아 더 랜덤에 대해 부정적인 경험이 있을 수 있음
  • 유물 삭제 : 유물도 랜덤요소의 하나지만 새로운 재미를 주지는 못함. 이부분은 제거하고 운명의 갈림길로 대체 한다. 운명의 갈림길 목록은 랜덤이지만, 기존과 달리 내가 원하는 것을 선택할 수 있다는 차이점이 있음. |
|  | **시인성 개선
  •** 다수 유닛, 이펙트 중첩 상황 : 유저에게 이펙트를 끄거나, 유닛들의 반투명도를 줄이는 옵션 제공. 몬스터 유닛의 표현정도를 약하게 해서 조금 더 유닛이 잘 보이게 대비 
  • 지휘관 강조 (아웃라인 오브젝트 추가로 만들어서 가장 위쪽에 랜더링하기)
  • 지휘관보다  조금 큰 사이즈로 컬라이더 만들어서 몬스터와 유닛이 안으로 못들어오게 하기. 그러면 지휘관 주변에 약간의 여유 공간이 생길 수 있음
  • 지휘관 피격 : 카메라 쉐이크 및 진동은 조금 약하게 변경. 30% 이하에서 스크린 이펙트 추가
  • 마지막 타격 시 카메라 줌 인 : 죽을 위험이 오면 슬로우 걸면서 확대해서 알려주기
  • 전체 유닛 범위 동적 표현 : 지휘관 뿐 아니라 전체 유닛이 부대인만큼 화면의 어느 영역까지가 부대의 피격범위인지 동적으로 사이즈를 체크해서 바닥에 그려주는 기능 추가
  • 경험치 보석획득 리듬감 :  10/30 단위로 묶음 처리, 날아오는 속도가 너무 빠르다는 의견이 많아, 이지값을 넣고, 속도를 줄여 체감되록 변경
  • 재화 획득 연출 변경 : 기존에 화면을 날아가는 연출을 지우고, 해당 위치에서 몇개를 획득했는지만 표기
  • 맵 오브젝트 제거 : 플레이 영역 안에 불필요한 오브젝트 최대한 제거
  • 몬스터 사망 이펙트 : 뼈가 튀어오르는 것과 전기가 터지는 등 너무 과한 이펙트 제거
  • 보스 및 특별 유닛 패턴 출현 연출 : 최소화
  • 설치형 스킬 : 아군 것인지 명확하게 컬러 구분
 |

  • 현재 파워 표기 추가 : 인 게임 플레이 중 새로운 전술 카드나 새로운 유닛을 얻어서 배치할 경우 파워의 변동을 보여주고 현재 기준에서 낮은지 높은지 가이드
  • 속성 덱이 유리한 컨텐츠 : 좀 더 다양하게 덱이 활용될 컨텐츠가 더 많이 필요함
  • 전설 유닛 추가 : 전설의 역할이 딜러뿐이 없기 때문에 서포터나 힐러 등 더 다양한 역할군의 추가 |
