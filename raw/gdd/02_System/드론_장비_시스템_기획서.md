# 폴리싱 - 장비개선(드론)

> **요약:** 기존 RPG 장비를 공격/폭격/방어 3종 역할군의 드론 시스템으로 전환하는 기획이다. 각 타입 2슬롯(총 6슬롯) 구조로, 챕터별 슬롯 해금, 동일 sub_type 중복 장착 방지, 상위 등급 교체 시 강화 레벨 이전 등의 규칙을 포함한다. 장비 타입 이넘 변경(AttackDrone/BombardDrone/DefenceDrone), 슬롯 선택 팝업, 추천 규칙 등 UI/UX 변경사항도 상세히 정의한다.
> **현재 상태:** 미적용

**폴리싱\_장비개선(드론)**

### ![](data:image/png;base64...)

**개요**

* 성장 체감 안되는 장비 성장 대신 '드론'이라는 새로운 파밍/육성처를 제공합니다.
  + 공격/폭격/방어 3가지 역할군의 드론을 조합하는 재미를 제공합니다.
  + 지휘관 주변을 비행하는 드론 군단을 통해 성장을 시각적으로 체감하게 합니다.

**요약**

| **구분** | **기존** | **변경** |
| --- | --- | --- |
| 장비 시스템 | 일반 RPG 장비 (투구/갑옷 등) | 배틀 슈트 (본체) + 6드론 (파츠) |
| 전투 방식 | 무기로 공격 | 드론 전투 및 지휘관 공격 |
| 성장 방식 | 단순 레벨업 + 승급 | 드론: 머지(합성) + 레벨업  지휘관: 승급 + 레벨업 + 큐브(지휘관) |

## **성장 시스템**

### 성장 구조 - 기존 장비와 같음

#### 레벨업

* + - 재화: 골드 + 드론 스크롤(타입별).
    - 효과: 드론의 깡공격력/깡체력 상승.
    - 제한: 현재 등급의 Max Level까지만 가능.

#### 승급

* + - 조건: 동일 ID + 동일 등급 2개 합성. (조합 테이블에 따라 제어)
    - 효과:
      1. 등급 상승 (일반 -> 고급 -> 희귀...)
      2. Max Level +10 확장.
      3. 잠재력 해금 (잠재력 활성화).

###

### 고정 옵션 테이블 - 기존 장비와 같음

* + 특정 등급에 도달하면 드론의 스킬들이 강화됩니다.
  + 등급별(일반~신화 등) 해금되는 효과 테이블.

**3. 장비 추천 규칙**

(1) 기준 : 추천 태그는 “해당 슬롯 기준으로 전투력이 상승할 때만"

**4. 추천 규칙 상세**

4-1. 빈 슬롯이 있을 때

(1) 추천 대상 :

- 해당 슬롯에 장착 가능한 장비 중 등급(티어 포함)이 가장 높은 장비

- 동급이면 강화 레벨이 높은 장비

- 강화도 같으면 모두 추천 태그 표시

(2) 예외 : **[확인불가]**

- 다른 슬롯에 동일 타입 장비가 이미 장착 중인 경우,

해당 타입의 현재 착용 장비보다

등급(티어 포함)이 높거나,

동급일 경우 강화 레벨이 높은 경우에만

추천 태그를 표시한다.

- 동급이면 강화 레벨이 높은 장비

(2) 예외 : **[위 빈 슬롯 예외 사항이 이곳에도 적용되어야 할 것 같습니다. 타입이 같을 때는 자동 교체해주어서요.]**

- 다른 슬롯에 동일 타입 장비가 이미 장착 중인 경우,

해당 타입의 현재 착용 장비보다

등급(티어 포함)이 높거나,

동급일 경우 강화 레벨이 높은 경우에만

추천 태그를 표시한다.

5. 추천 규칙 통합 요약

추천 태그는 “현재 슬롯에 장착했을 때, 타입 충돌을 고려한 결과 존 장비보다 확실히 강해지는 장비”에만 표시한다.
