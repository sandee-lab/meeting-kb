# 튜토리얼 & 온보딩

> 출처: [[raw/meetings/[1220] 튜토리얼 예시]], [[raw/meetings/[1223] 튜토리얼 제작]], [[raw/meetings/[0103] 폴리싱 리스트 Alpha 1]], [[raw/meetings/[0110] 폴리싱 리스트 Alpha 2]], [[raw/meetings/[1125] 튜토리얼 및 택틱 정리]], [[raw/meetings/[0915] 플레이 시나리오]], [[raw/feedback/Minicraft_0.0.6 사업팀 피드백 취합]], [[raw/feedback/미니크래프트_0.0.6_B팀 테스트 피드백]], [[raw/feedback/미니크래프트_0.0.6_럭키배틀 피드백 by PCELL]], [[raw/feedback/comtus_타이니캣어셈블_옥석감정단평가서]], [[raw/gdd/01_Core/초반_플레이_시퀀스]] *(대체됨)*, [[raw/gdd/05_UX/성장_가이드_시스템_기획서]], [[raw/gdd/02_System/달성_임무_조건_기획서]]

## 개요

신규 유저가 게임의 핵심 메커니즘(유닛 소환, 카드 머지, 골드 강화 등)을 자연스럽게 학습할 수 있도록 설계된 튜토리얼 및 온보딩 시스템이다. 초기 버전(Alpha)에서는 인게임 도입부 연출과 손가락 가이드 중심으로 설계되었으며, 이후 스토리 컷신과 다이얼로그 기반의 학습 흐름으로 발전하였다. 게임을 강제로 멈추지 않고 자연스러운 플레이 속에서 학습이 이루어지는 것을 원칙으로 한다.

## 상세 내용

### 1. 초기 튜토리얼 레퍼런스 (2024.12.20)

- Clash Mini의 튜토리얼을 참고 자료로 검토
- 시퀀스 이미지 기반의 튜토리얼 흐름을 벤치마크

### 2. 인게임 튜토리얼 설계 (2024.12.23)

튜토리얼은 **도입부**와 **본 튜토리얼** 두 파트로 구성된다.

**도입부:**
- 저글링 2~3마리가 우측에서 접근하며 메시지 표시
- 카메라가 우측으로 이동하여 몬스터 생성 장면과 콜로니를 보여줌
- 카메라가 좌측으로 복귀하며 비행기가 나타나 총냥이 유닛 10여 마리를 연속 생산
- 전투로 인해 유닛이 모두 사망하는 장면 연출

**본 튜토리얼:**
- 게임 플레이 가능 시점으로 전환 후 카드 UI와 메시지 창 팝업
- 1초 후 웨이브 정보 표시, 동시에 우측에서 카드 진입
- 웨이브 정보 사라지면 상단에 웨이브 정보 고정 표시
- 튜토리얼 초반에는 미션 정보를 표시하지 않음
- **2초 내 입력이 없을 경우** 손가락 가이드 표시 (게임을 멈추거나 강제하지 않음)
- 유닛 카드 생산 시간 관련 툴팁은 3번째 카드 생성 시까지 유지 후 자동 소멸
- 카드 3장 이상 드로우 시 유닛 부스팅 버튼 가이드 진행
- **머지 가능 상황 발생 시** 화면을 멈추고 머지 관련 튜토리얼을 강제 표시 (유일하게 강제 진행되는 구간)

### 3. Alpha 1 폴리싱 - 튜토리얼 구현 (2025.01.03)

- 튜토리얼 앞부분 구현 착수
- 일반 시민 디자인 작업 포함

### 4. Alpha 2 폴리싱 - 튜토리얼 및 아웃게임 온보딩 (2025.01.10)

**아웃게임 플로우 설계:**
- 처음 게임 시작 시 하단 메뉴에서 카드와 강화는 잠금 처리
- 배틀 1회 진행 후 골드 강화만 언락, 손가락 가이드 진행
- 배틀 2회차 진행 후 카드 메뉴 언락
- 배틀 2회차 이후에는 골드 강화 가이드 중단
- 스테이지 클리어 보상 수령 가능 시 해당 메뉴로 손가락 가이드
- 카드 뽑기 가능 재화 확보 시 카드 메뉴로 손가락 가이드

**인트로 개선:**
- 스킵 기능 추가
- 카메라 이동 시간 및 이벤트 사이 시간 단축
- 전투 종료 지점 앞당기기

**인게임 플로우 디테일:**
- UI가 아래에서 위로 올라온 후 0.5초 대기 후 툴팁 표시
- "now it's your turn" 대사 시 웨이브 정보 미표시 (1.5초간 대사 후 웨이브 정보)
- 최종적으로 2초 후 웨이브 정보 확인 가능
- 광폭화 관련 설명 유지시간 3초
- 광고 버튼 활성화: 1~2 웨이브 콜로니에 기지 도착 시 표시, 첫 광고는 무료 제공
- 레이지 모드 설명은 2챕터 진입 이후 (1챕터에서는 레이지 모드 제거)

### 5. 플레이 시나리오 기반 온보딩 구조 (2025.09.15)

챕터별 기능 언락을 통한 단계적 온보딩 설계:

| 챕터 | 플레이 시간 | 구성 | 언락 요소 |
|------|------------|------|-----------|
| 1챕터 | 5분 | 10웨이브, 1보스 | 소환 언락 |
| 2챕터 | 10분 | 10웨이브, 1보스 | 클리어 보상 언락 → 영웅 강화 → 가이드 완료 → 영웅 뽑기 |
| 3챕터 | 20분 | 15웨이브, 2보스 | 골드 강화 언락 → 7일 미션 → 인벤토리 → 상점 |
| 4챕터 | 30분 | 20웨이브, 3보스 | - |

### 6. 최신 튜토리얼 설계 (2025.11.25)

**핵심 컨셉: "가벼운 이야기에서 학습의 자연스러운 연결"**

튜토리얼 흐름:
1. **스토리 컷신:** 2~3페이지 분량, 첫 플레이 배경 행성 도착 과정 전달
2. **인게임 진입:** 다이얼로그를 통한 상황 설명
3. **이동 가이드:** 터치 이동을 5초 이상 유지할 때까지 강제 이동 가이드 표시
4. **최초 전투:** 시간 제한 없음. 몬스터 처리 중 다이얼로그로 위험 알림, 지원군(불냥이 2마리) 도착
5. **미네랄 채집:** 지원군 요청을 위한 미네랄 채집 유도, 근처에 1개 생성 후 자유 전투 (15초)
6. **택틱 선택:** 첫 택틱 획득, 레벨업 시 다이얼로그로 선택 가이드. 택틱 종류 3가지 고정 (체감 확실한 스킬)
7. **아머리 진입:** 지원 요청, 배치, 머지까지 한번에 가이드
8. **보스 전투:** 택배 획득

**재미 사이클 설계 (5웨이브 기준):**
- 택틱을 자주 선택하게 유도 (웨이브당 2~3개, 특정 이벤트에 여러 개 동시 지급)
- 택틱 지급 직후 몬스터 몰려오게 하여 효과 체감
- 일반 유닛 3마리 지급, 신규 유닛 설명 및 필드 언락
- 아머리 진입은 2회 초과 금지
- 처음에는 유닛을 너무 다양하게 지급하지 않음
- 유닛 강화카드 한번에 여러 장 지급
- 전설 유닛 빠르게 조합할 수 있도록 구성

### GDD 기획서 — 온보딩 상세 (2026년 3월)

- 챕터 1~2의 콘텐츠 언락 순서와 소환 튜토리얼 설계 초안
- 이후 [1125] 튜토리얼 및 택틱 정리 문서로 대체됨

**성장 가이드 시스템 (미적용):**
- 던전 실패 시 병목 진단 → 덱 분석 → 가이드 분기를 자동 제공
- 7개 지표(덱 저장 스테이지, 평점, 유닛/장비/지휘관 레벨, 업그레이드 레벨) 종합 평가
- 덱 구성(1순위) → 업그레이드(2순위) → 장비(3순위) → 유닛(4순위) → 지휘관(5순위) 우선순위 분기
- 조건 충족 상위 3개 항목 우선 출력, 각 분기에서 관련 콘텐츠/패키지로 자연스럽게 유도
- 기준값: 기획자 디폴트 → 서버 통계 → 하이브리드 3단계 운영 방식
- 상세 설계: [[wiki/concepts/성장-가이드-시스템]] 참고

**달성 임무 조건:**
- 프로필 레벨 언락용 달성 조건 추가
- 누적 처치, 누적 소환, 스테이지 클리어 등 초보자 미션 체계
- 달성 보상으로 성장 재화 제공하여 자연스러운 온보딩 유도

### GDD 기획서 — 스토리 기반 온보딩 (2026년 3월)

> 출처: [[raw/gdd/01_Core/스토리_설정]], [[raw/gdd/01_Core/초반_플레이_시퀀스]] *(대체됨)*

**스토리 컨셉 (스토리_설정 V5):**
- SF 블랙코미디: 우주 대기업 카마존 말단 로봇이 한정판 츄르를 찾아 외계 행성 탐사
- 슈뢰딩거 상자 = 가챠 메타포 ("열어봐야 아는 랜덤 상자")
- 칼냥이(전투 유닛) = 슈뢰딩거 시스템 적용 전투 파트너 → 소환의 내러티브 근거

**튜토리얼 컷신 구성:**
1. 오프닝: 우주 배경 비행선 → 상자 분실 사고 → 전시 프로토콜 발동
2. 맵 도착: 이동 가이드 (지직- 매니저 무전 "전투 유닛 지원, 상자를 찾으세요")
3. 미네랄 가이드: 매니저 대화 ("현장 조달이에요")
4. 아머리 > 고용: 매니저 "열어봐야 알죠. 슈뢰딩거니까요"
5. 보스 처치 > 유물 확인: 상자 열기, 본사 확인 후 "아니래요"
6. 챕터 클리어 컷신: 상자 훔친 행성 폭파 — 전시 프로토콜 7조 3항

### M5 초보자 퀘스트 시스템 (아이디어 단계, 미구현)

> 출처: [[raw/gdd/02_System/M5_아웃게임_초보자퀘스트]]

신규 유저의 초반 단기 목표 제시 및 구매 유도를 목적으로 설계된 별도 퀘스트 세트 시스템. (아이디어 단계, 미구현)

**일자별 그룹 잠금 해제 구조:**
- 퀘스트는 일자(Day)를 기준으로 그룹화되며, 해당 일차가 도래해야 그룹 해제
- 유저 접속 여부와 무관하게 일차 도래 시 자동 해제 (사전 달성은 가능, 표시만 잠금)
- 기간 내 모든 퀘스트 보상 수령 + 진행도 보상 + 상품 구매 완료 시 조기 종료

**전용 토큰 수집 → 진행도 보상:**
- 각 세트마다 전용 재화(토큰)가 지정됨
- 퀘스트 완수 및 상품 구매 시 전용 토큰 지급
- 보유 토큰이 특정 수량 이상이 되면 진행도 보상 수령 가능 (고가치 아이템 배치)

**기간 한정 패키지 배치:**
- 각 일자별로 기간 한정 패키지 상품이 배치됨
- 일반 패키지와 달리 무료 판매 또는 복수 구매 허용 가능
- 패키지 구매 시 추가 전용 토큰 획득 → 진행도 보상으로 이중 유도

**신규 유저 초반 목표 제시 설계 의도:**
- 퀘스트 파트와 전용 상점 파트를 하나의 [세트]로 구성하여 통합 진행
- 최초 해금 시 강제 팝업으로 신규 유저에게 즉시 노출
- 누적 접속일에 따라 초보자퀘스트1, 2 등 연속 운영 가능
