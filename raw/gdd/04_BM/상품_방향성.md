## 2. 카테고리별 의도 정의

### 2-1. 일일 상점 (Daily Shop)

| 항목 | 내용 |
|------|------|
| **핵심 역할** | 다이아 소비 유도 (보석 싱크) |
| **밸런스 통제** | 할인 상품(20%/40%)의 출현 가중치로 공급량 조절 |
| **과금 재화 창구** | 황금 디스크, 전설칩 조각 등 과금 전용 재화를 무료 다이아로 구매 가능한 유일한 경로 |

현행 구조:
- 86개 상품 풀에서 가중치(`select_weight`)로 랜덤 진열
- 가중치 10,000 (무료 보석) / 1,000 (정가) / 300 (20% 할인) / 100 (40% 할인)
- 정가 = 기준 가치 (마진 0%), 할인 상품이 실질적 이득
- 초기화: 골드 1,000 (일3회) 또는 광고 (일3회)

### 2-2. 패키지 (Package)

| 항목 | 내용 |
|------|------|
| **핵심 역할** | 챕터 진행에 따른 고효율 구매 기회 제공 |
| **해금 빈도** | 초기 매 챕터 → 이후 3~5챕터 간격으로 점진적 확대 |
| **가격대** | 소과금 ~ 중과금 (다양한 가격대 설정) |
| **보상 구성** | 특정 컨셉 아닌 범용 재화 다양 조합 |

### 2-3. 소환 (Draw/Summon)

| 소환 타입 | 핵심 역할 | 타겟 |
|----------|----------|------|
| **유닛 소환** | 유닛 기본 수급처, 초반 다이아 소모 유도 | 전체 |
| **일반 장비 소환** | 장비 기본 수급처, 무과금 유저를 위한 장비 수급 | 무과금 |
| **스페셜 장비 소환** | 상위 등급 장비 수급처, 과금 유저를 위한 장비 수급 | 과금 |

### 2-4. 보석 상점 (Currency)

| 항목 | 내용 |
|------|------|
| **핵심 역할** | 다이아 가치의 기준 제시 |

현행: 6개 상품 (1,400원~140,000원), 첫구매 2배, 볼륨 디스카운트

### 2-5. 골드 상점 (Gold)

| 항목 | 내용 |
|------|------|
| **핵심 역할** | 부족한 골드 수급처 |
| **가격 설계** | 골드 가치가 진행도에 따라 변동하므로 절대량이 아닌 시간 기준 티켓 상품으로 제공 |

현행: 3개 상품 (2시간/6시간/24시간 골드 스윕)

### 2-10. 명예의 전당 (골든벨)

**골든벨 패키지** (최초 돌파 유저 대상)

| 항목 | 내용 |
|------|------|
| **타겟** | 선착순 랭킹 등록 유저 (과금 유저 가능성 높음) |
| **핵심 역할** | 과금 유저에게 상품 노출 + 구매 시 다른 유저에게 보상 제공 |
| **심리 설계** | 과금이 사회적 명예로 전환되는 보상 구조 |

**골든벨 특별 패키지** (전체 유저 대상)

| 항목 | 내용 |
|------|------|
| **차별화** | 선착순 달성 유저의 이름이 패키지에 부여 |
| **가치 설계** | 일반 챕터 클리어 패키지보다 고효율 |
| **보상 구성** | 선택 상자 등 효용 가치 높은 아이템 (유닛 선택 상자, 장비 선택 상자 등) |

구조:
```
1. 유저가 챕터 20/25/.../60 글로벌 최초 돌파
    ↓
2. 골든벨 상품 푸쉬 (48시간 한정, 5,000원)
    ├─ 구매 시 → 본인에게 buy_reward 지급
    └─ 구매 시 → 다른 유저에게 treat_reward 지급 (서버 우편)
    ↓
3. 기록 달성 시 → 모든 유저 대상 특가 상품 3종 등록 (골든벨 구매와 무관)
```
