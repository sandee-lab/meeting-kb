# 폴리싱 - 데미지 통계에 배틀슈트 추가

> **요약:** 배틀슈트(구 지휘관)에 공격 스킬이 부여됨에 따라, 일시 정지 통계와 챕터 결과창 통계에 배틀슈트의 데미지 정보를 추가한다. 누적 피해량과 초당 피해량(소수점 1자리)을 전투 유닛과 동일하게 처리하며, 정렬 시 전투 유닛과 구분하지 않고 통합 정렬한다. CommanderData ID 1~5에 대응하는 UnitData ID 901~905를 매핑한다.
> **현재 상태:** 미적용

**폴리싱\_데미지 통계에 배틀슈트(구 지휘관) 추가**

![](data:image/png;base64...)![](data:image/png;base64...)

**개요**

* 배틀슈트(구 지휘관)에 공격 스킬을 부여함에 따라 통계에 배틀슈트 정보를 추가한다.

**대상**

* 일시 정지 > 통계
* 챕터 결과창 > 통계

**처리**

* 배틀슈트 유닛의 데미지 정보
  + 전투 유닛과 동일하게 처리한다
    - 누적 피해량 : 피해량 합산
    - 초당 피해량 : 피해량 / 참여 시간 > 초당 피해량 환산
      * 소수점 최대 1자리까지 표기 (2자리 이하 절사)

| 배틀슈트 ID (CommanderData) | 해당 배틀슈트의 유닛 ID (UnitData) |
| --- | --- |
| 1 | 901 |
| 2 | 902 |
| 3 | 903 |
| 4 | 904 |
| 5 | 905 |

* 정렬 규칙
  + 기존 정렬 규칙에 포함시킨다.
    - 전투 유닛과 배틀슈트를 구분하지 않는다.
* 아이콘
  + UnitCostumeData@해당 배틀 슈트 id의 icon 참조
