# 전략성 강화 아웃게임 성장 요소 정리

> **요약:** 전략성 강화에 따른 아웃게임 성장 요소를 장비/유닛/지휘관/메인UI 4개 콘텐츠로 정리한 기획서이다. 장비는 무기/장신구에만 스킬을 배치하고 슬롯 구조를 변경하며, 유닛 강화 효과는 해당 유닛에게만 적용하고 시작 택틱 부여 방식으로 변경한다. 지휘관은 계수 성장 방식으로 전환하고 승급 택틱을 시작 택틱으로 전환한다. 메인 UI는 훈련 모드 추가와 콘텐츠 배치를 재정리한다.
> **현재 상태:** 미적용
