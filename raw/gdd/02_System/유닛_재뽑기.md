# 유닛 재뽑기(리롤)

> **요약:** 인게임에서 유닛을 동일 등급 내 랜덤 교체하는 재뽑기 시스템. 등급별 재뽑기 비용과 재뽑기 대상 유닛 풀을 정의한다.
> **현재 상태:** 이후 버전으로 대체됨

M6\_유닛 재뽑기

# 목표

* 유닛 판매 기능은 리스트가 커서 게임 플레이에서 잘 사용되지 않는 수준이다.
* 판매 기능 보다 효용성이 있는 재뽑기 기능을 제공하여 게임 플레이를 보다 원활하게 한다.

# 시스템 동작 구조

## 재뽑 시도

* + 포메이션 슬롯 혹은 인벤토리 슬롯의 유닛을 [재뽑 영역]으로 드래그& 드랍 하여, 재뽑기를 시도할 수 있다.

## 재뽑 시도 비용

* + 재뽑 시도 시 재화를 소모한다.
  + 소모하는 재화량은 개별 유닛 별로 설정하되, 각 유닛의 등급에 따라서 밸런스를 적용한다.
  + 재화 소 모시 UI 상에서 재화가 소모되는 연출을 간략하게 추가해준다.

## 재뽑 시도 대상

* + 재뽑 시도는 지정한 등급의 유닛만 가능하며, 재뽑 시도 비용 값이 -1인 경우 시도할 수 없는 유닛으로 판정한다.

| 등급 | reroll\_fee |  |
| --- | --- | --- |
| Common | 1 |  |
| Uncommon | 2 |  |
| Rare | 3 |  |
| Epic | -1 | 재뽑 시도 불가 |
| Legendary | -1 | 재뽑 시도 불가 |

## 재뽑 시도 결과

### 유닛 선택

* + - 재뽑 시도 대상 유닛과 동일한 등급 중 하나의 유닛을 랜덤하게 결정한다.
    - 단, 동일한 유닛은 선택지에서 제외한다.

### 유닛 생성

* + - 재뽑 시도 대상 유닛이 배치되었던 슬롯에 유닛을 생성한다.
    - 생성 시 등급에 맞는 생성 연출을 실행한다.
