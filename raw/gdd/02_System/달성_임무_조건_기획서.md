# 달성 임무 조건 추가

> **요약:** 초보자 미션, 프로필 언락 조건 등에 활용되는 달성 조건 목록을 정의한다. 누적 광고 시청, 던전 단계, 덱 평점, 전투력, 계정 레벨, 유닛/장비 레벨, 재화 획득/소모, 소환 횟수 등 17종의 달성 조건을 포함하며, 달성 조건은 초기화되지 않는 누적 방식으로 처리된다.
> **현재 상태:** 미적용

개요

* 초보자 미션, 프로필 언락 조건 등으로 활용
* 달성 조건은 초기화되지 않는다.
* 기존에 구현한 조건 체크 후 리스트 fix

달성 조건

* 누적 광고 시청 횟수
* 골드 던전 단계 달성
* 유닛 던전 단계 달성
* 슈트 던전 단계 달성
* 덱 평점 달성
* 일일 임무 완료 횟수
* 전투력 달성
* 계정 레벨 n 달성
* 유닛 레벨 누적 n 달성
* 등급 n 이상 장비 n개 장착
* 골드 - 일반 업그레이드 n 단계 달성
* 골드 - 특별 업그레이드 n 단계 달성
* 골드 누적 n 획득
* 다이아 누적 n 획득
* 골드 누적 n 소모
* 다이아 누적 n 소모
* 유닛 누적 소환 횟수
* 장비 누적 소환 횟수
