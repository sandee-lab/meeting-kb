# 일시정지

* 유저는 게임 진행 중 [일시정지] 버튼을 눌러 진행을 멈출 수 있다.
* [일시정지] 시, 게임 진행을 중단하고 [일시정지 팝업]을 출력한다.
* 앱이 켜져 있는 동안에는 상태를 유지한다.
* 유저가 게임 진행 중 앱을 내린 경우, 일정 시간 동안은 [일시정지] 상태로 유지한다.
  + 10분 - 논의 필요 (10분 유지 또는 시간 연장)
    - 현재 세션 시간은 약 10분, 유저가 약 10분간 서버랑 통신하지 않으면 끊깁니다.
  + 이후에는 강제 종료로 분류한다. (이어하기)

# 이어하기

* 게임 강제 종료 후 새로 앱을 열었을 때, [이어하기] 옵션을 제공한다.
  + 게임 기록 저장 필요 - 논의 필요 (서버 저장 여부, 새 게임 시작할때마다 날리는 정보)
    - 스테이지 : 아머리 진입 시 진행 내용 저장. (챕터-최종 웨이브 정보)
    - 던전 : (던전 진행 방식에 따라 다름)
* 유저가 [이어하기] 선택 시, 해당 게임을 불러온다.
  + 스테이지 : 다음 웨이브 준비 화면 (ArmoryUI)를 열어준다.
    - 예 : 스테이지 1챕터 19웨이브 도중 강제 종료했다면, 1챕터 19웨이브 시작 ArmoryUI 에서 시작한다.
      * 현재 마지막 전투 상황 저장은 어렵습니다. (클라에서 서버에 요청을 보내야 하여, [종료] 같은 액션을 유저가 직접 취해야 함)
  + 던전 : (던전에 따라 결정)
* 유저가 [이어하기] 미선택 시, 해당 기록은 삭제한다.
