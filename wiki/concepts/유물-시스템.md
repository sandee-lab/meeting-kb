# 유물 시스템

> 출처: [[raw/gdd/02_System/유물_시스템_기획서]], [[raw/meetings/[1027] 던전 및 컨텐츠 제안]], [[raw/meetings/[1124] 골드 특별 성장]]

## 개요

인게임 보상으로 유물 조각을 수집하고 강화하여 계정 전체에 영구 능력치를 적용하는 아웃게임 성장 시스템이다. 등급 없이 강화만 존재하며, 유물 분해/삭제는 불가하다. 향후 초월 유물로 확장 가능한 구조로 장기 수집 목표를 제공한다.

> **참고:** 기존 인게임 "유물 상자"(유닛 아이템 뽑기)는 제거되며, "유물"은 본 아웃게임 성장 시스템의 전용 명칭이 된다.

## 상세 내용

### 시스템 구조 (GDD v1.0, 2026.03)

> 출처: [[raw/gdd/02_System/유물_시스템_기획서]]

**핵심 제한사항:**
- 등급 없음, 강화(Lv.0~10)만 존재
- 유물 분해/삭제 불가 (획득하면 영구 보유)
- 세트 보너스 없음
- 강화 비용: 골드만 소모 (다이아 등 추가 재화 없음)
- 런칭 시 조각 교환 시스템 미포함 (잉여 조각은 재고 누적)

**기본 구성:**
- 일반 유물 27종 (런칭)
- 해금 조건: **25챕터 클리어** (ContentsUnlockData - Artifact 타입)
- 유물 조각 드롭 시작: StageChapterData `reward_item_type_2` 컬럼에 유물 상자 배치

**보유 효과:**
- 유물 획득 즉시 계정 전체에 능력치 적용 (장비/장착 개념 없음)
- Lv.1 해금 시: 최대 효과의 **50%** 즉시 적용

**획득 경로:**

| 경로 | 보상 | 수량 |
|------|------|------|
| 챕터 클리어/순찰 | 유물 상자 | 약 1.5개/일 |
| 보스 모드 | 유물 상자 | 10개/주 |
| 이벤트 패키지/핫딜 | 유물 조각 지정 판매 | 운영 시점 별도 기획 |

- 상자 개봉 시: 전체 27종 균등 확률로 유물 1종 결정
  - 미보유 유물: Lv.1 해금 + 50% 효과 즉시 적용
  - 보유 유물: 해당 유물 조각 +1

### 강화 시스템

**능력치 배분 구조:**

| 레벨 | 상태 | 누적 효과 |
|------|------|----------|
| Lv.0 | 미획득 | 0% |
| Lv.1 | 해금 | **50%** |
| Lv.2~9 | 강화 | +5%씩 (55%~90%) |
| Lv.10 | **만렙** | **100%** |

**강화 비용 구조 (`GrowArtifactEnchantData`):**

| 강화 레벨 | 필요 조각 | 필요 골드 |
|----------|----------|----------|
| Lv.0→1 | 1 | 0 |
| Lv.1→2 | 2 | 2,000 |
| Lv.2→3 | 3 | 3,000 |
| Lv.3→4 | 5 | 5,000 |
| Lv.4→5 | 7 | 7,000 |
| Lv.5→6 | 10 | 10,000 |
| Lv.6→7 | 12 | 12,000 |
| Lv.7→8 | 15 | 15,000 |
| Lv.8→9 | 20 | 20,000 |
| Lv.9→10 | 25 | 25,000 |

- 유물 1개 완성(만렙)에 총 **100조각** + 약 99,000골드 필요
- 만렙 이후 잉여 조각은 재고 누적 (향후 초월 유물 업데이트 시 변환 활용)
- 유물 조각은 Consumable 타입으로 관리 (`fee_id_1`로 참조)

### 데이터 테이블 구조

**신규 테이블 2개:**
- `GrowArtifactData`: id / artifact_type(Normal/Transcend) / artifact_effect_type / local_name / local_desc / stat_type
- `GrowArtifactEnchantData`: id + enchant_level(복합키) / fee_type_1,2 / fee_id_1,2 / fee_amount_1,2 / stat_value

**기존 테이블 확장:**
- `ContentsUnlockData`: Artifact 콘텐츠 타입 추가

**Enum 정의:**
- ArtifactType: Normal(런칭) / Transcend(향후)
- ArtifactEffectType: Stat(기본) / 향후 TacticBoost 등 확장 가능

### BM 목표

- 유물 조각 1개 = **3,000원** 가치 기준 설정
- 이벤트 패키지/핫딜을 통한 핵심 유물 조각 판매 고려 (P2)
- 기본은 게임 플레이를 통한 획득 (인게임 보상)

### 회의록 기반 논의

[1027] 던전 및 컨텐츠 제안에서:
- 챕터마다 다른 컨셉의 유물 존재
- 특정 유물 획득을 위한 챕터별 조합 존재
- 속성덱 강화가 유물의 핵심 스탯

[1124] 골드 특별 성장에서 (이전 설계 — 인게임 유물 기반):
- 유물 등급 확률 증가 / 유물 슬롯 +1 (개별 룰렛 적용)
- 신규 유물 해금 (챕터별) / 유물 저주 확률 감소 / 축복 확률 증가
- 합성 시 저주 해제 확률 증가
- (미확인) 이 내용은 아웃게임 유물 시스템과 별개의 이전 인게임 유물 기획 내용일 수 있음

### 확장 방향

- 초월 유물로 확장 가능한 구조 (`artifact_type = Transcend`)
- 영구 누적 성장 동기 제공
- 핵심 상품 매력도를 보존하면서 콘텐츠 보상으로 활용 (Ncsoft 제안)
