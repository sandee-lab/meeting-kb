# 골드 던전 인게임 폴리싱

> **요약:** 골드 던전(황금 행성)의 인게임 폴리싱 기획서로, 1분 내 황금 돼지 50마리 처치 목표의 상세 규칙을 정의한다. 일반 돼지(도망 패턴, UnitData@score=1)와 대형 돼지킹(10마리 카운트), 스폰 시스템, 돼지 행동 패턴(감지/도망/경계충돌), 결과 화면 UI, 폴리싱 항목(중앙 스폰, 아이들 이동, 웨이브 스킵, 코인 드랍 연출 변경)을 포함한다.
> **현재 상태:** 미적용

## **UI**

### **상단 (고정)**

**황금 돼지 카운터 및 게이지**

잡아야하는 돼지 수 출력

* 처음 : 50
* X마리 처치 시 50-X
* 50마리 잡을 시 0
* 바의 길이도 처치율 (현재 잡아야하는 돼지 수 / 전체 목표 킬카운트)

업데이트 : 돼지 처치 즉시

* UnitData@score로 제어
  + 일반 황금 돼지 잡으면 1점
  + 황금 돼지킹 잡으면 10점

**시간 게이지 -** 기존 보스 타이머 게이지와 같음

**기획의도:** 타이머와 같이 줄어드는 것으로 경주와 같은 긴장감 제공

**일시 정지**

* 버튼 터치 시
* 일반 스테이지와 마찬가지로 일시 멈춤
* 보유 택틱 표시, 유물 표시, 나가기, 이어하기, 딜미터기 버튼 그대로 사용

**골드 보유량 표시**

* 삭제
* 삭제 이유 : 던전 실패 시 보상 없어야함 안그러면 악용할 수 있음

### **플레이어**

**이동**

* 자동 이동 없음
* 수동이동만 가능
* 화면 터치 드래그 (기존 스테이지와 동일)
* 이동 속도 (기존 스테이지와 동일)

**유닛 배치**

* 배치된 덱 그대로 사용
* 배치 변경 불가
* 유닛들은 대형 유지 (기존 스테이지와 동일)

**공격**

* 유닛들이 자동 공격 (기존 스테이지와 동일)
* 사거리 내 돼지 자동 타게팅

**일반 황금 돼지**

던전 몬스터 공용

* 공격력 (황금돼지는 사용안함)

UnitData@atk\_weight \*
DungeonStageData@add\_atk \*
GoldDungeonWaveData@atk\_weight

* 체력 : UnitData@max\_hp\_weight \*
  DungeonStageData@add\_max\_hp \*
  GoldDungeonWaveData@max\_hp\_weight
* 이동 속도 : UnitData@move\_speed\_weight\*
  DungeonStageData@add\_move\_speed

행동 패턴

1. 기본 : Idle 애니메이션으로 제자리 (50%확률), 배회(50% 확률)
2. 감지 : 플레이어가 범위 5이내 진입 시
3. 도망 : 플레이어 반대 방향으로 최대 속도 이동
4. 경계 충돌 : 벽에 부딪히면 90도 방향 전환 후(벽에서 먼 방향) 계속 도망
5. 벗어남: 플레이어 범위 10 이상 멀어지면 1로 복귀

이펙트

* 반짝이 이펙트 (황금이니까)
* 도망칠 때 먼지 파티클

사운드

* 감지: 꿀꿀! (높은 톤)
* 도망: 발소리
* 처치: 동전 소리

처치 보상

* 처치 시 +1 텍스트 팝업 (UnitData@score)

**대형 황금 돼지 (돼지킹)**

* 출현 확률: 5% (데이터로 제어)
* 스폰 위치: 맵 중앙 (고정)
* 스폰 연출: 땅에서 솟아오름 + 반짝임 이펙트 + 사운드

행동 패턴

* 일반 돼지와 동일하지만 이동속도가 느림

이펙트

* 반짝이 이펙트 (황금이니까)
* 발자국 이펙트

사운드

* 감지: 꿀꿀! (낮은 톤)
* 도망: 쿵쿵 소리
* 처치: 큰 동전 소리 + 환호

처치 보상

* 화면에 +10 텍스트 팝업 (UnitData@score)

스폰 시스템(GoldDungeonWaveData 참조) - 데이터로 제어중

* 초기 스폰 (데이터에서 첫 웨이브 스폰 몬스터를 20)
  + 20마리 스폰 (랜덤 스폰)
  + 모두 일반 돼지
* 10초 주기 스폰 : 게임 시작 후 10초마다 실행 (데이터로 제어)
  + 대형 돼지: 5% 확률로 1마리 (맵 중앙)

동시 존재 가능: 최대 50마리

* 50마리 도달 시: 추가 스폰 중지
  + 처치되면: 다시 스폰 시작

### 특수 상황 처리

돼지 뭉침

* 여러 돼지가 겹치지 않도록 충돌 처리
* 서로 밀어내기 (약하게)

플레이어 정지

* 돼지가 범위 밖으로 도망간 후 Idle
* 다시 접근하면 다시 도망

맵 모서리 몰이

* 돼지들이 모서리에 몰릴 수 있음
* 처리: 벽 충돌 후 다른 방향 도망 (부딪히는 벽에서 먼 방향으로 90도 방향 전환)

## 폴리싱 항목 및 추가 작업 요청 리스트

* 돼지가 죽으면 주는 점수 데이터로 제어
  + UnitData@score로 제어
* 돼지가 가운데 주변에 돼지가 리젠되어야 함
  + 기존 : 모든 맵의 지역에서 랜덤 소환됨
  + 변경 후 : 중앙에서 5칸 거리 이내에서 랜덤 소환
* 돼지가 일정 거리 안에 들어가도 맞지 않으면 이동안함
  + 기존 : 맞으면 도망침
  + 변경 후 : 5칸 거리 안에 있거나 맞으면 도망침
* 돼지 아이들 상태 액션 변경
  + 기존 : 돼지가 아이들 상태일 때 가만히 있음 (움직여도 랜덤으로 조금)
  + 변경 후: 돼지가 아이들 상태일 때 중앙을 향해 이동하도록 변경
  + 기대 결과: 모서리나 구석에 박혀있는 현상 감소
* 웨이브 스킵 도입
  + 몬스터가 전멸하면 남은 리젠 시간을 기다리지 않음
  + **즉시 다음 웨이브가 스폰**되도록 로직 수정 부탁드립니다. (루즈한 구간 제거)
* 코인 드랍 연출 변경
  + 기존 돼지 사망 시 코인이 돼지 조각상 UI로 움직여서 채워짐
  + 돼지 사망 시 노란색 혹은 갈색(돼지 조각상 색깔)으로 +score 출력
