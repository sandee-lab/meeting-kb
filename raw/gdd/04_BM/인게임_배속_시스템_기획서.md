# 인게임 배속 시스템

> **요약:** 반복 플레이 피로도를 낮추고 파밍 효율을 높이기 위한 인게임 배속 기능으로, 기본 x1.0에서 x1.2/x1.5까지 3단계 토글 방식으로 작동한다. x1.2와 x1.5는 컴포트 패키지(product_id 60002) 구매 시 동시 해금되며, HUD 우측 상단에 배치된다. 게임 로직 전체의 시간 델타를 조절하되 UI 애니메이션과 사운드는 제외하며, 매 판 x1 배속으로 초기화된다.
> **현재 상태:** 미적용

인게임 배속 시스템

## **1. 개요**

* **목적:** 반복 플레이의 피로도를 낮추고, 파밍 효율을 높여 유저 만족도 증대.
* **핵심:** 배속 기능을 (1.2x,1.5x) 으로 매출 유도.

## **2. 배속 단계 및 해금 조건**

배속은 **3단계 토글** 방식으로 작동하며, 각 단계별 해금 조건은 다음과 같습니다.

| **배속 단계** | **상태** | **해금 조건** | **비고** |
| --- | --- | --- | --- |
| **x 1.0** | 기본 | **기본 제공** | 기본 속도 |
| **x 1.2** | 유료 | **[컴포트 패키지] 구매**  product\_id 60002  (상점 내 영구 적용 상품) | 과금 유저 혜택  1.5배는 너무 빨라 컨트롤이 불가할 수 있으니, 중간 단계 추가 |
| **x 1.5** | 유료 | **[컴포트 패키지] 구매**  product\_id 60002  (상점 내 영구 적용 상품) | 과금 유저 혜택 |

**특이 사항:** [컴포트 패키지] 구매 시, **x 1.2 및 x 1.5 기능이 즉시 모두 해금**됩니다.

### **3.2 상호작용**

버튼 터치 시 다음 순서로 순환(Loop)합니다.

* **Case A: 배속 기능 해금 (패키지 구매자)**
  + x1.0 → x1.2 → x1.5 → x1.0 ...
* **Case B: 배속 기능 비활성화**

### **3.3 잠금 상태 안내**

유저가 해금되지 않은 배속을 인지할 수 있도록 처리합니다.

* **버튼 표기:** x1 텍스트가 써진 버튼을 딤드처리
* **터치 시 반응:Local\_Toast\_Locked\_Fast\_Foward**
  + **미해금 유저:** "컴포트 패키지 구매 시 사용 가능합니다." (토스트 메시지)

![](data:image/png;base64...)

### **4.1 속도 적용 대상**

단순 애니메이션 가속이 아닌, 게임 로직 전체의 시간 델타를 조절해야 합니다.

* **적용 대상:** 아군 공격 속도, 이동 속도, 쿨타임, 적군 스폰 주기, 이동 속도, 스테이지 타이머 등 **모든 인게임 요소**.
* **제외 대상:** UI 애니메이션, 일시정지 메뉴, 사운드 피치는 왜곡되지 않도록 주의

### **4.2 적용 범위**

* 메인 스테이지
* 모의전
* 모든 던전

### **4.2 설정 저장**

* 매판 x1배속으로 초기화 합니다.
  + 프리팹으로 설정을 저장 시킬 수는 있지만, 엔드 게임과 초반 상황이 다른 상황
    - 보통, 엔드 게임은 몹 스펙이 높을 것이라, x1배속 플레이 예상
    - 초반엔 게임 템포가 느리기 때문에, x1.5배속 플레이 예상
  + 시간 경과로 컴포트 패키지가 해제되는 상황 생각하면 x1배속 초기화가 더 안전
