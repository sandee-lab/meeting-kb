# 신규 랜덤 플레이 미션 (전술 실험)

> **요약:** 스테이지 진입 전 랜덤 미션을 부여하여 다양한 덱 조합 유도 및 광고 수익원을 확보하는 신규 콘텐츠 기획. 카마존 연구소의 전술 실험이라는 컨셉으로 메인 태스크(필수 유닛 배치)와 추천 태스크(플레이 가이드)를 제공하며, 일일 3회 실험 기회와 광고 리롤을 통해 반복 플레이와 수익화를 동시에 달성한다. 확률 보정 시스템으로 미션 관련 유닛/택틱 출현율을 상향하여 클리어를 지원한다.
> **현재 상태:** 구현됨

**탭 1**

**신규\_랜덤 플레이 미션**

# 1. 목표

1. 덱 구성 가이드 : 복잡한 선택 없이 시스템이 제안하는 방향성을 수용하도록 한다.
2. 반복 플레이 지루함 해소 : 매번 동일한 최적화 덱만 사용하는 유저에게 새로운 유닛 조합과 전략적 시도를 유도한다.
3. 능동적 전략 설계 유도 : 스테이지 진입 전 미션을 노출함으로써, 유저가 미션에 최적화된 지휘관이나 초기 유닛 덱을 고민하게 만드는 사전 전략 구성을 강화한다.
4. 신규 수익원 확보 : 미션 리롤 및 보상 강화 모델을 통해 자연스러운 광고 시청을 유도한다.

# 2. 컨셉

## 카마존 전술 실험

* 1. 배경
     1. ‘탐사대(덱)’이 전투 데이터를 수집하는 컨셉을 사용하는 데서 착안. 카마존 실험실의 요구를 받아 전투 데이터를 수집하고, 전투 프로토콜 2.0을 준비한다는 내용
  2. 플레이버 텍스트 :
     1. 전시 프로토콜 하에서 모든 전투는 본사의 자산 가치 측정 실험을 겸합니다. 카마존 실험실은 위기 상황을 공짜 임상 시험 기회로 정의했습니다. 고도의 (혹은 근거 없는) 유닛 조합 지침 데이터를 수집하고 전시 프로토콜 2.0을 준비하세요.
     2. 카마존 연구소에서 '오늘의 희생... 아니, 전술 데이터'를 요청했습니다. 연구소에서 전송된 실험 중 마음에 드는 것을 골라서 실천하십시오. 피실험자가 실험 내용을 고르는 건 건방진 일이지만, 한 번은 바꿀 수 있게 해드립니다.

# 3. 실험의 구성

## 실험은 크게 **메인(필수)**과 **추천**(**선택)** 속성으로 관리된다.

* 1. 메인 태스크 : 실험의 핵심 목표(=권장 덱 완성)
     1. 필수 유닛 : 지정된 유닛 ID와 마릿수를 모두 배치해야 함.
     2. 권장 덱 명칭 : 화염 폭발 덱, 중기갑 보병단 등 컨셉 별로 지정
  2. 추천 태스크 : 메인과 별개로 표기
     1. 종류 : 특정 유닛 배치, 특정 택틱 수집
     2. 선택 사항 : 덱 관련 정보 전달 목적이 강하기 때문에 수행 여부는 유저의 선택.

## 실험 성공 판정은 인게임에서, 보상 지급은 아웃게임에서 진행된다.

     1. 해당 챕터 종료 후 전용 UI에서 획득한다.
  4. 미수령 보상
     1. 초기화 시간까지 수령하지 않은 보상은 소멸한다.

     2. 해당 실험 보수를 수령해야만 다음 실험을 확인할 수 있다
        1. 실패해도 실험은 사라지지 않음 → 성공할 때까지 반복 도전 유도
     3. 실험 횟수는 매일 오전 9시 서버 초기화 시간에 전체 초기화된다.

# 4. 태스크의 구성

1. 유저의 성장 단계에 따라 실험 난이도와 종류를 조절한다.
   1. 그룹 언락 조건 : 개별 태스크는 그룹 단위로 관리되며, 각 그룹은 특정 챕터 클리어를 조건으로 한다.
   2. 태스크 노출 조건 : 개별 태스크는 노출 조건을 설정하지 않는다.
2. 태스크는 유저가 플레이 가능한 환경 내에서만 생성된다.
   1. 유저의 진행도를 고려하여 그룹 언락 조건을 설정해야 한다.

##

## 1단계 : 실험 확인

* 1. 로비에서 실험 오픈
     1. 메인 화면 우측에 숏컷 아이콘 배치
     2. 숏컷 터치하면 전용 팝업 노출
        1. 최초 실험 : 간단한 **“실험 할당 연출”** 출력 후 전용 팝업 노출

## 2단계 : 전략 준비

* 1. 유저는 확인한 실험에 맞춰 장비, 지휘관 등을 변경한다.
     1. 리롤(선택) : 미션이 자신의 현재 전력으로 불가능하거나 마음에 들지 않을 경우, 광고 시청으로 실험을 리롤한다.
        1. 리롤 1회 후에는 버튼 비활성화
        2. 터치 시 토스트 메시지 출력 : “리롤은 실험 당 1회 가능합니다”
           1. local\_toast\_random\_play\_mission\_cant\_reroll

## 3단계 : 인게임 플레이

* 1. 진행도 안내
     1. 인게임 및 아머리 UI에 진행도 안내 아이콘을 배치한다.
        1. 인게임 진행 구분
           1. 배치 : 필수 유닛 배치 단계
           2. wait\_completion : 유닛 배치 후 1웨이브 생존 대기
           3. completion : 유닛 배치 후 1웨이브 생존 성공. 챕터 종료 시 아웃게임으로 보상 지급 명령
     2. 미션 진행도 상승 시마다 진행도 표기 및 말풍선을 통해 피드백을 준다.
     3. 아머리 UI에 한하여, 아이콘 클릭 시 미션 팝업을 열어준다.
        1. 인게임에서는 게임 흐름 깨지므로 피드백만 출력
        2. 팝업 구성은 아웃게임과 차이를 둔다.

##

## 4단계 : 보상 수령

* 1. 위에서 설명한 대로, 메인 태스크를 모두 달성한 상태에서 해당 챕터를 정상 종료했을 때 해당 실험 완료 처리한다.
  2. 성공 판정된 챕터 정상 종료 시, 아웃게임에서 보상을 획득할 수 있다.
  3. 일일 실험 기회가 남았다면, 보상 수령 시 자동으로 실험을 새로 뽑는다.
  4. 일일 실험 기회가 소진됐다면, 보상 수령 후 팝업을 닫는다.
     1. 이후 진입 아이콘 숏컷 터치 시 토스트 메시지 출력
        1. “오늘의 전술 실험을 모두 완료했습니다”
           1. local\_toast\_random\_play\_mission\_max\_daily\_trial
