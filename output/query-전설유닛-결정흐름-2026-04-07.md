# 전설유닛 결정 흐름 + 피드백 방향

---

## 결정의 흐름 — 4단계

### 1단계: 강한 유닛으로서의 전설 (2025.02~07)

초기 전설 유닛에는 별도의 설계 원칙이 없었다. 소환 확률 12%로 설정됐고, 부대 시너지를 바이오닉/메카닉 2분류로 나눈 것이 전부였다. 전설은 그냥 "강한 유닛"이었다.

### 2단계: 조합식·시너지 중심 재설계 (2025.09~11)

M8에서 전설의 역할이 근본적으로 바뀌었다. 핵심 방향은 **"하위 유닛 기능을 합쳐 덱을 압축하는 최종 유닛"**.

- 아웃게임 진행도 연동: 처음부터 전설 1~2마리 보유
- 속성별 상태이상 체계 도입 → 전설이 시너지의 정점
- 조합식 세부 확정: 1티어(에픽1+고급1) vs 2티어(에픽2)
- [계승]/[진화] 이원화 (계승: 재료 속성 계승, 진화: 카오스 포함 시 강화형)

> **남은 문제**: 전설이 시너지 없이는 효용 거의 없는 구조 — 기본 성능 0~20%, 시너지 의존 80~100%

### 3단계: 역할 재배치, 시너지 체계 개편 (2026.01~02)

전설과 하위 유닛의 역할을 다시 분리했다.

- 속성별 시너지 빌드 체계 확정 (화염/전기/냉기/카오스 각 루트)
- 재료 단순화 방향 논의 시작 (에픽1 + 낮은 등급1)
- 물리 속성 → 무속성으로 전환

> **남은 문제**: 전설 접근성 허들(레시피 복잡), 등급 간 성능 체감 부족

### 4단계: NCsoft 피드백 반영 — 최종 원칙 확정 (2026.03)

NCsoft의 3/3, 3/10 피드백이 전설 설계를 뒤집는 계기가 됐다.

> *"전설이 '선택지'에 가깝게 작동 → '명확한 목표'로 재정의 필요"*

- **기본:시너지 = 50:50** (기존 0~20%:80~100% → 50%:50%)
- 14종 최종 확정: 10점(무속성 6종) / 14점(속성 4종) / 24점(속성 4종)
- 10점 전설 이중 효과: ① 하위 유닛 시너지 ② 범용 버퍼 (속성 전환 후에도 유지)
- 인접 슬롯 패시브 도입 (배치 전략성 강화)
- 등급별 파워 상향: 전설 10~16 → **16~24**
- 소환 확률 최종 확정: **1%**

## 피드백 취합 — 방향 도출

NCsoft 피드백에서 반복된 4가지 핵심 문제와 그에 대응하는 방향.

| 피드백 문제 | 확정된 방향 | 미결 과제 |
|------------|-----------|---------|
| 레시피 복잡, 허들 과다 | 에픽1+재료1 단순화, 초반 배치 2종 우선 체험 | — |
| 시너지 없으면 전설도 "꽝" | 기본 50% 성능 원칙, 포어맨·간호순 개편 | 캡틴·시즈·메카 루트 상세 설계 |
| 등급 간 성능 차이 체감 안 됨 | 전설 16~24, 영웅 12, 희귀 5로 등급 격차 확대 | — |
| 머지 시 카드 투자 소멸 상실감 | 유닛 승계 시스템 (3/20 빌드 테스트 예정) | 검증 후 확정 여부 결정 |

### 취합 방향 요약 (4개 축)

**1. 목표 명확화**
전설 = 게임의 핵심 목표. 하위 유닛 = 전설로 가는 과정으로 포지셔닝. 등급 간 성능 격차를 압박감으로 체감시켜야 함.

**2. 단독 효능감 확보**
기본 50% 원칙은 이미 확정됐지만, 적용이 안 된 유닛들(간호순·포어맨)의 To-Be 설계를 완료해야 함. "소환하면 바로 뭔가 달라진다"는 체감이 핵심.

**3. 접근성 + 리워드**
레시피 단순화는 진입 장벽만 낮추는 것이 아니라, "가장 빨리 전설을 만드는 루트"를 명시적으로 안내하는 튜토리얼 연계가 필요.

**4. 투자 보존**
유닛 승계 시스템이 핵심 변수. 머지 시 카드 소멸 문제는 "다시 쌓으면 된다"는 게임적 설득이 없으면 이탈 원인. 승계 테스트 결과에 따라 카드 확률 체계 전체를 재조정할 가능성 있음.
