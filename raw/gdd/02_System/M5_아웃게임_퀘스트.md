# M5 아웃게임 퀘스트

> **요약:** 일일/주간 퀘스트로 지속 플레이를 유도하는 달성 점수+진행도 보상 시스템.
> **현재 상태:** 미적용

**M5\_아웃게임\_퀘스트**

# 개요

* 꾸준한 플레이를 유도하여 게임에 지속적인 몰입과 목표를 유지시킨다.

# 구성

## 퀘스트

* + 일일 플레이 동선을 가이드하는 역할
  + 매일 서버 초기화 시간에 진행 상황이 초기화된다.

## 달성 점수

* + 매 퀘스트 완료 시 주어지는 보상

## 진행도

* + 일일 진행도
    - 일일 누적 달성 점수
    - 매일 서버 초기화 시간에 진행 상황이 초기화된다.
  + 주간 진행도
    - 주간 누적 달성 점수
    - 매주 서버 초기화 시간에 진행 상황이 초기화된다.

## 진행도 보상

* + 각 진행도마다 일정 점수 달성 시 보상이 지급된다.

# 동작

## 퀘스트 해금

* + 콘텐츠 타입
    - Contents\_Type = Quest

## 퀘스트 진입

* + 메인 화면에 바로가기 아이콘 배치
  + 아이콘 선택 시 팝업 출력

## 퀘스트 할당

* + 게임 최초 진입 시 퀘스트 할당
  + 퀘스트는 전용 테이블에서 관리한다.

## 퀘스트 상태

* + 퀘스트 진행 중
    - 최초 할당 시의 상태
    - 퀘스트 별로 할당 시점부터 달성 여부를 카운트한다.
    - 진행 중인 퀘스트에는 [이동] 버튼이 배치되며, 선택 시 해당 콘텐츠 UI로 이동한다.
  + 퀘스트 완료
    - 유저가 퀘스트를 완료한 상태
    - 완료 시점 이후로는 더 이상 달성 여부를 카운트하지 않는다.
    - 유저는 퀘스트 팝업에서 보상(달성 점수)을 얻을 수 있다.
      * 보상은 각 퀘스트 별로 획득할 수 있다.
    - 완료 퀘스트는 목록 최상단으로 올려준다.
  + 퀘스트 보상 획득 완료
    - 유저가 퀘스트 팝업에서 [완료](또는 보상 받기 버튼)을 선택하여 달성 점수를 얻은 상태
    - 더 이상 퀘스트 달성 여부를 카운트 하지 않는다.
    - 보상을 획득한 퀘스트는 상호작용이 없다.
    - 보상을 획득한 퀘스트는 목록 하단으로 내려준다. (단, 진행 불가 퀘스트가 있는 경우, 진행 불가 퀘스트보다 위에 정렬한다.
  + 진행 불가 퀘스트
    - 퀘스트에 할당된 조건을 만족하지 않은 경우, 해당 퀘스트는 잠금 상태로 제공된다.
    - 퀘스트에는 잠금 해제 조건이 출력되며, 조건 만족 이후 즉시 수행할 수 있다.
    - 진행 불가 퀘스트는 목록 최하단에 출력된다.

## 초기화

* + 퀘스트 목록은 일 단위로 갱신한다
    - 수령하지 않은 보상은 보존해주지 않는다.
  + 다음 초기화까지 남은 시간은 팝업에 안내한다.

## 진행도 보상

* + 유저가 획득한 달성 점수를 일간, 주간 단위로 누적하여 출력한다.
  + 특정 점수 도달 시, 유저는 보상을 획득할 수 있다.
    - 보상은 최대 3종까지 연결할 수 있다.
    - UI 연출
  + 유저는 진행도에서 보상 아이콘을 선택하여 보상을 획득할 수 있다.
    - 보상 획득은 공용 연출 사용
  + 획득한 보상은 아이콘 표현을 달리 한다.

## 진행도 초기화

* + 일일, 주간 진행도 별로 초기화 시간을 구분한다.
