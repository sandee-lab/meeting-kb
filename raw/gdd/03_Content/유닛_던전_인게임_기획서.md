# 유닛 던전 인게임 기획서

> **요약:** 유닛 던전의 방벽 디펜스 인게임 기획서로, 위쪽에서 몰려오는 몬스터를 막아 아래쪽 방벽을 1분간 사수하는 것이 목표이다. 일반 몬스터(물량형 자폭), 탱커 몬스터(범위 자폭), 공중 몬스터(수직 낙하)의 3종 몬스터와 방벽 HP 시스템, 전투/스폰/아군 구역 정의, 오브젝트 상호작용 매트릭스, 프로그래머 로직 명세를 포함한다.
> **현재 상태:** 미적용

# **유닛 던전 인게임 기획서**

## **개요**

* 기존 던전들과 차별화: 나를 지키는 것이 아닌 벽(제3자)을 방어
* 골드 던전(추격), 지휘관 던전(보스 단일 딜)와 구별되는 방어 콘셉트
* 상황에 따라 능동적으로 적에게 부딪히는 공격적 플레이 (방어 스탯 중심)

### **목표**

위쪽에서 몰려오는 몬스터들을 막아내어 아래쪽쪽 방벽을 1분간 사수

### **맵**

* **구조:**
  + **하단:** 방벽 (HP 존재, 파괴 시 패배)
  + **상단:** 몬스터 스폰 구역
  + **하단~중단 아래:** 자유 이동 전투 공간 (상하좌우 이동 가능)

### **몬스터**

**일반 몬스터 (물량형)**

* **스폰:** 다수 출현
* **이동:** 아래로 직진 (보통)
* **패턴:** 방벽으로 계속 이동
* **공격:** 플레이어 혹은 벽에 자폭하여 피해
* **대응:** 광역 공격으로 빠르게 정리

**탱커 몬스터 (자폭형)**

* **스폰:** 소수 출현
* **스펙:** 체력 매우 높음, 넉백 불가, 상태이상 저항
* **이동:** 아래로 직진 (느림)
* **패턴:**
  + **플레이어 접촉 시:** 즉시 자폭하며 범위안에 피해를 줌
    (일반 몬스터도 같이 폭발에 휘말려서서 죽음)
  + **방벽 도달 시:** 자폭하여 방벽에 큰 피해
* **대응:**
  + **몸통 박치기:** 유닛으로 부딪혀 자폭 피해를 적군 일반 유닛도 처리
  + **극딜:** 원거리에서 화력을 집중하여 접근 전 처치 (피해 없음)

**공중 몬스터 (추적형)

* **스폰:** 보통 출현
* **이동:** 빠름

### **승리조건 / 패배 조건**

* **승리:** 제한 시간 1분 생존
* **패배:** 방벽 HP 0 도달 혹은 지휘관 사망

## **플로우**

1. **진입** → 던전 선택 창 → 유닛 던전 선택 → 단계 선택 → 덱(탐사대) 선택 (탱커/딜러 비율 고려) → [도전] 버튼 터치
2. **게임 시작** → 로딩 (맵 로드) → 카메라 페이드 인 (좌측 방벽부터 우측 스폰 지역까지 훑음) → "DEFENSE!" 문구 출력 → 타이머 시작 (1:00)
3. **게임 진행** **[반복]**
   1. **몬스터 스폰:** 우측 끝 랜덤한 상하 위치에서 몬스터 등장
   2. **전투 및 이동:**
      * 플레이어 터치 드래그로 유닛 이동 (상하좌우 자유)
      * 유닛들 자동 공격 및 스킬 사용
   3. **대응 (상황별 액션):**
      * **일반 몬스터:** 광역 스킬로 쓸어담기
      * **공중 몬스터:** 이동 경로 확인 후 회피 기동
      * **탱커 몬스터:** 아군 탱커로 돌진하여 피해 흡수 OR 원거리 일점사
   4. **방어:** 방벽에 도달한 몬스터 자폭(공중 몬스터 제외) (방벽 HP 감소)
4. **게임 종료**
   1. **승리 시** → 1분 경과 → 남은 몬스터 일제 소멸 → "DEFENSE SUCCESS" 연출 → 결과 화면 (보상 획득)
   2. **패배 시** → 방벽 HP 0 도달 → 방벽 붕괴 연출 → "DEFENSE FAILED" 연출 → 결과 화면 (팁 제공: 탱커 육성 필요 등)

2. 💻 To. 프로그래머 (New Logic Specification)

"공중 몬스터의 패턴 단순화: 일직선 낙하"

A. 공중 몬스터 (Kamikaze) 로직 변경

기존의 '유저 위치 조준 -> 대기 -> 폭격' 방식을 삭제하고, 고전 슈팅 게임의 '폭탄 투하' 방식으로 변경합니다.

* [Spawn]:
  + X축: 5.0f ~ 25.0f (플레이 가능 구역 내 랜덤)
  + Y축: 10.0f (화면 상단 공중)
* [Move]:
  + 생성 직후 Vector3.down (수직 아래) 방향으로 등속 낙하.
  + 속도: MoveSpeed (빠름, 예: 5.0f).
* [Collision]:
  + Hit Player: 아군 유닛과 충돌(OnTriggerEnter) 시, 즉시 자폭하여 범위 데미지를 주고 소멸.
  + Hit Ground: 바닥(Y <= -5.0f)에 닿을 때까지 플레이어와 충돌하지 않으면, 아무런 피해 없이 소멸(Destroy).
    - *기획 의도:* 유저가 무빙으로 피하면 아무 일도 일어나지 않는 '회피 보상'을 확실하게 줌. 방벽 피해 없음.

B. 기타 로직 확인 (기존 유지)

* 지상 몬스터 (워리어/가스백): 기존대로 우측(X=32)에서 생성되어 좌측(Vector3.left)으로 이동.
* 장판 스킬: 가스백 자폭 시 생성되는 독 장판 등의 TriggerStay 로직 유지.
* 충돌 판정:
  + Kamikaze는 Wall 레이어와 충돌 무시 (방벽 피해 없음).
  + Kamikaze는 Ground 레이어(바닥) 도달 시 소멸 처리.

## **상상하는 화면 (리소스 제작 참고)**

* **배경:** 어두운 황무지, 아래에 낡은 강철 방벽이 서 있음.
* **전투:**
  + 위쪽에서 떼로 몰려오는 **일반 몬스터**들을 아군 딜러들이 쓸어버림.
  + 갑자기 덩치 큰 **탱커 몬스터**가 다가오자, 아군이 달려가 '쾅!' 하고 부딪힘. 폭발이 일어나지만 버텨냄.
  + 하늘에서 **공중 몬스터**가 강하함. 플레이어가 급하게 옆으로로 드래그하자, 유닛들이 간발의 차로 피함.
* **이펙트:** 몬스터 자폭 시 파편이 튀고 바닥이 그을림. 방벽 피격 시 화면이 덜컹거림.

### 외계 행성의 황량한 전장과 기괴한 곤충 군단

A. 배경 디자인 (Environment)

* 컨셉: 척박한 외계 행성의 지표면. (스타쉽 트루퍼스, 스타크래프트 차 행성 느낌)
* 톤앤매너:
  + 하늘: 보랏빛이나 짙은 남색의 어두운 밤하늘. 거대한 위성(달)이나 오로라가 은은하게 보임.
  + 지면: 회색빛 암석과 붉은 모래가 섞인 황무지. 곳곳에 기괴한 외계 식물이나 점막이 보임.
* 구조물:
  + 좌측 (아군): [플라즈마 방벽]. 투명한 푸른색 에너지 쉴드가 육각형 벌집 모양으로 웅웅거리며 켜져 있음. 뒤쪽에는 방어 포탑이나 기지 구조물이 살짝 보임.
  + 우측 (적군): [하이브 입구]. 땅이 갈라지거나 거대한 곤충 둥지의 입구. 붉은색 빛과 연기가 새어 나옴.

B. 몬스터 디자인 (Creatures)

* 공통: 외골격(갑각)을 가진 곤충형 외계 생명체. 징그럽지만 타격감 있게 터지는 질감.
* 1. 워리어 (일반):
  + 외형: 저글링, 개미, 사마귀 등을 섞은 소형 곤충. 날카로운 앞발과 턱 강조.
  + 특징: 떼를 지어 빠르게 기어가는 모션. 죽을 때 녹색 체액을 튀기며 조각남.
* 2. 가스백 (탱커):
  + 외형: 뚱뚱하고 둔해 보이는 딱정벌레/진드기 형태.
  + 핵심 포인트: 등 뒤에 거대하고 투명한 녹색 가스 주머니가 달려 있음. 주머니 안에서 액체가 출렁거리거나 맥동(Pump)하는 애니메이션. (딱 봐도 터트리고 싶게 생겨야 함)
* 3. 카미카제 (공중):
  + 외형: 배 부분이 붉게 부풀어 오른 맹독성 말벌/모기. 혹은 자폭용 생체 드론.
  + 특징: 날개가 보이지 않을 정도로 빠르게 윙윙거림. 배 부분(폭탄)이 강조되어야 함.
