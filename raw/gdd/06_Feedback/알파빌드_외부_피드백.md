# 알파빌드 외부 피드백 (찬중)

> **요약:** 알파빌드 외부 플레이테스터의 인게임·UX 피드백 정리. 충돌 데미지, 머지 성능 저하, UI 흐름 단절 등 핵심 이슈를 기록한다.
> **현재 상태:** 구현 반영 참고용

[전체 인상]

- 직관적 플레이: 뱀서류 특유의 단순·직관적 조작으로 진입 장벽이 낮음.

- 조합 시스템: 플레이어와 유닛의 조합은 차별화를 위한 시도는 돋보이나, 전장 화면이 난잡해져 탄막 액션 장르와는 다소 어울리지 않는 인상이 있음.

- 아트 퀄리티: 정돈되고 통일감 있는 아트 스타일은 긍정적.

- 정보 전달성: 유닛 개별 역할을 상세 정보 확인 없이는 학습하기 어려워, 직관적 이해도가 낮음.

[인게임 플레이]

1. 충돌 대미지 중복 적용

- 여러 마리 적과 충돌 시 끔살하는 상황이 발생.

- 초반 충돌 대미지는 중복 상한을 두어 케어를 해주면 좋지 않을까하는 생각

2. 유닛 머지 후 성능 저하

- 유닛 머지로 등급이 올라가면, 공격 범위나 속도 등에 의해 잡몹 처리 성능이 떨어질때가 자주 있음

- 성장이 부족해 몹처리가 안되는 상황은 납득 가능하지만, 머지 결과에 따라 초반 플레이 난이도가 너무 크게 좌우되는 느낌이 들었음.

3. 일부 적의 공격 가시성 부족

- 번개와 같은 공격은 후반 다수 유닛이 전장에 있을 때 인지가 전혀 되지 않음.

- 몬스터의 공격은 투사체 혹은 이펙트 트레이스를 추가하여 가시성, 예측 가능성을 강화해야하지 않을까 싶음.

4. 보스전 타임오버 불인지

- 첫 플레이 시, 보스전 종료 사유를 알 수 없어 승리 / 패배 여부에 혼동이 옴

- 타임오버 시, 시각/사운드/UI 노출 해줘야 할 듯.

- 단순 타임 오버보다는 시간 경과에 따른 보스 패턴 강화와 같은 점진적 압박 기믹으로 세련되게 푸는게 어떨까 하는 생각.

[인게임 UI/UX]

1. 카드 선택, 유닛 뽑기 및 머지, 배치와 전투 흐름 단절

- 해당 기능 들이 별도 씬으로 제작되어 게임 흐름이 끊김.

- 동일 씬 내 오버레이로 처리하여 흐름 유지하는게 어떨까

2. 뽑기 및 배치의 반복 피로

- 조합/배치가 반복되어 귀찮고 피로감을 유발함.

- 숙련 유저가 아니면 조합 배치에 전략적인 선택을 하기도 힘들어 보임.

- 자동 뽑기, 자동 배치를 디폴트로 원할 경우만 수동으로 조정할 수 있는 옵션 제공이 어떨까 하는 생각.

3. 유닛 속성 키워드 인지 문제

- 피지컬/에너지 같은 핵심 속성 키워드를 확인하려면 상세보기로 진입 해야함.

- 유닛 아이콘 프레임 혹은 내부 아이콘으로 속성을 직관적으로 표시해주면 어떨까

Q/A

* 일반 몹을 잘 정리하지 못하는 상황이 되었을 때, 회피 컨트롤을 통해서 생존해나가는 플레이를 하기 어려운 부분이 안되는 것은 아쉬움.
  + 어떤 룰을 추가하는 것은 복잡성을 높이는 방향이기도 하고, 현재 구성의 완성도는 높다고 느껴져서 다른 방향으로 바꾸는 것은 쉽지 않아 보임.
* 지휘관을 중심으로 조작하는 느낌 자체는 자연스럽게 느껴서 조작감 자체는 나쁘지 않았음.
* 유닛으로 조합하다보니 스킬에 대한 학습 문제 등이 불편.
* UX 개선이 가장 중요하게 느낌. (1, 2번)
